## 核心故事

OWL 的视觉识别系统讲述的是**从好奇到创造的涌现之旅**。

### 品牌故事内核

**起点 → 过程 → 终点**

```
🦉 凝视（观察）
    ↓
  好奇心的种子
    ↓
✨ 灵光（奇思妙想）
    ↓
  偶然的连接
    ↓
🔗 连接（跨界融合）
    ↓
  创意的涌现
    ↓
🪶 飞翔（勇敢探索）
    ↓
  作品的诞生
    ↓
🌟 分享（开放共享）
```

### 设计哲学

> **不是冷冰冰的科技感，而是充满玩耍精神的探索**

- **有趣**：视觉元素充满趣味性和惊喜感
- **好奇心**：鼓励提问、探索、发现
- **奇思妙想**：允许非常规、跨界、随机的组合
- **偶然连接**：强调意外发现的价值
- **创意**：每个元素都是创造力的表达
- **前沿问题**：关注未知和挑战
- **探索**：过程比结果更重要

## 色彩系统

### 核心双色

**Neon Pink #D91A7A**
- **情感**: 热情、创造、行动、玩耍精神
- **象征**: 创意火花、勇敢探索、动手实践
- **应用**: 行动按钮、高亮元素、创意工具

**Blue #2563EB**
- **情感**: 洞察、理性、深度、知识
- **象征**: 好奇的眼睛、思考、理论框架
- **应用**: 文本、结构线框、知识内容

**Dark #0E0E14**
- **情感**: 深邃、专注、未知、探索空间
- **象征**: 创新的土壤、未知的领域
- **应用**: 背景、深色模式、沉浸式场景

### 辅助色系（克制使用）

| 颜色 | 色值 | 情感 | 使用场景 | 使用原则 |
|------|------|------|---------|---------|
| **Neon Cyan** | `#00D9FF` | 创新、活力、连接 | M02 治理、M05 工具、创新区域 | 与 Blue 配合使用 |
| **Violet** | `#8B5CF6` | 融合、思考、深度 | M01 理念、跨界场景 | Pink 和 Blue 的过渡色 |
| **Emerald** | `#10B981` | 安全、生长、可持续 | M03 空间、M06 安全 | 仅用于特定功能指示 |
| **Orange** | `#F97316` | 活力、创造、热情 | M04 项目、Living Modules | 点缀使用，不大面积 |
| **Amber** | `#F59E0B` | 温暖、传递、点燃 | M07 人员、导师相关 | 温暖场景的辅助 |
| **Purple** | `#A855F7` | 成长、评价、反思 | M09 评价 | 评估相关场景 |

### 色彩使用原则

**原则 1: 核心双色优先**
- 80% 的场景只使用 Pink + Blue + Dark
- 辅助色仅在需要明确功能区分时使用

**原则 2: 渐变光谱**
- Pink → Violet → Blue → Cyan 形成连续光谱
- 渐变表达"融合"和"过渡"的概念

**原则 3: 对比与和谐**
- 高对比：Pink on Dark、Blue on Dark（可读性）
- 低对比：渐变、氛围、背景（沉浸感）

**原则 4: 动态变化**
- 粒子可以是 Pink、Blue 或两者混合
- 交互时产生颜色变化（Blue → Pink = 从观察到行动）

## 符号系统

### 核心符号：好奇猫头鹰的五种状态

#### 🦉 状态 1: 凝视 (Gaze)

**概念**: 好奇的起点，观察与提问

**形态描述**
- 眼睛睁大、瞳孔扩张
- 周围有探测波纹（声呐式）
- 几何框架：六边形或圆形轮廓

**粒子行为**
- 从眼睛向外发射好奇光束
- 遇到"知识节点"时产生涟漪
- 缓慢的脉冲呼吸（4-6s 周期）

**色彩配置**
- 主色：Blue（洞察）
- 点缀：Pink 闪烁（兴奋）
- 背景：Dark with subtle gradient

**应用场景**
- 知识库入口、"发现问题"阶段
- M01 理念模块、M06 安全模块
- 引导用户探索的场景

**动态效果**
- 静态：眼睛缓慢眨动
- 悬停：瞳孔跟随鼠标移动
- 点击：产生扩散波纹

#### ✨ 状态 2: 灵光 (Spark)

**概念**: 奇思妙想的瞬间，灵感的爆发

**形态描述**
- 眼睛周围爆发星状粒子
- 几何碎片飞散
- 不规则的、爆炸性的构图

**粒子行为**
- 粒子在空中做布朗运动
- 偶然碰撞产生"连接火花"
- 快速的、随机的运动

**色彩配置**
- Pink 和 Blue 粒子随机碰撞
- 碰撞点产生 Violet 闪光
- 高对比、高饱和度

**应用场景**
- 创意工具、AI Lab
- 头脑风暴阶段、灵感时刻
- "啊哈"瞬间的视觉表达

**动态效果**
- 静态：粒子缓慢飘浮
- 悬停：粒子加速碰撞
- 点击：粒子爆发后重新聚集成新图案

#### 🔗 状态 3: 连接 (Connect)

**概念**: 偶然的发现，跨界的连接

**形态描述**
- 多个眼睛通过光束相互连接
- 形成网络状结构
- 节点和边的组合

**粒子行为**
- 粒子沿着连线流动
- 在节点处聚集形成"知识晶体"
- 连接随机生成和消失

**色彩配置**
- 渐变光谱：Pink → Violet → Blue → Cyan
- 连接线使用渐变
- 节点发光效果

**应用场景**
- 社区网络、协作项目
- M02 治理模块、M08 运营模块
- 跨界融合的场景

**动态效果**
- 静态：网络缓慢呼吸
- 悬停：连接线加粗发光
- 点击：触发涟漪传播动画

#### 🪶 状态 4: 飞翔 (Flight)

**概念**: 探索未知，勇敢前行

**形态描述**
- 翅膀完全展开
- 羽毛飘散、留下轨迹
- 动态的、向前的姿态

**粒子行为**
- 粒子形成羽毛轮廓
- 边缘不断"脱落"又重新生长
- 向前飞行的轨迹

**色彩配置**
- 主色：Pink（行动）
- 轨迹：Cyan（创新路径）
- 羽毛渐变：Pink → Violet → Blue

**应用场景**
- 项目启动、Living Modules
- 探索阶段、实践阶段
- M03 空间、M05 工具

**动态效果**
- 静态：翅膀缓慢扇动
- 悬停：羽毛飘落动画
- 点击：飞翔轨迹绘制

#### 🌟 状态 5: 分享 (Share)

**概念**: 创造的成果，开放共享

**形态描述**
- 眼睛变成发光源
- 向外辐射同心圆波纹
- 开放的、扩散的构图

**粒子行为**
- 粒子从中心向四周扩散
- 形成"知识涟漪"
- 触碰边界时反弹回来（循环）

**色彩配置**
- 全光谱：Pink + Blue + Cyan + Violet
- 波纹渐变透明
- 象征多样性和包容性

**应用场景**
- 作品展示、开源分享
- M07 人员、M09 评价
- 社区成果展示

**动态效果**
- 静态：波纹持续扩散
- 悬停：波纹加速
- 点击：产生新的扩散中心

## 动态规则

### 三级动态强度

#### Level 1: 微动 (Micro-motion)

**场景**
- 导航图标、按钮、小卡片
- 日常交互、频繁使用的元素

**效果**
- 悬停时轻微缩放（scale: 1 → 1.05）
- 色彩渐变（Blue → Pink，0.3s ease）
- 边框发光（box-shadow 变化）

**实现**
- 纯 CSS transition/animation
- 无 JavaScript 依赖

**性能**
- 极低开销，60fps 稳定

**代码示例**
```css
.micro-motion {
  transition: all 0.3s ease;
}

.micro-motion:hover {
  transform: scale(1.05);
  box-shadow: 0 0 20px rgba(217, 26, 122, 0.5);
}
```

#### Level 2: 交互动画 (Interactive Animation)

**场景**
- 页面加载、滚动触发
- 点击反馈、状态切换

**效果**
- 粒子爆发（10-50 粒子）
- 路径绘制（SVG stroke-dasharray）
- 形态变换（morphing）

**实现**
- Framer Motion
- SVG 动画
- React Spring

**性能**
- 中等开销，需优化
- 避免同时多个动画

**代码示例**
```tsx
<motion.div
  initial={{ opacity: 0, y: 20 }}
  animate={{ opacity: 1, y: 0 }}
  transition={{ duration: 0.5 }}
>
  {content}
</motion.div>
```

#### Level 3: 生成艺术 (Generative Art)

**场景**
- 首页英雄区
- Living Modules 配图
- 特殊展示页

**效果**
- 实时粒子系统（100-1000 粒子）
- 参数化图形
- WebGL 渲染

**实现**
- React Three Fiber
- Canvas API
- 自定义粒子引擎

**性能**
- 较高开销
- 限制使用场景
- 提供降级方案

**代码示例**
```tsx
<Canvas>
  <ParticleSystem count={500} />
</Canvas>
```

### 粒子运动规则

#### 基础运动：布朗运动

**物理规则**
- 随机游走（Random Walk）
- 速度：0.5-2 px/frame
- 方向：每 30-60 帧随机改变

**代码逻辑**
```javascript
particle.vx += (Math.random() - 0.5) * 0.2;
particle.vy += (Math.random() - 0.5) * 0.2;
particle.x += particle.vx;
particle.y += particle.vy;
```

#### 吸引力场：鼠标交互

**物理规则**
- 鼠标周围产生引力/斥力场
- 距离越近，力越强
- 引力半径：100-200px

**代码逻辑**
```javascript
const dx = mouse.x - particle.x;
const dy = mouse.y - particle.y;
const distance = Math.sqrt(dx * dx + dy * dy);

if (distance < 200) {
  const force = (200 - distance) / 200;
  particle.vx += dx * force * 0.01;
  particle.vy += dy * force * 0.01;
}
```

#### 偶然连接：知识涌现

**规则**
- 粒子距离 < 阈值时产生连线
- 连线持续 0.5-2s 后消失
- 最多同时显示 20 条连线

**视觉效果**
- 连线透明度随距离衰减
- 连线颜色：Violet（Pink + Blue）
- 连线宽度：1-2px

**代码逻辑**
```javascript
particles.forEach((p1, i) => {
  particles.slice(i + 1).forEach(p2 => {
    const distance = getDistance(p1, p2);
    if (distance < threshold) {
      drawConnection(p1, p2, distance);
    }
  });
});
```

### 几何变换规则

#### 入场动画：从虚到实

**阶段 1**: 线框显现（0-0.3s）
- opacity: 0 → 1
- stroke-dasharray 动画（描边绘制）

**阶段 2**: 填充渐现（0.3-0.6s）
- fill opacity: 0 → 1
- 粒子从四周聚集

**阶段 3**: 稳定态（0.6s+）
- 进入呼吸动画
- 准备接受交互

#### 悬停效果：3D 旋转

**效果**
- transform: rotateY(0deg) → rotateY(15deg)
- box-shadow 增强
- 发光效果

**时长**: 0.3s ease-out

**代码**
```css

  transform: rotateY(15deg) rotateX(5deg);
  box-shadow:
    0 10px 40px rgba(217, 26, 122, 0.3),
    0 0 20px rgba(37, 99, 235, 0.3);
}
```

#### 点击反馈：脉冲缩放

**效果**
- scale: 1 → 0.95 → 1.05 → 1
- 持续时间：0.4s

**附加效果**
- 粒子爆发
- 波纹扩散

**代码**
```javascript
const pulseAnimation = {
  scale: [1, 0.95, 1.05, 1],
  transition: { duration: 0.4 }
};
```

#### 退场动画：粒子化消散

**阶段 1**: 粒子化（0-0.3s）
- 实体分解为粒子
- 粒子向外扩散

**阶段 2**: 消散（0.3-0.6s）
- 粒子透明度降低
- 粒子速度减慢

**阶段 3**: 消失（0.6s+）
- opacity: 0
- 从 DOM 移除

**用途**

**效果**
- 更明显的缩放变化

#### 快速闪烁（0.5-1s）

- 高亮重点

**效果**

- 配合声音/震动
