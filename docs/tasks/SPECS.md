| 功能 | 描述 | 验收标准 |
|------|------|---------|
| **智能组队** | 基于兴趣标签和能力画像的跨校匹配 | 推荐准确率 > 70% |
| **角色分配** | AI 建议课题组角色分工（负责人/研究员/工程师/设计师） | 支持自定义角色 |

| **导师连接** | 匹配领域专家作为远程指导导师 | 可查看导师档案 |

#### 用户场景
1. **学生A**：在北京，对合成生物学感兴趣，想找小伙伴一起做研究
2. **学生B**：在深圳，擅长编程，希望参与有趣的项目
3. **系统匹配**：根据兴趣和能力推荐组队，AI 生成角色分工建议
4. **协作研究**：在线协作完成研究，导师远程指导
5. **成果展示**：生成研究报告，可导出到 T05 作品展示生成器

### 技术方案

- **前端**：React + TypeScript + Zustand + TanStack Query
- **协作**：WebSocket 实时通信 + CRDT 协同编辑
- **匹配算法**：基于向量相似度的兴趣匹配 + 能力互补推荐
- **数据模型**：用户画像、课题组、任务、里程碑
- **存储**：PostgreSQL + Redis (缓存) + S3 (文件)

### 关联模块

- M02 治理与网络：跨节点协作模式
- M04 课程与项目：研究性学习方法
- M07 人员与能力：角色能力模型

## 013 - 全球案例地图

**Global Case Map** | 优先级：P3

### 实现目标

可视化浏览全球创客空间案例，让想建设 OWL 的人能够学习他人经验，找到参考和灵感。

### 交付效果

#### 核心功能
| 功能 | 描述 | 验收标准 |
|------|------|---------|
| **地图浏览** | 在地图上浏览全球案例分布 | 支持缩放/聚合 |
| **多维筛选** | 按地区、类型、规模、预算筛选 | 组合筛选 + 即时更新 |
| **案例详情** | 查看详细的案例介绍和图片 | 标准化案例模板 |
| **联系方式** | 获取案例负责人的联系方式 | 支持预约访问 |

#### 案例分类
| 类型 | 描述 | 数据字段 |
|------|------|---------|
| 学校型 | 嵌入学校的创客空间 | 面积、预算、服务人数、课程特色 |
| 社区型 | 开放的公共创客空间 | 开放时间、收费模式、特色项目 |
| 专业型 | 面向专业领域的空间 | 专业方向、设备配置、合作模式 |
| 企业型 | 企业内部创新空间 | 行业、创新模式、开放程度 |

### 技术方案

- **前端**：React + TypeScript + Mapbox GL / Leaflet
- **数据管理**：Supabase / PostgreSQL + PostGIS
- **案例提交**：表单系统 + 管理员审核
- **数据模型**：案例档案、地理位置、标签分类、媒体资源

### 关联模块

- M02 治理与网络：全球网络、案例数据
