### 002 - T02 虚拟课题组

**Virtual Research Team** | 优先级：P1

#### 实现目标

打破地域限制，让不同学校、不同城市的学生能够组建虚拟研究小组，共同探究真实的科研问题。

#### 核心功能

| 功能 | 描述 | 验收标准 |
|------|------|---------|
| **智能组队** | 基于兴趣标签和能力画像的跨校匹配 | 推荐准确率 > 70% |
| **角色分配** | AI 建议课题组角色分工 | 支持自定义角色 |
| **任务看板** | 可视化的任务拆解和进度追踪 | 支持甘特图视图 |
| **协作空间** | 在线文档、白板、资源共享 | 实时协同编辑 |
| **导师连接** | 匹配领域专家作为远程指导导师 | 可查看导师档案 |

#### 技术方案

- **前端**：React + TypeScript + Zustand + TanStack Query
- **协作**：WebSocket 实时通信 + CRDT 协同编辑
- **匹配算法**：向量相似度 + 能力互补推荐

#### 里程碑

| 阶段 | 目标 | 交付物 |
|------|------|--------|
| Phase 1 | MVP 组队功能 | 手动组队 + 任务看板 |
| Phase 2 | 智能匹配 | AI 推荐组队 + 角色建议 |
| Phase 3 | 协作空间 | 在线文档 + 白板 |
| Phase 4 | 导师网络 | 导师匹配 + 预约系统 |
