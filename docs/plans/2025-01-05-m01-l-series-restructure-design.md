# M01 与 L序列 结构重构设计

**版本**: 1.0.0
**日期**: 2025-01-05
**状态**: 待实施

---

## 1. 问题背景

M01（理念与理论基础）与 L序列（前沿理念）之间存在以下问题：

1. **内容重复**：同样的理论或概念在两边都出现（如失败友好、AI能力、社区驱动）
2. **命名冲突**：M01 中的"五个转变"使用 L01-L05 编号，与 L序列的 L01-L04 混淆

---

## 2. 定位重新定义

### 2.1 核心区分

| 维度 | M01 理念与理论基础 | L序列 前沿理念 |
|------|-------------------|---------------|
| **本质** | 稳定的地基 | 活跃的触角 |
| **内容性质** | 已验证、经典的教育理论 | 前沿、激进、探索性研究 |
| **理论时间线** | 经典理论为主 | 2024-2025 最新研究 |
| **抽象层级** | "是什么" + "为什么重要" | "最新怎么理解" + "前沿怎么做" |
| **确定性** | 高，可直接应用 | 中低，需实验验证 |
| **篇幅风格** | 精简、核心原则 | 深度研究、详细论证 |
| **更新频率** | 稳定，偶尔更新 | 活跃，持续演进 |

### 2.2 隐喻说明

- **M01** 是知识库的"骨骼"——提供稳定的结构支撑
- **L序列** 是思想的"DNA"——定义探索的方向和边界

### 2.3 读者定位

- **M01 读者心态**："告诉我核心原则是什么"
- **L序列读者心态**："启发我用最新研究如何思考"

采用**主题词式**命名，去除 L01-L05 编号，避免与 L序列混淆。

| 04 从教师中心到学生中心 | **学生中心** |

|------|------|
| L01 | 空间的塑造 |
| L02 | 思维的延伸 |

## 4. 内容边界

### 4.1 M01 内容结构（精简版）

```
M01 理念与理论基础
├── 模块概述（ModuleSummary 组件）
├── 教育的根本性转变（5个主题词，精简阐述）
│   ├── 能力导向
│   ├── 思维培养
│   ├── 个性化
│   ├── 学生中心
│   └── 开放生态
├── 核心原则（6条，简明定义）
│   ├── 做中学
│   ├── 开放共享
│   ├── 社区驱动
│   ├── 失败友好
│   ├── 跨界整合
│   └── 赋能非替代
├── 使命与愿景
├── 核心价值观
├── 设计原则（7条，简明版）
├── 理论框架概述（建构主义、4P、情境学习、设计思维）
├── 21世纪能力框架
├── 与其他模块的关系
├── 扩展阅读（链接到 extend 文件夹和 L序列）
└── 参考文献（统一格式 [1][2][3]）
```

### 4.2 L序列内容结构（保持深度）

```
L序列 前沿理念
├── L01 空间的塑造
├── L02 思维的延伸
├── L03 涌现的智慧
├── L04 技术的诗意
└── 每个模块包含：
    ├── 核心理念（理论基石、元隐喻、范式转变）
    ├── 前沿研究方向（5-6个方向，深度论述）
    ├── 设计原则（6条，基于最新研究）
    ├── 反模式警示
    ├── 实践启示
    ├── 与其他L模块的关联
    ├── 从理念到行动（链接到M序列）
    └── 延伸阅读
```

### 4.3 内容差异化原则

| 主题 | M01 处理方式 | L序列 处理方式 |
|------|-------------|---------------|
| 失败友好 | 作为核心价值观简明阐述 | L01 深入探讨"失败剧场"设计 |
| AI能力 | 21世纪能力框架中简要提及 | L02 完整探讨认知增强前沿 |
| 社区驱动 | 作为核心原则简明阐述 | L03 深入探讨涌现智慧机制 |
| 空间设计 | 不深入，指向 L01 和 M03 | L01 深入探讨神经建筑学 |
| 技术选型 | 不深入，指向 L04 和 M05 | L04 深入探讨技术哲学 |

## 5. 参考文献格式规范

### 5.1 引用格式

- 正文中使用数字引用：`[1]`、`[2]`、`[3]`
- 所有参考文献统一放在文档末尾
- 按引用顺序编号

### 5.2 参考文献条目格式

```markdown
## 参考文献

[1] 作者. (年份). 标题. 来源/期刊. URL（如适用）

[2] 作者. (年份). 《书名》. 出版社.
```
