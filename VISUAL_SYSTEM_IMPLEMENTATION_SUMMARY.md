**粒子场组件**: `src/components/brand/ParticleField.tsx`
```typescript
功能特性:
✅ 布朗运动的粒子系统
✅ 鼠标影响力场
✅ 偶然连接（粒子距离 < 阈值时产生连线）
✅ 可配置数量、速度、连接阈值
✅ 使用 Canvas API，高性能
✅ 完全响应式

参数:
- count: 粒子数量 (默认 80)
- connectionThreshold: 连接距离 (默认 150px)
- maxConnections: 最大连接数 (默认 20)
- speed: 运动速度 (默认 0.3)
- mouseInfluence: 鼠标交互 (默认 true)
```

## 🎨 设计哲学总结

> **不是冷冰冰的科技感，而是充满玩耍精神的探索**

### 关键词
- **有趣**: 粒子随机运动、偶然连接
- **好奇心**: 猫头鹰凝视、问号隐喻
- **奇思妙想**: 灵光状态、粒子爆发
- **偶然连接**: 粒子网络、跨界融合
- **创意**: 色彩渐变、动态生成
- **前沿问题**: 探测器符号、未知领域
- **探索**: 飞翔状态、开放路径

### 平衡点
- **几何 ⚖️ 有机**: 框架提供秩序，粒子带来活力
- **理性 ⚖️ 感性**: 蓝色洞察，粉色热情
- **静态 ⚖️ 动态**: 结构稳定，细节动人
- **克制 ⚖️ 表达**: 核心色简洁，辅助色丰富

## 💡 实施建议

### 对于开发者

1. **保持克制**: 不要过度使用动画和辅助色
2. **性能优先**: 在低端设备测试，提供降级方案
3. **一致性**: 使用设计 Token，不要硬编码颜色
4. **可访问性**: 保证文本对比度，支持键盘导航

### 对于设计师

1. **理解基因**: 三大视觉基因是设计语法，不是限制
2. **讲述故事**: 每个应用都应该讲述"从好奇到创造"的旅程
3. **自适应**: 同样的基因，不同的表现形式
4. **有趣第一**: 如果设计不好玩，重新开始
