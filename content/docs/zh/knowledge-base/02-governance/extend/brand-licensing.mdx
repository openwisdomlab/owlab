# 品牌授权指南

## 概述

OWL 品牌是网络共同资产，承载着所有节点的信誉和价值。本文档规范品牌的使用方式，确保品牌在各节点的一致呈现，同时保护品牌不被滥用。品牌授权是信任的象征，被授权者有责任维护品牌声誉。

---

## 1. 品牌体系

### 1.1 品牌架构

```
OWL (主品牌)
├── OWL (核心实验室) - 完整品牌
├── OWL-U (联合实验室) - 联合品牌
│   ├── OWL-U · 区域名
│   ├── OWL-U · 学校名
│   └── ...
├── OWL-X (社区实验室) - 社区品牌
│   └── OWL-X · 空间名
└── 子品牌/项目品牌
    ├── OWL Camp (研学营)
    ├── OWL Challenge (挑战赛)
    └── ...
```

### 1.2 品牌要素

| 要素 | 说明 | 使用场景 |
|------|------|----------|
| **Logo** | 品牌标识 | 所有官方呈现 |
| **名称** | OWL 及变体 | 文字描述 |
| **标语** | 核心口号 | 宣传材料 |
| **色彩** | 品牌色系 | 设计物料 |
| **字体** | 指定字体 | 设计物料 |
| **图形元素** | 辅助图形 | 设计物料 |

## 2. 品牌标识规范

### 2.1 Logo 使用规范

#### 标准 Logo
- 主 Logo：完整标识，包含图形和文字
- 简化 Logo：仅图形部分
- 文字 Logo：仅文字部分

#### 使用规则

| 规则 | 说明 |
|------|------|
| **最小尺寸** | 宽度≥24px (数字)，≥15mm (印刷) |
| **安全区域** | Logo 周围保留 1/4 Logo 高度的空白 |
| **背景对比** | 确保足够的对比度 |
| **比例** | 不得改变 Logo 比例 |
| **完整性** | 不得裁切或遮挡 Logo 任何部分 |

#### 禁止的使用方式

| 禁止 | 说明 |
|------|------|
| ❌ 拉伸变形 | 不得改变 Logo 比例 |
| ❌ 改变颜色 | 不得自行更改品牌色 |
| ❌ 添加效果 | 不得添加阴影、渐变、3D 效果 |
| ❌ 旋转倾斜 | 不得旋转或倾斜 Logo |
| ❌ 更换字体 | 不得更换 Logo 中的字体 |
| ❌ 与其他元素组合 | 未经授权不得与其他 Logo 组合 |

### 2.2 品牌色彩

| 色彩 | 名称 | Hex | 用途 |
|------|------|-----|------|
| 主色 | OWL Blue | #2563EB | 主要标识、标题 |
| 辅色 | OWL Green | #10B981 | 辅助、强调 |
| 背景 | Light Gray | #F3F4F6 | 背景色 |
| 文字 | Dark | #1F2937 | 正文 |
| 警示 | Red | #EF4444 | 警示、错误 |

### 2.3 品牌字体

| 用途 | 中文 | 英文 |
|------|------|------|
| 标题 | 思源黑体 | Inter Bold |
| 正文 | 思源黑体 | Inter Regular |
| 代码 | 等宽字体 | Fira Code |

## 3. 不同节点的品牌使用

### 3.1 授权级别

| 节点类型 | 品牌使用 | Logo 形式 | 命名规则 |
|----------|----------|-----------|----------|
| **OWL** | 完整品牌 | 主 Logo | "OWL [名称]" |
| **OWL-U** | 联合品牌 | 联合 Logo | "OWL-U · [名称]" |
| **OWL-X** | 社区品牌 | 社区 Logo | "OWL-X · [名称]" |

### 3.2 联合实验室 (OWL-U) 品牌规范

#### 联合 Logo 构成
```
┌─────────────────────────────────────────┐
│                                         │
│   [OWL Logo]  ·  [合作方名称/Logo]       │
│                                         │
└─────────────────────────────────────────┘
```

#### 使用规则
- OWL Logo 与合作方标识以"·"连接
- 两者视觉权重相当
- 遵循联合 Logo 模板
- 需经 OWL 审核后使用

### 3.3 社区实验室 (OWL-X) 品牌规范

#### 社区 Logo 构成
```
┌─────────────────────────────────────────┐
│                                         │
│   [OWL-X Logo]  ·  [空间名称]            │
│                                         │
└─────────────────────────────────────────┘
```

#### 使用规则
- 使用 OWL-X 专用 Logo
- 空间名称置于 Logo 后
- 可保留原有空间 Logo，与 OWL-X 标识分开展示
- 需标明"OWL 生态伙伴"身份

## 4. 品牌使用场景

### 4.1 必须使用场景

| 场景 | 要求 |
|------|------|
| **空间门头** | 需展示授权标识 |
| **官方材料** | 课程材料、证书需有标识 |
| **对外宣传** | 使用授权的品牌描述 |
| **数字平台** | 网站、社交媒体展示标识 |

### 4.2 使用示例

│                                         │

│        OWL 联合实验室                    │

### 4.3 禁止使用场景

| 禁止场景 | 说明 |
|----------|------|
| ❌ 未授权商业活动 | 超出授权范围的商业使用 |
| ❌ 违背价值观内容 | 与 OWL 价值观相悖的内容 |
| ❌ 误导性使用 | 可能造成误解的使用方式 |
| ❌ 授权到期后 | 授权终止后继续使用 |

## 5. 授权协议

### 5.1 授权类型

| 类型 | 适用节点 | 期限 | 范围 |
|------|----------|------|------|
| **标准授权** | OWL-U | 1年 | 基础品牌使用 |
| **扩展授权** | OWL-U (进阶) | 1年 | 品牌+课程使用 |
| **生态授权** | OWL-X | 1年 | 社区标识使用 |

### 5.2 授权协议要点

#### 授权内容
- 品牌标识使用权
- 指定名称使用权
- 相关物料模板使用权

#### 授权条件
- 遵守品牌使用规范
- 按要求使用标准物料
- 接受品牌使用监督
- 维护品牌声誉

#### 授权限制
- 不得转授权
- 不得修改品牌标识
- 不得用于未授权场景
- 不得损害品牌声誉

### 5.3 授权协议模板

```
OWL 品牌授权协议

甲方：OWL 总部
乙方：[节点名称]

一、授权内容
甲方授权乙方在以下范围内使用 OWL 品牌：
1. 使用 [OWL-U/OWL-X] 标识
2. 使用"[节点全称]"名称
3. 使用授权的物料模板

二、授权期限
本授权自 [开始日期] 至 [结束日期]，为期一年。
授权到期前30天可申请续期。

三、使用规范
乙方同意遵守《OWL 品牌使用规范》，具体包括：
1. 按规范使用品牌标识
2. 使用授权的物料模板
3. 接受品牌使用监督
4. 维护品牌声誉

四、权利义务
甲方权利义务：
1. 提供品牌使用指南和物料
2. 监督品牌使用情况
3. 维护品牌权益

乙方权利义务：
1. 在授权范围内使用品牌
2. 遵守使用规范
3. 报告品牌使用情况
4. 发现侵权及时通知甲方

五、违约责任
乙方违反本协议规定，甲方有权：
1. 要求整改
2. 暂停授权
3. 终止授权
4. 追究法律责任

六、授权终止
以下情况授权终止：
1. 授权期满未续期
2. 乙方申请终止
3. 乙方严重违约
4. 乙方退出 OWL 网络

授权终止后，乙方应：
1. 停止使用品牌标识
2. 移除品牌相关物料
3. 停止使用授权名称

甲方代表：_______________  日期：_______________
乙方代表：_______________  日期：_______________
```

## 6. 品牌监督与保护

### 6.1 自我监督

| 检查项 | 频率 |
|--------|------|
| 品牌使用规范性 | 持续 |
| 物料更新情况 | 每季度 |
| 使用场景合规性 | 持续 |

### 6.2 网络监督

| 监督方式 | 频率 | 负责方 |
|----------|------|--------|
| 抽查 | 每季度 | 区域/总部 |
| 年度评估 | 每年 | 总部 |
| 投诉处理 | 即时 | 总部 |

### 6.3 侵权处理

| 侵权类型 | 处理方式 |
|----------|----------|
| **内部节点违规** | 警告 → 整改 → 暂停 → 终止 |
| **外部侵权** | 通知 → 法律手段 |
| **网络假冒** | 投诉 + 法律手段 |

## 7. 常见问题

| 问题 | 回答 |
|------|------|
| 能修改 Logo 颜色吗？ | 不能，请使用标准色 |
| 联合 Logo 需要审批吗？ | 需要，提交设计稿审批 |
| 授权到期如何续期？ | 提前30天申请，符合条件自动续期 |
| 退出后能保留品牌吗？ | 不能，需在30天内移除 |
| 发现侵权怎么办？ | 第一时间通知 OWL 总部 |
