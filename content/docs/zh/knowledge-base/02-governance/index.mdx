# M02 治理与网络

> **一句话定位**: OWL 网络的组织形式与协作规则

---

## 核心原则

| 原则 | 内涵 | 体现 |
|------|------|------|
| **分布式网络** | 去中心化，节点自主协作 | 节点独立运营 |
| **标准与灵活** | 核心统一，本地灵活 | MVS + 本地化 |
| **共建共享** | 资源共享，风险共担 | 网络资源池 |
| **透明可信** | 决策透明，信息公开 | 公开治理 |
| **持续进化** | 机制随网络发展迭代 | 定期复盘 |

## 实验室分类体系

### 按管理关系分类

| 类型 | 代码 | 定位 | 运营方 | 典型场景 |
|------|------|------|--------|----------|
| **核心实验室** | OWL | 官方直营，网络大脑 | OWL 团队 | 旗舰实验室 |
| **联合实验室** | OWL-U | 深度合作共建 | 合作方运营 | 区域/学校/大学 |
| **社区实验室** | OWL-X | 生态伙伴 | 独立运营 | 创客空间/工作室 |
| **虚拟实验室** | OWL-Virtual | 纯线上运营 | 线上社区 | 远程协作 |
| **移动实验室** | OWL-Mobile | 可移动空间 | 项目制 | 科技大篷车 |
| **分布式实验室** | OWL-DAO | 社区自治 | DAO 机制 | 去中心化实验 |

### 联合实验室 (OWL-U) 细分

| 子类型 | 合作方 | 特点 |
|--------|--------|------|
| **区域联建** | 市教科院、科技馆 | 政府资源支持 |
| **学校联建** | 中小学、集团校 | 融入校园教育 |
| **高校联建** | 大学/研究机构 | 高中-大学衔接 |
| **企业联建** | 科技企业 | 产业资源连接 |
| **国际合作** | 海外机构 | 全球化扩展 |

## 加盟准则与流程

### 准入条件

| 维度 | 要求 | 验证方式 |
|------|------|----------|
| **理念认同** | 认同 OWL 核心价值观 | 价值公约签署 |
| **开放承诺** | 承诺向社区开放并参与知识共享 | 书面承诺 |
| **法人资格** | 合法运营主体 | 资质审核 |
| **负责人** | 至少1人全职/半全职 | 人员面谈 |
| **基础培训** | 完成入网培训 | 培训认证 |
| **空间条件** | 符合 Lite 级最低标准 | 现场评估 |

### 申请流程

```
1. 意向咨询 → 2. 在线申请 → 3. 资质审核 → 4. 现场评估
     → 5. 入网培训 → 6. 签署协议 → 7. 品牌授权 → 8. 正式运营
```

### 品牌授权规范

| 项目 | OWL | OWL-U | OWL-X |
|------|-----|-------|-------|
| 品牌使用 | 完整 | 联合标识 | 社区标识 |
| 课程体系 | 完整授权 | 选择性使用 | 参考借鉴 |
| 资源共享 | 完整访问 | 核心资源 | 公开资源 |
| 网络参与 | 核心成员 | 正式成员 | 生态伙伴 |
| 年度评估 | 内部评估 | 认证评估 | 自评报告 |

## 组织架构与角色

### 管理团队

| 角色 | 职责 | 最低配置 |
|------|------|----------|
| **实验室负责人** | 整体运营、战略规划 | 1人 |
| **技术导师/教研员** | 课程设计、技术支持 | 按规模 |
| **社区经理** | 社区运营、会员服务 | 按规模 |
| **运营助理** | 日常运营、行政支持 | 可选 |

### 决策权责矩阵 (RACI)

| 决策事项 | 节点 | 区域 | 总部 |
|----------|------|------|------|
| 日常运营 | R | I | - |
| 课程选用 | R | C | I |
| 活动策划 | R | C | I |
| 品牌使用 | C | C | A |
| 新节点准入 | C | R | A |
| 重大变更 | I | C | R/A |
| 网络标准 | I | C | R/A |

**R**=负责 **A**=批准 **C**=咨询 **I**=知会

## 退出机制

| 情况 | 流程 | 后续 |
|------|------|------|
| **主动退出** | 提前3个月通知，完成交接 | 友好退出，可重新申请 |
| **考核未达标** | 整改期 → 降级 → 退出 | 支持整改 |
| **严重违规** | 立即终止授权 | 品牌保护措施 |

## 常见误区

- ❌ "分布式"="无管理" → ✅ 有治理规则和底线
- ❌ "准入标准"="淘汰机制" → ✅ 以赋能为主
- ❌ "品牌授权"="加盟收费" → ✅ 价值导向非利润
- ❌ "联合实验室"="完全自主" → ✅ 需遵循核心标准
