# OWL实验室建设方案统一术语表

本术语表旨在统一OWL实验室建设方案中各模块的术语使用，确保表述口径一致，提高文档的可读性和逻辑性。

| 术语 | 统一称谓 | 定义 |
|------|----------|------|
| 创客空间 | 实验室 | 指OWL网络中的各类创客教育空间，包括核心实验室、联合实验室、社区实验室等 |
| 学员 | 学习者 | 参与OWL实验室学习活动的人员，包括儿童、青少年、成人等 |
| 导师 | 教育者 | 在OWL实验室中指导学习者的人员，包括全职导师、兼职导师、志愿者等 |
| 课程 | 学习体验 | 指OWL实验室提供的各类学习活动，包括体验课、工作坊、系列课、项目制课程等 |
| 项目 | 学习项目 | 以真实问题为驱动，通过动手实践完成的学习活动 |
| 设备 | 工具 | 指实验室中用于学习和创作的各类硬件设备和软件工具 |
| 空间 | 环境 | 指实验室的物理空间、数字环境和文化氛围的总和 |
| 运营 | 管理 | 指实验室的日常管理和运营活动，包括会员管理、活动策划、设备维护等 |
| 评价 | 评估 | 指对学习者能力发展、实验室运营质量和社会影响力的评估活动 |
| 节点 | 实验室 | 指OWL网络中的各个实验室，包括核心实验室、联合实验室、社区实验室等 |
| 治理 | 管理 | 指OWL网络的组织形式和协作规则 |
| 安全 | 安全合规 | 指实验室的物理安全、数据安全和伦理边界 |
| AI | 人工智能 | 指用于辅助学习和创作的各类人工智能工具和技术 |
| MVS | 最小可运行标准 | 指实验室正常运行所需的最低标准和要求 |
| SOP | 标准操作程序 | 指实验室运营的标准流程和操作规范 |
| PBL | 基于问题的学习 | 以真实问题为驱动，通过持续探究完成的学习方式。OWL采用CBL（基于挑战）、PBL（基于问题）、RBL（基于研究）三种探究方法 |
| UbD | 逆向设计 | 从学习目标出发倒推评价方式，再设计学习体验的课程设计方法 |
| PDSA | 持续改进循环 | 计划-执行-评估-改进的持续改进方法 |
| STEAM | 跨学科教育 | 科学、技术、工程、艺术、数学的跨学科整合教育 |
