# M03 空间与环境

> **一句话定位**: 创造激发创意、支持创造、包容多元的学习环境

---

## 核心原则

| 原则 | 内涵 | 设计体现 |
|------|------|----------|
| **学习可见** | 让过程可见，激发好奇 | 透明工作区、作品展示墙 |
| **灵活可变** | 支持快速重组 | 可移动家具、模块化设计 |
| **安全包容** | 安全底线 + 多元包容 | 无障碍设计、儿童友好 |
| **低门槛高上限** | 易入门，可深耕 | 分区设置、进阶设备 |
| **创新友好** | 激发创意，容错试错 | 实验区、草稿区 |

---

## 空间设计三层模型

```
┌─────────────────────────────────────────────┐
│  物理层 Physical                            │
│  空间布局、功能区、设备配置                   │
├─────────────────────────────────────────────┤
│  数字层 Digital                             │
│  数字工具、在线平台、混合体验                 │
├─────────────────────────────────────────────┤
│  文化层 Culture                             │
│  氛围营造、行为规范、社区感                   │
└─────────────────────────────────────────────┘
```

## 核心功能区

| 功能区 | 目的 | 最小面积 | 关键配置 |
|--------|------|----------|----------|
| **创作区** | 动手制作 | 40% | 工作台、工具、材料 |
| **展示区** | 成果展示 | 15% | 展示架、投影、灯光 |
| **讨论区** | 交流协作 | 20% | 可移动座椅、白板 |
| **储物区** | 物资存放 | 15% | 储物柜、材料架 |
| **缓冲区** | 过渡休息 | 10% | 入口、休息角 |

## 常见误区

- ❌ "高大上"="好空间" → ✅ 适用性比美观更重要
- ❌ 设计完成后固定不变 → ✅ 空间应持续迭代
- ❌ 只考虑成人视角 → ✅ 需考虑儿童体验
- ❌ 功能区边界严格 → ✅ 需支持灵活混用
