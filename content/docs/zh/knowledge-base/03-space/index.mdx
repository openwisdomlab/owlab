# M03 空间与环境

> **一句话定位**: 创造激发创意、支持创造、包容多元的学习环境

> *"空间是组织的'肢体语言'——它有意或无意地传递着文化价值观。"*
> — Make Space, Stanford d.school

---

## 设计哲学：空间即工具

传统观念将空间视为既定条件，而现代创客教育主张将其视为一种**可以塑造协作与创新的动态工具**。

| 核心理念 | 内涵 | 设计启示 |
|---------|------|---------|
| **情境即内容** | 空间的形态、功能、质感反映并引导身处其中人的文化与行为 | 每个设计决策都要问：这将如何促进协作？如何打破层级？如何降低尝试的门槛？ |
| **可破解性** | 理想的协作空间应可被使用者"破解"和改造 | 赋予用户改造环境的权利和工具，培养主人翁意识 |
| **为演变留白** | 空间不应被过度设计或固化 | 保留物理和概念上的"空白地带"，允许空间随需求生长 |

---

## 核心原则

| 原则 | 内涵 | 设计体现 |
|------|------|----------|
| **学习可见** | 让过程可见，激发好奇 | 透明工作区、作品展示墙 |
| **灵活可变** | 支持快速重组 | 可移动家具、模块化设计 |
| **安全包容** | 安全底线 + 多元包容 | 无障碍设计、儿童友好 |
| **低门槛高上限** | 易入门，可深耕 | 分区设置、进阶设备 |
| **创新友好** | 激发创意，容错试错 | 实验区、草稿区 |
| **可持续** | 环保材料，绿色运营 | 节能设计、循环利用 |

## 空间设计三层模型

```
┌─────────────────────────────────────────────┐
│  物理层 Physical                            │
│  空间布局、功能区、设备配置、材料选择         │
├─────────────────────────────────────────────┤
│  数字层 Digital                             │
│  数字工具、在线平台、混合体验、智能设备       │
├─────────────────────────────────────────────┤
│  文化层 Culture                             │
│  氛围营造、行为规范、社区感、仪式感          │
└─────────────────────────────────────────────┘
```

### 三层交互关系

| 层级 | 支撑 | 被支撑 | 关键设计点 |
|------|------|--------|------------|
| **物理层** | - | 数字层、文化层 | 功能分区、人体工程学 |
| **数字层** | 物理层 | 文化层 | 平台、工具、可视化 |
| **文化层** | 物理层、数字层 | - | 社区感、归属感、仪式 |

## d.school 空间设计语法

斯坦福 d.school 提出的四维度框架，为创客空间设计提供了系统性的"语法"。三层模型描述空间"**是什么**"，四维度框架指导"**如何设计**"。

### 维度一：场所 Places

具有不同功能和情感意义的空间区域，共同构成完整的生态系统。

| 场所类型 | 定义 | OWL 对应设计 |
|---------|------|-------------|
| **归属基站** Home Base | 团队工作和身份的锚点 | 项目团队专属角落，存放专属工具、资料、进行中的原型 |
| **聚集空间** Gathering Spaces | 促进社区成员互动的公共区域 | 开放厨房、咖啡角、分享舞台 |
| **过渡区** Thresholds | 连接不同活动区域的通道 | 标志心境/节奏转变的入口设计、缓冲区 |
| **支撑结构** Support Structure | 支持创造性工作的服务性空间 | 储物区、材料库、原型制作室 |

### 维度二：属性 Properties

可调整以改变行为和情绪的空间特征。

| 属性 | 定义 | 设计杠杆 |
|-----|------|---------|
| **姿态** Posture | 身体的物理位置 | 站立→活跃参与；坐下→深度讨论；躺卧→放松思考 |
| **朝向** Orientation | 人与人、人与物的相对位置 | 圆形→平等对话；面向墙→聚焦工作；面对面→辩论 |
| **表面** Surface | 工作平面的朝向 | 垂直白板→发散展示；水平桌面→动手制作 |
| **氛围** Ambience | 环境的整体气氛 | 灯光、色彩、声学、气味 |
| **密度** Density | 空间与活动的相对关系 | 紧凑→能量集中；宽敞→思维发散 |
| **存储** Storage | 非活动对象的状态 | 可见存储→激发灵感；隐藏存储→减少干扰 |

### 维度三：行动 Actions

空间应支持的创造性活动类型。

| 行动 | 定义 | 空间支持 |
|-----|------|---------|
| **沉浸** Saturate | 通过故事、照片、图表分享信息 | 四面可写墙、海报展示区、投影沉浸室 |
| **整合** Synthesize | 从复杂信息中提炼观点，发现模式 | 安静思考角、"白屋"封闭空间 |
| **聚焦** Focus | 持续专注于单一任务 | 个人"藏身处"、降噪隔间 |
| **发散** Flare | 突破限制，产生大量新概念 | 站立高桌、移动白板、开放头脑风暴区 |
| **实现** Realize | 将想法转化为有形实物 | 原型制作台、工具墙、材料开放陈列 |

### 维度四：态度 Attitudes

希望通过空间设计培养的文化价值观。

| 态度 | 内涵 | 空间表达 |
|-----|------|---------|
| **深度协作** Radical Collaboration | 汇集不同观点检验概念 | 无主位的圆桌、混合层级的座位 |
| **展示胜于言说** Show Don't Tell | 通过创造体验和实物传达想法 | 原型展示架、"储物画廊" |
| **行动偏好** Bias Toward Action | 鼓励尝试探索，从行动中学习 | 低门槛材料区、"草稿墙" |
| **关注人的价值** Focus on Human Values | 从用户真实需求获取设计灵感 | 用户研究墙、同理心地图展示 |
| **关注过程** Be Mindful of Process | 认识到失败是学习的一部分 | 失败展示区、迭代历程墙 |

### 四维度自检表

在设计或评估空间时，逐一检查：

- [ ] **场所**：是否有明确的归属基站？聚集空间是否促进交流？过渡区是否有仪式感？
- [ ] **属性**：是否支持多种姿态？氛围是否可调节？存储是否激发灵感？
- [ ] **行动**：是否同时支持发散与聚焦？是否有实现想法的空间？
- [ ] **态度**：空间是否传递协作、行动、包容失败的文化信号？

## 可塑性设计原则

### 责任-可调节性滑动标尺

空间的灵活性越高，使用者的维护责任就越大。

```
低灵活性 ◀━━━━━━━━━━━━━━━━━━━━━━━▶ 高灵活性
低责任   ◀━━━━━━━━━━━━━━━━━━━━━━━▶ 高责任

传统教室        │        OWL 实验室        │        d.school
固定桌椅        │      可移动+可重组       │     完全可破解
无需重置        │      团队自行重置        │     社区共同维护
```

### 赋权与所有权

| 策略 | 实施方式 | 预期效果 |
|-----|---------|---------|
| **提供改造工具** | 脚轮、连接件、移动白板随时可用 | 降低改造门槛 |
| **鼓励"破解"** | 观察用户自发改造，将其放大整合 | 发现未满足需求 |
| **建立重置规范** | 明确"默认状态"，团队用后恢复 | 平衡自由与秩序 |
| **表彰空间贡献** | 记录空间创新，纳入社区故事 | 强化主人翁意识 |

### 全尺寸原型验证

在项目早期进行真实尺度的空间原型测试：

1. **用廉价材料搭建**：泡沫板、瓦楞纸板、批量纸张
2. **亲身体验而非猜测**：走进去、坐下来、模拟工作流
3. **快速迭代**：发现问题后立即调整，当天多次迭代
4. **记录洞察**：拍照、标注、收集使用者反馈

## 空间分级标准

### 按实验室级别

| 级别 | 面积 | 功能区 | 设备配置 | 适用场景 |
|------|------|--------|----------|----------|
| **Lite** | ≥30m² | 3个核心区 | 基础套件 | 社区嵌入、学校角落 |
| **Pro** | ≥100m² | 5个功能区 | 高阶套件 | 独立空间、专业创客空间 |
| **Ultra** | ≥200m² | 7+功能区 | 专业仪器 | 旗舰实验室、研究中心 |

### 空间类型对照

| 类型 | 代码 | 特点 | 面积要求 |
|------|------|------|----------|
| **标准实验室** | OWL | 固定场所，完整功能 | ≥50m² |
| **联合实验室** | OWL-U | 共享空间，时段使用 | ≥30m² |
| **社区实验室** | OWL-X | 灵活配置，社区嵌入 | ≥20m² |
| **虚拟实验室** | OWL-Virtual | 纯线上 | N/A |
| **移动实验室** | OWL-Mobile | 可移动，临时部署 | 车载/集装箱 |

## 环境心理学应用

### 创新空间心理要素

| 要素 | 心理效应 | 设计应用 |
|------|----------|----------|
| **开放视野** | 激发创造力 | 高天花板、开放布局 |
| **自然元素** | 减压、专注 | 绿植、自然光、木材 |
| **色彩** | 情绪调节 | 创作区暖色、专注区冷色 |
| **声学** | 减少干扰 | 吸音材料、分区隔离 |
| **个性化** | 归属感 | 允许学员布置个人空间 |

### 年龄适配设计

| 年龄组 | 设计考量 | 特殊需求 |
|--------|----------|----------|
| **4-6岁** | 圆角、低矮家具 | 亲子区、安全材料 |
| **7-12岁** | 色彩丰富、激发好奇 | 可触摸展品、游戏化 |
| **13-17岁** | 独立空间、社交区 | 小组协作区、隐私角 |
| **18+岁** | 专业设备、灵活配置 | 深夜开放、咖啡区 |

## 无障碍设计

### 无障碍基本标准

| 项目 | 标准 | 法规依据 |
|------|------|----------|
| **通道宽度** | ≥1.2m | 轮椅通行 |
| **门宽** | ≥0.9m | 无障碍通行 |
| **坡道** | 坡度≤1:12 | 轮椅可达 |
| **工作台** | 可调高度 | 适应轮椅高度 |
| **卫生间** | 无障碍位 | 必须 |
| **标识** | 触觉+视觉 | 视障友好 |

### 多元包容设计

| 群体 | 设计考量 | 具体措施 |
|------|----------|----------|
| **轮椅使用者** | 可达性 | 坡道、升降台、低位开关 |
| **视障人士** | 触觉导引 | 盲道、触觉标识、语音提示 |
| **听障人士** | 视觉信息 | 闪光警报、字幕显示 |
| **自闭症谱系** | 感官友好 | 安静角、可预测环境 |
| **老年人** | 防滑防摔 | 扶手、防滑地面、放大标识 |

## 可持续设计

### 绿色运营原则

| 原则 | 实施方式 | 效果 |
|------|----------|------|
| **节能** | LED照明、智能控制 | 降低能耗30%+ |
| **节水** | 节水龙头、雨水收集 | 降低水耗 |
| **材料** | 环保认证材料 | 健康环境 |
| **循环** | 废料回收再利用 | 减少浪费 |
| **教育** | 可持续设计融入课程 | 培养意识 |

### 材料环保要求

| 材料 | 要求 | 认证 |
|------|------|------|

## 常见误区

- ❌ "高大上"="好空间" → ✅ 适用性比美观更重要
- ❌ 设计完成后固定不变 → ✅ 空间应持续迭代
- ❌ 只考虑成人视角 → ✅ 需考虑儿童体验
- ❌ 功能区边界严格 → ✅ 需支持灵活混用
- ❌ 忽视隐性环境 → ✅ 声光气味同样重要
- ❌ 只看硬件不看软件 → ✅ 文化氛围同样重要
