# M03 空间与环境

> **一句话定位**: 创造激发创意、支持创造、包容多元的学习环境

---

## 核心原则

| 原则 | 内涵 | 设计体现 |
|------|------|----------|
| **学习可见** | 让过程可见，激发好奇 | 透明工作区、作品展示墙 |
| **灵活可变** | 支持快速重组 | 可移动家具、模块化设计 |
| **安全包容** | 安全底线 + 多元包容 | 无障碍设计、儿童友好 |
| **低门槛高上限** | 易入门，可深耕 | 分区设置、进阶设备 |
| **创新友好** | 激发创意，容错试错 | 实验区、草稿区 |
| **可持续** | 环保材料，绿色运营 | 节能设计、循环利用 |

---

## 空间设计三层模型

```
┌─────────────────────────────────────────────┐
│  物理层 Physical                            │
│  空间布局、功能区、设备配置、材料选择         │
├─────────────────────────────────────────────┤
│  数字层 Digital                             │
│  数字工具、在线平台、混合体验、智能设备       │
├─────────────────────────────────────────────┤
│  文化层 Culture                             │
│  氛围营造、行为规范、社区感、仪式感          │
└─────────────────────────────────────────────┘
```

### 三层交互关系

| 层级 | 支撑 | 被支撑 | 关键设计点 |
|------|------|--------|------------|
| **物理层** | - | 数字层、文化层 | 功能分区、人体工程学 |
| **数字层** | 物理层 | 文化层 | 平台、工具、可视化 |
| **文化层** | 物理层、数字层 | - | 社区感、归属感、仪式 |

## 空间分级标准

### 按实验室级别

| 级别 | 面积 | 功能区 | 设备配置 | 适用场景 |
|------|------|--------|----------|----------|
| **Lite** | ≥30m² | 3个核心区 | 基础套件 | 社区嵌入、学校角落 |
| **Pro** | ≥100m² | 5个功能区 | 高阶套件 | 独立空间、专业创客空间 |
| **Ultra** | ≥200m² | 7+功能区 | 专业仪器 | 旗舰实验室、研究中心 |

### 空间类型对照

| 类型 | 代码 | 特点 | 面积要求 |
|------|------|------|----------|
| **标准实验室** | OWL | 固定场所，完整功能 | ≥50m² |
| **联合实验室** | OWL-U | 共享空间，时段使用 | ≥30m² |
| **社区实验室** | OWL-X | 灵活配置，社区嵌入 | ≥20m² |
| **虚拟实验室** | OWL-Virtual | 纯线上 | N/A |
| **移动实验室** | OWL-Mobile | 可移动，临时部署 | 车载/集装箱 |

## 核心功能区

### 功能区配置矩阵

| 功能区 | 目的 | Lite | Pro | Ultra | 关键配置 |
|--------|------|------|-----|-------|----------|
| **创作区** | 动手制作 | 40% | 35% | 30% | 工作台、工具、材料 |
| **展示区** | 成果展示 | 15% | 15% | 15% | 展示架、投影、灯光 |
| **讨论区** | 交流协作 | 20% | 15% | 15% | 可移动座椅、白板 |
| **储物区** | 物资存放 | 15% | 15% | 10% | 储物柜、材料架 |
| **缓冲区** | 过渡休息 | 10% | 10% | 10% | 入口、休息角 |
| **专业区** | 高阶设备 | - | 10% | 15% | 激光、CNC等 |
| **录制区** | 媒体制作 | - | - | 5% | 绿幕、灯光、声学 |

### 创作区设计要点

| 要素 | 标准 | 说明 |
|------|------|------|
| **工作台** | 高度可调 80-100cm | 适应不同身高 |
| **照明** | ≥500lux | 精细工作需要 |
| **电源** | 每工位2个插座 | 工具供电 |
| **通风** | 换气≥6次/小时 | 3D打印、焊接等 |
| **地面** | 耐磨防滑 | 安全清洁 |

### 展示区设计要点

| 要素 | 标准 | 说明 |
|------|------|------|
| **展示柜** | 透明可锁 | 保护作品 |
| **灯光** | 轨道灯可调 | 突出作品 |
| **标签** | 统一格式 | 作者、日期、说明 |
| **互动** | 二维码链接 | 数字延伸 |

## 环境心理学应用

### 创新空间心理要素

| 要素 | 心理效应 | 设计应用 |
|------|----------|----------|
| **开放视野** | 激发创造力 | 高天花板、开放布局 |
| **自然元素** | 减压、专注 | 绿植、自然光、木材 |
| **色彩** | 情绪调节 | 创作区暖色、专注区冷色 |
| **声学** | 减少干扰 | 吸音材料、分区隔离 |
| **个性化** | 归属感 | 允许学员布置个人空间 |

### 年龄适配设计

| 年龄组 | 设计考量 | 特殊需求 |
|--------|----------|----------|
| **4-6岁** | 圆角、低矮家具 | 亲子区、安全材料 |
| **7-12岁** | 色彩丰富、激发好奇 | 可触摸展品、游戏化 |
| **13-17岁** | 独立空间、社交区 | 小组协作区、隐私角 |
| **18+岁** | 专业设备、灵活配置 | 深夜开放、咖啡区 |

## 无障碍设计

### 无障碍基本标准

| 项目 | 标准 | 法规依据 |
|------|------|----------|
| **通道宽度** | ≥1.2m | 轮椅通行 |
| **门宽** | ≥0.9m | 无障碍通行 |
| **坡道** | 坡度≤1:12 | 轮椅可达 |
| **工作台** | 可调高度 | 适应轮椅高度 |
| **卫生间** | 无障碍位 | 必须 |
| **标识** | 触觉+视觉 | 视障友好 |

### 多元包容设计

| 群体 | 设计考量 | 具体措施 |
|------|----------|----------|
| **轮椅使用者** | 可达性 | 坡道、升降台、低位开关 |
| **视障人士** | 触觉导引 | 盲道、触觉标识、语音提示 |
| **听障人士** | 视觉信息 | 闪光警报、字幕显示 |
| **自闭症谱系** | 感官友好 | 安静角、可预测环境 |
| **老年人** | 防滑防摔 | 扶手、防滑地面、放大标识 |

## 选址与装修

### 选址评估清单

| 维度 | 评估项 | 权重 | 说明 |
|------|--------|------|------|
| **交通** | 公共交通可达性 | 高 | 地铁/公交站距离 |
| **安全** | 周边环境安全 | 高 | 治安、消防通道 |
| **成本** | 租金可持续性 | 高 | 控制在预算内 |
| **面积** | 满足MVS要求 | 高 | ≥30m² |
| **层高** | ≥2.8m | 中 | 设备安装需求 |
| **承重** | 满足设备需求 | 中 | 重型设备要求 |
| **水电** | 容量满足需求 | 高 | 电力、上下水 |
| **社区** | 目标用户集中 | 中 | 学校、社区 |

### 装修预算分配

| 项目 | 占比 | 说明 |
|------|------|------|
| **硬装** | 30-40% | 墙面、地面、水电 |
| **家具** | 20-30% | 工作台、储物柜 |
| **设备** | 20-30% | 见M05 |
| **软装** | 10-15% | 灯光、装饰 |

## 常见误区

- ❌ "高大上"="好空间" → ✅ 适用性比美观更重要
- ❌ 设计完成后固定不变 → ✅ 空间应持续迭代
- ❌ 只考虑成人视角 → ✅ 需考虑儿童体验
- ❌ 功能区边界严格 → ✅ 需支持灵活混用
- ❌ 忽视隐性环境 → ✅ 声光气味同样重要
- ❌ 只看硬件不看软件 → ✅ 文化氛围同样重要
