# 安全空间设计

## 概述

创客空间的安全设计需要平衡两个看似矛盾的目标：保障人员安全和激发探索创造。本文档提供物理安全设计的系统性指南，帮助节点创建既安全又充满活力的学习环境。

**核心原则**:
1. 安全是设计的起点，不是限制
2. 分级管理：不同区域不同风险等级
3. 设计消除危险，而非仅靠规则
4. 清晰的视觉指引和信息传达

## 1. 安全分区规划

### 1.1 风险等级分区

| 区域等级 | 风险水平 | 典型区域 | 准入要求 |
|---------|---------|---------|---------|
| **绿区** | 低风险 | 大厅、讨论区、阅读区 | 开放 |
| **黄区** | 中风险 | 手工区、电子区、编程区 | 基础培训 |
| **橙区** | 较高风险 | 木工区、焊接区 | 专项认证 |
| **红区** | 高风险 | 激光切割、CNC、化学区 | 高级认证+监督 |

## 2. 人流动线设计

### 2.1 动线规划原则

| 原则 | 说明 |
|------|------|
| **单向流动** | 高风险区域设置明确的进出方向 |
| **宽度充足** | 主通道≥1.5m，次通道≥1.2m |
| **视线通透** | 避免死角，便于监督和求助 |
| **紧急疏散** | 任何位置到安全出口≤30m |

### 2.2 交叉隔离

- 洁净区与脏污区分离
- 高噪音区与安静区隔离
- 危险品存储远离主通道
- 电气设备远离水源

## 3. 应急设施配置

### 3.1 消防设施

| 设施 | 配置要求 | 位置 |
|------|---------|------|
| **灭火器** | 每50m²至少1个 | 显眼处，距地1.5m以下 |
| **消防栓** | 按消防规范 | 走廊、操作区入口 |
| **烟雾报警** | 全覆盖 | 天花板 |
| **喷淋系统** | 高风险区 | 木工、焊接等区域 |

### 3.2 急救设施

| 设施 | 内容 | 位置 |
|------|------|------|
| **急救箱** | 创可贴、绷带、消毒液、烫伤膏等 | 每个操作区 |
| **洗眼器** | 化学区、焊接区必备 | 距工作台15秒步行 |
| **应急淋浴** | 化学区必备 | 化学操作区内 |
| **AED** | 自动体外除颤器 | 大厅显眼处 |

### 3.3 紧急出口

- 至少2个独立出口
- 出口宽度≥1.4m
- 常亮应急指示灯
- 门向外开启，不得上锁
- 疏散路线清晰标识

## 4. 标识系统

### 4.1 安全标识类型

| 类型 | 颜色 | 用途 | 示例 |
|------|------|------|------|
| **禁止标识** | 红色 | 禁止行为 | 禁止烟火、禁止触摸 |
| **警告标识** | 黄色 | 危险提醒 | 当心触电、当心高温 |
| **指令标识** | 蓝色 | 必须遵守 | 必须戴护目镜 |
| **提示标识** | 绿色 | 安全信息 | 紧急出口、急救箱 |

### 4.2 标识规范

- 标识高度：1.5-2m（成人视线）
- 儿童区域：0.8-1.2m
- 文字简洁，配图标
- 多语言（如需）
- 定期检查更新

## 5. 设备安全配置

### 5.1 设备布局

| 要求 | 说明 |
|------|------|
| **操作空间** | 设备周围留足操作和维护空间 |
| **安全距离** | 相邻设备间距≥1m |
| **稳固安装** | 大型设备固定于地面 |
| **接地保护** | 所有电气设备可靠接地 |

### 5.2 紧急停止

- 每台动力设备配备急停按钮
- 急停按钮红色，易于触及
- 定期测试急停功能
- 区域总电源开关明显标识

## 6. 年龄适配设计

### 6.1 儿童区域特殊要求

| 要素 | 儿童适配要求 |
|------|-------------|
| **家具** | 圆角处理，稳定不易倾倒 |
| **高度** | 工作台高度适配儿童身高 |
| **电源** | 使用安全插座盖 |
| **材料** | 无毒、无尖锐边缘 |
| **监督** | 成人可视范围内 |

### 6.2 视线管理

- 成人工作区与儿童区可视
- 高风险区有物理隔离
- 监控覆盖主要区域

## 常见问题

<details>
<summary>小空间如何实现安全分区？</summary>

即使空间有限，也可以通过以下方式实现分区：
1. 使用颜色地面标识区分区域
2. 移动隔断或工作台作为边界
3. 时间分区——不同时段不同用途
4. 高风险设备加装围栏或罩子
</details>

<details>
<summary>如何平衡安全与自由探索？</summary>

1. 低风险区完全开放，高风险区逐级准入
2. 用设计消除危险，而非仅靠规则限制
3. 安全培训让学习者有能力自主判断
4. 监督是赋能，不是看管
</details>
