# 无障碍设计指南

## 概述

> 无障碍设计不是为少数人的特殊考虑，而是让空间对所有人更友好的通用设计。

创客空间的核心理念是"人人可创造"。但如果物理空间本身成为障碍，这一理念就无法真正实现。无障碍设计不仅是法规要求，更是创客教育普惠性的体现。

**核心观点**：
1. **通用设计**：为一个群体设计的无障碍功能往往让所有人受益（"路沿坡道效应"）
2. **全流程考量**：无障碍不只是"能进来"，而是"能完整参与"
3. **成本效益**：早期规划的无障碍设计成本远低于后期改造
4. **文化包容**：物理无障碍是心理包容的基础

## 1. 无障碍基本标准

### 1.1 空间通行标准

| 项目 | 最低标准 | 推荐标准 | 法规依据 | 说明 |
|------|----------|----------|----------|------|
| **主通道宽度** | ≥1.2m | ≥1.5m | GB 50763 | 容纳轮椅双向通行 |
| **次通道宽度** | ≥0.9m | ≥1.2m | GB 50763 | 单向轮椅通行 |
| **门净宽** | ≥0.9m | ≥1.0m | GB 50763 | 建议推拉门或感应门 |
| **门槛高度** | ≤15mm | 无门槛 | GB 50763 | 门槛需做斜坡处理 |
| **坡道坡度** | ≤1:12 | ≤1:16 | GB 50763 | 每升高0.75m设休息平台 |
| **轮椅回转** | ∅1.5m | ∅1.8m | GB 50763 | 各功能区需留回转空间 |

### 1.2 工作台与设备标准

| 项目 | 标准 | 说明 |
|------|------|------|
| **台面高度** | 0.7-0.85m可调 | 适应轮椅和站立 |
| **膝部空间** | ≥0.65m深×0.6m宽×0.65m高 | 轮椅使用者必需 |
| **操作区域** | 伸手范围内 | 避免需要弯腰或攀高 |
| **开关高度** | 0.9-1.2m | 轮椅可及范围 |
| **插座高度** | 0.4-1.0m | 避免地面插座 |
| **显示屏倾斜** | 可调节 | 适应不同视角 |

### 1.3 卫生间标准

| 项目 | 标准 | 说明 |
|------|------|------|
| **无障碍位面积** | ≥2.0m×1.5m | 独立无障碍卫生间更佳 |
| **扶手配置** | L型+水平 | 马桶两侧及洗手台 |
| **紧急呼叫** | 必须 | 拉绳或按钮，连接到监控处 |
| **门开启方向** | 外开或推拉 | 便于紧急救援 |
| **镜子高度** | 底边≤0.9m | 轮椅使用者可用 |

## 2. 多元包容设计策略

### 2.1 肢体障碍（轮椅/拐杖使用者）

**设计要点**：

```
┌─────────────────────────────────────────────────────────────┐
│                    轮椅使用者动线设计                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   入口 ──→ 前台 ──→ 储物 ──→ 工作区 ──→ 展示区            │
│     ↓        ↓        ↓         ↓         ↓                │
│   坡道    低柜台   低位柜   可调桌   低位展架              │
│   感应门  轮椅高度  腿部空间  回转空间  通道宽度            │
│                                                             │
│   关键节点检查：                                             │
│   □ 每个节点是否有≥1.5m回转空间                            │
│   □ 是否有无障碍替代路径                                    │
│   □ 紧急出口是否轮椅可达                                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**实施清单**：
- [ ] 入口坡道坡度≤1:12，宽度≥1.2m
- [ ] 所有工作区至少1个可调节高度工作台
- [ ] 设备操作面板高度0.9-1.2m
- [ ] 紧急出口配备无障碍通道
- [ ] 停车区预留无障碍车位（如有）

### 2.2 视觉障碍（全盲/低视力）

**设计要点**：

| 策略 | 实施方法 | 成本 |
|------|----------|------|
| **触觉导引** | 入口至主要区域铺设盲道砖或纹理地面 | 中 |
| **高对比度** | 地面/墙面/家具使用对比色（>70%对比度） | 低 |
| **盲文标识** | 关键位置（门牌、楼层、电梯）配盲文 | 低 |
| **语音辅助** | 智能助手语音导览（可选） | 中高 |
| **照明充足** | 工作区≥500lux，避免眩光 | 低 |

**标识系统设计**：
- 字体：无衬线体，笔画清晰
- 字号：主标识≥50mm，次标识≥25mm
- 颜色：深色底+浅色字或反之，避免红绿搭配
- 盲文：主要标识配备盲文或触觉图案

### 2.3 听觉障碍（聋/重听）

**设计要点**：

| 策略 | 实施方法 | 必要性 |
|------|----------|--------|
| **视觉警报** | 火警配闪光灯（≥75cd） | 必须 |
| **振动提醒** | 重要设备完成提醒可配振动 | 推荐 |
| **字幕显示** | 公共屏幕播放内容配字幕 | 推荐 |
| **视线通透** | 避免完全封闭的隔断，保持视觉联系 | 推荐 |
| **助听系统** | 感应线圈系统（T-coil）用于讲座区 | 高级 |

### 2.4 神经多样性（自闭症谱系/ADHD/感觉处理差异）

**这是创客空间中常被忽视但重要的群体**。

| 需求 | 设计策略 | 实施方法 |
|------|----------|----------|
| **感官避难** | 低刺激区 | 设置"安静角"，柔和灯光，隔音材料 |
| **可预测性** | 清晰分区 | 明确的区域划分和视觉提示 |
| **感官调节** | 可控刺激 | 可调灯光、降噪耳机可借用 |
| **运动需求** | 运动友好 | 站立工位、运动球椅选项 |
| **过渡支持** | 视觉时间表 | 活动流程可视化 |

**安静角设计建议**：
- 位置：远离主要活动区和设备噪音
- 尺寸：至少2m×2m的独立空间
- 灯光：可调暗，暖色调
- 材料：吸音墙面，地毯或软垫
- 设施：舒适座椅，可选加重毯

### 2.5 全龄适配（儿童/老年人）

**儿童友好**：
- 家具边角倒圆（R≥10mm）
- 低矮材料架（≤1.2m）
- 儿童专用工具（尺寸适配）
- 防夹手设计（门、抽屉）

**老年人友好**：
- 防滑地面（R10级以上）
- 充足照明（≥500lux）
- 带扶手座椅
- 大号字体标识
- 休息座椅（每30m动线）

### 3.2 分级实施建议

| 级别 | 预算 | 优先实施项 |
|------|------|------------|
| **基础级** | <5000元 | 坡道、高对比标识、可调工作台(1个)、降噪耳机 |
| **标准级** | 5000-2万元 | 无障碍卫生间改造、感应门、盲道、安静角 |
| **完善级** | 2-5万元 | 全面照明升级、助听系统、智能门禁、环境监测 |

## 5. 常见误区与正解

| 误区 | 正解 |
|------|------|
| ❌ "无障碍就是加个坡道" | ✅ 无障碍是全流程体验，包括到达、使用设备、参与社交、紧急疏散 |
| ❌ "没有残障用户就不用做" | ✅ "路沿坡道效应"：坡道也方便推车、行李箱、婴儿车；低刺激区也帮助需要安静的普通用户 |
| ❌ "无障碍设计很贵" | ✅ 新建时规划无障碍增加成本<1%；后期改造才昂贵 |
| ❌ "符合法规就够了" | ✅ 法规是最低标准；真正的包容需要超越法规的用心设计 |
| ❌ "神经多样性不算无障碍" | ✅ 感官友好设计是现代无障碍设计的重要组成部分 |

## 常见问题

<details>
<summary>小型空间（<50㎡）如何做无障碍？</summary>

**空间有限不等于放弃无障碍**。优先级建议：

1. **入口可达**（必须）：至少有一条无障碍入口路径
2. **通道畅通**（必须）：保持≥0.9m的核心通道
3. **一个可调工位**（推荐）：至少一个轮椅可用工位
4. **清晰标识**（推荐）：高对比度、大字体

小空间可能无法设置独立无障碍卫生间，但应确保最近的无障碍卫生间信息明确告知。
</details>

<details>
<summary>如何了解特殊需求用户的真实需求？</summary>

**最佳实践是让用户参与设计**：

1. **邀请体验**：请轮椅使用者、视障人士等实际体验空间
2. **用户测试**：在改造前后请目标用户反馈
3. **咨询组织**：联系当地残障人士协会获取建议
4. **模拟体验**：工作人员坐轮椅或蒙眼体验自己的空间

记住：不同个体需求差异很大，没有"标准残障人士"。
</details>

<details>
<summary>现有空间改造优先做什么？</summary>

**按影响面积和受益人数排序**：

1. **入口无障碍**（影响所有人能否进入）
2. **主通道畅通**（影响所有人的移动）
3. **卫生间改造**（影响能否长时间停留）
4. **工作台适配**（影响能否参与核心活动）
5. **感官友好区**（影响神经多样性群体体验）
</details>
