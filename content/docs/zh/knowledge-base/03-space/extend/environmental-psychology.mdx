# 环境心理学应用指南

## 概述

环境心理学研究人与物理环境之间的相互作用，揭示空间如何影响我们的认知、情绪和行为。对于创客空间而言，理解这些原理可以帮助我们设计出更能激发创造力、促进学习、支持协作的环境。

**核心应用领域**:
1. **注意力管理** — 支持专注，促进恢复
2. **压力调节** — 创造舒适，减少焦虑
3. **创造力激发** — 环境刺激与认知联结
4. **行为引导** — 空间布局影响使用模式

> **社会物质性视角 (Sociomateriality Perspective)**
>
> 本指南主要聚焦于**物理环境**对人的影响。但需要注意，空间从来不是中性的"容器"——它在使用中被**社会性地建构**。同样的物理布局，在不同的社群文化、权力关系、使用实践中，会产生截然不同的心理效果。
>
> Wanda Orlikowski 等学者提出的"社会物质性"理论提醒我们：**技术、物质环境与社会实践是相互构成的**。一张"协作大桌"是否真能促进协作，取决于社群的对话规范、权力距离、信任水平——而非仅仅桌子的形状。因此，空间设计必须与**社群培育**同步进行。
>
> 延伸阅读: Orlikowski, W. (2007). Sociomaterial Practices: Exploring Technology at Work. *Organization Studies*.

## 第一部分：理论框架

### 1.1 注意力恢复理论

**Kaplan 夫妇的 ART 理论** [E2]

```
定向注意力 (Directed Attention)
  │  特点: 需要努力维持，易疲劳
  │  应用: 专注工作、学习任务
  │
  ↓  疲劳后需要...

非自主注意力 (Involuntary Attention)
  │  特点: 不需努力，自然吸引
  │  应用: 自然环境、柔和刺激
  │
  ↓  促进...

注意力恢复 (Attention Restoration)
     特点: 定向注意力得到休息
     效果: 恢复专注能力
```

**恢复性环境的四个特征**:

| 特征 | 定义 | 空间设计策略 |
|-----|------|-------------|
| **远离** (Being Away) | 从日常事务中抽离 | 视觉/心理边界、主题分区 |
| **延伸** (Extent) | 足够丰富的内容探索 | 层次丰富的环境、发现元素 |
| **迷人** (Fascination) | 轻松吸引注意力 | 自然元素、柔和视觉焦点 |
| **兼容** (Compatibility) | 与个人需求相符 | 多样化选择、可控环境 |

### 1.2 压力恢复理论

**Roger Ulrich 的 SRT 理论** [E2]

```
压力源暴露
     │
     ↓
负面情绪反应
     │
     ├─→ 支持性环境 ─→ 快速恢复
     │   (自然、积极)      正向情绪↑ 生理指标↓
     │
     └─→ 非支持环境 ─→ 持续压力
         (拥挤、混乱)      焦虑↑ 表现↓
```

**压力恢复的环境特征**:

| 特征 | 具体表现 | 设计应用 |
|-----|---------|---------|
| 视觉复杂度适中 | 不过于单调或混乱 | 有节奏的视觉变化 |
| 自然元素 | 植物、水、自然光 | 绿化、天窗、水景 |
| 空间开阔感 | 视野开放、不压抑 | 高天花、透视感 |
| 安全感 | 有退路、可监视 | 背靠墙、可控视野 |
| 秩序感 | 整洁、可预测 | 清晰分区、收纳系统 |

### 1.3 情绪-行为模型

**Mehrabian-Russell PAD 模型** [E2]

```
环境刺激 → 情绪反应 → 行为倾向

情绪三维度:
├─ 愉悦度 (Pleasure): 快乐 ↔ 不快
├─ 唤醒度 (Arousal): 兴奋 ↔ 平静
└─ 支配感 (Dominance): 控制 ↔ 被控制

行为倾向:
├─ 趋近 (Approach): 停留、探索、交流
└─ 回避 (Avoidance): 离开、逃避、封闭
```

**创客空间情绪设计目标**:

| 区域 | 情绪目标 | 设计策略 |
|-----|---------|---------|
| 入口 | 高愉悦、中唤醒 | 明亮、友好、有吸引力 |
| 工作区 | 中愉悦、适中唤醒 | 舒适、专注、不过度刺激 |
| 协作区 | 高愉悦、高唤醒 | 活跃、开放、激发互动 |
| 休息区 | 高愉悦、低唤醒 | 放松、安静、恢复性 |

### 1.4 可供性理论

**James Gibson 的 Affordance 理论**

```
可供性 (Affordance):
  环境提供的行动可能性

  物理可供性: 椅子 → 坐
  感知可供性: 门把 → 推/拉
  社会可供性: 圆桌 → 平等对话
```

**创客空间可供性设计**:

| 设计元素 | 可供性 | 期望行为 |
|---------|-------|---------|
| 白板墙 | 书写、展示 | 头脑风暴、分享想法 |
| 高脚桌 | 站立、快速交流 | 非正式讨论、碰撞 |
| 工具墙 | 拿取、选择 | 动手制作、实验 |
| 沙发区 | 休憩、放松 | 休息、深度交流 |
| 开放大桌 | 协作、展开 | 团队项目、大型作品 |

## 第 1.5 节：2024-2025 前沿研究与标准

### 1.5.1 神经建筑学新进展

**Academy of Neuroscience for Architecture (ANFA) 研究重点** [E2]

神经建筑学运用脑科学方法研究建筑环境对人类认知和情绪的影响：

| 研究领域 | 发现 | 设计启示 |
|---------|------|---------|
| **空间导航** | 海马体对空间复杂度敏感 | 提供可理解的空间结构 |
| **视觉处理** | 研究表明人们倾向于偏好曲线形态，这可能与降低心理防御有关 | 减少直角，增加曲线 |
| **自然偏好** | 分形图案激活奖励回路 | 融入自然纹理和图案 |
| **光线影响** | 蓝光调节昼夜节律和警觉 | 动态照明，模拟日光变化 |
| **高度感知** | 高天花激活额叶创意区 | 创意区采用高天花设计 |

**脑电/眼动研究对设计的指导**:

```
实验室到实地的转化:

实验室发现                      设计应用
─────────────                   ─────────
高天花→抽象思维脑波        →    头脑风暴区 3m+ 层高
红色→警觉/细节关注         →    精密工作区点缀红色
曲线→放松/开放性思维       →    休息区使用曲线家具
自然视野→压力激素下降      →    工作区朝向窗户/绿植
节律照明→认知表现提升      →    可调色温照明系统
```

### 1.5.2 WELL v2 Mind 概念要求

**WELL Building Standard v2 - Mind Concept (2024)** [E2]

WELL v2 的 Mind 概念聚焦认知与情绪健康，包含以下条目：

| 条目 | 内容 | 创客空间应用 |
|-----|------|-------------|
| M01 | 心理健康促进 | 压力管理资源、社群支持 |
| M02 | 心理健康教育 | 心理健康工作坊、意识提升 |
| M03 | 心理健康服务 | 咨询渠道、危机支持 |
| M04 | 心理恢复空间 | 冥想室、安静角落 |
| M05 | 压力管理空间 | 恢复性环境、自然元素 |
| M06 | 恢复性自然 | 室内绿植、自然视野 |
| M07 | 可控性增强 | 个人环境控制、选择自由 |
| M11 | 睡眠支持 | 休息空间、节律照明 |

**WELL 认证对创客空间的启示**:

```
核心要求转化:

1. 恢复性自然 (M06)
   最低要求: 每 100m² 至少 1 盆 ≥1m 高植物
   推荐: 绿视率 ≥10%，自然景观视野

2. 心理恢复空间 (M04)
   最低要求: 每 300m² 提供 1 个安静区
   推荐: 声学隔离、可调光、自然元素

3. 可控性增强 (M07)
   最低要求: 50% 工位有照明控制
   推荐: 温度、声音、家具可个人调节
```

| 类别 | 模式 | 创客空间应用示例 |
|-----|------|-----------------|

| | 2. 非视觉自然联结 | 自然音景、木材触感 |

| | 9. 材质与质地联结 | 木材、石材、天然织物 |

| | 12. 神秘与诱惑 | 部分遮挡的景观、探索感 |

**亲生物设计效益实证**:

```
研究证据汇总 (注: 具体数值来自特定研究情境，实际效果因环境而异):

自然视野        → 有助于加快康复过程 (Ulrich 1984)
室内绿植        → 研究表明有助于提升工作效率和专注力 (Nieuwenhuis 2014)
自然音景        → 有助于认知恢复和注意力恢复 (Ratcliffe 2016)
木材内装        → 研究表明有助于降低压力水平 (Tsunetsugu 2007)
生物形态设计    → 研究表明有助于激发创造性思维 (Kellert 2015)
```

## 第二部分：认知与学习

### 2.1 认知负荷管理

**环境认知负荷来源**:

| 负荷类型 | 来源 | 管理策略 |
|---------|------|---------|
| 内在负荷 | 任务本身复杂性 | 分步骤指导、脚手架 |
| 外在负荷 | 无关信息干扰 | 减少视觉噪音、降噪 |
| 相关负荷 | 学习加工过程 | 关联提示、类比支持 |

**降低外在认知负荷的环境设计**:

```
视觉简化:
├─ 统一的色彩系统
├─ 清晰的信息层级
├─ 隐藏非必要信息
└─ 留白空间

听觉简化:
├─ 背景噪音控制
├─ 声学分区
├─ 白噪音/自然音景
└─ 静音区设置

空间简化:
├─ 功能分区明确
├─ 导视系统清晰
├─ 动线直观
└─ 存储系统有序
```

### 2.2 多感官学习环境

**感官通道与学习**:

| 感官 | 学习作用 | 环境设计 |
|-----|---------|---------|
| 视觉 | 信息获取主通道 | 照明、色彩、展示 |
| 听觉 | 语言学习、环境感知 | 声学、背景音 |
| 触觉 | 动手体验、精细技能 | 材质、温度、工具 |
| 嗅觉 | 情绪记忆、状态切换 | 气味管理、空气质量 |
| 本体觉 | 身体姿势、空间定位 | 家具、动线、活动空间 |

**感官丰富但不过载**:

```
感官平衡设计:

  刺激强度
      │
  高  │      * 协作区
      │   * 展示区
  中  │      * 工作区
      │   * 安静区
  低  │      * 冥想角
      └─────────────────→
          单一      多元
          感官参与度
```

### 2.3 具身认知与空间

**具身认知观点**: 认知不仅发生在大脑，还与身体和环境密切相关

**空间设计启示**:

| 原理 | 空间应用 |
|-----|---------|
| 身体姿势影响思维 | 站立讨论→更积极；躺卧→更创意 |
| 空间隐喻影响认知 | 高天花→抽象思维；低天花→细节关注 |
| 移动促进思维 | 走动空间、散步通道 |
| 手势支持理解 | 足够的手势空间、演示区 |
| 环境线索触发知识 | 情境化学习环境、实物参照 |

**实践建议**:

```
头脑风暴:
├─ 站立围圈
├─ 大面积白板
├─ 移动自由
└─ 高天花空间

细节工作:
├─ 坐姿专注
├─ 任务照明
├─ 封闭隔间
└─ 低刺激环境

创意发想:
├─ 躺卧/休闲姿势
├─ 视觉丰富
├─ 音乐背景
└─ 户外/自然视野
```

## 第三部分：情绪与幸福感

### 3.1 积极情绪设计

**积极情绪的环境触发器**:

| 情绪 | 触发元素 | 设计应用 |
|-----|---------|---------|
| 愉悦 | 美感、秩序、自然 | 艺术装饰、绿化、整洁 |
| 兴趣 | 新奇、复杂度 | 展示变化、发现元素 |
| 希望 | 目标可见、进步感 | 进度展示、成就墙 |
| 自豪 | 成就展示、认可 | 作品展示、荣誉墙 |
| 感激 | 社群连接、共享 | 协作空间、社区感 |

**拓展-建构理论应用** (Barbara Fredrickson):

```
积极情绪 → 拓展注意力 → 建构资源
                ↓
    创造力↑  学习效率↑  人际关系↑
                ↓
         长期幸福感和韧性
```

### 3.2 压力缓冲区设计

**压力缓冲空间类型**:

| 类型 | 功能 | 设计特点 |
|-----|------|---------|
| 自然视野区 | 视觉恢复 | 窗户朝向绿地、远景 |
| 绿植角落 | 微恢复 | 密集绿植、自然材质 |
| 安静舱 | 感官隔离 | 隔音、调光、私密 |
| 户外连接 | 全面恢复 | 露台、花园、庭院 |
| 水元素 | 镇静效果 | 水景、流水声 |

**微恢复设计策略**:

```
3-5分钟微恢复:
├─ 站起来看窗外绿植
├─ 走到饮水区喝水
├─ 触摸桌上盆栽
└─ 听几分钟自然音

15-30分钟恢复:
├─ 休息区小憩
├─ 户外短暂散步
├─ 冥想角静坐
└─ 切换到低强度任务
```

### 3.3 社会支持环境

**归属感的空间建构**:

| 要素 | 空间表现 | 心理效果 |
|-----|---------|---------|
| 社群照片墙 | 成员可见 | "我是其中一员" |
| 公共仪式空间 | 聚会场所 | 共同体验 |

### 4.5 空间属性的心理学基础

**d.school Properties 维度与行为心理** [make-space-2012]

Stanford d.school 提出的"属性"维度揭示了空间特征如何影响心理和行为：

| 属性 | 心理学原理 | 设计应用 |
|-----|-----------|---------|
| **姿态** Posture | 体态反馈影响认知状态 | 站立→唤醒度↑→参与度↑；坐下→深度思考 |
| **朝向** Orientation | 空间位置影响社会动力 | 圆形→平等感↑；面向墙→聚焦↑ |
| **表面** Surface | 垂直vs水平影响思维模式 | 垂直白板→发散展示；水平桌面→动手操作 |
| **氛围** Ambience | 环境刺激影响情绪状态 | 灯光、色彩、声学、气味综合作用 |
| **密度** Density | 空间紧凑度影响能量水平 | 紧凑→能量集中；宽敞→思维发散 |
| **存储** Storage | 可见vs隐藏影响心理暗示 | 可见工具→激发灵感；隐藏杂物→减少干扰 |

**姿态与参与度的具体关系**:

```
站立姿态的心理效应:
├─ 生理唤醒水平提升 (心率、皮肤电导)
├─ 领导力感知更平等分布
├─ 讨论节奏加快、更高效
├─ 主动贡献门槛降低
└─ 领地意识减弱

设计应用:
├─ 站立高桌 (约 1050mm) 用于快速讨论
├─ Scrum 站会专用区
├─ 液压升降桌支持姿态切换
└─ 高脚凳作为站立的过渡选择
```

**灯光调光器的情绪调节作用**:

| 照明状态 | 心理效应 | 适用活动 |
|---------|---------|---------|
| 明亮白光 (5000K+) | 警觉、专注 | 精细工作、检查 |
| 中性白光 (4000K) | 平衡、中性 | 日常工作 |
| 暖黄光 (2700-3500K) | 放松、舒适 | 讨论、社交 |
| 可调暗淡 | 亲密、反思 | 冥想、恢复 |

**设计要点**:
- 所有区域必须安装调光器
- 提供多种色温光源组合
- 允许使用者自行调节 (个人控制感)
- 场景预设快捷切换 (工作/展示/社交/清洁)

**创意孵化区设计**:

```
配置要素:
├─ 舒适座椅 (吊椅、懒人沙发)
├─ 自然视野或绿植
├─ 柔和的环境光
├─ 可选背景音乐/自然音
├─ 书籍/杂志/灵感素材
├─ 笔记工具 (随时记录)
└─ 无明显时钟/任务提示
```

## 第五部分：行为设计

### 5.1 助推理论应用

**Thaler & Sunstein 助推 (Nudge)** [E2]

```
定义: 在保留选择自由的前提下，
      通过改变选择架构引导更好的决策

创客空间应用:
├─ 默认选项: 设备默认安全设置
├─ 显著性: 安全装备放在显眼位置
├─ 社会证明: 展示他人良好行为
├─ 即时反馈: 能耗/使用时长可视化
└─ 简化选择: 推荐入门项目路径
```

**行为助推设计示例**:

| 目标行为 | 助推设计 |
|---------|---------|
| 佩戴防护设备 | 护目镜架设在设备旁，与开机联动提醒 |
| 工具归位 | 工具轮廓描边，缺失一目了然 |
| 节约用电 | 设备使用成本可视化显示 |
| 清理工位 | 离开前自动播放清理提示音 |
| 分享项目 | 展示墙预留空框，邀请填满 |

### 5.2 空间语法与流动

**空间语法分析**:

```
整合度 (Integration):
  高整合: 容易到达，人流汇聚
  低整合: 相对隔离，私密安静

应用:
  高整合区域 → 公共服务、社交功能
  低整合区域 → 专注工作、存储

连接度 (Connectivity):
  多连接: 灵活、选择多
  少连接: 明确、引导强
```

**动线设计心理学**:

| 动线类型 | 心理效果 | 适用场景 |
|---------|---------|---------|
| 直线 | 效率、目的性 | 主通道、紧急出口 |
| 曲线 | 探索、发现 | 展示区、休闲动线 |
| 网格 | 灵活、民主 | 开放工作区 |
| 环形 | 浏览、游逛 | 展览、商店 |

### 5.3 领域行为管理

**领域性 (Territoriality)**:

```
领域类型:

└─ 公共领域: 共享空间、公共设备

设计考量:
├─ 允许适度个性化 (主要领域)
├─ 明确使用规则 (次要领域)

```

**领域边界设计**:

| 边界类型 | 实现方式 | 心理效果 |

| 物理边界 | 隔墙、屏风 | 明确分隔 |
| 象征边界 | 地面颜色、灯光 | 提示区分 |
| 社会边界 | 规则、文化 | 行为规范 |
| 时间边界 | 预约制、时段 | 使用秩序 |

## 第七部分：环境评估工具

### 7.1 环境心理学测量

**常用评估量表**:

| 量表 | 测量内容 | 应用场景 |
|-----|---------|---------|
| PRS (恢复性量表) | 注意力恢复潜力 | 休息区评估 |
| POMS (情绪量表) | 情绪状态变化 | 前后测对比 |
| NASA-TLX | 认知负荷 | 工作区评估 |
| POE (使用后评估) | 综合满意度 | 整体评估 |

**自编评估问卷示例**:

```
空间感受评估 (1-7分):

认知维度:
□ 这个空间帮助我集中注意力
□ 我在这里能高效完成任务
□ 这里的信息容易理解

情感维度:
□ 我在这个空间感到舒适
□ 这里让我感到有创造力
□ 我喜欢在这里度过时间

社会维度:
□ 这里容易与他人交流
□ 我感到自己是这里的一员
□ 这里有适合独处的地方
```

### 7.2 行为观察方法

**行为地图法 (Behavior Mapping)**:

```
记录内容:
├─ 位置: 人在哪里
├─ 行为: 在做什么
├─ 时间: 什么时候
├─ 人数: 多少人
└─ 特征: 年龄、性别等

分析维度:
├─ 空间使用率
├─ 行为类型分布
├─ 高峰/低谷时段
├─ 热点/冷点区域
└─ 社交互动模式
```

**追踪观察法**:

```
追踪一个用户的完整旅程:
入口 → 签到 → 工作区选择 → 取材料 →
工作过程 → 休息 → 社交 → 清理 → 离开

记录:
├─ 动线路径
├─ 停留时长
├─ 困惑点
├─ 情绪变化
└─ 求助行为
```

### 7.3 数据驱动优化

**环境传感器数据**:

| 数据类型 | 传感器 | 应用 |
|---------|-------|------|
| 温湿度 | 温湿度传感器 | 舒适度监控 |
| 光照 | 照度传感器 | 照明调节 |
| 噪音 | 分贝计 | 声学管理 |
| 空气质量 | CO2/PM2.5 | 通风控制 |
| 人流量 | 红外/视觉 | 空间使用分析 |
| 设备使用 | 智能插座 | 资源利用 |

**数据分析与洞察**:

```
关联分析示例:

环境参数     →    用户行为
CO2 > 1000ppm    →    专注度下降
照度 < 300lux   →    离开率上升
噪音 > 65dB     →    协作区使用↑
温度 > 26°C     →    停留时间↓

优化决策:
├─ CO2 超标自动增加新风
├─ 光照不足自动补光
├─ 噪音地图指导分区
└─ 个人舒适度偏好记忆
```

### 8.1 环境心理学设计清单

```
□ 美感与秩序感
```

**创造力支持**:
```

□ 收敛思维空间 (低刺激、专注)
□ 创意孵化区 (放松、灵感)
□ 原型制作空间 (动手、测试)
□ 跨领域碰撞的物理邻近

```

**行为引导**:
```

□ 良好习惯的默认选项

□ 即时反馈机制
□ 动线引导期望行为
□ 领域边界清晰合理
```

### 8.2 常见问题诊断

| 问题现象 | 可能原因 | 环境心理学解释 | 改善建议 |
|---------|---------|---------------|---------|
| 使用者容易疲劳 | 缺乏恢复性环境 | 注意力过度消耗 | 增加自然元素和休息空间 |
| 创意产出不足 | 环境过于单调 | 刺激不足，联想受限 | 增加视觉复杂度和材质变化 |
| 协作较少发生 | 缺乏碰撞机会 | 功能邻近不足 | 设置碰撞点，调整动线 |
| 新人难以融入 | 领域边界过强 | 归属感建立困难 | 增加公共区域，欢迎仪式 |
| 噪音投诉多 | 声学分区不当 | 认知负荷过高 | 重新规划声学分区 |
