## 理论基础

<Callout type="info">
**核心理论来源**：空间设计的理论基础详见 [L01 空间的塑造](/docs/living-modules/01-space-as-educator)。

L 序列是"道"（理念层），M 序列是"术"（执行层）。理念指引实践，实践验证理念。
</Callout>

### L01 核心理论概览

| 理论方向 | 核心概念 | M03 落地应用 |
|---------|---------|-------------|
| **神经建筑学** | 空间影响大脑活动与认知 | 天花板高度、光线色温参数 |
| **第三空间 2.0** | 物理-数字混合社交场所 | 混合学习节点设计 |
| **可供性生态** | 环境"邀请"的行为 | 空间功能规划 |
| **时间景观** | 空间的时间节奏设计 | 快/慢节奏分区 |
| **失败剧场** | 让失败可见可学习 | 失败墙、原型坟场 |
| **边界溶解** | 促进跨界交流 | Allen 曲线布局 |

### 关键引用
