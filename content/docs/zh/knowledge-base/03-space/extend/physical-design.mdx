# 物理空间设计深度指南

## 概述

物理空间是创客活动的载体，其设计质量直接影响使用者的安全、舒适、效率和创造力。本文档从专业角度深入探讨空间设计的核心要素，提供可操作的技术规范。

**核心设计原则**:
1. **安全优先** — 所有设计决策以人身安全为第一考量
2. **人因适配** — 空间尺度、工作面高度、动线设计符合人体工程学
3. **感官舒适** — 光、声、热、空气质量达到舒适标准
4. **灵活适应** — 空间可重构以适应不同活动需求

### 空间面积标准对比

| 标准来源 | 人均面积 | 适用场景 | 说明 |

WELL 认证覆盖全球 130+ 国家，超过 50 亿平方英尺空间：

| 概念 | 关键指标 | 创客空间应用 |
|-----|---------|-------------|
| **Light (光)** | 日光模拟 sDA ≥55%，照度 300-1000 lux | 工作区自然采光优化 |
| **Sound (声)** | 背景噪音 40-45 dBA，隔声 STC ≥45 | 噪声分区设计 |
| **Thermal (热)** | PMV -0.5~0.5，满意度 ≥80% | 分区温控系统 |
| **Air (空气)** | CO₂ &lt;1000ppm，PM2.5 &lt;15μg/m³ | 通风与净化系统 |

- 强调科技教育空间的普惠性
- 支持物理空间与数字空间融合
- 鼓励社会力量参与空间建设

## 第一部分：人因工程基础

### 1.1 人体尺寸与空间规划

**参考标准**: GB 10000-88《中国成年人人体尺寸》[E1]

**关键人体尺寸（第5-95百分位）**:

| 尺寸参数 | 男性 (mm) | 女性 (mm) | 设计启示 |
|---------|----------|----------|----------|
| 身高 | 1583-1775 | 1484-1662 | 最小净空高度 ≥2400mm |
| 眼高（站姿） | 1475-1660 | 1376-1550 | 展示面高度 1400-1600mm |
| 坐高 | 855-955 | 800-895 | 座椅+桌面高度组合 |
| 肩宽 | 420-480 | 380-435 | 最小通道宽度 ≥600mm |
| 臂展 | 1630-1850 | 1495-1705 | 工作台最大深度 ≤800mm |

**工作台高度设计**:

```
站立工作（精细操作）: 1000-1100mm
站立工作（普通操作）: 900-1000mm
站立工作（重型操作）: 750-900mm
坐姿工作: 700-750mm
可调节工作台: 680-1200mm（推荐）
```

**最佳实践**:
```
□ 工作台高度可调节，适应不同身高使用者
□ 座椅高度可调，配合工作台使用
□ 儿童专用工作区降低工作台高度至 600-700mm
□ 轮椅使用者工作台下方净空 ≥700mm
```

### 1.2 动线设计规范

**通道宽度标准**:

| 通道类型 | 最小宽度 | 推荐宽度 | 适用场景 |
|---------|---------|---------|---------|
| 主通道 | 1500mm | 1800mm | 人流主干道，需满足轮椅双向通行 |
| 次通道 | 1200mm | 1500mm | 设备区域间通道 |
| 工作区内通道 | 900mm | 1200mm | 工作台之间 |
| 紧急疏散通道 | 1400mm | 1800mm | 通向安全出口 |

**动线设计原则**:

```
工作流程动线:
  材料存储 → 加工区 → 组装区 → 测试区 → 成品存储
       ↓         ↓         ↓         ↓
     取料便捷   加工高效   协作方便   反馈快速

人流动线:
  入口 → 签到/更衣 → 工作区 → 休息区 → 出口
            ↓
       安全培训/导览
```

**避免的动线问题**:
- ❌ 死角和盲区
- ❌ 交叉干扰的人流和物流
- ❌ 通道被设备或物料阻挡
- ❌ 紧急出口被锁闭或遮挡

### 1.3 工作姿势与疲劳预防

**站立工作优化**:

| 因素 | 问题 | 解决方案 |
|-----|------|----------|
| 地面硬度 | 长时间站立导致腿部疲劳 | 抗疲劳地垫（厚度 15-20mm） |
| 工作高度 | 过高/过低导致肩颈疲劳 | 可调节工作台或脚踏板 |
| 工具位置 | 反复弯腰/伸展取工具 | 工具悬挂系统，常用工具置于肘部高度 |

**坐姿工作优化**:

```
理想坐姿几何关系:
  显示器距离: 500-700mm
  视线角度: 水平向下 15-30°
  键盘高度: 肘部弯曲 90-110°
  大腿与地面: 平行或略向下倾斜
  脚部: 平放地面或脚踏板
```

**轮换工作制建议**:
- 每 45-60 分钟变换工作姿势
- 站立/坐姿工作比例约 60:40
- 设置定时提醒系统

## 第二部分：光环境设计

### 2.1 照度标准

**参考标准**: GB 50034-2013《建筑照明设计标准》、IES Lighting Handbook [E2]

**不同区域照度要求**:

| 区域 | 照度 (lux) | 色温 (K) | 显色指数 (Ra) |
|-----|-----------|---------|--------------|
| 一般工作区 | 300-500 | 4000-5000 | ≥80 |
| 精细操作区 | 500-1000 | 5000-6000 | ≥90 |
| 电子焊接区 | 750-1500 | 5500-6500 | ≥90 |
| 木工/金工区 | 300-500 | 4000-5000 | ≥80 |
| 喷涂/上色区 | 1000-2000 | 6500 | ≥95 |
| 展示区 | 200-500 | 3000-4000 | ≥90 |
| 休息讨论区 | 150-300 | 2700-3500 | ≥80 |
| 通道 | 100-200 | 4000 | ≥70 |
| 仓储区 | 100-150 | 4000 | ≥60 |

### 2.2 自然采光设计

**采光系数要求**:

| 空间类型 | 采光系数 | 窗地面积比 |
|---------|---------|-----------|
| 工作区 | ≥2% | 1:5 - 1:7 |
| 休息区 | ≥1% | 1:7 - 1:10 |

**采光优化策略**:

```
顶部采光:
  ├─ 天窗: 均匀照明，需配遮阳
  ├─ 锯齿形屋顶: 北向采光，避免眩光
  └─ 光导管: 深部空间引入自然光

侧面采光:
  ├─ 高侧窗: 深入照明，减少遮挡
  ├─ 落地窗: 视野开阔，需控制眩光
  └─ 光线架: 将直射光反射到天花板
```

**眩光控制**:

| 眩光类型 | 来源 | 控制措施 |
|---------|------|----------|
| 直接眩光 | 太阳/灯具直射 | 遮阳百叶、灯具遮光罩 |
| 反射眩光 | 光滑表面反射 | 哑光材料、调整工作面角度 |
| 对比眩光 | 亮度差异过大 | 均匀环境照明、任务照明补充 |

### 2.3 人工照明设计

**照明层次**:

```
三层照明系统:

环境照明 (Ambient)
  │  功能: 提供基础照度
  │  灯具: 面板灯、线性灯
  │  控制: 分区调光
  │
  ↓
重点照明 (Accent)
  │  功能: 突出展示、指引方向
  │  灯具: 射灯、轨道灯
  │  控制: 独立调光
  │
  ↓
任务照明 (Task)
     功能: 精细操作补光
     灯具: 台灯、悬臂灯
     控制: 个人可调
```

**LED 照明选型**:

| 参数 | 推荐范围 | 说明 |
|-----|---------|------|
| 光效 | ≥100 lm/W | 能效等级 |
| 显色指数 | ≥80 (一般) / ≥90 (精细) | 颜色还原能力 |
| 色温 | 4000-5000K | 日间工作适宜 |
| 眩光指数 UGR | ≤19 (一般) / ≤16 (精细) | 视觉舒适度 |
| 频闪 | PstLM ≤1, SVM ≤0.4 | 避免视觉疲劳 |

**智能照明控制**:
- 日光联动调光（保持恒定照度）
- 人员感应（无人区域自动降低）
- 场景预设（工作/展示/清洁模式）
- 色温日变化（模拟自然光节律）

## 第三部分：声环境设计

### 3.1 噪声标准与分区

**参考标准**: GB 50118-2010《民用建筑隔声设计规范》[E2]

**噪声限值**:

| 区域类型 | 等效声级 LAeq (dB) | 峰值声级 (dB) |
|---------|-------------------|--------------|
| 安静工作区 | ≤45 | ≤55 |
| 一般工作区 | ≤55 | ≤65 |
| 机械加工区 | ≤75 | ≤85 |
| 噪声隔离区 | ≤35 | ≤45 |

**噪声分区策略**:

```
空间布局噪声分区:

  高噪声区 ←────── 缓冲区 ──────→ 低噪声区
  (机械加工)        (通道/存储)      (设计/讨论)

  └─ CNC、激光切割    └─ 材料库        └─ 会议室
     金属加工            工具间           安静工作区
     木工车床            更衣区           图书角
```

### 3.2 建筑声学设计

**隔声设计**:

| 构件 | 隔声量 Rw (dB) | 构造示例 |
|-----|--------------|---------|
| 高噪声区隔墙 | ≥50 | 双层石膏板+岩棉+空气层 |
| 一般隔墙 | ≥40 | 单层石膏板+岩棉 |
| 门 | ≥30 | 隔声门，带门槛密封 |
| 窗 | ≥35 | 双层中空玻璃，断桥铝框 |
| 楼板 | ≥50 | 浮筑地面，弹性垫层 |

**吸声设计**:

```
混响时间控制:
  工作区: 0.6-0.8s
  讨论区: 0.4-0.6s
  展示区: 0.8-1.0s

吸声材料布置:
  天花: 穿孔吸声板 (αw ≥0.7)
  墙面: 木丝水泥板、织物包覆板
  地面: 地毯、橡胶地板
```

**常用吸声材料**:

| 材料 | 吸声系数 (500Hz) | 适用位置 | 特点 |
|-----|-----------------|---------|------|
| 穿孔石膏板 | 0.6-0.8 | 天花 | 经济、防火 |
| 木丝水泥板 | 0.5-0.7 | 墙面 | 耐撞、工业感 |
| 聚酯纤维板 | 0.8-0.95 | 墙面 | 环保、可定制 |
| 岩棉玻璃布 | 0.9+ | 天花/墙面 | 高效、需保护 |
| 地毯 | 0.3-0.5 | 地面 | 舒适、难清洁 |

### 3.3 设备噪声控制

**噪声源控制**:

| 设备类型 | 典型噪声 (dB) | 控制措施 |
|---------|-------------|----------|
| CNC 加工中心 | 75-85 | 全封闭隔声罩，负压排气 |
| 激光切割机 | 70-80 | 半封闭隔声罩，排烟净化 |
| 木工机械 | 80-95 | 隔声房，吸尘系统 |
| 空压机 | 75-90 | 室外设置或隔声房 |
| 通风设备 | 50-70 | 管道消声器，减振支架 |

**隔声罩设计要点**:
- 结构: 钢板+阻尼层+吸声层
- 开口: 观察窗双层玻璃，进出料口迷宫设计
- 通风: 消声百叶，保证设备散热
- 密封: 软性密封条，避免缝隙传声

## 第四部分：热环境与通风

### 4.1 热舒适标准

**参考标准**: GB 50736-2012、ASHRAE Standard 55 [E2]

**温湿度控制目标**:

| 季节 | 温度 (°C) | 相对湿度 (%) | 风速 (m/s) |
|-----|----------|-------------|-----------|
| 夏季 | 24-26 | 40-60 | 0.2-0.3 |
| 冬季 | 20-22 | 30-50 | 0.1-0.2 |
| 过渡季 | 22-24 | 40-60 | 0.1-0.3 |

**特殊区域要求**:

| 区域 | 温度要求 | 湿度要求 | 说明 |
|-----|---------|---------|------|
| 电子操作区 | 22±2°C | 40-60% | 防静电 |
| 3D 打印区 | 20-25°C | ≤50% | 材料特性 |
| 喷涂区 | 18-25°C | 50-70% | 涂料性能 |
| 材料存储区 | 15-25°C | 30-50% | 材料保存 |

### 4.2 通风系统设计

**换气次数标准**:

| 空间类型 | 换气次数 (次/h) | 新风量 (m³/h·人) |
|---------|---------------|-----------------|
| 一般工作区 | 6-10 | 30 |
| 木工区 | 15-20 | - |
| 喷涂区 | 20-30 | - |
| 焊接区 | 10-15 | - |
| 化学实验区 | 15-25 | - |

**局部排风设计**:

```
污染源 → 捕集罩 → 风管 → 净化 → 排放
            │
            └─ 关键参数:
               · 控制风速: 0.25-1.0 m/s
               · 捕集效率: ≥90%
               · 罩口距离: 尽量靠近源头
```

**常见局部排风形式**:

| 形式 | 适用场景 | 捕集效率 | 能耗 |
|-----|---------|---------|------|
| 密闭罩 | 自动化设备 | 最高 | 最低 |
| 柜式罩 | 化学操作 | 高 | 中 |
| 槽边罩 | 表面处理 | 中高 | 中高 |
| 吹吸罩 | 焊接工位 | 中 | 高 |
| 伞形罩 | 热源上方 | 低 | 高 |

### 4.3 空气质量控制

**室内空气质量标准**:

| 污染物 | 限值 | 来源 | 控制措施 |
|-------|-----|------|----------|
| PM2.5 | ≤35 μg/m³ | 加工粉尘 | HEPA 过滤 |
| PM10 | ≤70 μg/m³ | 木工、金工 | 除尘系统 |
| TVOC | ≤0.6 mg/m³ | 涂料、胶水 | 通风排气 |
| 甲醛 | ≤0.08 mg/m³ | 板材、家具 | 源头控制 |
| CO₂ | ≤1000 ppm | 人员呼吸 | 新风补充 |
| CO | ≤10 mg/m³ | 燃烧设备 | 燃烧排气 |

**空气净化策略**:

```
多级净化系统:

新风 → 初效过滤 → 中效过滤 → 高效过滤 → 室内
(G4)         (F7)         (H13)

回风 → 活性炭 → 光催化 → 混合送风
      (TVOC)   (杀菌)
```

## 第五部分：材料选择指南

### 5.1 地面材料

| 材料 | 耐磨性 | 耐化学品 | 舒适性 | 清洁性 | 造价 | 适用区域 |
|-----|-------|---------|-------|-------|------|---------|
| 环氧自流平 | ★★★★★ | ★★★★★ | ★★ | ★★★★★ | ★★★ | 机械加工区 |
| PU 地坪 | ★★★★ | ★★★★ | ★★★★ | ★★★★ | ★★★★ | 电子操作区 |
| PVC 卷材 | ★★★ | ★★★ | ★★★★ | ★★★★ | ★★ | 一般工作区 |
| 橡胶地板 | ★★★★ | ★★★ | ★★★★★ | ★★★ | ★★★★ | 装配区 |
| 抛光混凝土 | ★★★★★ | ★★★ | ★★ | ★★★★ | ★★ | 重型工作区 |
| 架空地板 | ★★★ | ★★★ | ★★★ | ★★★ | ★★★★★ | 电子实验区 |

**选型要点**:
```
□ 承重区域: 耐压强度 ≥25 MPa
□ 化学区域: 耐酸碱腐蚀
□ 静电敏感区: 防静电 (10⁶-10⁹ Ω)
□ 站立作业区: 弹性地面或抗疲劳垫
□ 潮湿区域: 防滑系数 R10-R12
```

### 7.1 施工顺序


### 8.1 典型问题诊断

| 问题现象 | 可能原因 | 解决方案 |

## 第十一部分：实用计算公式

### 11.1 空间容量计算

```
最大容纳人数 = 总面积 (m²) ÷ 人均面积系数

人均面积系数:
  MIT 标准:     5.6 m²/人
  国内中小学:   3-4 m²/人
  社区创客空间: 4-5 m²/人

示例: 100m² 空间
  MIT 标准:    100 ÷ 5.6 ≈ 18 人
  国内标准:    100 ÷ 3.5 ≈ 28 人
```

### 11.2 照明计算

```
所需灯具数量 = (目标照度 × 面积) ÷ (单灯光通量 × 利用系数 × 维护系数)

典型参数:
  目标照度:    500 lux (一般工作区)
  利用系数:    0.5-0.7 (取决于房间反射)
  维护系数:    0.7-0.8 (考虑灰尘积累)

示例: 60m² 工作区，使用 3000lm LED 面板灯
  灯具数量 = (500 × 60) ÷ (3000 × 0.6 × 0.75)
           = 30000 ÷ 1350
           ≈ 22 盏
```

### 11.3 通风量计算

```
所需通风量 = max(按换气次数, 按人数)

按换气次数: Q₁ = 房间体积 × 换气次数
按人数:     Q₂ = 人数 × 人均新风量 (30 m³/h·人)

示例: 100m² × 3m 高，20 人，一般工作区 (8次/h)
  Q₁ = 100 × 3 × 8 = 2400 m³/h
  Q₂ = 20 × 30 = 600 m³/h
  所需通风量 = max(2400, 600) = 2400 m³/h
```

### 11.4 电力负荷计算

```
总电力需求 = Σ(设备功率 × 需用系数) + 照明负荷 + 空调负荷 + 预留

需用系数参考:
  生产设备:   0.3-0.5 (非同时使用)
  照明:       0.8-1.0
  空调:       0.8-1.0

示例: Lite 型空间
  设备: (激光切割 1500W + 3D打印 500W × 2) × 0.4 = 1000W
  照明: 60m² × 15W/m² = 900W
  空调: 60m² × 150W/m² = 9000W
  预留: 20%
  总计 ≈ (1000 + 900 + 9000) × 1.2 = 13 kW
```
