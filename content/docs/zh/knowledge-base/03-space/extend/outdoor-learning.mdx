# 户外学习空间

## 概述

创客教育不局限于室内。户外空间提供了独特的学习机会：与自然互动、更大的活动范围、不同的材料和挑战。本文档探讨如何利用户外空间丰富创客教育体验。

**核心价值**:
1. 自然是最好的材料库和灵感来源
2. 户外活动培养环境意识
3. 更大空间支持大型项目
4. 身心健康的综合发展

---

## 1. 户外创客活动类型

### 1.1 活动分类

| 类型 | 内容示例 | 空间需求 |
|------|---------|---------|
| **自然创客** | 树屋、鸟巢、生态系统 | 自然环境 |
| **大型制作** | 飞行器、大型装置 | 开阔空地 |
| **环境项目** | 气象站、太阳能、雨水收集 | 半固定区域 |
| **临时活动** | 户外工作坊、创客集市 | 临时场地 |
| **探险挑战** | 野外生存、户外技能 | 野外环境 |

### 1.2 季节性活动规划

| 季节 | 适合活动 | 注意事项 |
|------|---------|---------|
| **春** | 种植、观察、轻量活动 | 花粉过敏 |
| **夏** | 水上项目、大型活动 | 防晒、补水 |
| **秋** | 收获、落叶艺术、飞行 | 天气变化 |
| **冬** | 雪地项目、室内为主 | 保暖、短时间 |

## 2. 户外空间类型

### 2.1 固定户外区

**校园/社区花园型**:
- 种植区：探索植物生长
- 观察区：自然观察记录
- 工作区：户外工作台

**屋顶/平台型**:
- 天文观测
- 太阳能项目
- 城市农业

### 2.2 临时户外空间

**帐篷/遮阳棚**:
```
临时户外工作站:
┌─────────────────────────────────────┐
│      ┌───────────────────┐          │
│      │   遮阳帐篷        │          │
│      │  ┌─────────────┐  │          │
│      │  │  工作区     │  │          │
│      │  └─────────────┘  │          │
│      └───────────────────┘          │
│                                      │
│    [材料区]  [展示区]  [休息区]      │
│                                      │
└─────────────────────────────────────┘
```

**移动创客车**:
- 改装货车或拖车
- 自带工具和材料
- 可到达不同社区

## 3. 户外安全管理

### 3.1 风险因素

| 风险 | 预防措施 |
|------|---------|
| **天气** | 天气预报、备用方案、遮蔽设施 |
| **昆虫/动物** | 驱虫、急救包、安全距离 |
| **地形** | 场地勘察、标识危险区 |
| **紫外线** | 防晒、遮阳、控制时长 |
| **走失** | 明确边界、点名制度 |

### 3.2 户外活动检查清单

```
活动前检查:
□ 天气预报确认
□ 场地安全勘察
□ 急救包和应急计划
□ 家长知情同意
□ 防晒/防雨准备
□ 充足饮用水
□ 通讯设备
□ 备用室内方案
```

## 4. 自然材料利用

### 4.1 可用自然材料

| 材料 | 来源 | 创作用途 |
|------|------|---------|
| 树枝 | 树下收集 | 结构、编织 |
| 树叶 | 季节性收集 | 拓印、堆肥 |
| 石头 | 河边、山地 | 建造、装饰 |
| 泥土 | 地面 | 泥塑、土砖 |
| 种子 | 植物果实 | 种植、艺术 |

### 4.2 自然创客项目示例

**鸟屋制作**: 观察鸟类行为，学习木工
**昆虫旅馆**: 了解生态系统，创意设计
**太阳能烤箱**: 能源科学，实用技能
**雨水收集器**: 水循环知识，工程思维

## 5. 与自然教育结合

### 5.1 跨领域连接

| 自然教育 | 创客技能 | 结合项目 |
|---------|---------|---------|
| 植物学 | 传感器编程 | 自动浇水系统 |
| 气象学 | 数据采集 | 校园气象站 |

- 本地生态探索
- 垃圾变材料
- 可持续设计思维
- 环境问题解决项目
