### 5. MVS 最小可运行版本

**Must 必须包含**:
- [ ] 必备设备清单（基础款）
- [ ] 采购前评估清单
- [ ] 设备登记基础表格
- [ ] 核心设备使用须知

**Must Not 不得包含**:
- [ ] 未经安全评估的高危设备
- [ ] 侵犯隐私的监控设备配置

**最小配置**:
- 设备类别: 5大类（计算、制造、电子、手工、展示）
- 必备设备: 10-15件核心设备
- 管理表格: 资产登记 + 借用记录 + 维护记录

**最小流程**:
1. 需求评估 → 2. 采购审批 → 3. 验收登记 → 4. 培训上架 → 5. 使用维护 → 6. 盘点报废

**最小角色**:
- 资产管理员: 登记盘点
- 设备培训员: 使用培训
- 维护人员: 保养维修

### 7. 核心理念 (3-7条)

1. **适度配置**: 不追求最新最贵，追求适合场景
2. **安全优先**: 设备选型首先考虑安全性
3. **开放兼容**: 优先选择开放标准、可替代的设备
4. **全生命周期**: 从采购到报废全程管理
5. **可维护性**: 优先选择易维护、有本地支持的设备
6. **赋能学习**: 设备是服务学习的工具，不是展示品

**常见误区**:
- ❌ 追求"高配"而非"适配"
- ❌ 只关注采购，忽视维护和培训
- ❌ 设备闲置，未真正用于教学

### 9. 指标与证据 (3-8个)

| 指标 | 口径定义 | 数据源 | 目标值 |
|------|----------|--------|--------|
| 设备完好率 | 可正常使用设备占比 | 盘点记录 | ≥95% |
| 设备利用率 | 月均使用时长/可用时长 | 使用记录 | ≥40% |
| 故障响应时间 | 从报修到修复的平均时长 | 维修记录 | ≤48小时 |
| 资产账实相符率 | 盘点一致率 | 盘点记录 | ≥98% |
| 培训覆盖率 | 使用前接受培训的人员占比 | 培训记录 | 100% |
| 耗材缺货率 | 耗材缺货导致活动受影响次数 | 运营记录 | ≤2次/月 |
