# 微项目设计

## 概述

微项目是30分钟到2小时内可完成的小型创客项目。它们是创客教育的重要组成部分，适用于新手入门、技能训练、开放日体验等场景。

**设计原则**:
1. 时间短、成就感强
2. 材料易得、成本低
3. 技能入门友好
4. 可独立完成或快速协作

**微项目的教育价值**:

| 价值维度 | 具体体现 | 适用场景 |
|---------|---------|---------|
| **低门槛体验** | 无需前置技能即可参与 | 开放日、招新活动 |
| **快速成就感** | 当场完成并带走作品 | 新手入门、信心建立 |
| **技能启蒙** | 接触新技术领域的起点 | 兴趣探索、能力测试 |
| **活动暖场** | 营造动手氛围 | 长期项目前热身 |
| **教学演示** | 展示核心概念 | 原理讲解、知识点验证 |
| **灵活填充** | 适应碎片时间 | 等待间隙、活动补充 |

## 1. 微项目分类体系

### 1.1 按时长分类

| 时长 | 类型 | 适用场景 | 典型结构 | 成品复杂度 |
|------|------|---------|---------|-----------|
| **30分钟** | 快闪项目 | 开放日体验、课前热身 | 引入3分+演示5分+制作20分+展示2分 | 单一功能 |
| **45分钟** | 短课项目 | 课堂活动、社团热身 | 引入5分+演示8分+制作28分+展示4分 | 基础功能+小变化 |
| **1小时** | 标准微项目 | 工作坊、社团活动 | 引入5分+演示10分+制作38分+分享7分 | 功能完整 |
| **1.5小时** | 扩展微项目 | 半日体验、入门课 | 引入8分+演示12分+制作55分+分享15分 | 可定制功能 |
| **2小时** | 深度微项目 | 半日营、技能课程 | 引入10分+演示15分+制作80分+分享15分 | 多功能/高完成度 |

### 1.2 按技能领域分类

| 技能领域 | 入门项目(30分) | 标准项目(1小时) | 进阶项目(2小时) |
|---------|---------------|----------------|----------------|
| **纸艺/结构** | 纸飞机优化、纸塔挑战 | 纸桥承重、弹珠轨道 | 纸板机械手、建筑模型 |
| **电子基础** | LED贺卡、简单电路 | 蜂鸣器音乐盒、闪光装置 | 声控LED、光敏夜灯 |
| **编程入门** | Scratch动画、表情显示 | micro:bit计步器、电子骰子 | 简易游戏、交互装置 |
| **机械原理** | 橡皮筋弹射器、陀螺 | 橡皮筋动力车、齿轮装置 | 简易机械臂、连杆机构 |
| **艺术创意** | 涂鸦机器人、拓印画 | 光影装置、动态雕塑 | LED灯光秀、互动装置 |
| **科学探究** | 简易显微镜、潜望镜 | 气压火箭、太阳能小车 | 风力发电机、水质检测 |

### 1.3 按难度分级

```
微项目难度分级标准:

★☆☆☆☆ 入门级 (成功率>95%)
├── 步骤数: ≤5步
├── 工具: 无需或仅需剪刀
├── 技能: 无前置要求
├── 示例: LED贺卡、纸飞机、涂鸦机器人
└── 适合: 首次体验者、低龄儿童

★★☆☆☆ 基础级 (成功率>90%)
├── 步骤数: 6-10步
├── 工具: 基础手工具
├── 技能: 基本手工能力
├── 示例: 橡皮筋车、弹珠轨道、蜂鸣器
└── 适合: 有少量动手经验者

★★★☆☆ 中等级 (成功率>80%)
├── 步骤数: 11-15步
├── 工具: 含电动工具或编程
├── 技能: 需要简单指导
├── 示例: micro:bit项目、机械臂、太阳能车
└── 适合: 有一定基础的学习者

★★★★☆ 挑战级 (成功率>70%)
├── 步骤数: 16-20步
├── 工具: 多种工具组合
├── 技能: 需要前置学习
├── 示例: 复杂编程项目、精密机械装置
└── 适合: 有经验的创客

★★★★★ 专家级 (成功率>60%)
├── 步骤数: >20步
├── 工具: 专业设备
├── 技能: 需要深入指导
├── 示例: 综合性创客作品
└── 适合: 进阶学习者、竞赛准备
```

### 2.3 时间估算方法

**时间估算公式**:

| 环节 | 基础时间 | 人数修正 | 年龄修正 | 难度修正 |
|------|---------|---------|---------|---------|
| **引入** | 3分钟 | 无 | 低龄+2分 | 无 |
| **演示** | 步骤数×1.5分 | 无 | 低龄×1.3 | 难度高×1.2 |
| **制作** | 测试时间×1.5 | 每10人+5% | 低龄×1.5 | 难度高×1.3 |
| **分享** | 人数×0.5分 | 上限15分 | 无 | 无 |
| **缓冲** | 总时间×15% | - | - | - |

**实际测试验证流程**:

```
时间验证三步法:

第一步: 导师制作

└── 收集反馈意见

第三步: 小组测试
├── 6-8人小规模测试
├── 记录不同进度差异
├── 验证指导策略
└── 最终确定时间方案

经验系数:
• 儿童(6-8岁): 导师时间 × 2.5-3
```

### 3.1 30分钟项目详解

#### LED贺卡 ★☆☆☆☆

| 项目信息 | 详情 |
|---------|------|
| **时长** | 25-30分钟 |
| **适合年龄** | 6岁以上 |
| **人数** | 1-30人 |
| **单人成本** | 3-5元 |
| **成功率** | >95% |

**材料清单(每人)**:
- 彩色卡纸 A5 × 1张
- 5mm LED × 1个
- CR2032纽扣电池 × 1个
- 铜箔导电胶带 × 30cm
- 透明胶带 × 适量
- 彩笔/贴纸(装饰用)

**制作步骤**:
```
LED贺卡制作流程:

1. 设计阶段 (5分钟)
   ├── 对折卡纸
   ├── 确定LED位置(如星星、灯泡图案中心)
   └── 用铅笔轻轻标记

2. 电路铺设 (10分钟)
   ├── 在卡纸背面贴铜箔胶带形成两条轨道
   ├── 两条轨道不能接触(留5mm间隙)
   ├── 轨道一端通向LED位置
   └── 另一端预留电池位置

3. 安装LED (5分钟)
   ├── 将LED两脚分开
   ├── 长脚(+)贴到一条轨道
   ├── 短脚(-)贴到另一条轨道
   └── 用透明胶带固定

4. 测试点亮 (3分钟)
   ├── 将电池放在轨道末端
   ├── 正极(+)朝上
   └── 确认LED亮起

5. 装饰完成 (5分钟)
   ├── 设计电池固定方式(可做成开关)
   ├── 正面绘画装饰
   └── 写上祝福语

电路原理图:
    ┌──────────────────┐
    │      卡纸正面      │
    │    ┌────┐        │
    │    │ ★ │ ← LED   │
    │    │LED │        │
    │    └────┘        │
    └──────────────────┘

    ┌──────────────────┐
    │      卡纸背面      │
    │  ══════╗         │
    │        ║ ← 铜箔+  │
    │  ┌─────╨─────┐   │
    │  │   LED    │   │
    │  └─────╥─────┘   │
    │        ║ ← 铜箔-  │
    │  ══════╝         │
    │  [电池位置]       │
    └──────────────────┘
```

**常见问题与解决**:

| 问题 | 原因 | 解决方法 |
|------|------|---------|
| LED不亮 | 电池方向反 | 翻转电池 |
| LED不亮 | 铜箔断开 | 检查并补贴 |
| LED不亮 | LED脚接反 | 调换LED方向 |
| 时亮时灭 | 接触不良 | 加固连接点 |

**变化版本**:
- 多LED版：并联2-3个LED
- 开关版：设计折叠开关
- 闪烁版：使用闪烁LED
- 节日主题：圣诞树、生日蛋糕等

**制作步骤**:
```
涂鸦机器人制作:

   ├── 将振动电机粘在杯底中央
   ├── 电机轴上可加小偏心块增强振动

   ├── 用毛根做触角

4. 测试运行 (5分钟)
   ├── 打开笔盖
   ├── 放在大纸上
   ├── 接通电源
   └── 观察绘画轨迹

调整技巧:
• 笔的高度影响轨迹粗细
• 偏心块大小影响振动幅度
• 电机位置影响运动方向
```

### 3.2 1小时项目详解

#### 橡皮筋动力车 ★★☆☆☆

| 项目信息 | 详情 |
|---------|------|
| **时长** | 50-60分钟 |
| **适合年龄** | 8岁以上 |
| **人数** | 1-15人 |
| **单人成本** | 5-10元 |
| **成功率** | >85% |

**材料清单(每人)**:
- 木棒/筷子 × 4根
- 瓶盖/圆形纸板(轮子) × 4个
- 橡皮筋 × 3-5根
- 吸管 × 2根
- 竹签 × 2根
- 热熔胶枪(公用)

**设计原理**:
```
橡皮筋动力车工作原理:

能量转换:
弹性势能 ──► 动能 ──► 车辆前进

结构示意:
        橡皮筋(动力源)
           ╭━━━╮
    ┌──────┤  ├──────┐
    │      ╰━━╯      │
   ○│               │○  ← 前轮(从动)
    │    车身框架    │
   ●│               │●  ← 后轮(驱动)
    └───────────────┘
         ↑
      后轴缠绕橡皮筋

工作过程:
1. 向后转动驱动轮
2. 橡皮筋缠绕在车轴上
3. 释放后橡皮筋回弹
4. 带动车轴旋转
5. 车辆向前行驶
```

**制作步骤**:
```
制作流程 (60分钟):

1. 制作车身框架 (15分钟)
   ├── 两根木棒平行放置(间距8cm)
   ├── 两根横木棒垂直连接
   └── 用热熔胶固定成矩形框架

2. 安装车轴系统 (15分钟)
   ├── 将吸管截成4cm两段
   ├── 粘在框架前后作为轴承
   ├── 竹签穿过吸管作为车轴
   └── 竹签两端伸出2cm

3. 安装轮子 (10分钟)
   ├── 在瓶盖中心打孔
   ├── 将瓶盖套在竹签两端
   └── 用热熔胶固定

4. 安装动力系统 (10分钟)
   ├── 将橡皮筋一端固定在前轴
   ├── 另一端固定在后轴
   └── 橡皮筋应有适当张力

5. 测试调整 (10分钟)
   ├── 向后转动后轮缠绕橡皮筋
   ├── 释放测试行驶距离
   └── 调整橡皮筋张力和轮子摩擦

优化方向:
• 增加橡皮筋数量/长度
• 改善轮子与地面摩擦
• 减轻车身重量
• 优化传动比
```

**竞赛变体**:
- 距离赛：看谁跑得最远

```

绳索控制版:
        ┌───关节1
        │   ╱
   固定点──╱
        ╲ ╱ ←拉绳使关节弯曲
         ╲
          ╲──关节2
           ╲
            ╲──夹爪

每个关节:
┌─────┐   拉绳   ┌─────┐
│     │←─────────│     │
│ 段1 │══════════│ 段2 │
│     │─────────→│     │
└─────┘  橡皮筋  └─────┘
         (复位)

控制方式:
• 拉绳 → 关节弯曲
• 松绳 → 橡皮筋复位

```

## 常见问题

<details>
<summary>如何确保所有人都能完成？</summary>

**提高完成率的策略**:

1. **设计层面**
   - 选择成功率>90%的项目
   - 预先测试验证可行性
   - 减少精细操作步骤
   - 避免需要等待的环节

2. **准备层面**
   - 材料预处理(如预剪裁、预打孔)
   - 准备不同难度版本
   - 备好半成品供落后者使用
   - 关键材料多备15%

3. **执行层面**
   - 设置进度检查点及时发现困难
   - 助教重点关注后进学员
   - 允许简化完成而非放弃
   - 延长收尾时间给缓冲

4. **心态层面**
   - 完成比完美更重要
   - 肯定过程而非只看结果
   - 未完成也能带走继续

</details>

<details>
<summary>材料不够或有人弄坏了怎么办？</summary>

**材料应急处理**:

1. **预防措施**
   - 关键材料备用+15%
   - 提前告知材料使用规范
   - 危险/易损材料集中保管

2. **现场应对**
   - 小问题: 从备用材料补充
   - 材料短缺: 调整为2人合作版本
   - 关键材料耗尽: 演示完成剩余步骤

3. **损坏处理**
   - 可修复: 协助修复，视为学习机会
   - 不可修复: 提供替代材料
   - 责任归属: 不追究，正向引导

4. **记录改进**
   - 记录消耗异常情况
   - 分析原因(设计问题/操作问题)
   - 调整下次材料配置

</details>

<details>
<summary>有学员不愿意动手怎么办？</summary>

**引导策略**:

1. **了解原因**
   - 怕失败/怕脏/不感兴趣/社交焦虑
   - 观察而非直接询问

2. **针对性引导**
   - 怕失败: 强调"没有失败，只有实验"
   - 怕脏: 提供手套、围裙
   - 不感兴趣: 允许观察，展示有趣成果
   - 社交焦虑: 安排独立位置，减少关注

3. **渐进参与**
   - 先邀请帮忙递材料
   - 再尝试最简单的一步
   - 逐步增加参与度

4. **尊重选择**
   - 不强迫，保持邀请姿态
   - 允许只观察也是一种参与
   - 可能下次就愿意尝试了

</details>
