# 微项目设计

## 概述

微项目是30分钟到2小时内可完成的小型创客项目。它们是创客教育的重要组成部分，适用于新手入门、技能训练、开放日体验等场景。

**设计原则**:
1. 时间短、成就感强
2. 材料易得、成本低
3. 技能入门友好
4. 可独立完成或快速协作

---

## 1. 微项目分类

### 1.1 按时长分类

| 时长 | 类型 | 适用场景 |
|------|------|---------|
| **30分钟** | 快闪项目 | 开放日体验、课前热身 |
| **1小时** | 标准微项目 | 工作坊、社团活动 |
| **2小时** | 深度微项目 | 半日营、入门课程 |

### 1.2 按技能分类

| 技能领域 | 微项目示例 |
|---------|-----------|
| **纸艺** | 纸飞机、折纸机器人、纸桥挑战 |
| **电子** | LED贺卡、简单电路、蜂鸣器音乐 |
| **编程** | Scratch动画、micro:bit表情 |
| **机械** | 橡皮筋车、弹射器、简单齿轮 |
| **艺术** | 涂鸦机器人、拓印画、光影装置 |

---

## 2. 设计框架

### 2.1 微项目结构

```
微项目标准流程:
┌─────────────────────────────────────────┐
│  引入 (5-10%)                           │
│  ├── 激发兴趣                           │
│  └── 展示目标成品                       │
├─────────────────────────────────────────┤
│  演示 (10-15%)                          │
│  ├── 关键步骤讲解                       │
│  └── 常见问题预防                       │
├─────────────────────────────────────────┤
│  制作 (60-70%)                          │
│  ├── 动手制作                           │
│  ├── 导师巡回指导                       │
│  └── 鼓励变化创新                       │
├─────────────────────────────────────────┤
│  完成 (10-15%)                          │
│  ├── 测试/展示                          │
│  ├── 简短分享                           │
│  └── 带走作品                           │
└─────────────────────────────────────────┘
```

### 2.2 设计检查清单

```
微项目设计检查:
□ 时间可控（含意外缓冲）
□ 材料已预先准备
□ 成功率高（>90%能完成）
□ 有可变化的空间
□ 成品可带走或展示
□ 导师讲解已简化
□ 安全风险已评估
```

## 3. 案例库

### 3.1 30分钟项目

**LED贺卡**
- 材料：卡纸、LED、纽扣电池、铜箔胶带
- 技能：简单电路
- 变化：多LED、开关设计

**纸桥承重挑战**
- 材料：A4纸若干、胶带
- 技能：结构、工程思维
- 变化：限定材料、竞赛形式

**涂鸦机器人**
- 材料：纸杯、马克笔、小电机、电池
- 技能：简单机械、电路
- 变化：外形装饰、轨迹设计

### 3.2 1小时项目

**橡皮筋动力车**
- 材料：木棒、轮子、橡皮筋
- 技能：能量转换、机械原理
- 变化：距离竞赛、爬坡挑战

**micro:bit表情显示器**
- 材料：micro:bit、电池盒
- 技能：编程入门
- 变化：摇晃显示、按钮交互

**弹珠轨道**
- 材料：纸板、纸管、胶带
- 技能：物理、设计迭代
- 变化：机关设计、团队接力

### 3.3 2小时项目

**简易机械臂**
- 材料：纸板、棍子、绳子
- 技能：机械原理、设计思维
- 变化：液压版、抓取挑战

**太阳能小车**
- 材料：太阳能板、电机、轮子
- 技能：能源、电路、机械
- 变化：造型设计、竞速比赛

**声控LED灯**
- 材料：Arduino/micro:bit、LED、声音传感器
- 技能：编程、传感器
- 变化：颜色变化、节奏响应

## 4. 材料管理

### 4.1 预包装策略

| 策略 | 优点 | 适用 |
|------|------|------|
| **个人包** | 分发快、公平 | 标准化项目 |
| **小组包** | 协作、省材料 | 团队项目 |
| **公共材料区** | 灵活、可创新 | 开放项目 |

### 4.2 材料清单模板

```
项目名称: ________________
时长: ____分钟
人数: ____人

每人材料:
- [ ] 材料1 x 数量
- [ ] 材料2 x 数量
- ...

公共材料:
- [ ] 工具1 x 数量
- ...

备用材料:
- [ ] 额外10%材料
```

## 5. 年龄适配

### 5.1 分龄设计要点

| 年龄段 | 时长建议 | 设计要点 |
|-------|---------|---------|
| **6-8岁** | 30-45分钟 | 步骤简单、多图示、高成功率 |
| **9-11岁** | 45-90分钟 | 可适度挑战、引入竞赛元素 |
| **12-14岁** | 60-120分钟 | 开放设计空间、技术深度 |
| **15+** | 90-120分钟 | 复杂问题、团队协作 |

## 常见问题

<details>
<summary>如何确保所有人都能完成？</summary>

1. 预先测试，确保设计可行
2. 准备多层次版本（基础版/进阶版）
3. 导师提前分工关注不同进度
4. 设置"检查点"及时发现困难
5. 准备半成品供落后者使用
</details>
