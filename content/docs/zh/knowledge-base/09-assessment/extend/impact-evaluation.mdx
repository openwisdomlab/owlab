# 影响力评估方法

## 概述

影响力评估（Impact Evaluation）旨在回答一个核心问题：**我们的干预是否真正带来了积极的变化？** 它超越了简单的满意度调查，深入探究因果关系和长期效应。

**核心挑战**: 如何证明观察到的变化确实是由我们的项目引起的，而非其他因素所致。

---

## 1. 评估方法概览

### 1.1 方法分类

| 方法类型 | 特点 | 适用场景 | 资源需求 |
|----------|------|----------|----------|
| **实验方法** | 随机对照，因果推断强 | 大规模项目、政策评估 | 高 |
| **准实验方法** | 对照组，但非随机 | 无法随机的场景 | 中-高 |
| **非实验方法** | 无对照组 | 资源有限、探索阶段 | 低-中 |
| **混合方法** | 定量+定性结合 | 需要深度理解机制 | 中-高 |

### 1.2 证据强度层级

```
┌─────────────────────────────────────────┐
│  随机对照试验 (RCT)                      │ ← 证据最强
├─────────────────────────────────────────┤
│  准实验设计（匹配、断点回归等）            │
├─────────────────────────────────────────┤
│  前后测对比                              │
├─────────────────────────────────────────┤
│  横截面调查                              │
├─────────────────────────────────────────┤
│  案例研究/访谈                           │ ← 证据较弱但有深度
└─────────────────────────────────────────┘
```

## 2. 前后测对比

### 2.1 基本设计

```
时间线:
O₁ ────── X ────── O₂
前测      干预      后测
```

- **O₁**: 干预前测量（基线数据）
- **X**: 干预/项目
- **O₂**: 干预后测量

### 2.2 实施步骤

| 步骤 | 关键活动 | 注意事项 |
|------|----------|----------|
| **1. 确定测量内容** | 选择与学习目标对齐的指标 | 指标需可测量、敏感 |
| **2. 设计测量工具** | 开发问卷、测试、量规 | 确保信效度 |
| **3. 收集基线数据** | 在干预开始前收集 | 时间点选择合适 |
| **4. 实施干预** | 记录干预过程 | 监测实施质量 |
| **5. 收集后测数据** | 在干预结束后收集 | 与前测保持一致 |
| **6. 分析变化** | 计算变化量和效应量 | 考虑测量误差 |

### 2.3 常用效应量

| 效应量 | 公式 | 解释标准 |
|--------|------|----------|
| **Cohen's d** | (后测均值-前测均值)/标准差 | 0.2小/0.5中/0.8大 |
| **增益得分** | 后测-前测 | 绝对提升量 |
| **增益百分比** | (后测-前测)/前测×100% | 相对提升比例 |

### 2.4 局限性与应对

| 局限 | 问题 | 应对策略 |
|------|------|----------|
| **成熟效应** | 变化可能是自然成长 | 设置对照组 |
| **历史效应** | 其他事件可能影响 | 记录并控制 |
| **测试效应** | 前测影响后测表现 | 使用不同版本 |
| **回归效应** | 极端值倾向回归均值 | 选择可靠测量 |

## 3. 对照组设计

### 3.1 设计类型

| 设计 | 描述 | 优势 | 劣势 |
|------|------|------|------|
| **随机对照** | 随机分配实验/对照组 | 因果推断最强 | 伦理和可行性问题 |
| **匹配对照** | 匹配相似个体作对照 | 较可行 | 可能遗漏重要变量 |
| **等待名单对照** | 对照组延后接受干预 | 伦理可接受 | 时间差异问题 |
| **自然对照** | 利用自然存在的对比 | 最可行 | 因果推断较弱 |

### 3.2 匹配变量选择

对于创客教育项目，常用匹配变量：

| 变量类别 | 具体变量 |
|----------|----------|
| **人口学** | 年龄、性别、年级 |
| **背景** | 家庭社经地位、学校类型 |
| **前置能力** | 先前知识/技能水平 |
| **动机态度** | 学习兴趣、自我效能感 |

### 3.3 实施注意事项

- **知情同意**: 确保参与者了解研究目的
- **伦理考量**: 对照组不应被剥夺有益干预
- **流失控制**: 尽量减少参与者流失
- **实施保真**: 确保干预按设计实施

## 4. 追踪调查

### 4.1 追踪设计

```
时间线:
O₁ ── X ── O₂ ────── O₃ ────── O₄
前测  干预 后测    追踪1      追踪2
                  (3个月)    (12个月)
```

### 4.2 追踪时间点选择

| 追踪时点 | 目的 | 关注焦点 |
|----------|------|----------|
| **即时后测** | 测量直接效果 | 知识、技能 |
| **短期追踪（1-3月）** | 行为改变 | 应用、迁移 |
| **中期追踪（6-12月）** | 持续性 | 保持、发展 |
| **长期追踪（1-3年）** | 持久影响 | 职业、生活 |

### 4.3 追踪调查方法

| 方法 | 适用情况 | 回收率策略 |
|------|----------|------------|
| **在线问卷** | 广泛分布的参与者 | 激励机制、多次提醒 |
| **电话访谈** | 重点对象 | 预约时间、简短问题 |
| **社交媒体追踪** | 年轻群体 | 保持联系、非侵入性 |
| **学校/机构合作** | 有稳定归属的参与者 | 机构协助 |
| **校友网络** | 长期追踪 | 建立社区 |

### 4.4 流失与偏差处理

| 问题 | 风险 | 应对 |
|------|------|------|
| **选择性流失** | 特定群体更易流失 | 分析流失者特征 |
| **回忆偏差** | 时间越久记忆越不准 | 使用客观指标 |
| **社会期望偏差** | 给出"正确"答案 | 匿名、中性措辞 |

## 5. 定性评估方法

### 5.1 深度访谈

| 要素 | 说明 |
|------|------|
| **目的** | 深入理解变化过程和机制 |
| **样本** | 6-15人，有意抽样 |
| **时长** | 45-90分钟 |
| **记录** | 录音+笔记 |
| **分析** | 主题分析 |

**访谈提纲示例**:

1. 请描述你参与创客项目的经历
2. 你觉得这段经历对你有什么影响？
3. 有哪些具体的变化你可以分享？
4. 什么因素促成了这些变化？
5. 如果没有参与这个项目，你觉得会有什么不同？

### 5.2 焦点小组

| 要素 | 说明 |
|------|------|
| **目的** | 探索群体观点和互动 |
| **规模** | 每组6-10人 |
| **时长** | 60-120分钟 |
| **组数** | 2-4组 |

### 5.3 最显著变化法 (MSC)

Most Significant Change 方法步骤：

1. **收集故事**: 请参与者讲述最显著的变化故事
2. **选择故事**: 层层筛选最具代表性的故事
3. **讨论原因**: 讨论为什么选择这些故事
4. **验证反馈**: 将结果反馈给利益相关方

## 6. 混合方法设计

### 6.1 设计类型

| 设计 | 顺序 | 整合时机 | 适用场景 |
|------|------|----------|----------|
| **收敛设计** | 同时 | 分析阶段 | 验证和补充 |
| **解释性序列** | 定量→定性 | 定性收集 | 解释定量结果 |
| **探索性序列** | 定性→定量 | 定量收集 | 探索后验证 |
| **嵌入设计** | 一种嵌入另一种 | 持续 | 复杂评估 |

### 6.2 整合策略

```
定量发现 ──────┬────── 整合分析 ────── 综合结论
               │
定性发现 ──────┘

整合问题:
• 定性发现如何解释定量结果？
• 是否有矛盾？如何理解？
• 整合后有什么新洞察？
```

## 7. Kirkpatrick 四级评估模型

### 7.1 模型概览

| 级别 | 关注点 | 问题 | 方法 |
|------|--------|------|------|
| **L1 反应** | 满意度 | 喜欢吗？ | 问卷、反馈 |
| **L2 学习** | 知识技能 | 学到了吗？ | 测试、评估 |
| **L3 行为** | 应用迁移 | 用起来了吗？ | 观察、追踪 |
| **L4 结果** | 组织影响 | 有什么成效？ | 指标追踪 |

### 7.2 各级别实施

**Level 1 反应评估**:
- 课程满意度调查
- NPS（净推荐值）
- 即时反馈收集

**Level 2 学习评估**:
- 技能实操考核
- 知识测试
- 作品评价

**Level 3 行为评估**:
- 课后行为追踪
- 导师/家长观察
- 自我报告

**Level 4 结果评估**:
- 长期成效追踪
- 社会影响指标
- ROI 分析

## 8. 社会投资回报率 (SROI)

> ⚠️ **使用警示**
>
> SROI 将教育成果货币化存在根本性争议。教育价值难以简单换算为人民币，过度依赖货币化指标可能导致"唯金钱论"，忽视无法量化的重要成效（如人格成长、创造力培养等）。
>
> **建议**：SROI 仅作为**估算参考值**之一，必须与非货币化评估方法（如 MSC 最显著变化法）配合使用，提供全面的影响力画像。

### 8.1 SROI 概念

SROI (Social Return on Investment) 将社会影响货币化，计算每单位投入产生的社会价值。这是一种**估算工具**，而非精确测量。

```
SROI 比率 = 社会价值总额 / 投入成本

⚠️ 注意：货币化估值具有高度主观性，不同估算方法可能产生差异显著的结果
```

### 8.2 计算步骤

| 步骤 | 活动 | 产出 |
|------|------|------|
| **1. 确定范围** | 界定分析边界 | 范围说明 |
| **2. 识别利益相关方** | 谁受到影响 | 利益相关方地图 |
| **3. 绘制成效地图** | 投入→活动→成效 | 成效链条 |
| **4. 确认成效并赋值** | 每个成效值多少 | 财务代理指标 |
| **5. 确认影响** | 扣除死重、归因等 | 净影响 |
| **6. 计算SROI** | 价值/投入 | SROI比率 |
| **7. 报告和使用** | 沟通结果 | SROI报告 |

### 8.3 创客教育 SROI 示例

> ⚠️ **估算参考值声明**：以下数值仅为**示例性估算**，实际价值因地区、项目特点而异。建议结合定性方法（如 MSC 故事收集）一同呈现，避免将教育成效简单等同于经济价值。

| 成效 | 财务代理 | 价值估算 | 不确定性 |
|------|----------|----------|----------|
| **技能提升** | 培训课程市场价格 | ¥XX/人 | 中 |
| **升学加分** | 预科班/辅导费用 | ¥XX/人 | 高 |
| **就业竞争力** | 起薪增量×就业年限 | ¥XX/人 | 很高 |
| **志愿服务** | 志愿者时间价值 | ¥XX/小时 | 中 |

**配套使用 MSC (最显著变化法)**:

除 SROI 外，强烈建议收集"最显著变化故事"作为补充：
1. 请学习者/家长描述参与项目后最显著的变化
2. 这些故事无法货币化，但能呈现教育的深层价值
3. SROI + MSC 组合使用，提供更完整的影响力证据

## 9. 评估报告撰写

### 9.1 报告结构

1. **执行摘要**: 关键发现和建议
2. **项目背景**: 项目介绍和评估目的
3. **方法论**: 评估设计和数据收集
4. **发现**: 按评估问题组织
5. **结论**: 回应评估问题
6. **建议**: 可操作的改进建议
7. **附录**: 工具、详细数据

### 9.2 可信度要素

| 要素 | 说明 |
|------|------|
| **透明性** | 明确说明方法和局限 |
| **三角验证** | 多种方法相互印证 |
| **证据链** | 从数据到结论的逻辑清晰 |
| **利益相关方验证** | 结论经过检验 |
