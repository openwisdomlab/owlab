# M09 评价与影响力

> **一句话定位**: 看见成长，证明价值

---

## 核心原则

| 原则 | 内涵 | 体现 |
|------|------|------|
| **评价即学习** | 评价过程是学习一部分 | 形成性评价 |
| **多元证据** | 不依赖单一指标 | 档案袋 |
| **成长导向** | 关注过程而非只看结果 | 过程记录 |
| **真实评价** | 真实情境评价真实能力 | 项目作品 |
| **为改进服务** | 评价驱动改进 | 行动研究 |

---

## 评价层次

```
┌─────────────────────────────────────────┐
│  学习评价 (Learning)                     │
│  学习者的成长与能力发展                   │
├─────────────────────────────────────────┤
│  运营评价 (Operations)                   │
│  节点运营的效能与质量                     │
├─────────────────────────────────────────┤
│  影响力评价 (Impact)                     │
│  对个人/社区/社会的更广泛影响             │
└─────────────────────────────────────────┘
```

---

## 学习评价方法

| 方法 | 适用场景 | 证据类型 |
|------|----------|----------|
| **档案袋** | 能力发展追踪 | 作品集 |
| **量规评价** | 项目/作品评估 | 评分表 |
| **观察法** | 过程行为 | 观察记录 |
| **自评互评** | 反思能力 | 反思表 |
| **展示答辩** | 综合能力 | 现场表现 |

---

## 运营核心指标

| 维度 | 指标 | 目标 |
|------|------|------|
| **参与** | 月活跃人次 | 趋势↑ |
| **留存** | 续费率 | ≥70% |
| **满意** | NPS | ≥50 |
| **质量** | 课程完成率 | ≥85% |
| **效率** | 空间利用率 | ≥60% |

---

## MVS 最小可运行标准

| 项目 | 要求 |
|------|------|
| 活动反馈 | ✅ 每次活动收集 |
| 核心指标 | ✅ 月度统计 |
| 定期复盘 | ✅ 至少季度 |
| 改进行动 | ✅ 复盘后有行动 |

---

## 影响力评估框架

```
变化理论 (Theory of Change)

投入 → 活动 → 产出 → 成效 → 影响
Input  Activity Output Outcome Impact
```

## 常见误区

- ❌ 用分数简化复杂学习 → ✅ 多元证据
- ❌ 只关注易量化指标 → ✅ 定性定量结合
- ❌ 评价只用于问责 → ✅ 评价驱动改进
- ❌ 评价与目标脱节 → ✅ 评价对齐目标
