### 5. MVS 最小可运行版本

**Must 必须包含**:
- [ ] 最小面积要求（按节点类型）
- [ ] 核心功能区定义（至少4个区）
- [ ] 基础品牌视觉元素
- [ ] 安全通道与消防要求
- [ ] 无障碍基础要求

**Must Not 不得包含**:
- [ ] 违反消防规范的设计
- [ ] 封闭不透明的空间（需可视化监督）

**最小配置**:
- 功能区: 4个核心区（创作区、展示区、讨论区、储物区）
- 面积: 因节点类型而异（最小 50m²）
- 品牌元素: Logo + 主色调 + 基础物料

**最小流程**:
1. 选址评估 → 2. 平面规划 → 3. 装修施工 → 4. 验收检查 → 5. 开业运营

**最小角色**:
- 空间负责人: 统筹规划与验收
- 安全检查员: 安全合规验收

### 7. 核心理念 (3-7条)

1. **学习可见**: 空间应让学习过程可见，激发好奇与模仿
2. **灵活可变**: 空间应支持快速重组，适应不同活动需求
3. **安全包容**: 安全是底线，同时要包容不同能力与背景
4. **低门槛高上限**: 入门简单，但能支持深度探索
5. **数字与物理融合**: 线上线下体验一体化设计
6. **社区归属感**: 空间应让使用者产生归属感

**常见误区**:
- ❌ 追求"高大上"而忽视实用性
- ❌ 空间设计完成后就不再调整（应持续迭代）
- ❌ 只考虑成人视角，忽视儿童体验

### 9. 指标与证据 (3-8个)

| 指标 | 口径定义 | 数据源 | 目标值 |
|------|----------|--------|--------|
| 空间利用率 | 高峰时段使用面积/总面积 | 观察记录 | ≥70% |

| 用户满意度 | 空间满意度评分 | 用户调查 | ≥4.2/5.0 |
| 无障碍达标 | 无障碍设施到位率 | 检查清单 | ≥80% |
| 品牌一致性 | 品牌元素规范使用率 | 视觉审核 | ≥90% |

### 10. 合规边界 (3-6条)

| 边界 | 说明 |
|------|------|
| **消防安全** | 必须通过消防验收 |
| **建筑规范** | 符合当地建筑使用规范 |
| **无障碍** | 满足无障碍设计基本要求 |
| **环保健康** | 装修材料符合环保标准 |
| **视觉监督** | 空间需有可视化监督条件 |
| **应急通道** | 紧急疏散通道畅通 |
