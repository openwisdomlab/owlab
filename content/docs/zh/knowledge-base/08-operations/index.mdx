# M08 运营手册

> **一句话定位**: 让运营像呼吸一样自然

---

## 核心原则

| 原则 | 内涵 | 体现 |
|------|------|------|
| **流程可复制** | 新人可快速上手 | SOP标准化 |
| **用户为中心** | 围绕用户体验设计 | 服务设计 |
| **数据驱动** | 记录数据支持决策 | 运营仪表盘 |
| **灵活与标准** | 核心标准，细节灵活 | 分层规范 |
| **持续优化** | 从运营中学习改进 | PDSA循环 |

---

## 日运营周期

```
┌─────────────────────────────────────────┐
│  开门准备 (开业前30分钟)                  │
│  安全检查 / 设备开启 / 环境准备           │
├─────────────────────────────────────────┤
│  日常运营                                │
│  接待 / 活动执行 / 会员服务               │
├─────────────────────────────────────────┤
│  日终收尾 (关门前30分钟)                  │
│  清洁 / 安全检查 / 记录归档               │
└─────────────────────────────────────────┘
```

## MVS 最小可运行标准

| 项目 | 要求 |
|------|------|
| 开关门SOP | ✅ 有标准流程 |
| 安全检查 | ✅ 每日执行 |
| 运营记录 | ✅ 每日填写 |

## 关键指标

| 指标 | 定义 | 目标 |
|------|------|------|
| 活动准时率 | 按时开始的活动占比 | ≥95% |
| 会员续费率 | 到期续费比例 | ≥70% |
| 投诉解决率 | 承诺时间内解决 | ≥90% |
| 日志完成率 | 每日填写比例 | 100% |
| 用户满意度 | 服务评分 | ≥4.2/5 |
