# 每日开门 SOP

**SOP编号**: SOP-M08-001
**版本**: 0.1.0
**执行时间**: 开业前30分钟
**执行人**: 当日值班人员

---

## 流程概览

```
到达 → 安全检查 → 设备开启 → 环境准备 → 开门迎客
(0分钟)  (5分钟)    (10分钟)    (25分钟)    (30分钟)
```

---

## 1. 到达与初步检查 (0-5分钟)

### 1.1 进入节点

| 步骤 | 操作 | 检查点 |
|------|------|--------|
| 1 | 到达节点门口 | 观察周围环境是否异常 |
| 2 | 检查门锁/门窗 | 无撬动痕迹 |
| 3 | 开门进入 | 无异味、无异响 |
| 4 | 关闭报警系统（如有） | 按规定操作 |

### 1.2 快速巡视

- [ ] 空间无明显异常
- [ ] 无漏水/积水
- [ ] 无不明人员/物品
- [ ] 昨日设备已断电（除指定设备）

**发现异常处理**:
> 如发现异常（入侵迹象、设备损坏、安全隐患），
> 立即联系节点负责人，不要触碰现场。

---

## 2. 安全检查 (5-10分钟)

### 2.1 消防安全

| 检查项 | 标准 | 结果 |
|--------|------|------|
| 灭火器在位 | 指定位置 | □ 正常 □ 异常 |
| 灭火器压力 | 指针在绿区 | □ 正常 □ 异常 |
| 消防通道 | 畅通无阻 | □ 正常 □ 异常 |
| 应急出口 | 可正常开启 | □ 正常 □ 异常 |
| 应急指示灯 | 正常亮起 | □ 正常 □ 异常 |

### 2.2 电气安全

| 检查项 | 标准 | 结果 |
|--------|------|------|
| 总电闸 | 正常位置 | □ 正常 □ 异常 |
| 插座外观 | 无烧焦/松动 | □ 正常 □ 异常 |
| 电线状态 | 无裸露/损坏 | □ 正常 □ 异常 |

### 2.3 通道安全

| 检查项 | 标准 | 结果 |
|--------|------|------|
| 主通道 | 无障碍物 | □ 正常 □ 异常 |
| 地面 | 干燥防滑 | □ 正常 □ 异常 |
| 照明 | 正常工作 | □ 正常 □ 异常 |

---

## 3. 设备开启 (10-20分钟)

### 3.1 基础设施

按以下顺序开启：

```
1. 总电源 → 2. 照明 → 3. 空调/通风 → 4. 网络设备
```

| 设备 | 操作 | 检查点 |
|------|------|--------|
| 总电源 | 开启 | 指示灯正常 |
| 区域照明 | 逐区开启 | 无闪烁/异响 |
| 空调/暖气 | 设定温度 | 正常运行 |
| 新风系统 | 开启 | 正常出风 |
| 路由器/网络 | 确认状态 | 指示灯正常 |

### 3.2 功能设备

根据当日需要开启：

| 区域 | 设备 | 操作 |
|------|------|------|
| 接待区 | 签到设备 | 开启并测试 |
| 工作区 | 公用电脑 | 开机待用 |
| 活动区 | 投影/音响 | 检查待用 |
| 制作区 | 按当日需要 | 预热/检查 |

### 3.3 高风险设备

> **重要**: 高风险设备（激光切割、3D打印等）
> 仅在有资质人员在场且有使用需求时开启

- [ ] 检查设备外观完好
- [ ] 检查安全防护装置
- [ ] 暂不开启，待需要时开启

---

## 4. 环境准备 (20-28分钟)

### 4.1 清洁检查

| 区域 | 检查项 | 结果 |
|------|--------|------|
| 接待区 | 整洁、有序 | □ OK □ 需清理 |
| 工作区 | 桌面清洁 | □ OK □ 需清理 |
| 卫生间 | 干净、有纸 | □ OK □ 需补充 |
| 公共区 | 垃圾已清 | □ OK □ 需清理 |

### 4.2 物品归位

- [ ] 工具归位（参照工具图示）
- [ ] 材料归位
- [ ] 桌椅归位
- [ ] 展示物品整理

### 4.3 信息准备

| 项目 | 操作 |
|------|------|
| 今日活动 | 查看日程，准备相关材料 |
| 预约情况 | 查看预约系统 |
| 公告更新 | 检查是否有新公告需张贴 |
| 特殊提醒 | 查看是否有交接事项 |

---

## 5. 开门迎客 (28-30分钟)

### 5.1 最终检查

- [ ] 入口区域整洁
- [ ] 欢迎信息/营业牌就位
- [ ] 背景音乐（可选）
- [ ] 接待人员就位

### 5.2 开门操作

1. 解锁大门
2. 打开门/调整至迎客状态
3. 开启门铃/通知系统（如有）

### 5.3 记录

在运营日志中记录：
- 开门时间: __________
- 值班人员: __________
- 安全检查: □ 正常 □ 有异常（备注）
- 设备状态: □ 正常 □ 有异常（备注）
- 今日重点: __________

---

## 异常处理

### 情况1: 发现入侵痕迹

```
发现异常 → 不进入/保护现场 → 报警 → 通知负责人 → 等待处理
```

### 情况2: 设备无法启动

```
记录问题 → 尝试重启一次 → 仍无法启动 → 记录并报修 → 使用备用方案
```

### 情况3: 消防设施异常

```
记录问题 → 评估风险等级 → R1以上立即报修 → 必要时暂停开业
```

### 情况4: 水电问题

| 问题 | 处理 |
|------|------|
| 停电 | 联系物业/电力公司，启用备用方案 |
| 停水 | 联系物业，评估是否影响运营 |
| 漏水 | 关闭水阀，清理积水，联系维修 |
