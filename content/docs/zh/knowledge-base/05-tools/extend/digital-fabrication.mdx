# 数字制造工具链

## 概述

数字制造（Digital Fabrication）是将数字设计转化为物理产品的过程，是 Fab Lab 和创客空间的核心能力。本文档系统梳理从创意概念到实体产品的完整工具链。

**核心洞察**:
1. **设计与制造一体化** — 数字文件直接驱动制造设备
2. **快速迭代** — 设计-制造-测试循环以小时而非月计
3. **个性化生产** — 每件产品都可以是独一无二的
4. **技能民主化** — 降低专业制造的门槛

### 1.2 关键文件格式

| 阶段 | 格式 | 用途 | 典型软件 |
|-----|------|------|----------|
| **3D建模** | .STEP, .IGES | 工程数据交换 | SolidWorks, Fusion 360 |
| | .STL, .OBJ | 3D打印/可视化 | 通用格式 |
| | .3MF | 增强型3D打印 | 新一代标准 |
| **2D设计** | .DXF, .DWG | CAD矢量图 | AutoCAD, LibreCAD |
| | .SVG | 网页/激光切割 | Inkscape, Illustrator |
| | .AI, .PDF | 设计稿/印刷 | Adobe系列 |

| | .NC | CNC加工 | CAM软件输出 |
| | .RML | Roland设备 | Roland专用 |

## 第六部分：工作流整合

### 6.1 典型项目工作流示例

#### 示例1：3D打印手机支架

```
1. 概念草图 (纸笔)
   ↓
2. 3D建模 (Fusion 360 / Tinkercad)
   ↓
3. 导出STL文件
   ↓
4. 切片设置 (Cura / PrusaSlicer)
   - 层高: 0.2mm
   - 填充: 15%
   - 支撑: 无
   ↓
5. 发送到打印机
   ↓
6. 打印完成 (约2小时)
   ↓
7. 后处理 (去毛刺)
   ↓
8. 测试与迭代
```

#### 示例2：激光切割收纳盒

```
1. 尺寸规划 (草图)
   ↓
2. 2D设计 (Inkscape / Fusion 360)
   - 使用盒子生成器
   - 设置榫卯接口
   ↓
3. 导出DXF/SVG文件
   ↓
4. 激光软件设置 (LightBurn)
   - 功率: 50%
   - 速度: 10mm/s
   ↓
5. 切割 (约10分钟)
   ↓
6. 组装 (胶粘/榫卯)
   ↓
7. 表面处理 (上漆/保持原色)
```

#### 示例3：Arduino 物联网设备

```
1. 需求定义 (传感器+执行器)
   ↓
2. 电路设计 (Fritzing)
   ↓
3. 外壳设计 (Fusion 360)
   ↓
4. PCB设计 (KiCad) - 可选
   ↓
5. 代码开发 (Arduino IDE / PlatformIO)
   ↓
6. 3D打印外壳
   ↓
7. 组装与测试
   ↓
8. 部署与迭代
```
