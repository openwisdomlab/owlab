# 工具借用管理

## 概述

允许会员借用工具可以延伸创客活动到家庭和其他场所，但需要良好的管理制度确保工具的可用性和安全性。

---

## 1. 借用规则

### 1.1 借用资格

| 资格要求 | 说明 |
|---------|------|
| 正式会员 | 有效会员资格 |
| 技能认证 | 借用工具需有对应认证 |
| 良好记录 | 无逾期未还记录 |
| 押金/担保 | 高价值工具需要 |

### 1.2 可借用物品分类

| 类别 | 示例 | 借用期限 |
|------|------|---------|
| **A类 (自由借)** | 基础手工具 | 7-14天 |
| **B类 (登记借)** | 电动工具 | 3-7天 |
| **C类 (申请借)** | 贵重设备 | 1-3天，需审批 |
| **D类 (不可借)** | 固定设备、危险品 | 仅限现场 |

---

## 2. 借用流程

### 2.1 标准流程

```
借用流程:
┌─────────────────────────────────────────┐
│  1. 查询可用性                          │
│     └── 在线/现场查询工具是否可借       │
├─────────────────────────────────────────┤
│  2. 提交申请                            │
│     └── 登记借用工具、用途、归还日期    │
├─────────────────────────────────────────┤
│  3. 检查确认                            │
│     └── 双方确认工具状态                │
├─────────────────────────────────────────┤
│  4. 领取工具                            │
│     └── 签字确认领取                    │
├─────────────────────────────────────────┤
│  5. 使用                                │
│     └── 按规范使用、妥善保管            │
├─────────────────────────────────────────┤
│  6. 归还                                │
│     └── 清洁后归还、确认完好            │
└─────────────────────────────────────────┘
```

### 2.2 借用记录

| 记录项 | 内容 |
|-------|------|
| 借用人 | 姓名、会员号 |
| 工具信息 | 名称、编号、状态 |
| 时间 | 借出时间、应还时间 |
| 用途 | 简述借用目的 |
| 归还 | 归还时间、状态 |

---

## 3. 逾期与损坏

### 3.1 逾期处理

| 逾期时长 | 处理措施 |
|---------|---------|
| 1-3天 | 提醒，限制继续借用 |
| 4-7天 | 暂停借用权30天 |
| 8-14天 | 暂停借用权90天，联系催还 |
| 15天+ | 暂停会员资格，按遗失处理 |

### 3.2 损坏赔偿

| 损坏程度 | 处理方式 |
|---------|---------|
| 正常损耗 | 不需赔偿 |
| 轻微损坏 | 承担维修费用 |
| 严重损坏 | 赔偿重置成本 |
| 遗失 | 赔偿重置成本 |

---

## 4. 管理工具

### 4.1 简易管理方案

**纸质登记本**:
适用于小型节点，工具数量少

**电子表格**:
Google Sheet/腾讯文档，可多人协作

### 4.2 数字化管理

- 资产管理软件
- 条码/二维码扫描
- 自助借还终端
- 到期自动提醒

---

## 5. 工具维护责任

### 5.1 借用者责任

- 按规范使用
- 妥善保管
- 及时报告问题
- 清洁后归还

### 5.2 空间责任

- 提供状态良好的工具
- 定期检查和维护
- 建立维护记录
- 及时维修或更换
