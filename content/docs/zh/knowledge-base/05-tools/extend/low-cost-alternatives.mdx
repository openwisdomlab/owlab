# 低成本替代方案

## 概述

创客教育不应受限于设备预算。许多高价设备都有低成本替代方案，或可通过DIY方式实现类似功能。本文档提供实用的替代思路和方案。

**核心理念**:
1. 约束激发创造力
2. DIY工具本身是学习
3. 从基础开始，逐步升级
4. 功能优先于品牌

---

## 1. 数字制造设备替代

### 1.1 3D打印替代

| 需求 | 高端方案 | 低成本替代 |
|------|---------|-----------|
| 快速原型 | 专业3D打印机 | 入门级打印机、使用公共服务 |
| 模型制作 | SLA打印机 | 纸模型、泡沫切割 |
| 精密部件 | 工业级打印 | 手工加工、购买成品件 |

### 1.2 激光切割替代

| 需求 | 替代方案 | 适用 |
|------|---------|------|
| 直线切割 | 美工刀+直尺 | 纸板、薄木板 |
| 复杂曲线 | 线锯、曲线锯 | 木材、亚克力 |
| 精密切割 | CNC雕刻机（入门级） | 多种材料 |
| 大批量 | 外包激光切割服务 | 最终产品 |

### 1.3 CNC替代

| 替代方案 | 成本 | 能力 |
|---------|------|------|
| 手工雕刻 | 低 | 基础造型 |
| 电动雕刻机 | 中低 | 简单2D加工 |
| 桌面CNC | 中 | 小型精密加工 |
| 外包加工 | 按需 | 复杂大型件 |

## 4. DIY工具制作

### 4.1 可自制的工具

| 工具 | DIY方法 | 难度 |
|------|---------|------|
| 工作台 | 木板+木腿 | 简单 |
| 绕线器 | 3D打印/木制 | 简单 |
| 焊接固定架 | 木板+夹子 | 简单 |
| 简易钻床台 | 木架+手电钻 | 中等 |
| 吸锡器 | 注射器改造 | 简单 |
| 零件收纳 | 食品容器利用 | 简单 |

### 4.2 DIY案例：简易真空成型机

```
材料:
- 木框架
- 烤箱/热风枪
- 吸尘器
- 钻孔板

原理:
加热塑料片 → 覆盖模具 → 抽真空成型

成本: ¥100-200
商用设备: ¥2000+
```

## 5. 服务替代购买

## 6. 采购策略

### 6.1 低成本采购渠道

| 渠道 | 优势 | 注意事项 |
|------|------|---------|
| **淘宝/1688** | 价格低、选择多 | 品质参差 |
| **二手平台** | 价格极低 | 需要检验 |
| **工厂直销** | 批量优惠 | 起订量要求 |
| **教育优惠** | 正品折扣 | 需资质 |
| **团购/集采** | 批量折扣 | 需要组织 |

### 6.2 采购优先级

```
采购决策流程:
1. 真的需要吗？→ 考虑借用/共享
2. 需要多高端？→ 够用就好
3. 有替代方案吗？→ 优先低成本替代
4. 可以DIY吗？→ DIY本身是学习
5. 最优采购渠道？→ 比价选择
```
