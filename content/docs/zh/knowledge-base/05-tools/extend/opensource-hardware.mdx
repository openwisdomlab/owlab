# 开源硬件平台指南

## 概述

开源硬件（Open Source Hardware, OSHW）是创客运动和 STEAM 教育的核心支柱。它打破了电子产品的"黑箱"，让学习者能够理解、修改和创造自己的电子设备。

**核心洞察**:
1. **透明性** — 开源硬件提供完整的设计文档，支持学习和改进
2. **社区驱动** — 庞大的开源社区提供丰富的资源和支持
3. **低门槛** — 精心设计的平台使初学者能够快速上手
4. **高天花板** — 同样的平台可以支持从入门到专业的项目
5. **跨学科** — 连接编程、电子、机械、艺术等多个领域

## 第二部分：micro:bit

### 2.1 平台简介

micro:bit 是英国 BBC 主导开发的教育微控制器，专为计算机科学入门设计。

**核心特性**:
- 25个LED点阵显示
- 2个可编程按钮
- 加速度计和磁力计
- 温度和光线传感器
- 蓝牙和无线通信
- 边缘引脚连接器

### 2.2 版本对比

| 特性 | micro:bit V1 | micro:bit V2 |
|-----|-------------|-------------|
| 处理器 | nRF51822 16MHz | nRF52833 64MHz |
| 内存 | 16KB RAM | 128KB RAM |
| 存储 | 256KB Flash | 512KB Flash |
| 扬声器 | 无 | 有 |
| 麦克风 | 无 | 有 |
| 触摸感应 | 无 | Logo触摸 |
| 电源管理 | 基础 | 休眠模式 |
| 价格 | ¥80-120 | ¥120-180 |

### 2.3 编程环境

| 环境 | 类型 | 难度 | 适用场景 |
|-----|------|------|----------|

| **Python** | 文本 | 中 | 进阶学习 |
| **MicroPython** | 文本 | 中 | 嵌入式Python |

## 第三部分：Arduino

### 3.1 平台简介

Arduino 是最成功的开源电子原型平台，由意大利 Ivrea 交互设计学院于2005年推出，旨在让设计师和艺术家能够轻松创建交互式电子项目。

**核心优势**:
- 完全开源的硬件和软件
- 极其丰富的社区资源
- 简洁的编程语言
- 庞大的扩展板(Shield)生态
- 跨平台开发环境

### 3.2 主流型号对比

| 型号 | 处理器 | 时钟 | I/O | 特点 | 价格 |
|-----|--------|------|-----|------|------|
| **Uno R3** | ATmega328P | 16MHz | 14数字/6模拟 | 入门标准 | ¥50-150 |
| **Uno R4** | RA4M1 48MHz | 48MHz | 14数字/6模拟 | Uno升级版 | ¥150-250 |
| **Nano** | ATmega328P | 16MHz | 14数字/8模拟 | 迷你尺寸 | ¥30-100 |
| **Mega 2560** | ATmega2560 | 16MHz | 54数字/16模拟 | 多接口 | ¥100-250 |
| **Leonardo** | ATmega32U4 | 16MHz | 20数字/12模拟 | USB设备 | ¥80-180 |
| **Due** | AT91SAM3X8E | 84MHz | 54数字/12模拟 | 32位ARM | ¥150-300 |
| **MKR系列** | SAMD21 | 48MHz | 8数字/7模拟 | IoT专用 | ¥200-400 |

### 3.3 教育选型建议

| 场景 | 推荐型号 | 理由 |
|-----|---------|------|
| 入门学习 | Uno R3/R4 | 资源最丰富，标准尺寸 |

| 复杂项目 | Mega 2560 | 更多I/O和内存 |

### 3.6 学习路径

```

├── 传感器应用
├── 显示屏使用 (LCD/OLED)
├── 舵机/电机控制
├── I2C/SPI通信

├── 无线通信 (蓝牙/WiFi)
├── 数据记录与上传
├── 自定义库开发
└── PCB设计与制作
```

| 蓝牙 | 无 | BT 4.2/BLE | BT 5.0/BLE |

| ADC | 1通道 | 18通道 | 20通道 |
| USB | 无 | 无 | 原生USB |
| AI加速 | 无 | 无 | 矢量指令 |
| 价格 | ¥10-30 | ¥20-50 | ¥30-80 |

## 第五部分：Raspberry Pi

### 5.1 平台简介

Raspberry Pi 是英国树莓派基金会开发的单板计算机，运行完整的Linux系统，是学习计算机科学的理想平台。

### 5.2 型号选择

| 型号 | CPU | RAM | 特点 | 价格 |
|-----|-----|-----|------|------|
| **Pi Zero 2 W** | 四核 1GHz | 512MB | 超小型，WiFi | $15 |
| **Pi 4 Model B** | 四核 1.5GHz | 2/4/8GB | 主力型号 | $35-75 |
| **Pi 5** | 四核 2.4GHz | 4/8GB | 最新旗舰 | $60-80 |
| **Pi Pico** | RP2040双核 | 264KB | 微控制器 | $4 |
| **Pi Pico W** | RP2040双核 | 264KB | Pico+WiFi | $6 |

### 5.3 Pi vs Arduino

| 特性 | Raspberry Pi | Arduino |
|-----|-------------|---------|
| **定位** | 微型计算机 | 微控制器 |
| **操作系统** | Linux | 裸机/RTOS |
| **编程语言** | 多语言 | C/C++ |
| **GPIO实时性** | 软实时 | 硬实时 |
| **功耗** | 较高 (3-5W) | 极低 (&lt;0.5W) |
| **启动时间** | 30-60秒 | 毫秒级 |
| **价格** | 较高 | 较低 |
| **适用场景** | 复杂处理、多媒体 | 实时控制、低功耗 |

### 5.4 操作系统选择

| 系统 | 特点 | 适用场景 |
|-----|------|----------|
| **Raspberry Pi OS** | 官方系统，最广泛支持 | 通用 |
| **Ubuntu Server** | 服务器应用 | 服务部署 |
| **LibreELEC** | Kodi媒体中心 | 家庭影院 |
| **RetroPie** | 游戏模拟器 | 复古游戏 |
| **Home Assistant** | 智能家居 | 家庭自动化 |
| **OctoPi** | 3D打印管理 | 3D打印 |

## 常见误区

| 误区 | 正确理解 |
|-----|----------|
| ❌ 一定要用官方板 | ✅ 兼容板适合学习，官方板适合可靠性要求高的场景 |
| ❌ 追求最新最强 | ✅ 选择适合项目需求的平台 |
| ❌ 只学一个平台 | ✅ 不同平台适合不同场景，应掌握多种 |
| ❌ 忽视软件生态 | ✅ 社区和资源与硬件同样重要 |
| ❌ 跳过基础直接做复杂项目 | ✅ 循序渐进，打好基础 |
| ❌ 只看教程不动手 | ✅ 动手实践是唯一的学习方式 |
