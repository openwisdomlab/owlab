# M05 工具与资产

> **一句话定位**: 管好设备，让创造成为可能

---

## 核心原则

| 原则 | 内涵 | 体现 |
|------|------|------|
| **适度配置** | 适合场景，非最新最贵 | 分级推荐 |
| **安全优先** | 选型首先考虑安全 | 安全认证 |
| **开放兼容** | 优先开放标准 | 可替代性 |
| **全生命周期** | 从采购到报废 | 流程完整 |
| **赋能学习** | 服务学习的工具 | 实际使用率 |

---

## 设备分级体系

```
┌─────────────────────────────────────────┐
│  必备 (Essential)                        │
│  所有节点必须配置                         │
├─────────────────────────────────────────┤
│  推荐 (Recommended)                      │
│  标准节点建议配置                         │
├─────────────────────────────────────────┤
│  进阶 (Advanced)                         │
│  成熟节点按需配置                         │
└─────────────────────────────────────────┘
```

---

## 核心设备分类

| 类别 | 必备示例 | 推荐示例 | 进阶示例 |
|------|----------|----------|----------|
| **计算** | 笔记本电脑 | 台式机 | 工作站 |
| **制造** | 手工工具 | 3D打印机 | 激光切割机 |
| **电子** | 万用表、烙铁 | Arduino套件 | 示波器 |
| **手工** | 基础工具箱 | 木工台 | 缝纫机 |
| **展示** | 投影仪 | 大屏 | 交互屏 |

## MVS 最小可运行标准

| 项目 | 要求 |
|------|------|
| 必备设备 | ✅ 全部到位 |
| 安全认证 | ✅ 电器有认证 |
| 资产登记 | ✅ 账实相符 |
| 使用培训 | ✅ 先培训后使用 |
| 维护计划 | ✅ 定期保养 |

---

## 资产生命周期

```
需求 → 采购 → 验收 → 使用 → 维护 → 盘点 → 报废
        ↓      ↓      ↓      ↓      ↓
      审批   登记   培训   保养   核对   处置
```

---

## 关键指标

| 指标 | 定义 | 目标 |
|------|------|------|
| 完好率 | 可用设备/总设备 | ≥95% |
| 利用率 | 使用时长/可用时长 | ≥40% |
| 账实相符 | 盘点一致率 | ≥98% |
| 培训覆盖 | 使用前培训比例 | 100% |

## 常见误区

- ❌ 追求"高配"而非"适配" → ✅ 按需配置
- ❌ 只关注采购，忽视维护 → ✅ 全生命周期
- ❌ 设备闲置成展品 → ✅ 以使用率衡量
