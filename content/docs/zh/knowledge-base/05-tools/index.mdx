# M05 工具与资产

> **一句话定位**: 管好设备，让创造成为可能

---

## 核心原则

| 原则 | 内涵 | 体现 |
|------|------|------|
| **适度配置** | 适合场景，非最新最贵 | 分级推荐 |
| **安全优先** | 选型首先考虑安全 | 安全认证 |
| **开放兼容** | 优先开放标准 | 可替代性 |
| **全生命周期** | 从采购到报废 | 流程完整 |
| **赋能学习** | 服务学习的工具 | 实际使用率 |
| **AI 赋能** | 利用 AI 提升效率 | 智能化管理 |

## 采购与供应链

### 采购流程

```
1. 需求评估 → 2. 预算审批 → 3. 供应商比选 → 4. 采购执行
     → 5. 验收入库 → 6. 资产登记 → 7. 使用培训
```

### 供应商管理

| 类型 | 选择标准 | 管理方式 |
|------|----------|----------|
| **主力供应商** | 质量稳定、售后好 | 长期合作 |
| **备选供应商** | 价格竞争力 | 弹性采购 |
| **认证供应商** | 网络集采资质 | 统一平台 |

### 网络集采优势

- 统一采购获取更优价格
- 统一验证产品质量
- 共享供应商评价
- 简化采购流程

## 资产管理

### 资产生命周期

```
需求 → 采购 → 验收 → 使用 → 维护 → 盘点 → 报废
        ↓      ↓      ↓      ↓      ↓
      审批   登记   培训   保养   核对   处置
```

### 资产管理系统 (LIMS)

| 功能 | 说明 | 实现方式 |
|------|------|----------|
| **资产登记** | 设备录入、二维码标签 | 数据库 |
| **预约借用** | 设备预约、使用记录 | 小程序/Web |
| **状态监控** | 设备状态、使用时长 | IoT传感器 |
| **维护提醒** | 保养计划、到期提醒 | 自动推送 |
| **盘点核对** | 扫码盘点、差异报告 | 移动端 |

### 关键指标

| 指标 | 定义 | 目标 |
|------|------|------|
| 完好率 | 可用设备/总设备 | ≥95% |
| 利用率 | 使用时长/可用时长 | ≥40% |
| 账实相符 | 盘点一致率 | ≥98% |
| 培训覆盖 | 使用前培训比例 | 100% |

## 常见误区

- ❌ 追求"高配"而非"适配" → ✅ 按需配置，够用优先
- ❌ 只关注采购，忽视维护 → ✅ 全生命周期管理
- ❌ 设备闲置成展品 → ✅ 以使用率衡量价值
- ❌ 重硬件轻软件 → ✅ 软硬件协同考虑
- ❌ 一次性采购到位 → ✅ 分阶段、按需扩展
