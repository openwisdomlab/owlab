# M05 工具与资产

## 1. 摘要与核心定位
> **一句话定位**: 从“采购清单”到“创造力资产”的全生命周期管理。

本模块提供了 OWL 实验室的设备选型与管理标准。我们不追求最昂贵的工业级设备，而是追求**高可用性 (Availability)**、**低门槛 (Low Floor)** 和 **广阔的可能性 (High Ceiling)**。设备不是摆设，而是要把想法变为现实的通道。

**核心观点**:
1.  **分级配置**: 根据 Lite/Pro/Ultra 不同定位配置设备，拒绝“一步到位”的浪费。
2.  **维护即教育**: 设备的维护过程也是学习过程，鼓励用户参与维护。
3.  **软件定义硬件**: 重视开源工具链与 AI 辅助设计软件的价值，它们决定了硬件的上限。

---

## 2. 核心设备分级策略

OWL 参考 MIT Fab Lab 标准，定义了三个级别的配置方案，以适应不同规模的节点。

### 2.1 分级概览

| 级别 | 定位 | 核心能力 | 典型预算 (RMB) | 适用场景 |
|-----|-----|---------|--------------|---------|
| **Lite** | **普及型** | 3D打印 + 基础手工 | 5-10万 | 教室角、社区图书室 |
| **Pro** | **标准型** | + 激光切割 + 电子制作 | 20-50万 | 学校创客教室、社区中心 |
| **Ultra** | **专业型** | + CNC加工 + 精密测量 | 100万+ | 区域创新中心、高校实验室 |

### 2.2 核心能力矩阵 (Fab Inventory)

无论哪个级别，我们都要求具备以下四大核心制造能力：

#### A. 减材制造 (Subtractive)
*   **核心设备**: 激光切割机（必选）、数控铣床(CNC)、刻字机。
*   **选型原则**: 激光切割机是利用率最高的设备，应优先保证功率（≥80W）和幅面（≥600x400mm）。

#### B. 增材制造 (Additive)
*   **核心设备**: FDM 3D打印机（必选）、光固化打印机。
*   **选型原则**: 稳定性优于速度。推荐 Prusa/Bambu Lab 等社区支持良好的品牌，避免封闭生态的工业机。

#### C. 电子工作台 (Electronics)
*   **核心设备**: 焊台、万用表、示波器、可调电源。
*   **耗材库**: Arduino/ESP32 开发板、传感器套件、面包板、连接线。
*   **原则**: 耗材应像“自助餐”一样开放，鼓励随时取用。

#### D. 手工与组装 (Assembly)
*   **核心设备**: 热胶枪、电钻、手锯、钳工套件。
*   **原则**: 这一区域通常被低估，却是最繁忙的区域。必须配备结实的工作台和台钳。

> *详细选型指南：* 关于具体的品牌推荐、参数指标和采购避坑指南，请查阅 [**Fab Lab 设备目录**](/docs/knowledge-base/05-tools/extend/equipment-catalog)。

## 3. 资产全生命周期管理

设备买回来只是开始，管理才是挑战。

### 3.1 采购与入库
*   **非标设备**: 很多开源硬件（如树莓派）在政府采购流程中较难界定，建议设立“耗材专用金”。
*   **验收**: 必须进行“试运行”——不仅是开机，而是完整制作一个标准测试件（Test Piece）。

### 3.2 维护与保养
我们推行 **TPM (全员生产维护)** 理念：
*   **谁使用谁清洁**: 每次使用后必须清理废料。
*   **维护日**: 每周五下午定为“设备维护日”，带领学生一起给导轨上油、校准光路。
*   **维护红牌**: 设备故障挂红牌，修好换绿牌，培养可视化管理习惯。

> *操作手册：* 详细的点检表、润滑周期和故障排查流程，请参阅 [**维护指南**](/docs/knowledge-base/05-tools/extend/maintenance-guide)。

### 3.3 报废与迭代
*   **3年法则**: 消费级 3D 打印机通常寿命为 3 年，应做好折旧预算。
*   **零件再生**: 报废的设备应拆解，电机、光轴等零件作为“尸体库”供学生进行拆解研究或二次创造。

## 4. 软件与 AI 工具链

硬件只是显露的冰山一角，水面下的软件工具链决定了创造的效率。

### 4.1 开源优先
OWL 强烈推荐使用开源或免费的工具链，以保证学生离校后仍能继续创造。

| 领域 | 商业软件 (参考) | OWL 推荐 (免费/开源) | 理由 |
|-----|--------------|-------------------|------|
| **2D 设计** | Adobe Illustrator | **Inkscape** | 矢量图标准工具，插件丰富。 |
| **3D 建模** | SolidWorks | **Fusion 360** (Edu) / **Blender** | 涵盖参数化设计与艺术建模。 |
| **电子设计** | Altium Designer | **KiCad** | 越来越多的开源硬件转向 KiCad。 |
| **编程** | Keil | **VS Code** + **PlatformIO** | 现代化的开发体验。 |

### 4.2 AI 辅助设计
*   **代码生成**: 使用 GitHub Copilot 辅助编写 Arduino/Python 代码。
*   **创意生成**: 使用 Midjourney 生成产品外观草图。
*   **知识检索**: 建立本地 LLM 知识库，回答“为什么打印头堵了”等常见问题。
