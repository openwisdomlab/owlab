## 设计理念

### 核心概念：「夜视者」Night Vision

猫头鹰能在黑暗中洞察光明，Open Wisdom Lab 帮助学习者在未知领域激发创意、发现可能性。

**设计关键词**：沉稳、深邃、激发创意、未来主义、极简主义

我们的视觉系统融合三重特质：

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│    Logo紫色          Logo蓝色          蓝紫渐变            │
│    Magenta       +   Blue          +   Gradient            │
│                                                             │
│    洋红色            蓝色               融合渐变            │
│    #D91A7A           #2563EB           #D91A7A→#2563EB     │
│    智慧、洞察         未来、科技         创新、融合          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Logo 规范

### 主 Logo

Logo 由两部分组成：
- **图形标识**：抽象猫头鹰眼睛
- **文字标识**：OPEN WISDOM LAB

### 使用规范

**最小尺寸**
- 数字媒体：图形标识最小 32px，完整 Logo 最小 120px
- 印刷媒体：图形标识最小 10mm，完整 Logo 最小 35mm

**安全区域**
- Logo 周围保持至少 1/4 Logo 高度的空白区域

**背景使用**

| 背景类型 | 推荐版本 |
|---------|---------|
| 深色背景 | 彩色 Logo（默认）|
| 浅色背景 | 彩色 Logo |
| 复杂背景 | 单色白/黑版本 |

### 禁止用法

- 不要改变 Logo 颜色比例
- 不要拉伸或压缩 Logo
- 不要在 Logo 上添加效果
- 不要将 Logo 放置在低对比度背景上
- 不要旋转 Logo

## 图形元素

### 光晕效果 Glow

品牌特有的「夜视」光晕效果 —— 柔和、不刺眼：

```css
/* Magenta Glow - 温暖智慧 */
.owl-glow-magenta {
  box-shadow:
    0 0 16px rgba(217, 26, 122, 0.2),
    0 0 32px rgba(217, 26, 122, 0.1);
}

/* Blue Glow - 未来科技 */
.owl-glow-blue {
  box-shadow:
    0 0 16px rgba(37, 99, 235, 0.2),
    0 0 32px rgba(37, 99, 235, 0.1);
}
```

### 圆角规范

体现「未来主义」与「极简主义」：

- 小元素（按钮、标签）: `8px`
- 中元素（卡片、输入框）: `12px`
- 大元素（模块、面板）: `16px`
- 特大元素（浮层、对话框）: `24px`

## 动效规范

### 时间曲线

| 类型 | 曲线 | 时长 | 用途 |
|-----|------|-----|-----|
| 快速响应 | ease-out | 150ms | 按钮点击、状态切换 |
| 标准过渡 | ease-in-out | 300ms | 卡片悬停、展开收起 |
| 平滑进入 | ease-out | 400ms | 页面元素入场 |

### 动效原则

- **克制**：动效服务于功能，不喧宾夺主
- **柔和**：避免突兀的跳变，保持流畅
- **有意义**：每个动效都应传达状态变化
