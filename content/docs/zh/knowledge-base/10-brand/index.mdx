## 设计理念

### 核心概念：「夜视者」Night Vision

猫头鹰能在黑暗中洞察光明，Open Wisdom Lab 帮助学习者在未知领域激发创意、发现可能性。

**设计关键词**：沉稳、深邃、激发创意、未来主义、极简主义

我们的视觉系统融合三重特质：

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│    Logo紫色          Logo蓝色          蓝紫渐变            │
│    Magenta       +   Blue          +   Gradient            │
│                                                             │
│    洋红色            蓝色               融合渐变            │
│    #D91A7A           #2563EB           #D91A7A→#2563EB     │
│    智慧、洞察         未来、科技         创新、融合          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Logo 规范

### 主 Logo

Logo 由两部分组成：
- **图形标识**：抽象猫头鹰眼睛
- **文字标识**：OPEN WISDOM LAB

### 使用规范

**最小尺寸**
- 数字媒体：图形标识最小 32px，完整 Logo 最小 120px
- 印刷媒体：图形标识最小 10mm，完整 Logo 最小 35mm

**安全区域**
- Logo 周围保持至少 1/4 Logo 高度的空白区域

**背景使用**

| 背景类型 | 推荐版本 |
|---------|---------|
| 深色背景 | 彩色 Logo（默认）|
| 浅色背景 | 彩色 Logo |
| 复杂背景 | 单色白/黑版本 |

### 禁止用法

- 不要改变 Logo 颜色比例
- 不要拉伸或压缩 Logo
- 不要在 Logo 上添加效果
- 不要将 Logo 放置在低对比度背景上
- 不要旋转 Logo

## 三大视觉基因 Visual DNA

OWL 品牌视觉系统由三个核心视觉基因构成，代表理性与感性、秩序与创造的融合：

### 1. 几何晶格 Geometric Lattice

**特征**：蓝色线框、立方体阵列、等轴测网格

**象征意义**：
- 理性框架：知识体系的结构化组织
- 清晰边界：模块化设计、系统化思维
- 精确性：科学方法论、技术严谨性

**应用场景**：
- 知识库结构图
- MVS 框架展示
- 模块关系图
- 技术架构背景

**视觉特点**：
```
蓝色 (#2563EB) 主导
几何线框构图
立体网格透视
清晰的结构美学
```

### 2. 能量粒子流 Energy Particle Flow

**特征**：粉蓝粒子、布朗运动、有机涌现

**象征意义**：
- 好奇心能量：主动探索的原始驱动力
- 偶然连接：跨界思维、意外发现
- 涌现特性：集体智慧、复杂系统

**应用场景**：
- 首页 Hero 背景
- 创意工具界面
- 社区网络可视化
- 动态交互效果

**视觉特点**：
```
粉色 (#D91A7A) 和蓝色粒子
随机运动、自然连接
紫色 (#8B5CF6) 碰撞火花
有机、流动、不可预测
```

### 3. 混合符号 Hybrid Symbols

**特征**：粉色框架包含蓝色粒子、刚柔并济

**象征意义**：
- 结构化玩耍：在规则中自由创造
- 有限游戏：边界内的无限可能
- 理性感性融合：科学与艺术的交汇

**应用场景**：
- 猫头鹰五种状态图标
- 模块图标设计
- 品牌核心视觉
- 特色组件设计

**视觉特点**：
```
外层：粉色几何框架
内层：蓝色能量粒子
边缘：渐变融合效果
体现"有序的混沌"
```

### 视觉基因组合原则

在设计中灵活运用三大基因：

| 场景 | 主导基因 | 辅助基因 | 效果 |
|------|---------|---------|------|
| **理念性内容** | 几何晶格 | 能量粒子 | 理性中的创意 |
| **创意性工具** | 能量粒子 | 混合符号 | 自由但有边界 |
| **品牌核心** | 混合符号 | 全部 | 完整品牌个性 |
| **技术文档** | 几何晶格 | - | 清晰结构化 |
| **社区互动** | 能量粒子 | 混合符号 | 活力与连接 |

## 猫头鹰五种状态 Owl States

基于 OWL 品牌理念，设计五种象征性的猫头鹰状态，代表创新学习的完整旅程：

### 🦉 凝视 (Gaze) - 好奇的起点

**视觉特征**：
- 双色渐变眼睛（粉蓝同心圆）
- 六边形几何框架包围
- 探测波纹（同心圆扩散）
- 从眼睛发出的好奇光束

**色彩配置**：
- 主色：Blue (#2563EB) 占 70%
- 点缀：Pink (#D91A7A) 占 30%
- 背景：Dark (#0E0E14)

**象征意义**：观察、发问、探索的起点

**适用场景**：
- 知识库首页 banner
- M01 理念模块图标
- M06 安全模块图标
- "发现问题"阶段引导

### ✨ 灵光 (Spark) - 奇思妙想的瞬间

**视觉特征**：
- 爆炸性的几何碎片（三角形、菱形）
- 粉色和蓝色粒子随机碰撞
- 紫色 (#8B5CF6) 碰撞火花
- 不规则、动态的构图

**色彩配置**：
- Pink 和 Blue 粒子各占 50%
- Violet 碰撞效果
- 高饱和度、高对比度

**象征意义**：创意爆发、顿悟时刻、野性想法

**适用场景**：
- AI Lab 工具入口
- 创意工具图标
- 头脑风暴功能
- "灵感时刻"通知

### 🔗 连接 (Connect) - 偶然的发现

**视觉特征**：
- 网络节点结构
- 多个"眼睛"通过光束连接
- 渐变色连接线（Pink → Violet → Blue → Cyan）
- 节点处形成发光晶体

**色彩配置**：
- 全光谱渐变
- 连接线透明度渐变
- 节点发光效果

**象征意义**：跨界融合、协作、知识网络

**适用场景**：
- 社区网络页面
- M02 治理模块
- M08 运营模块
- 协作项目展示

### 🪶 飞翔 (Flight) - 勇敢探索

**视觉特征**：
- 展开的翅膀轮廓
- 羽毛飘散轨迹
- 向前飞行的动态姿态
- Cyan (#00D9FF) 路径轨迹

**色彩配置**：
- 主色：Pink（行动）
- 轨迹：Cyan（创新路径）
- 羽毛渐变：Pink → Violet → Blue

**象征意义**：实践、行动、勇气

**适用场景**：
- 项目启动页
- M03 空间模块
- M05 工具模块
- "探索"功能
- 实践阶段引导

### 🌟 分享 (Share) - 开放共享

**视觉特征**：
- 同心圆波纹扩散
- 眼睛变成光源
- 向外辐射的开放构图
- 全光谱色彩

**色彩配置**：
- Pink + Blue + Cyan + Violet 全光谱
- 多样性和包容性
- 波纹由实到虚

**象征意义**：开源精神、知识传播、社区贡献

**适用场景**：
- 作品展示页
- M07 人员模块
- M09 评价模块
- 开源分享功能
- 社区成果展示

## 动效规范

### 时间曲线

| 类型 | 曲线 | 时长 | 用途 |
|-----|------|-----|-----|
| 快速响应 | ease-out | 150ms | 按钮点击、状态切换 |
| 标准过渡 | ease-in-out | 300ms | 卡片悬停、展开收起 |
| 平滑进入 | ease-out | 400ms | 页面元素入场 |

### 动效原则

- **克制**：动效服务于功能，不喧宾夺主
- **柔和**：避免突兀的跳变，保持流畅
- **有意义**：每个动效都应传达状态变化
