# 社区专家资源

## 概述

每个社区都有丰富的专业人才资源。有效整合这些资源，可以极大丰富创客教育的内容和深度。

---

## 1. 专家类型

### 1.1 可邀请的专家

| 类型 | 来源 | 贡献方式 |
|------|------|---------|
| **行业专家** | 企业、研究机构 | 讲座、指导 |
| **手工艺人** | 社区、工坊 | 技艺传授 |
| **退休人员** | 社区 | 经验分享、指导 |
| **大学生** | 高校 | 辅导、陪伴 |
| **家长** | 会员家庭 | 职业分享 |

---

## 2. 发掘策略

### 2.1 寻找渠道

| 渠道 | 方法 |
|------|------|
| 会员网络 | 问卷调查家长技能 |
| 社区组织 | 对接社区服务中心 |
| 高校 | 联系志愿服务部门 |
| 企业 | 企业CSR/员工志愿 |
| 专业协会 | 行业协会、学会 |

### 2.2 邀请方法

- 说明参与的价值和意义
- 灵活安排时间和形式
- 提供必要的支持和资源
- 表达认可和感谢

---

## 3. 管理机制

### 3.1 专家档案

| 信息 | 内容 |
|------|------|
| 基本信息 | 姓名、联系方式 |
| 专业领域 | 技能、经验 |
| 可用时间 | 参与意愿、时间 |
| 参与记录 | 历史参与情况 |

### 3.2 支持专家

- 提供必要的培训
- 明确角色和期望
- 配备助手支持
- 定期反馈交流

---

## 4. 活动形式

| 形式 | 时长 | 适用 |
|------|------|------|
| 专题讲座 | 1-2小时 | 知识分享 |
| 工作坊 | 半天-全天 | 技能传授 |
| 项目指导 | 持续 | 深度参与 |
| 职业分享 | 1小时 | 视野拓展 |
| 在线问答 | 灵活 | 远程支持 |

---

## 常见问题

<details>
<summary>如何让忙碌的专家愿意参与？</summary>

1. 灵活安排时间（午餐、晚间）
2. 最小化准备工作
3. 提供有意义的体验
4. 表达真诚的感谢
5. 展示参与的影响力
