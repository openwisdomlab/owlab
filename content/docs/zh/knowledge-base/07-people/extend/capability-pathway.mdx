# 能力提升路径

## 概述

持续的能力提升是个人和节点发展的基础。本文档提供系统的能力发展框架。

---

## 1. 能力框架

### 1.1 核心能力域

| 能力域 | 内容 |
|-------|------|
| **技术能力** | 创客技能、工具使用 |
| **教学能力** | 课程设计、引导技巧 |
| **管理能力** | 项目管理、团队协作 |
| **社区能力** | 沟通、关系建立 |

### 1.2 能力等级

| 等级 | 描述 |
|------|------|
| L1 入门 | 基本了解，需要指导 |
| L2 熟练 | 独立操作，完成任务 |
| L3 精通 | 灵活应用，解决复杂问题 |
| L4 专家 | 创新改进，指导他人 |

---

## 2. 发展路径

### 2.1 导师发展路径

```
导师发展阶梯:
实习导师 → 初级导师 → 资深导师 → 首席导师
   ↓           ↓           ↓          ↓
  助教      独立带课    课程开发    培训导师
```

### 2.2 发展方式

| 方式 | 内容 |
|------|------|
| 培训课程 | 参加内外部培训 |
| 实践锻炼 | 在工作中学习 |
| 导师指导 | 资深导师带教 |
| 交流学习 | 跨节点交流 |
| 自主学习 | 在线课程、阅读 |

---

## 3. 发展支持

### 3.1 组织支持

| 支持 | 内容 |
|------|------|
| 培训资源 | 课程、资料、机会 |
| 时间保障 | 学习时间安排 |
| 经费支持 | 培训费用支持 |
| 晋升通道 | 清晰的发展通道 |

### 3.2 个人规划

| 步骤 | 内容 |
|------|------|
| 自我评估 | 识别当前能力水平 |
| 目标设定 | 确定发展方向 |
| 计划制定 | 具体行动计划 |
| 执行跟进 | 定期检视进展 |
