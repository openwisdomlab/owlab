# 培训课程体系

## 概述

OWL 培训体系基于"70-20-10"学习法则设计：70%能力发展来自工作实践，20%来自同伴互动，10%来自正式培训。正式培训的作用是"点火"——激发动机、建立框架、创造连接。

---

## 1. 培训体系架构

### 1.1 培训层级

```
                 ┌───────────────────┐
                 │    专项培训        │ ← 特定技能提升
                 │   (按需/选修)       │
                 ├───────────────────┤
                 │    进阶培训        │ ← E1→E2 发展
                 │   (每年/选修)       │
                 ├───────────────────┤
                 │    岗位培训        │ ← 岗位专业能力
                 │   (入职后/必修)     │
                 ├───────────────────┤
                 │    入职培训        │ ← 基础入门
                 │   (新入职/必修)     │
                 └───────────────────┘
```

### 1.2 培训类型概览

| 类型 | 目标人群 | 时长 | 形式 | 必修/选修 |
|------|----------|------|------|-----------|
| **入职培训** | 所有新人 | 8-16小时 | 线下集中 | 必修 |
| **岗位培训** | 各岗位人员 | 8-24小时 | 线下+实践 | 必修 |
| **进阶培训** | E1+人员 | 16-40小时 | 混合式 | 选修 |
| **专项培训** | 有需求者 | 4-16小时 | 灵活 | 选修 |
| **认证培训** | 认证申请者 | 依级别 | 集中+考核 | 认证必需 |

## 2. 入职培训 (新人必修)

### 2.1 培训目标

完成入职培训后，学员应能够：
- 理解并认同 OWL 的使命和价值观
- 遵守安全规程，通过安全考核
- 使用基础设备并了解操作规范
- 了解运营流程和自身职责

### 2.2 课程大纲

| 模块 | 内容 | 时长 | 方式 |
|------|------|------|------|
| **M1 欢迎与概览** | OWL 使命愿景、组织介绍 | 1h | 讲授+讨论 |
| **M2 价值观与文化** | 核心价值观、创客精神 | 1.5h | 案例+工作坊 |
| **M3 安全第一** | 安全规程、应急处理 | 3h | 讲授+演练 |
| **M4 设备入门** | 核心设备操作 | 3h | 示范+实操 |
| **M5 运营基础** | 日常流程、系统使用 | 1.5h | 讲授+演示 |
| **M6 角色职责** | 岗位职责、协作方式 | 1h | 讲授+讨论 |
| **M7 实践体验** | 参与一次完整活动 | 3h | 观摩+协助 |
| **M8 考核与反馈** | 安全考核、反馈交流 | 1h | 测试+讨论 |

**总时长**: 约16小时（2天）

### 2.3 各模块详情

#### M1 欢迎与概览 (1h)

| 内容 | 时长 | 要点 |
|------|------|------|
| 欢迎致辞 | 10min | 团队介绍、氛围营造 |
| OWL 故事 | 20min | 发展历程、愿景使命 |
| 网络介绍 | 15min | 网络结构、节点角色 |
| 培训说明 | 15min | 培训安排、预期产出 |

#### M3 安全第一 (3h)

| 内容 | 时长 | 要点 |
|------|------|------|
| 安全理念 | 20min | 安全第一文化 |
| 安全规程 | 40min | 通用规程、设备分级 |
| 设备安全 | 40min | 各设备安全要点 |
| 急救基础 | 30min | 常见伤害处理 |
| 应急处理 | 20min | 应急预案、疏散演练 |
| 安全考核 | 30min | 笔试+情景 |

### 2.4 考核标准

| 项目 | 方式 | 通过标准 |
|------|------|----------|
| 安全知识 | 笔试 | ≥80分 |
| 设备操作 | 实操 | 完成基础操作 |
| 参与态度 | 观察 | 积极参与 |

## 3. 岗位培训

### 3.1 导师培训 (E0→E1)

**目标**: 能独立执行标准课程

| 模块 | 内容 | 时长 | 方式 |
|------|------|------|------|
| **T1 学习理论** | 建构主义、做中学、4P理论 | 3h | 讲授+讨论 |
| **T2 教学设计** | 逆向设计、PBL基础 | 4h | 讲授+练习 |
| **T3 课堂管理** | 时间管理、纪律、安全 | 2h | 案例+演练 |
| **T4 引导技巧** | 提问、倾听、反馈 | 3h | 工作坊 |
| **T5 评价方法** | 过程评价、量规使用 | 2h | 讲授+练习 |
| **T6 课程实践** | 课程观摩、试讲 | 6h | 观摩+实践 |
| **T7 反思与认证** | 教学反思、认证评估 | 4h | 反思+考核 |

**总时长**: 约24小时

### 3.2 运营人员培训

| 模块 | 内容 | 时长 |
|------|------|------|
| **O1 会员服务** | 会员管理、客户沟通 | 3h |
| **O2 活动运营** | 活动策划、执行、复盘 | 4h |
| **O3 空间管理** | 空间维护、设备管理 | 3h |
| **O4 数据管理** | 数据采集、系统使用 | 2h |
| **O5 财务基础** | 收费、采购、账务 | 2h |

**总时长**: 约14小时

### 3.3 技术支持培训

| 模块 | 内容 | 时长 |
|------|------|------|
| **S1 设备维护** | 日常维护、故障排查 | 4h |
| **S2 技术支持** | 常见问题解决 | 3h |
| **S3 安全监督** | 设备安全、操作监督 | 2h |
| **S4 耗材管理** | 耗材采购、库存管理 | 2h |

**总时长**: 约11小时

## 4. 进阶培训 (E1→E2)

### 4.1 课程设计师培训

**目标**: 能设计高质量课程

| 模块 | 内容 | 时长 | 方式 |
|------|------|------|------|
| **D1 高阶PBL设计** | 驱动问题、深度探究 | 4h | 工作坊 |
| **D2 跨学科整合** | STEAM整合设计 | 3h | 案例+练习 |
| **D3 差异化设计** | 学习者差异应对 | 3h | 讲授+设计 |
| **D4 AI融入设计** | AI工具教学应用 | 3h | 实操+设计 |
| **D5 评价设计** | 量规设计、档案袋 | 3h | 工作坊 |
| **D6 课程开发实践** | 完成一门课程设计 | 8h | 项目制 |
| **D7 同行评审** | 课程评审与改进 | 2h | 互评 |

**总时长**: 约26小时

### 4.2 导师培训师培训

**目标**: 能培训和指导新导师

| 模块 | 内容 | 时长 |
|------|------|------|

| **M2 培训设计** | 培训课程设计 | 4h |
| **M3 培训技巧** | 讲授、引导、评价 | 4h |
| **M4 辅导技能** | 一对一辅导、反馈 | 3h |
| **M5 培训实践** | 实施培训模块 | 6h |
| **M6 认证评审** | 评审新导师认证 | 4h |

**总时长**: 约24小时

## 6. 培训实施

### 6.1 培训方式

| 方式 | 适用场景 | 优势 | 注意事项 |
|------|----------|------|----------|
| **集中面授** | 入职、安全培训 | 互动强、氛围好 | 时间协调 |
| **工作坊** | 技能训练 | 动手实践 | 小班制 |
| **在线学习** | 知识类内容 | 灵活、可回看 | 自律要求 |
| **师徒制** | 实践技能 | 个性化 | 导师时间 |
| **行动学习** | 问题解决 | 真实情境 | 需要引导 |
| **同伴学习** | 经验分享 | 低成本 | 质量控制 |

### 6.2 培训周期

| 培训类型 | 建议时机 | 频率 |
|----------|----------|------|
| 入职培训 | 入职首周 | 新人入职时 |
| 岗位培训 | 入职首月 | 新人入职时 |
| 进阶培训 | 年度 | 每年1-2次 |
| 专项培训 | 按需 | 不定期 |
| 复训 | 年度 | 每年1次 |

### 6.3 培训评估

| 层级 | 评估内容 | 方法 | 时机 |
|------|----------|------|------|
| **反应** | 满意度 | 问卷 | 培训后即时 |
| **学习** | 知识技能 | 测试、实操 | 培训结束 |
| **行为** | 行为改变 | 观察、360反馈 | 3个月后 |
| **结果** | 绩效影响 | 指标追踪 | 6个月后 |
