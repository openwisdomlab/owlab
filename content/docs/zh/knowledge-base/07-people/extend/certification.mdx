# 认证评估标准

## 概述

OWL 教育者认证体系建立了可信的能力评估和认证机制。认证不是终点，而是持续发展的里程碑。每个认证级别都有明确的能力要求、评估方式和有效期限。

---

## 1. 认证体系概览

### 1.1 认证级别

| 级别 | 名称 | 定位 | 有效期 | 续期要求 |
|------|------|------|--------|----------|
| **E0** | 见习导师 | 学习中，需督导 | - | 6个月内晋升E1 |
| **E1** | 认证导师 | 能独立执行 | 2年 | 持续授课+复训 |
| **E2** | 骨干导师 | 能设计和指导 | 2年 | 持续贡献+复训 |
| **E3** | 专家导师 | 能创新和引领 | 3年 | 持续贡献+评审 |

### 1.2 认证原则

| 原则 | 说明 |
|------|------|
| **能力导向** | 评估实际能力，非资历 |
| **证据为本** | 基于可观察的证据 |
| **多元评估** | 多种方式综合评估 |
| **持续有效** | 需定期复审和续期 |
| **发展支持** | 认证伴随发展支持 |

## 2. E0 见习导师

### 2.1 申请条件

| 条件 | 要求 | 验证方式 |
|------|------|----------|
| **基本资格** | 年满18周岁 | 身份证明 |
| **背景调查** | 通过背景调查（涉未成年人岗位） | 调查报告 |
| **入职培训** | 完成入职培训 | 培训证明 |
| **安全考核** | 通过安全考核 ≥80分 | 考核成绩 |

### 2.2 评估内容

| 维度 | 评估标准 | 权重 |
|------|----------|------|
| **理念认同** | 理解并认同OWL价值观 | 20% |
| **基础知识** | 完成培训内容 | 30% |
| **安全意识** | 通过安全考核 | 30% |
| **学习态度** | 积极参与、虚心学习 | 20% |

### 2.3 评估方式

| 方式 | 内容 | 通过标准 |
|------|------|----------|
| 笔试 | 安全知识、基础概念 | ≥80分 |
| 面谈 | 理念认同、学习态度 | 通过 |
| 培训参与 | 出勤和参与度 | ≥90%出勤 |

### 2.4 E0 期间要求

| 要求 | 说明 |
|------|------|
| **督导制度** | 每次授课需E1+导师在场 |
| **观摩记录** | 记录观摩心得 |
| **学习计划** | 有明确的E1晋升计划 |
| **时限** | 6个月内申请E1认证 |

## 3. E1 认证导师

### 3.1 申请条件

| 条件 | 要求 | 验证方式 |
|------|------|----------|
| **E0经历** | E0阶段≥3个月 | 记录 |
| **实践时数** | 协助授课≥30小时 | 授课记录 |
| **导师培训** | 完成岗位培训 | 培训证明 |
| **督导评价** | 督导评价合格 | 评价表 |

### 3.2 评估量规

| 维度 | 1分(不通过) | 2分(基本) | 3分(良好) | 4分(优秀) |
|------|-------------|-----------|-----------|-----------|
| **专业知识** | 知识缺失明显 | 掌握基础知识 | 知识扎实 | 知识广博 |
| **教学技能** | 无法独立授课 | 能执行标准课程 | 执行流畅 | 有个人风格 |
| **引导能力** | 缺乏引导意识 | 能基础引导 | 引导有效 | 引导出色 |
| **技术应用** | 操作不熟练 | 熟练使用 | 灵活应用 | 解决问题 |
| **反思能力** | 缺乏反思 | 能自我反思 | 反思有深度 | 主动改进 |

**通过标准**: 5个维度平均分≥2.5，无1分项

### 3.3 评估方式

| 方式 | 内容 | 权重 | 说明 |
|------|------|------|------|
| **试讲** | 15-20分钟微课 | 40% | 评委评分 |
| **笔试** | 理论知识测试 | 20% | ≥70分 |
| **材料评审** | 反思日志、观摩记录 | 20% | 材料完整性和质量 |
| **督导评价** | E1督导评价 | 20% | 综合评价 |

### 3.4 试讲评估表

| 评估项 | 1-4分 | 评价要点 |
|--------|-------|----------|
| 目标清晰 | | 学习目标明确可评价 |
| 内容准确 | | 知识准确、无错误 |
| 结构合理 | | 导入-展开-总结清晰 |
| 讲解清晰 | | 语言清楚、节奏适中 |
| 示范规范 | | 操作示范正确安全 |
| 互动有效 | | 与学习者有效互动 |
| 时间控制 | | 时间把控合理 |
| 整体印象 | | 综合表现 |

## 4. E2 骨干导师

### 4.1 申请条件

| 条件 | 要求 | 验证方式 |
|------|------|----------|
| **E1经历** | E1认证≥2年 | 认证记录 |
| **授课时数** | 独立授课≥200小时 | 授课记录 |
| **课程开发** | 参与开发≥1门课程 | 课程材料 |
| **培训经历** | 完成进阶培训 | 培训证明 |
| **同行认可** | 获得2位E2+导师推荐 | 推荐信 |

### 4.2 评估量规

| 维度 | 2分(基本) | 3分(良好) | 4分(优秀) |
|------|-----------|-----------|-----------|
| **专业知识** | 精通1个领域 | 精通2个领域 | 跨领域整合 |
| **教学技能** | 能设计课程 | 课程质量高 | 创新教学法 |
| **引导能力** | 应对复杂情境 | 引导深度探究 | 培训他人 |
| **技术应用** | 创新应用 | 解决复杂问题 | 技术整合 |
| **反思能力** | 系统反思 | 形成方法论 | 分享推广 |

**通过标准**: 5个维度平均分≥3，至少2个维度达4分

### 4.3 评估方式

| 方式 | 内容 | 权重 |
|------|------|------|
| **作品评审** | 课程设计作品集 | 30% |
| **公开课** | 30分钟课程展示 | 30% |
| **答辩** | 专业问答 | 20% |
| **360评价** | 同事、学员评价 | 20% |

### 4.4 课程作品评审标准

| 维度 | 权重 | 评价要点 |
|------|------|----------|
| **目标设计** | 20% | 目标清晰、对齐标准 |
| **内容设计** | 25% | 内容科学、适合受众 |
| **活动设计** | 25% | 活动有效、体验佳 |
| **评价设计** | 15% | 评价合理、可操作 |
| **创新性** | 15% | 有创新点、有特色 |

## 5. E3 专家导师

### 5.1 申请条件

| 条件 | 要求 | 验证方式 |
|------|------|----------|
| **E2经历** | E2认证≥3年 | 认证记录 |
| **授课时数** | 授课≥500小时 | 授课记录 |
| **课程开发** | 主导开发≥3门课程 | 课程材料 |
| **师资培训** | 培训导师≥10人 | 培训记录 |
| **网络贡献** | 显著贡献（发表、分享等） | 贡献记录 |
| **同行推荐** | 3位E3导师或总部推荐 | 推荐信 |

### 5.2 评估方式

| 方式 | 内容 | 说明 |
|------|------|------|
| **材料评审** | 贡献记录、作品集 | 评审委员会审核 |
| **答辩** | 专业深度答辩 | 评审委员会主持 |
| **公开展示** | 培训或讲座展示 | 现场评估 |
| **同行评议** | E3同行评价 | 书面评议 |

### 5.3 E3评审要点

| 维度 | 评价要点 |
|------|----------|
| **专业深度** | 在领域内有公认专业地位 |
| **创新贡献** | 发展了新方法、新课程 |
| **知识分享** | 积极分享、培训他人 |
| **网络贡献** | 对OWL网络有显著贡献 |
| **专业影响** | 在更广范围有影响力 |

## 6. 认证流程

### 6.1 申请流程

```
1. 自我评估 → 2. 准备材料 → 3. 提交申请 → 4. 资格审查
       ↓              ↓              ↓              ↓
   能力自评        收集证据        线上提交       形式审核

5. 评估安排 → 6. 评估实施 → 7. 结果通知 → 8. 证书发放
       ↓              ↓              ↓              ↓
   确定时间        多方式评估      反馈结果       发放证书
```

### 6.2 时间安排

| 认证级别 | 受理时间 | 评估周期 | 结果通知 |
|----------|----------|----------|----------|
| E0→E1 | 随时 | 2周内 | 1周内 |
| E1→E2 | 每季度 | 1个月内 | 2周内 |
| E2→E3 | 每半年 | 2个月内 | 1个月内 |

### 6.3 申请材料清单

**E1认证**:
- [ ] 认证申请表
- [ ] E0期间记录（观摩、协助）
- [ ] 反思日志（≥10篇）
- [ ] 督导评价表
- [ ] 培训完成证明

**E2认证**:
- [ ] 认证申请表
- [ ] 授课记录统计
- [ ] 课程设计作品（≥1门）
- [ ] 进阶培训证明
- [ ] 推荐信（2封）
- [ ] 教学反思集

**E3认证**:
- [ ] 认证申请表
- [ ] 贡献记录汇总
- [ ] 课程作品集（≥3门）
- [ ] 师资培训记录
- [ ] 推荐信（3封）
- [ ] 专业成果材料

## 7. 续期与复审

### 7.1 续期要求

| 级别 | 有效期 | 续期要求 |
|------|--------|----------|
| **E1** | 2年 | 持续授课≥60小时/年 + 年度复训 |
| **E2** | 2年 | 持续贡献 + 年度复训 + 课程更新 |
| **E3** | 3年 | 持续贡献 + 同行评议 |

### 7.2 续期流程

```
1. 续期提醒 (提前3个月)
2. 提交续期材料
3. 材料审核
4. 复训（如需）
5. 续期确认
6. 更新证书
```

### 7.3 年度复训

| 级别 | 复训时长 | 内容 |
|------|----------|------|
| E1 | 4小时 | 安全更新、新技术、案例分享 |
| E2 | 8小时 | 上述 + 教学法更新、同行研讨 |
| E3 | 按需 | 参与年度交流、贡献分享 |

## 8. 认证管理

### 8.1 证书管理

| 项目 | 说明 |
|------|------|
| **证书形式** | 电子证书 + 纸质证书（选） |
| **证书内容** | 姓名、级别、有效期、编号 |
| **验证方式** | 在线查询系统 |
| **变更处理** | 升级自动更新 |

### 8.2 认证撤销

| 情形 | 处理 |
|------|------|
| 续期未通过 | 降级或失效 |
| 严重违规 | 撤销认证 |
| 主动放弃 | 注销 |

### 8.3 申诉机制

| 情形 | 申诉渠道 | 处理时限 |
|------|----------|----------|
| 评估异议 | 提交书面申诉 | 2周内复核 |
| 撤销异议 | 申请复议 | 1个月内处理 |
