# 远程导师支持

## 概述

远程导师可以突破地理限制，为节点带来更丰富的专业指导资源。

---

## 1. 远程导师类型

| 类型 | 来源 | 形式 |
|------|------|------|
| **项目导师** | 专业人士 | 项目全程指导 |
| **专题顾问** | 领域专家 | 特定问题咨询 |
| **嘉宾讲师** | 各界人士 | 在线讲座 |
| **网络导师** | OWL网络 | 跨节点支持 |

---

## 2. 协作工具

### 2.1 常用工具

| 用途 | 工具选择 |
|------|---------|
| 视频会议 | 腾讯会议、Zoom |
| 即时通讯 | 微信群、飞书 |
| 协作文档 | 腾讯文档、Notion |
| 项目展示 | 视频、照片分享 |

### 2.2 有效沟通

| 要点 | 做法 |
|------|------|
| 准备充分 | 提前发送背景资料 |
| 问题具体 | 准备明确的问题 |
| 记录整理 | 及时记录要点 |
| 跟进反馈 | 汇报进展 |

---

## 3. 远程指导模式

### 3.1 同步模式

- 定期视频会议
- 实时问答
- 远程演示

### 3.2 异步模式

- 视频留言
- 文档反馈
- 论坛讨论

---

## 4. 管理机制

| 方面 | 做法 |
|------|------|
| 匹配 | 根据需求匹配导师 |
| 协议 | 明确时间、方式、期望 |
| 跟进 | 定期检视效果 |
| 感谢 | 适当表达认可 |
