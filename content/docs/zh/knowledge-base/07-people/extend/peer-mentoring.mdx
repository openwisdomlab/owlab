# 同伴导师计划

## 概述

同伴导师让经验丰富的学习者指导新成员，实现双向成长：新成员获得支持，导师锻炼能力。

---

## 1. 计划设计

### 1.1 核心价值

| 对象 | 收获 |
|------|------|
| **新成员** | 亲和的指导、榜样激励 |
| **同伴导师** | 领导力、教学能力、成就感 |
| **社区** | 文化传承、导师减负 |

### 1.2 选拔标准

| 标准 | 说明 |
|------|------|
| 技能水平 | 掌握一定技能 |
| 沟通能力 | 能够清晰表达 |
| 热心助人 | 愿意帮助他人 |
| 责任心 | 能够履行承诺 |

---

## 2. 培训内容

### 2.1 培训模块

| 模块 | 内容 |
|------|------|
| 角色认知 | 同伴导师vs老师 |
| 沟通技巧 | 如何有效沟通 |
| 指导方法 | 引导而非代做 |
| 边界意识 | 知道何时求助 |

---

## 3. 实施机制

### 3.1 配对方式

| 方式 | 适用 |
|------|------|
| 1对1 | 深度指导 |
| 1对多 | 扩大覆盖 |
| 小组互助 | 相近水平 |

### 3.2 运作模式

- 固定时间段的指导
- 项目中的协作
- 日常的互助
- 定期的检视

---

## 4. 激励机制

| 激励 | 方式 |
|------|------|
| 认可 | 导师徽章、证书 |
| 发展 | 进阶培训机会 |
| 记录 | 服务时长记录 |
| 推荐 | 升学/实习推荐 |
