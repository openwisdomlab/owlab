# 家长参与指南

## 概述

家长是创客教育的重要合作伙伴。有效的家长参与可以延伸学习到家庭，获得支持和理解。

---

## 1. 家长角色定位

### 1.1 角色类型

| 角色 | 行为 |
|------|------|
| **支持者** | 提供时间、资源、鼓励 |
| **学习伙伴** | 一起探索、共同学习 |
| **资源贡献者** | 分享专业技能、提供资源 |
| **倡导者** | 传播创客教育理念 |

### 1.2 避免的角色

| 避免 | 原因 |
|------|------|
| **代劳者** | 剥夺孩子学习机会 |
| **过度干预者** | 压制孩子自主性 |
| **成果导向者** | 过分关注结果而非过程 |

---

## 2. 参与方式

### 2.1 在家支持

| 方式 | 具体做法 |
|------|---------|
| 创建家庭创客角 | 提供探索空间 |
| 提供材料工具 | 基础材料支持 |
| 表达兴趣 | 询问学习情况 |
| 不评判失败 | 鼓励尝试 |

### 2.2 在空间参与

| 方式 | 说明 |
|------|------|
| 开放日参观 | 了解学习环境 |
| 亲子活动 | 共同参与项目 |
| 志愿服务 | 贡献时间技能 |
| 资源分享 | 分享专业知识 |

---

## 3. 沟通策略

### 3.1 与家长沟通

| 情境 | 方法 |
|------|------|
| 入门说明 | 解释创客教育理念 |
| 进度分享 | 定期展示学习成果 |
| 问题沟通 | 及时、诚恳沟通 |
| 邀请参与 | 提供参与机会 |

### 3.2 常见疑虑回应

| 疑虑 | 回应 |
|------|------|
| "玩有什么用？" | 玩中学的科学依据 |
| "影响学习怎么办？" | 技能与学业的相互促进 |
| "安全吗？" | 安全管理措施说明 |

---

## 4. 家长能力建设

### 4.1 家长工作坊

- 创客教育理念入门
- 家庭创客活动指南
- 与孩子有效沟通

### 4.2 资源支持

- 家庭项目建议清单
- 在线学习资源推荐
- 家长社群支持
