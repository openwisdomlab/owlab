# M07 人员与能力

> **一句话定位**: 培养能点燃学习者的人

---

## 核心原则

| 原则 | 内涵 | 体现 |
|------|------|------|
| **成长型思维** | 每个人都可以成长 | 持续学习 |
| **以身作则** | 导师首先是学习者 | 示范学习 |
| **赋能他人** | 让学习者能独立探索 | 脚手架思维 |
| **多元包容** | 不同背景皆可成师 | 开放标准 |
| **反思迭代** | 通过反思提升能力 | 反思实践 |
| **社区驱动** | 志愿者是核心力量 | 社区参与 |

---

## 角色体系

### 组织架构

```
┌─────────────────────────────────────────┐
│  管理团队                                │
│  实验室负责人 / 教研员 / 社区经理          │
├─────────────────────────────────────────┤
│  核心团队                                │
│  全职导师 / 技术支持                      │
├─────────────────────────────────────────┤
│  延伸团队                                │
│  兼职导师 / 专家顾问 / 外聘讲师            │
├─────────────────────────────────────────┤
│  社区力量                                │
│  志愿者 / 实习生 / 家长志愿者 / 大使       │
└─────────────────────────────────────────┘
```

### 管理团队角色

| 角色 | 职责 | 能力要求 | 配置 |
|------|------|----------|------|
| **实验室负责人** | 整体运营、战略规划、对外合作 | E2+管理能力 | 1人 |
| **技术导师/教研员** | 课程设计、技术支持、师资培训 | E2+专业技术 | 按规模 |
| **社区经理** | 社区运营、会员服务、活动策划 | 沟通+服务意识 | 按规模 |
| **运营助理** | 日常运营、行政支持、物资管理 | 执行力 | 可选 |

### 学习者角色 (游戏化身份)

| 角色 | 代码 | 定义 | 特征 |
|------|------|------|------|
| **探险家** | Explorer | 新加入的学习者 | 好奇心强 |
| **收藏家** | Collector | 积累一定经验 | 系统学习 |
| **创造者** | Creator | 能独立完成项目 | 动手能力 |
| **导师** | Mentor | 能帮助他人 | 乐于分享 |
| **大师** | Master | 在某领域专精 | 深度专业 |

## 教育者能力分级

### 四级能力体系

| 等级 | 代码 | 定义 | 资质要求 | 授权范围 |
|------|------|------|----------|----------|
| **见习** | E0 | 正在学习，需督导 | 完成基础培训 | 协助执行 |
| **合格** | E1 | 能独立执行标准课程 | 通过认证考核 | 独立授课 |
| **骨干** | E2 | 能设计课程并指导他人 | 2年+设计认证 | 课程开发 |
| **专家** | E3 | 能进行体系设计与师培 | 5年+专家认证 | 体系设计 |

### 能力模型 (4级×5维度)

| 维度 | E0 | E1 | E2 | E3 |
|------|-----|-----|-----|-----|
| **专业知识** | 了解基础 | 掌握核心 | 精通领域 | 创新突破 |
| **教学技能** | 模仿执行 | 标准执行 | 灵活设计 | 培训他人 |
| **引导能力** | 跟随流程 | 基础引导 | 灵活应变 | 复杂情境 |
| **技术应用** | 基础操作 | 熟练使用 | 创新应用 | 技术整合 |
| **反思能力** | 接受反馈 | 自我反思 | 改进实践 | 创造方法 |

## 导师体系

### X-导师选拔标准

| 维度 | 要求 | 验证方式 |
|------|------|----------|
| **理念认同** | 认同 OWL 核心价值观 | 面谈评估 |
| **专业能力** | 在某一领域有专长 | 作品/经历 |
| **教学意愿** | 愿意分享和引导他人 | 试讲评估 |
| **沟通能力** | 能有效与学习者互动 | 观察评估 |
| **时间承诺** | 能保证一定投入时间 | 协议约定 |

### 导师发展路径

```
见习导师 (E0) → 认证导师 (E1) → 骨干导师 (E2) → 专家导师 (E3)
    ↓              ↓              ↓              ↓
  3个月          1年+           3年+           5年+
  观摩+协助      独立授课       课程开发        体系设计
```

## 引导者 (Facilitator)

### 核心职责

| 职责 | 说明 | 关键技能 |
|------|------|----------|
| **创造安全环境** | 让学习者敢于尝试 | 氛围营造 |
| **提问而非告知** | 引导思考而非给答案 | 提问技巧 |
| **观察与支持** | 适时介入、适度退出 | 观察力 |
| **促进协作** | 帮助团队有效合作 | 协调能力 |
| **记录与反馈** | 记录过程、提供反馈 | 反馈技巧 |

### 引导技能清单

| 技能 | 应用场景 | 注意事项 |
|------|----------|----------|
| **积极倾听** | 了解学习者需求 | 不打断、不评判 |
| **开放式提问** | 引导深度思考 | 避免诱导性问题 |
| **重述与澄清** | 确认理解正确 | 避免加入个人解读 |
| **适时沉默** | 给思考空间 | 不急于填补沉默 |
| **正向反馈** | 强化积极行为 | 具体而非笼统 |

## 志愿者体系

### 志愿者类型

| 类型 | 来源 | 特点 | 适合任务 |
|------|------|------|----------|
| **大学生志愿者** | 高校合作 | 专业技能、活力 | 活动执行、技术支持 |
| **家长志愿者** | 会员家庭 | 责任心、关注安全 | 活动协助、安全监督 |
| **行业志愿者** | 企业员工 | 专业背景 | 专题分享、职业指导 |
| **退休志愿者** | 社区招募 | 经验丰富、时间充足 | 日常支持、传统工艺 |

### 志愿者管理流程

```
1. 招募 → 2. 面试 → 3. 背调 → 4. 培训 → 5. 服务 → 6. 评估 → 7. 激励
```

### 志愿者激励机制

| 激励方式 | 说明 |
|----------|------|
| **服务时数证明** | 官方志愿服务证明 |
| **技能培训** | 免费参加培训课程 |
| **优先体验** | 新活动优先参与 |
| **荣誉徽章** | 数字徽章与实体勋章 |
| **社区认可** | 优秀志愿者表彰 |

## 培训体系

### 培训层级

```
入职培训 (必修) ─── 所有新人必须完成
    ↓
岗位培训 (按角色) ─ 根据岗位定制
    ↓
进阶培训 (按发展) ─ 能力提升
    ↓
专项培训 (按需求) ─ 特定技能
```

### 培训内容框架

| 培训类型 | 内容 | 时长 | 考核方式 |
|----------|------|------|----------|
| **入职培训** | 理念、安全、基础技能 | 8-16小时 | 笔试+实操 |
| **导师培训** | 教学技能、引导技巧 | 16-24小时 | 试讲评估 |
| **安全培训** | 设备安全、应急处理 | 4-8小时 | 情景考核 |
| **进阶培训** | 课程设计、项目管理 | 按需 | 作品评审 |

## 认证体系

### 认证类型

| 认证 | 对象 | 有效期 | 续认条件 |
|------|------|--------|----------|
| **导师认证** | 教育者 | 2年 | 持续授课+复训 |
| **助教认证** | 助教 | 1年 | 服务时数+评价 |
| **志愿者认证** | 志愿者 | 1年 | 服务时数 |
| **课程认证** | 课程 | 2年 | 更新迭代 |
| **实验室认证** | 节点 | 1年 | 年度评估 |

### 认证流程

```
申请 → 资料审核 → 能力评估 → 试讲/实操 → 认证颁发 → 年度复审
```

## 常见误区

- ❌ "专家"="好导师" → ✅ 教学能力≠专业能力
- ❌ "培训"="发展" → ✅ 发展需要实践和反思
- ❌ 忽视志愿者价值 → ✅ 志愿者带来独特视角
- ❌ 一次培训终身有效 → ✅ 需持续学习和复训
- ❌ 重招募轻培养 → ✅ 内部培养是核心
