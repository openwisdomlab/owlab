# M07 人员与能力

> **一句话定位**: 培养能点燃学习者的人

---

## 核心原则

| 原则 | 内涵 | 体现 |
|------|------|------|
| **成长型思维** | 每个人都可以成长 | 持续学习 |
| **以身作则** | 导师首先是学习者 | 示范学习 |
| **赋能他人** | 让学习者能独立探索 | 脚手架思维 |
| **多元包容** | 不同背景皆可成师 | 开放标准 |
| **反思迭代** | 通过反思提升能力 | 反思实践 |

---

## 角色体系

```
┌─────────────────────────────────────────┐
│  核心团队                                │
│  节点负责人 / 全职导师                    │
├─────────────────────────────────────────┤
│  延伸团队                                │
│  兼职导师 / 专家顾问                      │
├─────────────────────────────────────────┤
│  社区力量                                │
│  志愿者 / 实习生 / 家长志愿者             │
└─────────────────────────────────────────┘
```

---

## 教育者能力分级

| 等级 | 代码 | 定义 | 资质要求 |
|------|------|------|----------|
| 见习 | E0 | 正在学习，需督导 | 完成基础培训 |
| 合格 | E1 | 能独立执行标准课程 | 通过认证考核 |
| 骨干 | E2 | 能设计课程并指导他人 | 2年+设计认证 |
| 专家 | E3 | 能进行体系设计与师培 | 5年+专家认证 |

---

## 能力模型 (简化)

| 维度 | E0 | E1 | E2 | E3 |
|------|-----|-----|-----|-----|
| 专业知识 | 了解 | 掌握 | 精通 | 创新 |
| 教学技能 | 模仿 | 执行 | 设计 | 培训 |
| 引导能力 | 跟随 | 基础引导 | 灵活引导 | 复杂情境 |
| 反思能力 | 接受反馈 | 自我反思 | 改进实践 | 创造方法 |

---

## MVS 最小可运行标准

| 项目 | 要求 |
|------|------|
| 负责人 | ✅ 至少1人E1以上 |
| 安全培训 | ✅ 全员完成 |
| 背景调查 | ✅ 涉及未成年人岗位 |
| 发展计划 | ✅ 每人有成长路径 |

---

## 培训体系

```
入职培训 (必修)
    ↓
岗位培训 (按角色)
    ↓
进阶培训 (按发展)
    ↓
专项培训 (按需求)
```

## 常见误区

- ❌ "专家"="好导师" → ✅ 教学能力≠专业能力
- ❌ "培训"="发展" → ✅ 发展需要实践和反思
- ❌ 忽视志愿者价值 → ✅ 志愿者带来独特视角
