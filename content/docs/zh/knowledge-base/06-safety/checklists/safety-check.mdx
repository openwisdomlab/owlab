# 安全检查清单

## 使用说明

| 检查类型 | 频率 | 执行人 | 耗时 |
|----------|------|--------|------|
| 日检 | 每日开门前 | 值班人员 | 5-10分钟 |
| 周检 | 每周一次 | 安全负责人 | 15-20分钟 |
| 月检 | 每月一次 | 节点负责人 | 30-45分钟 |

---

## 一、日检清单 (Daily Check)

**执行时间**: 开门前
**执行人**: 当日值班人员
**记录**: 运营日志

### 1.1 入口与通道

- [ ] 入口门锁正常开启
- [ ] 应急出口畅通无阻
- [ ] 通道无杂物堆放
- [ ] 地面无积水/滑倒风险
- [ ] 照明正常

### 1.2 消防设施

- [ ] 灭火器在位且压力正常（指针在绿区）
- [ ] 消防通道畅通
- [ ] 应急指示灯亮
- [ ] 烟感/喷淋无遮挡

### 1.3 电气安全

- [ ] 总电闸正常
- [ ] 无裸露电线
- [ ] 插座无烧焦痕迹
- [ ] 大功率设备无异常

### 1.4 设备状态

- [ ] 高风险设备已断电/上锁
- [ ] 工具归位
- [ ] 无遗留未完成的危险操作

### 1.5 环境检查

- [ ] 通风正常
- [ ] 温度适宜
- [ ] 无异味
- [ ] 垃圾已清理

**日检完成签字**: __________ **日期**: __________

---

## 二、周检清单 (Weekly Check)

**执行时间**: 每周固定时间
**执行人**: 安全负责人
**记录**: 安全检查表

### 2.1 消防设施 (详细)

- [ ] 灭火器外观完好
- [ ] 灭火器有效期检查
- [ ] 消防栓可用
- [ ] 应急照明电池测试
- [ ] 疏散图张贴完好

### 2.2 急救设备

- [ ] 急救箱位置明确
- [ ] 急救箱物品齐全
- [ ] 药品/物品未过期
- [ ] 急救联系方式更新

### 2.3 设备安全

- [ ] 电动工具绝缘检查
- [ ] 高风险设备防护罩完好
- [ ] 安全警示标识清晰
- [ ] 设备操作指南可读

### 2.4 化学品/材料

- [ ] 化学品存放规范
- [ ] 标识清晰
- [ ] 无泄漏
- [ ] MSDS资料可查

### 2.5 人员安全

- [ ] 紧急联系人信息更新
- [ ] 本周活动风险回顾
- [ ] 安全事件记录检查

**周检完成签字**: __________ **日期**: __________

---

## 三、月检清单 (Monthly Check)

**执行时间**: 每月固定日期
**执行人**: 节点负责人
**记录**: 月度安全报告

### 3.1 消防系统

- [ ] 消防设施专业检测（如需要）
- [ ] 应急演练执行情况回顾
- [ ] 消防器材维保记录
- [ ] 疏散路线有效性评估

### 3.2 设备维护

- [ ] 高风险设备维保记录
- [ ] 设备故障统计
- [ ] 维修工单跟进
- [ ] 报废设备处理

### 3.3 培训与资质

- [ ] 新员工安全培训完成
- [ ] 安全培训记录更新
- [ ] 特种设备操作资质检查
- [ ] 应急响应培训需求评估

### 3.4 制度与文档

- [ ] 安全制度更新检查
- [ ] 应急预案有效性评估
- [ ] 安全事件报告归档
- [ ] 保险有效期检查

### 3.5 合规性

- [ ] 监管要求变化检查
- [ ] 许可证/资质有效期
- [ ] 外部检查整改跟进
- [ ] 供应商安全资质

### 3.6 改进计划

- [ ] 上月问题整改情况
- [ ] 新发现问题清单
- [ ] 下月改进计划
- [ ] 资源需求确认

**月检完成签字**: __________ **日期**: __________

## 四、特殊检查

### 4.1 活动前检查

适用于大型活动、高风险活动

- [ ] 活动风险评估完成
- [ ] 应急预案准备
- [ ] 急救人员/设备到位
- [ ] 参与者安全告知
- [ ] 家长同意书（涉及未成年人）

### 4.2 节假日前检查

长假前的额外检查

- [ ] 所有电源关闭（必要的除外）
- [ ] 水阀关闭
- [ ] 门窗锁好
- [ ] 贵重物品妥善保管
- [ ] 应急联系人安排

### 4.3 恶劣天气检查

台风、暴雨等极端天气

- [ ] 门窗加固
- [ ] 户外物品收回
- [ ] 排水通畅
- [ ] 电气设备防护
- [ ] 人员疏散准备

## 六、检查工具

|------|------|
