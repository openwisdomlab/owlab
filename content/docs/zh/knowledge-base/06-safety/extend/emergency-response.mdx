# 应急响应流程

## 概述

应急响应是安全管理的最后一道防线。本文档建立了创新空间各类紧急情况的标准化响应流程，确保在紧急情况发生时，所有人员能够快速、正确地响应，最大限度减少人员伤害和财产损失。

**核心原则**: 生命第一，快速响应，有序处置，事后改进。

---

## 1. 应急管理体系

### 1.1 应急组织架构

```
应急指挥 (节点负责人)
    │
    ├── 疏散引导组 (导师)
    │   └── 引导疏散、清点人数
    │
    ├── 通讯联络组 (前台/运营)
    │   └── 报警、联络、信息发布
    │
    ├── 现场处置组 (安全员/持证人员)
    │   └── 初期灭火、急救、设备隔离
    │
    └── 后勤保障组 (运营)
        └── 物资调配、家长联络
```

### 1.2 应急职责分工

| 角色 | 日常职责 | 应急职责 |
|------|----------|----------|
| **节点负责人** | 安全管理总负责 | 应急指挥、决策、外部协调 |
| **安全员** | 安全检查、培训 | 现场处置、技术支持 |
| **值班导师** | 活动安全管理 | 人员疏散、急救 |
| **前台/运营** | 日常运营 | 报警、通讯、记录 |
| **所有人员** | 遵守安全规定 | 服从指挥、有序疏散 |

### 1.3 应急联系信息

**紧急电话** (应张贴于显眼位置):

| 类型 | 电话 | 用途 |
|------|------|------|
| 火警 | 119 | 火灾、化学品泄漏 |
| 急救 | 120 | 人员伤病 |
| 报警 | 110 | 治安、刑事 |
| 节点负责人 | [填写] | 内部应急指挥 |
| 总部应急热线 | [填写] | 重大事故上报 |
| 物业/消防中心 | [填写] | 建筑相关应急 |

## 2. 火灾应急响应

### 2.1 火灾响应流程

```
发现火情
    ↓
┌─────────────────────────────────────────┐
│  评估：是否可控？(小火、初期)              │
└─────────────────────────────────────────┘
    ↓是                    ↓否
┌─────────────┐      ┌─────────────────┐
│ 初期灭火     │      │ 立即疏散        │
│ (如有把握)   │      │ 拨打 119        │
└─────────────┘      └─────────────────┘
    ↓成功              ↓
┌─────────────┐      ┌─────────────────┐
│ 通风、监控   │      │ 引导疏散至集合点 │
│ 上报、检查   │      │ 清点人数        │
└─────────────┘      │ 等待消防救援     │
                     └─────────────────┘
```

### 2.2 火灾响应行动指南

#### 发现者行动

| 步骤 | 行动 | 要点 |
|------|------|------|
| 1 | 大声呼喊"着火了！" | 提醒周围人员 |
| 2 | 按下手动报警按钮(如有) | 触发警报 |
| 3 | 评估火势 | 能否安全灭火？ |
| 4A | 小火：用灭火器灭火 | 确保退路、保持安全距离 |
| 4B | 大火：立即撤离 | 不要冒险 |
| 5 | 拨打 119 | 说清地址、火情 |

#### 灭火器使用 (PASS 法则)

| 步骤 | 动作 | 说明 |
|------|------|------|
| **P**ull | 拔 | 拔出保险销 |
| **A**im | 瞄 | 瞄准火焰根部 |
| **S**queeze | 压 | 压下压把 |
| **S**weep | 扫 | 左右扫射 |

**注意**: 站在上风处，保持2-3米距离，确保退路畅通。

#### 疏散行动

| 角色 | 行动 |
|------|------|
| **导师** | 组织本区域人员有序撤离，不要跑、不要喊 |
| **学员** | 听从指挥，弯腰低姿，用湿毛巾捂口鼻(如有烟) |
| **门口人员** | 打开紧急出口，引导方向 |
| **最后离开者** | 关闭房门(不锁)，减缓烟火蔓延 |

#### 疏散路线原则

- 使用最近的安全出口
- 不使用电梯
- 遇烟雾大时，贴近地面爬行
- 不要返回取物品
- 到达室外集合点后清点人数

### 2.3 疏散集合与清点

| 步骤 | 行动 | 责任人 |
|------|------|--------|
| 1 | 在预定集合点集合 | 全员 |
| 2 | 导师清点本组人数 | 各导师 |
| 3 | 汇报给应急指挥 | 导师 |
| 4 | 缺人立即报告消防人员 | 应急指挥 |
| 5 | 禁止擅自返回建筑 | 全员 |

## 3. 人身伤害应急响应

### 3.1 伤害类型与初步处置

#### 切割伤/擦伤

| 情况 | 处置 | 后续 |
|------|------|------|
| 轻微(表皮) | 清洗 → 消毒 → 创可贴 | 观察 |
| 中度(较深) | 清洗 → 压迫止血 → 包扎 | 就医评估 |
| 严重(大量出血) | 直接压迫止血 → 呼叫120 | 紧急送医 |

#### 烫伤/灼伤

| 情况 | 处置 | 禁止 |
|------|------|------|
| I度(红肿) | 冷水冲洗20分钟 → 保护 | 涂牙膏等偏方 |
| II度(水泡) | 冷水冲洗 → 无菌覆盖 → 就医 | 挑破水泡 |
| III度(焦痂) | 覆盖 → 呼叫120 | 冲洗(防感染) |

**冷水冲洗要点**: 流动冷水(非冰水)，15-20分钟，减轻疼痛和损伤。

#### 触电

| 步骤 | 行动 | 要点 |
|------|------|------|
| 1 | 切断电源 | 拔插头或关总闸 |
| 2 | 如无法断电 | 用干燥绝缘物分离触电者 |
| 3 | 评估伤者状态 | 呼吸、心跳、意识 |
| 4 | 必要时 CPR | 等待120到达 |

**警告**: 不要直接接触触电者，防止二次触电！

#### 化学品伤害

| 类型 | 处置 | 特殊情况 |
|------|------|----------|
| 皮肤接触 | 大量清水冲洗15分钟 | 除去污染衣物 |
| 眼睛接触 | 清水冲洗15分钟 | 撑开眼睑，从内眼角向外冲 |
| 吸入 | 移至新鲜空气处 | 呼吸困难则呼叫120 |
| 误食 | 不要催吐 → 立即就医 | 带上化学品标签 |

### 3.2 急救响应流程

```
发现伤病情况
    ↓
评估现场安全
    ↓
评估伤者状态 (意识、呼吸、出血)
    ↓
┌─────────────────┬─────────────────┐
│ 严重/不确定      │ 轻微且稳定       │
│ 呼叫120         │ 现场急救处理     │
│ 通知负责人       │ 观察+记录       │
└────────┬────────┴────────┬────────┘
         ↓                  ↓
    等待急救到达         观察是否恶化
    同时进行急救         通知家长(未成年人)
```

### 3.3 CPR 心肺复苏 (仅供参考，建议持证操作)

**适用情况**: 无反应、无呼吸或仅有濒死喘息

| 步骤 | 动作 | 要点 |
|------|------|------|
| 1 | 确认安全、呼叫帮助 | 让人拨打120、取AED |
| 2 | 判断反应 | 拍肩、呼唤 |
| 3 | 开放气道 | 仰头抬颌法 |
| 4 | 判断呼吸 | 10秒内看听感 |
| 5 | 胸外按压 | 双手交叉，两乳头连线中点，5-6cm深，100-120次/分 |
| 6 | 人工呼吸(如培训过) | 30:2 (按压:呼吸) |
| 7 | AED到达 | 按语音提示操作 |

**持续至**: 专业救护到达、伤者恢复、或施救者无法继续

## 4. 设备故障应急响应

### 4.1 设备故障分类

| 类别 | 定义 | 示例 | 响应级别 |
|------|------|------|----------|
| **A类** | 可能导致人身伤害 | 激光失控、电火花、高温失控 | 立即停机、撤离 |
| **B类** | 可能导致火灾/财产损失 | 过热、冒烟、异常噪音 | 紧急停机、评估 |
| **C类** | 影响正常使用 | 精度下降、故障报警 | 停机检查 |

### 4.2 设备紧急停机流程

#### 一般设备

| 步骤 | 行动 |
|------|------|
| 1 | 按下紧急停止按钮(红色蘑菇头) |
| 2 | 关闭电源开关 |
| 3 | 拔出电源插头(如安全) |
| 4 | 警示标识、隔离区域 |
| 5 | 报告值班导师/安全员 |

#### 激光切割机紧急处理

| 情况 | 行动 |
|------|------|
| **火焰** | 紧急停止 → 关闭舱门 → 等待熄灭 → 如不灭则使用CO2灭火器 |
| **大量烟雾** | 紧急停止 → 不打开舱门 → 疏散人员 → 加强通风 |
| **激光异常** | 紧急停止 → 不直视 → 撤离 → 报告 |

#### 3D打印机紧急处理

| 情况 | 行动 |
|------|------|
| **冒烟/焦味** | 关闭电源 → 不触碰喷头 → 通风 |
| **喷头堵塞无法停止** | 拔电源 → 等待冷却 |
| **材料打结/缠绕** | 暂停打印 → 剪断材料 → 重新装载 |

### 4.3 故障后处理

| 步骤 | 行动 | 责任人 |
|------|------|--------|
| 1 | 填写故障报告 | 发现人 |
| 2 | 张贴"停用"标识 | 安全员 |
| 3 | 通知维修/供应商 | 运营 |
| 4 | 评估是否影响活动 | 导师 |
| 5 | 修复后测试验收 | 安全员 |
| 6 | 恢复使用 | 安全员确认 |

## 5. 人员走失/失踪应急响应

### 5.1 适用场景

- 活动期间发现学员不在应在位置
- 家长接送时找不到孩子
- 户外活动人员走散

### 5.2 响应流程

```
发现人员失踪
    ↓
确认最后见到时间/地点
    ↓
立即通知负责人
    ↓
┌─────────────────────────────────────────┐
│  5分钟内搜索                             │
│  - 洗手间、储物间、隐蔽角落               │
│  - 建筑内全面搜索                        │
│  - 询问其他人员                          │
└─────────────────────────────────────────┘
    ↓找到                    ↓未找到
┌─────────────┐      ┌─────────────────┐
│ 确认安全     │      │ 扩大搜索范围     │
│ 安抚、记录   │      │ 通知家长        │
│ 家长沟通     │      │ 15分钟仍未找到   │
└─────────────┘      │ → 报警 110      │
                     └─────────────────┘
```

### 5.3 预防措施

| 措施 | 说明 |
|------|------|
| **签到签退** | 所有活动必须签到，结束时清点 |
| **分组负责** | 每个导师负责固定学员 |
| **着装标识** | 统一标识(如胸牌、手环) |
| **安全边界** | 明确活动区域，禁止擅自离开 |
| **特别关注** | 年幼、特殊需求学员重点看护 |

## 6. 自然灾害应急响应

### 6.1 台风/暴雨

#### 预警阶段

| 预警等级 | 行动 |
|----------|------|
| 蓝色/黄色 | 关注预报、检查设施、准备应急物资 |
| 橙色 | 考虑取消户外活动、通知家长 |
| 红色 | 停止一切活动、安全闭馆 |

#### 应对措施

| 项目 | 措施 |
|------|------|
| **门窗** | 关闭并加固 |
| **户外物品** | 收回室内或固定 |
| **排水** | 检查排水通畅 |
| **电气** | 必要时断电 |
| **人员** | 留在室内安全区域 |

### 6.2 地震

#### 震中行动 (室内)

| 阶段 | 行动 | 要点 |
|------|------|------|
| **震动时** | 就近躲避 | 桌下、承重墙角、卫生间 |
|  | 保护头部 | 用手或书包保护 |
|  | 远离危险 | 窗户、玻璃、悬挂物 |
| **震动停止后** | 评估安全 | 检查有无受伤 |
|  | 有序疏散 | 不乘电梯、走楼梯 |
|  | 到达开阔地 | 远离建筑物、电线 |

### 6.3 停电

| 情况 | 行动 |
|------|------|
| **突然停电** | 保持冷静、就地等待、使用手机照明 |
| **应急照明** | 检查应急灯是否亮起 |
| **设备处理** | 关闭运行中设备开关(防止恢复供电时突然启动) |
| **人员疏散** | 如长时间停电，有序疏散 |
| **电梯困人** | 使用对讲机或电话求助、等待救援 |

## 7. 应急演练

### 7.1 演练计划

| 演练类型 | 频率 | 内容 | 参与人员 |
|----------|------|------|----------|
| **消防疏散** | 半年 | 火灾报警、疏散、集合、清点 | 全员 |
| **急救演练** | 年度 | CPR、AED、创伤处理 | 导师、安全员 |
| **设备故障** | 年度 | 紧急停机、初期处置 | 设备操作人员 |
| **桌面推演** | 年度 | 各类场景讨论决策 | 管理层 |

### 7.2 演练流程

| 阶段 | 内容 |
|------|------|
| **准备** | 确定日期、通知参与者、准备器材、明确分工 |
| **简报** | 说明演练目的、场景、注意事项 |
| **执行** | 模拟场景、按预案行动、记录过程 |
| **总结** | 收集反馈、分析问题、记录改进点 |
| **改进** | 更新预案、补充培训 |

### 7.3 演练评估

| 评估项 | 评估标准 | 记录 |
|--------|----------|------|
| **响应时间** | 从报警到开始疏散 | 秒 |
| **疏散时间** | 从开始到全员到达集合点 | 分钟 |
| **人员覆盖** | 参与演练人员比例 | % |
| **程序执行** | 是否按预案执行 | 偏差记录 |
| **设备有效性** | 报警、消防设备是否正常 | 问题清单 |

## 9. 事故报告与调查

### 9.1 报告要求

| 事故类型 | 报告时限 | 报告对象 |
|----------|----------|----------|
| 轻微伤(无需就医) | 当日 | 安全员 |
| 需就医治疗 | 2小时内 | 安全员 → 负责人 |
| 严重伤害/火灾 | 立即 | 负责人 → 总部 → 相关部门 |

### 9.2 事故报告内容

| 项目 | 内容 |
|------|------|
| **基本信息** | 时间、地点、人员 |
| **事故经过** | 发生了什么、怎么发生的 |
| **伤害情况** | 伤害类型、程度、处置情况 |
| **财产损失** | 设备损坏、其他损失 |
| **现场保护** | 是否保护现场、证据收集 |
| **初步原因** | 直接原因、初步分析 |
| **已采取措施** | 应急处置、后续安排 |

### 9.3 事故调查

| 调查步骤 | 内容 |
|----------|------|
| **收集信息** | 现场勘查、人员访谈、资料收集 |
| **时间线重建** | 还原事故发生过程 |
| **原因分析** | 直接原因、间接原因、根本原因 |
| **改进建议** | 针对性的预防措施 |
| **报告撰写** | 完整记录调查结果 |
