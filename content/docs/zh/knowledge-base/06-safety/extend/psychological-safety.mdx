# 心理安全框架

## 概述

心理安全是创造力和学习的基础。在心理安全的环境中，学习者敢于提问、敢于尝试、敢于失败，从而实现真正的成长。

---

## 1. 心理安全的定义

**心理安全感**：相信自己不会因为提出问题、承认错误或提出不同意见而受到惩罚或羞辱。

### 1.1 心理安全的表现

| 有心理安全感 | 缺乏心理安全感 |
|-------------|---------------|
| 愿意提问 | 不敢提问，怕显得"笨" |
| 敢于尝试新事物 | 只做确定能成功的事 |
| 承认不知道 | 假装知道 |
| 分享失败经验 | 隐藏失败 |
| 提出不同意见 | 沉默顺从 |

---

## 2. 建设策略

### 2.1 领导者示范

| 行为 | 效果 |
|------|------|
| 承认自己也不知道 | 允许无知 |
| 分享自己的失败 | 允许失败 |
| 认真对待每个问题 | 鼓励提问 |
| 感谢不同意见 | 允许异议 |

### 2.2 规范建立

```
心理安全公约:
┌─────────────────────────────────────────┐
│  ✓ 所有问题都是好问题                   │
│  ✓ 失败是学习的一部分                   │
│  ✓ 不同意见值得倾听                     │
│  ✓ 我们互相帮助而非评判                 │
│  ✓ 尊重每个人的学习节奏                 │
└─────────────────────────────────────────┘
```

---

## 3. 失败文化

### 3.1 重塑失败叙事

| 传统观念 | 创客观念 |
|---------|---------|
| 失败是可耻的 | 失败是学习 |
| 成功 = 一次做对 | 成功 = 迭代改进 |
| 隐藏失败 | 分享失败经验 |

### 3.2 具体实践

- "失败墙"：展示有价值的失败
- 失败分享会：定期分享和反思
- "最佳失败奖"：奖励有学习价值的失败

---

## 4. 预防与干预

### 4.1 识别信号

| 信号 | 可能原因 |
|------|---------|
| 突然退缩 | 受到打击或排斥 |
| 不再提问 | 曾被否定 |
| 回避活动 | 焦虑或不适 |
| 行为变化 | 需要关注 |

### 4.2 干预原则

1. 私下了解情况
2. 不评判，倾听为主
3. 必要时寻求专业帮助
4. 保护隐私
5. 跟进支持

---

## 5. 特殊群体关注

| 群体 | 特别考虑 |
|------|---------|
| 内向者 | 提供非公开表达方式 |
| 初学者 | 额外鼓励和支持 |
| 少数群体 | 确保包容和代表性 |
| 有心理困扰者 | 识别信号，专业转介 |
