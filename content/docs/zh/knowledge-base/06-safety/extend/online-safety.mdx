# 在线安全守则

## 概述

创客活动越来越多地涉及网络和数字工具。本文档提供在线安全的基本规范和教育指南。

---

## 1. 网络使用规范

### 1.1 基本规则

| 规则 | 说明 |
|------|------|
| 合法使用 | 不访问非法内容 |
| 尊重他人 | 文明网络交流 |
| 保护隐私 | 不泄露个人信息 |
| 安全意识 | 识别网络风险 |

### 1.2 设备使用

- 使用强密码
- 不安装不明软件
- 定期更新系统
- 使用安全网络

---

## 2. 在线交流规范

### 2.1 社区平台

| 行为 | 规范 |
|------|------|
| 发言 | 友善、建设性 |
| 分享 | 注意版权和隐私 |
| 互动 | 尊重不同意见 |
| 冲突 | 寻求调解 |

### 2.2 禁止行为

- 网络欺凌
- 散布谣言
- 骚扰他人
- 不当内容

---

## 3. 数字素养教育

### 3.1 核心能力

| 能力 | 内容 |
|------|------|
| 信息素养 | 识别信息真伪 |
| 安全意识 | 保护个人安全 |
| 道德规范 | 尊重版权和隐私 |
| 数字足迹 | 了解网络痕迹 |

### 3.2 教育融入

将数字素养融入创客活动：
- 编程项目讨论安全编码
- 网络项目讨论数据保护
- 社交媒体分享讨论隐私

---

## 4. 事件应对

### 4.1 发现问题时

1. 停止不当行为
2. 保留证据
3. 报告导师/管理员
4. 严重情况报告监护人/有关部门
