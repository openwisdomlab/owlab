# 隐私保护指南

## 概述

创客空间收集和使用成员信息时必须遵守隐私保护原则，特别是对未成年人的信息要格外谨慎。

---

## 1. 信息收集原则

### 1.1 最小化原则

| 原则 | 说明 |
|------|------|
| **必要性** | 只收集必需信息 |
| **目的限定** | 仅用于声明用途 |
| **知情同意** | 获得明确同意 |
| **安全保管** | 妥善保护数据 |

### 1.2 收集的信息类型

| 信息类型 | 必要性 | 保护级别 |
|---------|--------|---------|
| 基本联系信息 | 必须 | 高 |
| 年龄/出生日期 | 必须 | 高 |
| 健康信息 | 可选 | 极高 |
| 照片/视频 | 可选 | 高 |
| 作品记录 | 可选 | 中 |

---

## 2. 未成年人特别保护

### 2.1 监护人同意

| 年龄 | 要求 |
|------|------|
| <14岁 | 监护人书面同意 |
| 14-18岁 | 监护人知情+本人同意 |
| ≥18岁 | 本人同意 |

### 2.2 照片使用规范

```
未成年人照片使用规则:
┌─────────────────────────────────────────┐
│  必须: 获得监护人书面授权               │
│  禁止: 公开可识别个人的照片             │
│  建议: 使用背影/侧面/集体照             │
│  删除: 收到要求后及时删除               │
└─────────────────────────────────────────┘
```

---

## 3. 数据安全

### 3.1 存储安全

| 措施 | 内容 |
|------|------|
| **访问控制** | 限制访问权限 |
| **加密存储** | 敏感数据加密 |
| **定期备份** | 防止数据丢失 |
| **到期清理** | 不再需要后删除 |

### 3.2 使用规范

- 不得向第三方提供个人信息
- 统计分析使用匿名数据
- 对外展示需获得同意
- 发生泄露需及时通知

---

## 4. 隐私声明

### 4.1 声明内容

应包含：
1. 收集哪些信息
2. 如何使用信息
3. 如何保护信息
4. 保留多长时间
5. 如何查询/删除
6. 联系方式

## 常见问题

<details>
<summary>可以在社交媒体发布活动照片吗？</summary>

需要：
1. 取得照片中人的同意
2. 未成年人需监护人同意
3. 可以使用不可识别个人的照片
4. 建议设置照片授权选项
</details>
