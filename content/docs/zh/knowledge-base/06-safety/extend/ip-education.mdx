# 知识产权教育

## 概述

创客既是知识产权的创造者，也是使用者。理解知识产权有助于保护自己的创作，同时尊重他人的劳动成果。

---

## 1. 基础概念

### 1.1 知识产权类型

| 类型 | 保护对象 | 示例 |
|------|---------|------|
| **著作权/版权** | 作品表达 | 代码、设计、文档 |
| **专利** | 发明创造 | 技术方案、产品 |
| **商标** | 商业标识 | 品牌名、Logo |
| **商业秘密** | 保密信息 | 配方、工艺 |

### 1.2 版权基础

- 创作即产生，无需注册
- 保护表达形式，不保护想法
- 持续时间：作者终身+50年（中国）

---

## 2. 使用他人作品

### 2.1 合法使用方式

| 方式 | 说明 |
|------|------|
| **公共领域** | 版权过期或放弃 |
| **开放许可** | CC等开放许可 |
| **获得授权** | 联系作者获得许可 |
| **合理使用** | 教育、评论等有限情况 |

### 2.2 常见误区

| 误区 | 真相 |
|------|------|
| "网上的都能用" | 网上作品同样有版权 |
| "非商业就可以" | 非商业使用也需授权 |
| "标注来源就行" | 署名不能替代授权 |
| "改了就是我的" | 改编也需要授权 |

---

## 3. 保护自己的创作

### 3.1 记录创作过程

- 保存原始文件和版本
- 记录创作时间
- 保留创作证据

### 3.2 选择许可方式

| 需求 | 推荐许可 |
|------|---------|
| 希望自由传播 | CC BY |
| 要求相同方式共享 | CC BY-SA |
| 不希望商业使用 | CC BY-NC |
| 保留所有权利 | 标准版权 |

---

## 4. 教学活动

### 4.1 适龄内容

| 年龄 | 内容 |
|------|------|
| 6-9岁 | "这是别人的作品，要尊重" |
| 10-12岁 | 基本版权概念，署名习惯 |
| 13-15岁 | 许可协议，开源概念 |
| 16+岁 | 深入理解，应用实践 |

### 4.2 实践活动

- 寻找可用的素材资源
- 为自己的作品选择许可
- 分析案例讨论

## 常见问题

<details>
<summary>学生作品的版权归谁？</summary>

一般而言：
- 学生个人创作 → 归学生
- 课堂作业 → 通常归学生
- 机构项目 → 需要明确约定
- 团队项目 → 共同所有或约定
</details>
