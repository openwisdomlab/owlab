# 年龄分级操作规范 (K-12)

## 概述

不同年龄段的儿童和青少年在认知能力、精细动作、风险意识和自我保护能力方面存在显著差异。本规范基于儿童发展规律，为 K-12 创新空间活动提供科学的年龄分级指南。

**核心原则**: 尊重发展规律，在安全边界内最大化探索空间。

---

## 1. 年龄发展特征概览

### 1.1 发展阶段划分

| 阶段 | 年龄 | 年级参考 | 发展关键词 |
|------|------|----------|------------|
| **幼儿期** | 5-6岁 | 幼儿园大班-小学1年级 | 感知探索、模仿学习 |
| **儿童早期** | 7-8岁 | 小学1-2年级 | 好奇心旺盛、规则意识萌芽 |
| **儿童中期** | 9-10岁 | 小学3-4年级 | 逻辑思维发展、协作意识 |
| **儿童晚期** | 11-12岁 | 小学5-6年级 | 抽象思维、自主性增强 |
| **青少年早期** | 13-14岁 | 初中1-2年级 | 独立意识、批判思维 |
| **青少年晚期** | 15-17岁 | 初中3年级-高中 | 接近成人能力、专业探索 |

### 1.2 安全相关能力发展

| 年龄段 | 精细动作 | 风险意识 | 自我保护 | 规则遵守 |
|--------|----------|----------|----------|----------|
| **5-6岁** | 发展中，需辅助 | 弱，需持续提醒 | 依赖成人 | 需反复强调 |
| **7-8岁** | 基本具备 | 初步形成 | 开始发展 | 能理解简单规则 |
| **9-10岁** | 较为成熟 | 逐渐增强 | 有一定能力 | 能自觉遵守 |
| **11-12岁** | 接近成人 | 较好 | 较强 | 能理解规则意义 |
| **13-14岁** | 成熟 | 良好 | 良好 | 能自主判断 |
| **15-17岁** | 成熟 | 成熟 | 成熟 | 能自我管理 |

## 2. 幼儿期 (5-6岁) 安全规范

### 2.1 发展特征

| 维度 | 特征 | 安全启示 |
|------|------|----------|
| **认知** | 具象思维为主，注意力短暂(15-20分钟) | 指令简单化、视觉化 |
| **动作** | 精细动作发展中，握力有限 | 大颗粒材料、圆角工具 |
| **社交** | 以自我为中心，分享意识弱 | 避免共用危险物品 |
| **情绪** | 情绪波动大，受挫易哭 | 高度关注，及时安抚 |

### 2.2 适宜活动

| 活动类型 | 具体活动 | 安全要点 |
|----------|----------|----------|
| **建构类** | 大颗粒积木、磁力片、纸板搭建 | 无小零件吞咽风险 |
| **艺术类** | 涂鸦、黏土、拼贴 | 无毒材料、易清洗 |
| **探索类** | 观察放大镜、磁铁实验 | 导师全程在场 |
| **编程启蒙** | 实物编程(如 Cubetto)、不插电活动 | 无屏幕时间限制 |

### 2.3 禁止接触

| 类别 | 具体事项 | 原因 |
|------|----------|------|
| **设备** | 所有电动工具、加热设备 | 无法安全操作 |
| **材料** | 小零件(&lt;3cm)、锋利材料、化学品 | 吞咽/伤害风险 |
| **区域** | 设备操作区、化学品存储区 | 物理隔离 |

### 2.4 监护要求

| 项目 | 要求 |
|------|------|
| **导师比例** | 1:3 (1名导师最多照看3名儿童) |
| **家长参与** | 鼓励家长陪同，尤其首次参与 |
| **空间安排** | 专用儿童区，与高风险区域物理隔离 |
| **时间安排** | 单次活动≤45分钟，含休息时间 |

### 2.5 安全教育方法

| 方法 | 示例 |
|------|------|
| **角色扮演** | "小猫要戴上眼睛保护罩才能看明亮的光" |
| **图示规则** | 用图画展示"可以"和"不可以" |
| **正向引导** | "我们用安全剪刀"而非"不许用那个剪刀" |
| **模仿学习** | 导师示范正确行为 |

## 3. 儿童早期 (7-8岁) 安全规范

### 3.1 发展特征

| 维度 | 特征 | 安全启示 |
|------|------|----------|
| **认知** | 逻辑思维萌芽，好奇心强 | 可解释简单因果关系 |
| **动作** | 精细动作改善，可使用简单工具 | 引入基础手工工具 |
| **社交** | 合作意识发展，能轮流等待 | 可小组活动 |
| **情绪** | 自尊心发展，害怕失败 | 强调过程而非结果 |

### 3.2 适宜活动

| 活动类型 | 具体活动 | 安全要点 |
|----------|----------|----------|
| **手工制作** | 剪纸、折纸、简单缝纫(塑料针) | 使用安全剪刀 |
| **电子入门** | 导电黏土电路、简单LED项目 | 低压电源(&lt;5V) |
| **编程基础** | Scratch Jr、图形化编程 | 屏幕时间限制 |
| **科学探索** | 简单物理实验、植物观察 | 无危险化学品 |

### 3.3 可引入设备

| 设备 | 使用条件 | 监护要求 |
|------|----------|----------|
| **安全剪刀** | 圆头、弹簧辅助 | 导师指导首次使用 |
| **热熔胶枪** | 低温款(&lt;120°C) | 导师操作，学员辅助 |
| **简单手工工具** | 尺子、铅笔、打孔器 | 教授正确使用方法 |

### 3.4 监护要求

| 项目 | 要求 |
|------|------|
| **导师比例** | 1:5 |
| **活动时长** | 60分钟，含休息 |
| **规则教育** | 活动开始前明确3-5条核心规则 |
| **求助机制** | 教会说"我需要帮助" |

## 4. 儿童中期 (9-10岁) 安全规范

### 4.1 发展特征

| 维度 | 特征 | 安全启示 |
|------|------|----------|
| **认知** | 逻辑思维发展，能理解因果链 | 可教授安全原理 |
| **动作** | 精细动作较成熟，手眼协调好 | 可使用更多工具 |
| **社交** | 团队协作能力强 | 可分工合作项目 |
| **责任** | 责任意识萌芽 | 可分配简单安全职责 |

### 4.2 适宜活动

| 活动类型 | 具体活动 | 安全要点 |
|----------|----------|----------|
| **电子制作** | Arduino基础、LED电路 | 低压电源、导师监督 |
| **3D设计** | TinkerCAD、3D打印(观察为主) | 不触碰运行中设备 |
| **木工入门** | 简单拼接、砂纸打磨 | 护目镜、导师在场 |
| **编程进阶** | Scratch、Python入门 | 适度屏幕时间 |

### 4.3 可引入设备

| 设备 | 使用条件 | 监护要求 |
|------|----------|----------|
| **电烙铁(低温)** | 经培训后 | 导师在场指导 |
| **3D打印机** | 设计软件自主，打印由导师操作 | 不触碰喷头 |
| **基础手工工具** | 锤子、螺丝刀、钳子 | 培训后导师在场 |
| **万用表** | 学习测量 | 低压电路使用 |

### 4.4 安全教育重点

| 主题 | 内容 |
|------|------|
| **工具安全** | 正确握持、使用后归位、损坏报告 |
| **电气安全** | 湿手不触电、拔插头而非拉线 |
| **化学安全** | 不尝、不闻、不混合 |
| **急救意识** | 知道去哪里找导师、急救箱在哪里 |

## 5. 儿童晚期 (11-12岁) 安全规范

### 5.1 发展特征

| 维度 | 特征 | 安全启示 |
|------|------|----------|
| **认知** | 抽象思维发展，能理解复杂系统 | 可教授系统性安全知识 |
| **判断** | 风险判断能力增强 | 可培养自我保护意识 |
| **自主** | 独立意识增强 | 可给予更多自主空间 |
| **责任** | 能承担一定责任 | 可担任"安全小助手" |

### 5.2 适宜活动

| 活动类型 | 具体活动 | 安全要点 |
|----------|----------|----------|
| **电子进阶** | 复杂电路、传感器项目 | 培训后可较独立操作 |
| **3D打印** | 设计到打印全流程(导师在场) | 了解高温风险 |
| **木工基础** | 手锯、钻孔(手电钻) | 护目镜、导师在场 |
| **机器人** | 简单机器人搭建与编程 | 机械安全意识 |

### 5.3 可引入设备

| 设备 | 使用条件 | 监护要求 |
|------|----------|----------|
| **3D打印机** | 培训认证后 | 导师在场，可自主操作 |
| **电烙铁** | 培训认证后 | 通风环境，导师在场 |
| **手电钻** | 培训后 | 护目镜，导师在场 |
| **小型台钻** | 演示观察为主 | 导师操作 |

### 5.4 渐进式放权

| 阶段 | 自主程度 | 导师角色 |
|------|----------|----------|
| **入门期** | 全程指导 | 手把手教学 |
| **成长期** | 部分自主 | 在场观察 |
| **成熟期** | 较多自主 | 随时可呼叫 |
| **进阶期** | 培训后自主 | 定期检查 |

## 6. 青少年早期 (13-14岁) 安全规范

### 6.1 发展特征

| 维度 | 特征 | 安全启示 |
|------|------|----------|
| **认知** | 批判性思维发展 | 可讨论安全原则而非仅规定 |
| **自我** | 自我意识强，追求独立 | 尊重自主性，给予信任 |
| **同伴** | 同伴影响大 | 建立安全文化，正向同伴压力 |
| **冒险** | 冒险行为可能增加 | 明确边界，后果教育 |

### 6.2 适宜活动

| 活动类型 | 具体活动 | 安全要点 |
|----------|----------|----------|
| **激光切割** | 培训认证后可操作 | 专项培训、防护措施 |
| **电子高阶** | SMD焊接、高级电路 | 烟雾处理、通风 |
| **木工进阶** | 电动工具入门 | 护目镜、手套、培训 |
| **机械设计** | 3D打印机械结构 | 机械风险意识 |

### 6.3 可引入设备

| 设备 | 使用条件 | 监护要求 |
|------|----------|----------|
| **激光切割机(小型)** | 专项培训认证 | 值班人员在场 |
| **热风枪** | 培训后 | 通风环境 |
| **角磨机(小型)** | 培训后 | 护目镜、手套、导师在场 |
| **化学蚀刻(简单)** | 专项培训 | 通风柜、导师指导 |

### 6.4 责任与自主

| 方面 | 做法 |
|------|------|
| **签署安全承诺书** | 本人签署，了解违规后果 |
| **自主规划项目** | 提交安全评估，导师审批 |
| **同伴监督** | 高风险操作时同伴在场 |
| **安全志愿者** | 可担任低年龄段活动的安全助理 |

## 7. 青少年晚期 (15-17岁) 安全规范

### 7.1 发展特征

| 维度 | 特征 | 安全启示 |
|------|------|----------|
| **能力** | 接近成人能力水平 | 可进行大多数操作 |
| **判断** | 较成熟的风险判断 | 可自主评估风险 |
| **专业** | 开始专业方向探索 | 可进行专业级培训 |
| **责任** | 具备完全责任意识 | 为自身行为负责 |

### 7.2 适宜活动

| 活动类型 | 具体活动 | 安全要点 |
|----------|----------|----------|
| **高级制造** | CNC入门(在监督下)、金属加工基础 | 专项培训、导师值守 |
| **电气高阶** | 高压电路(≤48V)、电机控制 | 专项培训、绝缘措施 |
| **化学实验** | 基础化学合成、PCB蚀刻 | 通风柜、防护装备 |
| **生物技术** | 基础分子生物学(如DNA提取) | 无害材料、规范操作 |

### 7.3 可引入设备

| 设备 | 使用条件 | 监护要求 |
|------|----------|----------|
| **CNC铣床** | 专项培训 + 导师值守 | 不可单独操作 |
| **大型激光切割机** | 专项认证 | 值守人员确认 |
| **化学实验设备** | 专项培训 | 导师指导 |
| **电焊设备(入门)** | 专项培训、18岁以下需家长同意 | 导师在场 |

### 7.4 预备成人管理

| 方面 | 做法 |
|------|------|
| **自主项目** | 可独立提出并执行复杂项目 |
| **风险自评** | 项目开始前提交风险评估 |
| **导师资格预备** | 可协助指导低年龄段学员 |
| **向成人过渡** | 18岁后可申请成人会员资格 |

## 8. 跨年龄段活动管理

### 8.1 混龄活动原则

| 原则 | 说明 |
|------|------|
| **就低原则** | 设备使用以最小年龄适用标准为准 |
| **分区活动** | 不同年龄段可在不同区域同时活动 |
| **大带小机制** | 高年龄段可协助照看低年龄段 |
| **差异化任务** | 同一项目中分配适龄任务 |

### 8.2 家庭活动安全

| 场景 | 管理要点 |
|------|----------|
| **亲子工作坊** | 家长与儿童配对，家长承担监护责任 |
| **开放日** | 家长陪同，导师增配 |
| **家庭项目** | 明确家长角色，提供安全指南 |

### 8.3 特殊需求适配

| 情况 | 适配措施 |
|------|----------|
| **身体障碍** | 个别化风险评估，必要时增加辅助 |
| **发展迟缓** | 按实际能力而非年龄分级 |
| **超常发展** | 可提前进入更高级别，但需评估 |
| **注意力困难** | 增加监护比例，缩短活动时长 |

## 9. 安全文化建设

### 9.1 年龄适宜的安全教育

| 年龄段 | 教育方式 | 内容重点 |
|--------|----------|----------|
| **5-8岁** | 游戏、故事、歌谣 | 基础规则、求助方法 |
| **9-12岁** | 案例讨论、动手演练 | 原理理解、技能培养 |
| **13-17岁** | 项目式学习、同伴讨论 | 责任意识、风险管理 |

### 9.2 正向安全文化

| 做法 | 说明 |
|------|------|
| **表扬安全行为** | "我看到你主动戴上了护目镜，做得好！" |
| **分享安全故事** | 讨论"差一点"的经历，不追究责任 |
| **安全小明星** | 表彰安全行为榜样 |
| **学员参与制定规则** | 高年龄段可参与安全规则讨论 |
