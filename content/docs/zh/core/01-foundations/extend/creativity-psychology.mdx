# 创造力心理学

## 概述

创造力是人类最珍贵的能力之一，是创客教育的核心目标。本文档综述创造力的心理学研究，帮助教育者理解创造力的本质，并提供培养创造力的科学方法。

**核心观点**:
1. 创造力不是少数人的天赋，而是人人具有的潜能
2. 创造力可以通过适当的环境和方法来培养
3. 创造力是知识、技能、动机和环境的综合产物
4. 小c（日常创造力）和大C（杰出创造力）同样重要

---

## 1. 创造力的定义与内涵

### 1.1 标准定义

创造力通常被定义为产生既**新颖**（original）又**有价值**（useful）的想法或产品的能力。

```
创造力的双重标准:
┌─────────────────────────────────────────┐
│                                          │
│        新颖性 (Originality)              │
│           ↑                              │
│           │                              │
│     ┌─────┼─────┐                       │
│     │  创 │ 造  │                       │
│     │  造 │ 力  │                       │
│     │  力 │     │                       │
│     └─────┼─────┘                       │
│           │                              │
│           └──────────→ 有用性 (Value)    │
│                                          │
│  仅新颖 = 古怪   仅有用 = 常规           │
│  既新颖又有用 = 创造性                   │
│                                          │
└─────────────────────────────────────────┘
```

### 1.2 创造力的层次

Kaufman & Beghetto 的 Four C 模型：

| 层次 | 定义 | 示例 |
|------|------|------|
| **mini-c** | 个人主观的新发现 | 学生第一次理解一个概念 |
| **little-c** | 日常生活中的创造 | 改进家中的收纳方法 |
| **Pro-c** | 专业领域的创新 | 专业设计师的作品 |
| **Big-C** | 改变领域的杰出贡献 | 爱因斯坦的相对论 |

**教育启示**: 教育应关注 mini-c 和 little-c，为未来的 Pro-c 和 Big-C 奠基。

### 1.3 创造力的类型

| 类型 | 特点 | 领域示例 |
|------|------|---------|
| **艺术创造力** | 表达性、审美性 | 绘画、音乐、文学 |
| **科学创造力** | 发现性、解释性 | 理论、发现、发明 |
| **技术创造力** | 实用性、功能性 | 产品、工艺、系统 |
| **社会创造力** | 关系性、组织性 | 制度、模式、方法 |

## 2. 创造力的心理学理论

### 2.1 Guilford 的发散思维理论

J.P. Guilford 区分了两种思维模式：

| 思维类型 | 特点 | 在创造中的作用 |
|---------|------|---------------|
| **聚合思维** | 寻找唯一正确答案 | 评估和选择方案 |
| **发散思维** | 产生多种可能答案 | 产生创意和选项 |

**发散思维的四个维度**:

| 维度 | 定义 | 测量方式 |
|------|------|---------|
| **流畅性** | 产生想法的数量 | 数量计分 |
| **灵活性** | 想法类别的多样性 | 类别数计分 |
| **独创性** | 想法的独特程度 | 稀有度计分 |
| **精细性** | 想法的详细程度 | 细节计分 |

### 2.2 Amabile 的创造力成分模型

Teresa Amabile 提出创造力由三个成分决定：

```
           ┌─────────────────────────────────┐
           │          创造力                  │
           │        (Creativity)              │
           └───────────────┬─────────────────┘
                           │
           ┌───────────────┼───────────────┐
           ↓               ↓               ↓
    ┌───────────┐   ┌───────────┐   ┌───────────┐
    │  领域技能  │   │ 创造力技能 │   │  任务动机  │
    │  Domain   │   │ Creativity│   │   Task    │
    │  Skills   │   │  Skills   │   │Motivation │
    └───────────┘   └───────────┘   └───────────┘
         │               │               │
         ↓               ↓               ↓
    知识和技术专长   认知风格和策略    内在动机最重要
    (可以学习)      (可以培养)       (环境影响大)
```

**关键洞察**: 内在动机是创造力最重要的驱动力，外在奖励可能损害创造力。

### 2.3 Csikszentmihalyi 的系统模型

Mihaly Csikszentmihalyi 强调创造力是个人、领域和场域互动的结果：

```
创造力的系统观:
                 ┌─────────────┐
                 │    文化     │
                 │  (Domain)   │
                 │  符号规则    │
                 └──────┬──────┘
                        │
        传递知识 ↙      │      ↘ 选择创新
                       │
    ┌───────────┐      │      ┌───────────┐
    │   个人     │←─────┼──────│   场域    │
    │ (Person)  │      │      │  (Field)  │
    │  创造者   │      │      │  守门人   │
    └───────────┘      │      └───────────┘
            │          │
            └──产出创新─┘
```

**教育启示**: 创造力培养需要关注知识传授、技能培养和社会认可系统。

### 2.4 投资理论

Sternberg & Lubart 将创造力比作投资：

> "创造性的人像精明的投资者一样，'低买高卖'——他们追求那些新颖、可能不受欢迎但有潜力的想法。"

创造力需要六种资源的汇合：
1. 智力
2. 知识
3. 思维风格
4. 人格
5. 动机
6. 环境

## 3. 创造力的影响因素

### 3.1 个人因素

| 因素 | 与创造力的关系 |
|------|---------------|
| **智力** | 中等正相关，存在阈值效应（IQ约120以上，相关减弱） |
| **知识** | 必要但非充分；过多知识可能导致功能固着 |
| **人格** | 开放性、好奇心、冒险性与创造力正相关 |
| **动机** | 内在动机是创造力的强预测因素 |
| **认知风格** | 偏好复杂性、容忍模糊性有利于创造 |

### 3.2 环境因素

**有利于创造力的环境**:

```
创造力友好环境的特征:
┌─────────────────────────────────────────┐
│  心理安全                                │
│  ├── 允许冒险和失败                      │
│  ├── 不因"愚蠢的想法"受到嘲笑           │
│  └── 鼓励表达不同意见                    │
├─────────────────────────────────────────┤
│  自主性                                  │
│  ├── 有选择做什么和怎么做的自由          │
│  ├── 减少不必要的限制                    │
│  └── 支持内在动机                        │
├─────────────────────────────────────────┤
│  资源支持                                │
│  ├── 充足的时间和空间                    │
│  ├── 必要的材料和工具                    │
│  └── 信息和专业知识的获取                │
├─────────────────────────────────────────┤
│  激励认可                                │
│  ├── 过程性的鼓励而非仅结果              │
│  ├── 建设性的反馈                        │
│  └── 对创新尝试的认可                    │
└─────────────────────────────────────────┘
```

**扼杀创造力的因素**:
- 过度的评价压力
- 过多的外部奖励
- 严格的时间限制
- 过度竞争
- 严苛的监督
- 拒绝新想法的文化

### 3.3 动机的关键作用

| 动机类型 | 对创造力的影响 |
|---------|---------------|
| **内在动机** | 强烈正向：为兴趣和挑战而做 |
| **协同外在动机** | 可以支持：提供信息和认可 |
| **控制外在动机** | 可能负向：为奖惩而做 |

**培养内在动机的方法**:
1. 提供选择和自主权
2. 匹配挑战与技能（心流状态）
3. 减少不必要的评价
4. 关注过程而非仅结果
5. 激发好奇心和兴趣

## 4. 创造力的测量

### 4.1 常用测量方法

| 方法 | 代表工具 | 测量内容 |
|------|---------|---------|
| **发散思维测试** | Torrance Tests of Creative Thinking | 流畅性、灵活性、独创性 |
| **远距离联想** | Remote Associates Test | 概念间的联想能力 |
| **创造性产品评估** | Consensual Assessment Technique | 专家评价产品创造性 |
| **创造性人格量表** | Creative Personality Scale | 创造性倾向和特质 |
| **创造性行为清单** | Creative Behavior Inventory | 日常创造性行为 |

### 4.2 Torrance 创造力测验

最广泛使用的创造力测试：

**言语测验示例**:
- 不寻常用途：列出砖块的尽可能多的用途
- 改进产品：如何改进一个玩具使其更有趣

**图形测验示例**:
- 不完整图形：完成并命名图形
- 平行线：用平行线创作尽可能多的图画

**评分维度**:
| 维度 | 定义 | 计分方式 |
|------|------|---------|
| 流畅性 | 有效回答的数量 | 计数 |
| 灵活性 | 不同类别的数量 | 类别计数 |
| 独创性 | 回答的稀有程度 | 统计稀有度 |
| 精细性 | 细节的丰富程度 | 细节计数 |

## 5. 创造力培养策略

### 5.1 创客教育中的创造力培养

| 创客活动特点 | 创造力培养机制 |
|-------------|---------------|
| **开放式问题** | 激发发散思维 |
| **动手制作** | 具身认知，外化想法 |
| **迭代改进** | 容忍失败，持续优化 |
| **跨学科** | 促进远距离联想 |
| **分享展示** | 获得反馈，社会认可 |

### 5.2 具体教学策略

**头脑风暴优化版**:

```
有效头脑风暴规则:
┌─────────────────────────────────────────┐
│  1. 延迟判断 - 先产生，后评估          │
│  2. 追求数量 - 数量带来质量            │
│  3. 欢迎疯狂 - 越奇特越好              │
│  4. 组合改进 - 借鉴他人想法            │
├─────────────────────────────────────────┤
│  增强策略:                              │
│  • 先独立思考再分享（避免群体思维）    │
│  • 使用 SCAMPER 等提示工具              │
│  • 强制连接不相关概念                   │
│  • 设置数量目标（如50个想法）           │
└─────────────────────────────────────────┘
```

**SCAMPER 创意工具**:

| 字母 | 策略 | 提问 |
|------|------|------|
| S | Substitute 替代 | 可以用什么代替？ |
| C | Combine 组合 | 可以和什么结合？ |
| A | Adapt 调整 | 可以如何调整？ |
| M | Modify/Magnify 修改/放大 | 可以改变什么？放大什么？ |
| P | Put to other uses 另作他用 | 可以有其他用途吗？ |
| E | Eliminate 消除 | 可以去掉什么？ |
| R | Reverse/Rearrange 颠倒/重排 | 可以颠倒或重新排列吗？ |

**强制联想**:
- 随机词联想：随机选词与问题联系
- 图片刺激：用随机图片激发想法
- 类比思维：从其他领域借鉴解决方案

### 5.3 培养创造力倾向

| 倾向 | 培养方法 |
|------|---------|
| **好奇心** | 鼓励提问，展示未知的魅力 |
| **开放性** | 接触多元观点，质疑假设 |
| **冒险性** | 安全地尝试新事物，庆祝"有益的失败" |
| **想象力** | "如果...会怎样"练习 |
| **坚持性** | 展示创造过程的艰辛和价值 |

## 6. 创造力发展的年龄特点

### 6.1 发展阶段

| 年龄 | 创造力特点 | 教育重点 |
|------|-----------|---------|
| **幼儿期** | 想象力丰富，无拘束 | 保护想象力，不过度纠正 |
| **小学低** | 创意峰值（"四年级下滑"前） | 鼓励表达，提供多种媒介 |
| **小学高** | 开始受社会规范影响 | 平衡规范与创意 |
| **初中** | 自我意识强，怕被评判 | 创造安全环境，同伴支持 |
| **高中** | 可进行复杂创意工作 | 深度项目，领域专精 |

### 6.2 四年级下滑

研究发现创造力在小学四年级左右可能下降：

**可能原因**:
- 学校更强调"正确答案"
- 社会化压力增加
- 对失败的恐惧加深
- 批判性思维发展，自我审查增加

**应对策略**:
- 维持创意活动时间
- 明确创造力的价值
- 减少过度评价
- 培养创造力身份认同

## 常见问题

<details>
<summary>创造力是天生的还是可以培养的？</summary>

研究表明创造力既有遗传成分，也可以通过培养提高。遗传影响约25-40%，环境和教育同样重要。关键是提供适当的环境、培养必要的技能和态度。
</details>

<details>
<summary>外在奖励一定会损害创造力吗？</summary>

不一定。Amabile的研究区分了两种外在动机：
- **控制性**外在动机（为获得奖励/避免惩罚）可能损害创造力
- **信息性**外在动机（提供能力反馈、认可成就）可以与内在动机协同

关键是奖励的设计和传达方式。
</details>
