## Core 论断

M03 的全部空间论述可压缩为五条第一性论断。任何选址、装修、家具采购或活动布展若违背其中之一，必须显式给出理由。

1. **空间是没有声音的教师。** 它通过可供性默默规训行为；改变行为最低成本的杠杆，往往不是规则而是空间。
2. **创造性需要不同状态的空间。** 准备需要专注、孵化需要游荡、启迪需要触发、验证需要展示——单一开放或单一封闭都不够。
3. **物理 / 社会 / 数字三层必须协同设计。** 只盯物理空间的项目最终会被数字工具或社群文化反噬。
4. **第三空间的舒适来自心理所有权。** 学习者参与空间命名、改造、布置，比家具品质更能决定他们是否真正"住进来"。
5. **可持续与可塑性同等重要。** 空间必须能被一代又一代学习者重新定义，而非一次性装修后冻结。

## 关键证据

近三年的神经建筑学综述、超混合空间研究与教学法-空间耦合工具把 M03 论断从经验提升到证据：

**学习空间应承载多种主动教学法的可供性。** Orihuela 在 *Discover Education* 提出可操作的工坊空间设计工具，把空间布局与 Reggio Emilia 的 atelier、Montessori 的整合教室、Waldorf 的工作坊三种活动教学法显式对齐，并把"安静室、原型室、展示室"作为创客生态的标准角色<Cite id="M03-2025-DiscEd-Active" />。这把"功能区怎么分"从直觉问题变成有据可循的设计选择。

**创造性可被翻译为空间状态。** Mosquera-González 的案例研究把一个 65 m² 工作空间组织为 Wallas 创造性四阶段（准备 / 孵化 / 启迪 / 验证）的四区，并对接 DMN（默认网络）/ECN（执行控制网络）/突显网络的状态调节，把光照、声学、生物嵌入、多感官输入与灵活布置作为可验证的设计杠杆<Cite id="M03-2025-CCD-Creativity" />。OWL 空间分区可直接借鉴这套映射。

**神经建筑学为学校设计提供可操作原则。** A4LE 综述指出灵活学习空间应支持独立学习、同侪辅导、项目制等多模式；biophilic design 与小型学习社区显著影响学业表现与情绪福祉；"sense of place"（场所感）作为身份与归属的设计要素须被显式建模<Cite id="M03-2023-A4LE-Neuroarch" />。

**超混合空间是物理 + 数字的共生而非叠加。** Bezegová 等基于 Collabits 的实证研究表明，一个真正的"超混合空间"应能让师生在「呈现 / 体验 / 重塑」三态间切换，让数字工具承载意义生成而不只是内容传递<Cite id="M03-2025-SLE-HyperHybrid" />。这为 M03 的"数字空间层"提供了直接可施工的范例。

**经典证据继续支撑分级与可塑性。** Doorley 与 Witthoft 的 d.school 空间设计原则<Cite id="M03-001" />、Boys 关于学习空间架构的论述<Cite id="M03-002" />、Mehta 等关于中等噪音水平最有利于创造性思维的实验<Cite id="M03-004" />、CAST 的 UDL 通用学习设计指南<Cite id="M03-008" />为 OWL 的可塑性、噪音控制与无障碍设计提供基础。
