# 数字环境标准

## 概述

> 数字环境是物理空间的延伸和放大器。好的数字环境让协作跨越时空，让学习无处不在。

创客空间的数字层不仅仅是"有网络"那么简单。它决定了：
- 学习者能否流畅地下载设计文件和上传作品
- 远程参与者能否与现场用户无缝协作
- 设备能否智能化管理，降低运营负担
- 知识能否有效积累和传承

**核心原则**：
1. **稳定优先**：网络稳定性比峰值速度更重要
2. **简单易用**：复杂的数字工具会成为学习障碍
3. **物理融合**：数字工具应增强而非替代物理交互
4. **开放标准**：优先选择开源和可迁移的方案

## 1. 网络基础设施

### 1.1 带宽与覆盖标准

| 项目 | Lite级 | Pro级 | Ultra级 | 说明 |
|------|--------|-------|---------|------|
| **下行带宽** | ≥100Mbps | ≥500Mbps | ≥1Gbps | 共享带宽 |
| **上行带宽** | ≥50Mbps | ≥200Mbps | ≥500Mbps | 上传作品需求 |
| **WiFi标准** | WiFi 5 | WiFi 6 | WiFi 6E | 并发能力差异大 |
| **覆盖率** | 100% | 100% | 100% | 无死角 |
| **并发设备** | 30台 | 100台 | 200+台 | 人均2-3设备计算 |
| **有线网口** | 2个 | 8个 | 按需 | 固定工作站使用 |
| **备份网络** | 可选 | 推荐 | 必须 | 4G/5G备份 |

### 1.3 设备配置清单

| 设备类型 | Lite配置 | Pro配置 | 参考价格 |
|----------|----------|---------|----------|
| **路由器** | 家用千兆 | 企业级路由 | 300-3000元 |
| **无线AP** | 1个双频AP | 2-3个企业AP | 200-1500元/个 |
| **交换机** | 8口千兆 | 24口PoE | 100-2000元 |
| **网线** | Cat5e | Cat6 | 2-5元/米 |
| **UPS** | 可选 | 推荐 | 500-2000元 |

### 1.4 常见问题排查

| 问题现象 | 可能原因 | 解决方案 |
|----------|----------|----------|
| 网速慢 | 带宽不足/信道拥挤 | 升级带宽/调整信道 |
| 信号弱 | AP覆盖不足/障碍物 | 增加AP/调整位置 |
| 频繁断线 | 设备过热/固件问题 | 散热/升级固件 |
| 某区域无信号 | 死角 | 增加AP或信号放大器 |
| 有线设备无法上网 | IP冲突/网线问题 | 检查DHCP/更换网线 |

## 3. 智能化设备 (IoT)

### 3.1 智能化分级

| 级别 | 设备类型 | 功能 | 投资 | 适用场景 |
|------|----------|------|------|----------|
| **基础级** | 智能插座 | 远程控制、用电统计 | \<500元 | 所有空间 |
| **标准级** | 智能门禁+环境监测 | 预约核验、空气质量 | 3000-8000元 | Pro级空间 |
| **高级** | 全屋智能 | 场景联动、自动化 | 1-3万元 | Ultra级空间 |

### 3.2 核心智能设备

| 设备 | 功能 | 配置建议 | 参考价格 |
|------|------|----------|----------|
| **智能门禁** | 预约核验、使用统计、24h无人值守 | Pro+必备 | 2000-5000元 |
| **环境监测** | 温湿度、PM2.5、CO2、VOC | 推荐 | 500-1500元 |
| **智能照明** | 场景切换、节能、调色温 | 推荐 | 50-200元/灯 |
| **智能插座** | 远程控制、定时、用电监测 | 推荐 | 50-100元/个 |
| **网络摄像头** | 远程监控、安全 | 可选 | 200-500元 |
| **烟雾/燃气** | 安全报警、自动断电 | 必须 | 100-300元 |

### 3.4 实施建议

| 原则 | 说明 |
|------|------|
| **价值优先** | 优先部署能显著降低人力或提升安全的设备 |
| **简单可靠** | 避免过度复杂的自动化，保留手动控制 |
| **隐私保护** | 摄像头避开敏感区域，数据本地存储 |
| **备用方案** | 智能设备故障时有手动操作方式 |
| **渐进部署** | 从核心设备开始，逐步扩展 |

## 4. 数据管理与安全

### 4.1 数据备份策略

| 数据类型 | 备份频率 | 存储位置 | 保留期限 |
|----------|----------|----------|----------|
| **用户作品** | 实时同步 | 云端+本地 | 永久 |
| **项目文档** | 每日 | 云端 | 永久 |
| **设备配置** | 每周 | 本地 | 1年 |
| **监控录像** | 循环覆盖 | 本地NAS | 30天 |
| **使用数据** | 每日 | 云端 | 1年 |

### 4.2 网络安全要点

| 安全措施 | 实施方法 | 优先级 |
|----------|----------|--------|
| **WiFi加密** | WPA3或WPA2-PSK | 必须 |
| **访客隔离** | 独立SSID和VLAN | 推荐 |
| **固件更新** | 定期检查路由器/AP固件 | 推荐 |
| **弱密码** | 定期更换默认密码 | 必须 |
| **防火墙** | 开启路由器防火墙 | 推荐 |
| **IoT隔离** | 智能设备独立网段 | 推荐 |

## 6. 常见问题

<details>
<summary>小型空间需要企业级网络设备吗？</summary>

**不一定**。关键看并发设备数量：

- **\<20台设备**：家用路由器通常够用，选择口碑好的品牌
- **20-50台设备**：考虑双频AP或Mesh路由器
- **>50台设备**：建议企业级AP

重点是**稳定性**而非峰值速度。便宜但稳定的设备优于昂贵但复杂的方案。
</details>

<details>
<summary>如何选择项目管理工具？</summary>

**根据团队习惯选择**，工具本身不是关键，坚持使用才重要：

1. **已有飞书/钉钉**：直接使用其内置文档和任务功能
2. **偏好简洁**：Notion、语雀
3. **需要开发协作**：GitHub Projects
4. **预算有限**：腾讯文档、石墨文档

建议**先试用再决定**，不要频繁切换工具。
</details>

<details>
<summary>智能化值得投资吗？</summary>

**看具体场景**。ROI最高的智能化投资：

1. **智能门禁**（节省人力值守，24h开放） ⭐⭐⭐⭐⭐
2. **智能插座**（用电监测，远程关闭遗忘设备） ⭐⭐⭐⭐
3. **环境监测**（安全预警，用户体验） ⭐⭐⭐

**低优先级**：智能窗帘、智能音箱等（锦上添花）

原则：**先解决痛点，再追求体验**。
</details>
