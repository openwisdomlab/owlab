# 每日关门 SOP

**SOP编号**: SOP-M08-002
**版本**: 0.1.0
**执行时间**: 结束营业前30分钟
**执行人**: 当日值班人员

---

## 流程概览

```
预告结束 → 人员清场 → 设备关闭 → 安全检查 → 锁门离开
(关门前30分) (15分)    (10分)      (5分)      (0分)
```

---

## 1. 结束预告 (关门前30-15分钟)

### 1.1 时间提醒

| 时间点 | 提醒内容 | 方式 |
|--------|----------|------|
| 关门前30分钟 | "提醒大家，我们将在30分钟后结束今天的开放" | 口头/广播 |
| 关门前15分钟 | "我们将在15分钟后关门，请开始收尾工作" | 口头/广播 |
| 关门前5分钟 | "我们即将关门，请确保个人物品" | 口头/广播 |

### 1.2 活动结束

- [ ] 正在进行的活动提前告知结束时间
- [ ] 使用中的设备提醒保存工作
- [ ] 未完成项目协助安全存放

---

## 2. 人员清场 (关门前15-10分钟)

### 2.1 引导离场

| 区域 | 检查动作 |
|------|----------|
| 工作区 | 确认所有人员已离开 |
| 制作区 | 确认设备已停用，人员已离开 |
| 休息区 | 确认人员已离开 |
| 卫生间 | 确认无人 |
| 储物间 | 确认已锁好 |

### 2.2 物品确认

- [ ] 提醒会员带走个人物品
- [ ] 遗留物品登记并妥善保管
- [ ] 借用物品已归还

**遗留物品处理**:
```
发现遗留物品 → 拍照记录 → 放入失物招领 → 记录日志 → 通知失主（如可识别）
```

---

## 3. 区域收尾 (关门前10-5分钟)

### 3.1 工作区收尾

| 项目 | 操作 | 检查 |
|------|------|------|
| 桌面 | 清理、擦拭 | □ 完成 |
| 椅子 | 归位 | □ 完成 |
| 工具 | 归位、清点 | □ 完成 |
| 材料 | 归位、整理 | □ 完成 |
| 垃圾 | 清理至指定位置 | □ 完成 |

### 3.2 设备关闭

**关闭顺序** (与开机相反):

```
1. 专业设备 → 2. 功能设备 → 3. 环境设备 → 4. 基础设施
```

| 类型 | 设备 | 操作 | 检查 |
|------|------|------|------|
| 专业设备 | 3D打印机、激光切割等 | 按规程关闭 | □ 完成 |
| 功能设备 | 电脑、投影、音响 | 正常关机 | □ 完成 |
| 环境设备 | 空调、新风 | 关闭 | □ 完成 |
| 照明 | 区域灯光 | 逐区关闭 | □ 完成 |

### 3.3 高风险设备

> **重要**: 高风险设备必须确认完全关闭

| 设备 | 检查项 |
|------|--------|
| 激光切割机 | □ 已关机 □ 已断电 □ 已上锁 |
| 3D打印机 | □ 已关机 □ 无进行中任务 |
| 电烙铁/热风枪 | □ 已关闭 □ 已冷却 □ 已归位 |
| 电动工具 | □ 已关闭 □ 电池取出（便携式） |

### 3.4 水电检查

| 项目 | 操作 | 检查 |
|------|------|------|
| 饮水机 | 按需关闭 | □ 完成 |
| 热水器 | 按需关闭 | □ 完成 |
| 水龙头 | 确认关闭 | □ 完成 |
| 非必要电源 | 断开 | □ 完成 |

---

## 4. 安全检查 (关门前5-2分钟)

### 4.1 消防安全

| 检查项 | 标准 | 结果 |
|--------|------|------|
| 无明火 | 全区无明火 | □ 正常 |
| 电器断电 | 非必要电器已断电 | □ 正常 |
| 灭火器 | 在位 | □ 正常 |
| 通道 | 畅通 | □ 正常 |

### 4.2 门窗检查

| 区域 | 门窗 | 状态 |
|------|------|------|
| 主入口 | 大门 | □ 待锁 |
| 后门 | 安全门 | □ 已锁 |
| 窗户 | 所有窗户 | □ 已关 |
| 阳台（如有） | 阳台门 | □ 已锁 |

### 4.3 最终巡视

按路线完成最终巡视：

```
入口 → 工作区 → 制作区 → 休息区 → 卫生间 → 储物间 → 返回入口
```

- [ ] 确认所有区域无人
- [ ] 确认无遗留危险（明火、未关设备）
- [ ] 确认贵重物品安全

---

## 5. 锁门离开 (关门前2-0分钟)

### 5.1 最后操作

1. 关闭剩余照明（保留必要夜灯/安全灯）
2. 开启报警系统（如有）
3. 锁好大门
4. 确认门锁状态

### 5.2 离开检查

- [ ] 门已锁好（拉门确认）
- [ ] 报警系统已启动（如有）
- [ ] 随身携带所有钥匙

## 特殊情况处理

### 情况1: 会员不愿离开

```
1. 礼貌提醒关门时间
2. 了解原因，提供帮助
3. 协助完成紧急事项
4. 仍不配合 → 联系负责人
```

### 情况2: 发现设备故障

```
1. 记录故障详情
2. 拍照留证
3. 确保设备安全（断电等）
4. 记录在日志
5. 通知维修/负责人
```

### 情况3: 最后时刻有人来访

```
营业时间已结束:
- 礼貌说明
- 告知明日营业时间
- 如紧急情况，酌情处理
```

### 情况4: 停电

```
1. 使用手电筒
2. 确认人员安全疏散
3. 检查关键设备（冰箱等）
4. 锁门时特别注意安全
5. 记录停电情况
```
