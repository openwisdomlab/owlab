# 器材与耗材管理

> **一句话定位**: 让每一件器材都在最需要的时候可用

---

## 概述

创新学习空间的器材和耗材管理是运营的基础工作。良好的物资管理不仅能保障教学活动的顺利开展，还能延长设备寿命、控制运营成本、保障使用安全。本文档从器材分类、采购、库存、维护、报废等环节提供系统的管理方法。

**核心观点**:
1. 器材管理的目标是"高可用性"——需要时能用、能用好
2. 耗材管理的核心是"不断不囤"——既不因缺料中断活动，也不过度积压资金
3. 数字化工具可以大幅提升管理效率，但不能替代定期的实物盘点

---

## 1. 物资分类体系

### 1.1 按性质分类

| 类别 | 定义 | 示例 | 管理重点 |
|------|------|------|----------|
| **固定设备** | 单价高、使用寿命长 | 3D打印机、激光切割机、CNC | 资产登记、定期维护 |
| **工具器材** | 反复使用的手工具 | 电烙铁、万用表、螺丝刀套装 | 借还管理、损耗补充 |
| **电子元器件** | 可回收复用的元件 | Arduino板、传感器模块、电机 | 借用追踪、损坏判定 |
| **耗材** | 一次性使用后消耗 | 3D打印线材、焊锡、胶水、木板 | 库存预警、定期采购 |
| **安全防护** | 个人防护和安全设备 | 护目镜、手套、灭火器 | 定期检查、及时更换 |

### 1.2 按管理级别分类

| 级别 | 标准 | 管理要求 | 示例 |
|------|------|----------|------|
| **A级（高值）** | 单价 ≥5000元 | 资产编号、使用日志、专人负责 | 激光切割机、3D打印机 |
| **B级（中值）** | 单价 500-5000元 | 资产编号、定期盘点 | 电烙铁、示波器 |
| **C级（低值）** | 单价 &lt;500元 | 批量管理、定期补充 | 手工具、元器件 |
| **D级（耗材）** | 消耗性物资 | 库存预警、定期采购 | 线材、焊锡、胶水 |

## 2. 采购管理

### 2.1 采购决策框架

```text
需求提出 → 评估审批 → 选型比价 → 下单采购 → 验收入库
    │           │           │           │           │
  使用者     运营主管     技术评估     财务审批     质量检查
```

### 2.2 选型评估维度

| 维度 | 权重 | 说明 |
|------|------|------|
| **教学适用性** | 30% | 是否适合目标年龄段和课程需求 |
| **安全性** | 25% | 是否符合安全标准（参见 [M06 安全舱](/docs/core/06-safety)） |
| **维护成本** | 20% | 耗材成本、维修便利性、配件可得性 |
| **耐用性** | 15% | 在高频使用场景下的使用寿命 |
| **扩展性** | 10% | 与现有设备和课程体系的兼容性 |

### 2.3 供应商管理

| 管理内容 | 具体措施 |
|----------|----------|
| 供应商档案 | 建立合格供应商名录，记录联系方式、产品范围、报价 |
| 定期评估 | 每年评估供应商的价格、质量、交付及时率 |
| 备选供应商 | 关键物资至少有2家备选供应商 |
| 长期协议 | 高频采购的耗材签订年度框架协议，争取更优价格 |

## 3. 入库与编码

### 3.1 资产编码规则

建议采用分层编码方式：

```text
编码结构: [类别]-[区域]-[序号]

示例:
  FIX-A01-001  → 固定设备-A区-001号（3D打印机）
  TOL-B02-015  → 工具器材-B区-015号（电烙铁）
  COM-C01-100  → 电子元器件-C区-100号（Arduino Uno）
  CON-D01-050  → 耗材-D区-050号（PLA线材白色）

类别代码:
  FIX = 固定设备 (Fixed Equipment)
  TOL = 工具器材 (Tools)
  COM = 电子元器件 (Components)
  CON = 耗材 (Consumables)
  SAF = 安全防护 (Safety)
```

### 3.2 入库流程

| 步骤 | 操作 | 负责人 |
|------|------|--------|
| 1. 到货验收 | 核对品名、数量、外观，拍照记录 | 仓管/采购 |
| 2. 质量检查 | 通电测试/功能测试 | 技术人员 |
| 3. 资产登记 | 录入管理系统，粘贴标签 | 仓管 |
| 4. 上架归位 | 放入指定存储位置 | 仓管 |
| 5. 信息更新 | 更新库存台账和位置地图 | 仓管 |

### 3.3 标签与标识

每件A/B级资产应粘贴包含以下信息的标签：

| 标签内容 | 说明 |
|----------|------|
| 资产编号 | 唯一识别码 |
| 设备名称 | 易读名称 |
| 入库日期 | 便于计算使用年限 |
| 负责人 | 主要维护责任人 |
| 使用说明二维码 | 扫码查看操作指南和安全须知 |

## 4. 借用与归还

### 4.1 借用管理规则

| 物资类型 | 借用方式 | 归还要求 | 损坏处理 |
|----------|----------|----------|----------|
| **A级设备** | 预约使用，不可外借 | 使用后登记状态 | 报修，评估修复成本 |
| **B级工具** | 登记借用 | 当日归还，检查完好 | 评估是否人为损坏 |
| **C级元器件** | 登记借用 | 项目结束归还 | 根据损耗情况判定 |
| **D级耗材** | 按需领用 | 不需归还 | 记录用量 |

### 4.2 数字化借还流程

```text
借用流程:
  学习者扫码 → 选择物品 → 确认借用 → 系统记录 → 超时提醒

归还流程:
  学习者扫码 → 自检状态 → 确认归还 → 系统更新 → 异常标记
```

## 5. 库存与耗材管理

### 5.1 耗材安全库存计算

| 参数 | 计算方法 | 说明 |
|------|----------|------|
| **日均消耗量** | 月消耗量 / 30 | 取近3个月平均值 |
| **采购周期** | 下单到到货天数 | 含物流和检验时间 |
| **安全库存** | 日均消耗量 x 采购周期 x 1.5 | 1.5为安全系数 |
| **补货点** | 安全库存 + 日均消耗量 x 采购周期 | 触发采购的库存量 |

### 5.2 常用耗材管理参考

| 耗材类型 | 月均消耗（参考） | 安全库存 | 存储要点 |
|----------|-----------------|----------|----------|
| PLA线材 | 5-10卷 | 10卷 | 防潮密封 |
| 焊锡丝 | 2-3卷 | 4卷 | 常温存放 |
| 热熔胶棒 | 2-3包 | 4包 | 避免高温 |
| 杜邦线 | 3-5包 | 6包 | 防静电 |
| 面包板 | 5-10块 | 10块 | 防潮 |
| A4纸 | 2-3包 | 4包 | 防潮 |
| 电池 | 视使用频率 | 2倍月用量 | 阴凉处 |

### 5.3 盘点制度

| 盘点类型 | 频率 | 范围 | 方法 |
|----------|------|------|------|
| **日常抽盘** | 每周 | 随机抽取10%的SKU | 实物计数 vs 系统数据 |
| **月度盘点** | 每月 | 耗材和C级物资 | 全面计数，更新台账 |
| **季度盘点** | 每季 | 所有物资 | 全面盘点，生成报告 |
| **年度盘点** | 每年 | 所有物资（含资产审计） | 详细盘点，报废处理 |

## 6. 维护与保养

### 6.1 设备维护等级

| 等级 | 内容 | 频率 | 执行人 |
|------|------|------|--------|
| **日常维护** | 清洁、目视检查、基本功能测试 | 每日/每次使用后 | 使用者/志愿者 |
| **定期保养** | 润滑、校准、零部件检查 | 每周/每月 | 技术人员 |
| **深度维护** | 拆机清洁、关键部件更换 | 每季/半年 | 专业技术人员 |
| **年度检修** | 全面检查和性能测试 | 每年 | 厂家/专业服务商 |

### 6.2 常见设备维护清单

**3D打印机（每周维护）**:
- [ ] 清洁打印平台
- [ ] 检查喷头是否堵塞
- [ ] 润滑导轨
- [ ] 检查线材余量和状态
- [ ] 校准打印平台水平

**激光切割机（每周维护）**:
- [ ] 清洁透镜和反射镜
- [ ] 清理切割残渣
- [ ] 检查排烟系统
- [ ] 检查冷却水位
- [ ] 测试紧急停止按钮

### 6.3 维修记录管理

每次维修应记录以下信息：

| 记录项 | 说明 |
|--------|------|
| 设备编号 | 资产编码 |
| 故障描述 | 故障现象和发现时间 |
| 原因分析 | 初步判断的故障原因 |
| 维修内容 | 具体操作和更换的零部件 |
| 维修成本 | 零部件和人工费用 |
| 维修结果 | 是否修复、残留问题 |
| 预防措施 | 避免同类故障的建议 |

## 7. 报废与处置

### 7.1 报废标准

| 条件 | 说明 |
|------|------|
| 维修成本 > 购置成本50% | 修不如换 |
| 安全隐患无法排除 | 安全第一 |
| 技术过时影响教学 | 不再适用 |
| 使用寿命到期 | 按预设年限 |
| 关键零件停产 | 无法维修 |

### 7.2 电子废弃物处理

```text
报废流程:
  申请报废 → 审核确认 → 数据清除 → 分类处置 → 台账更新

处置方式:
├── 可复用部件 → 拆解回收用于维修其他设备
├── 电子废弃物 → 交给有资质的回收企业
├── 一般废弃物 → 按当地规定分类处理
└── 有害废弃物 → 专业回收（电池、墨盒等）
```

## 常见问题

<details>
<summary>问题1: 小型空间没有专职仓管怎么办？</summary>

可以采用"值日制"，每天由当班志愿者兼任简单的物资管理工作。关键是做好数字化系统（哪怕是共享表格），让物资状态透明可查。A级设备由运营负责人亲自管理。

</details>

<details>
<summary>问题2: 学习者损坏设备如何处理？</summary>

区分"正常使用损耗"和"违规操作损坏"。前者由空间承担，后者需要与学习者/家长沟通。重要的是建立事前预防机制：培训合格才能使用、操作时有人指导、高危设备有保护装置。参见 [M06 安全舱](/docs/core/06-safety)。

</details>

<details>
<summary>问题3: 如何控制耗材浪费？</summary>

设定每个项目的耗材预算，让学习者在预算内规划材料使用。在设计阶段鼓励使用模拟工具（如3D建模软件）减少试错耗材。同时收集废料用于其他项目（如3D打印失败品可做艺术创作材料）。

</details>
