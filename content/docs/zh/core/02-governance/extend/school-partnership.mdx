# 学校联建指南

## 概述

学校是创客教育最重要的阵地之一。与学校合作建设创客空间（OWL-U 学校联建），既能让创客教育惠及更多学生，也能充分利用学校的场地、生源和教师资源。本文档提供学校联建的完整指南，帮助建立互惠互利、可持续的合作关系。

---

## 1. 合作背景与政策

### 1.1 政策支持

| 政策 | 时间 | 核心要点 |
|------|------|----------|
| 《关于加强新时代中小学科学教育工作的意见》 | 2023 | 做好科学教育"加法"，拓展校外科学教育资源 |
| 《关于加强中小学科技教育的意见》 | 2025.11 | 构建大中小一体化科技教育，"教联体"模式 |
| 《义务教育课程方案》 | 2022 | 10% 跨学科主题学习 |
| 双减政策 | 2021 | 丰富课后服务，提升素质教育 |

### 1.2 学校需求分析

| 需求 | 说明 | OWL 价值 |
|------|------|----------|
| **课程资源** | 缺乏系统的创客/STEAM 课程 | 提供成熟课程体系 |
| **师资支持** | 教师缺乏专业培训 | 提供培训和协作教学 |
| **设备指导** | 不知道买什么、怎么用 | 提供配置方案和培训 |
| **活动策划** | 需要丰富的科技活动 | 提供活动方案和支持 |
| **品牌背书** | 需要专业认可 | 提供品牌和资质背书 |

## 2. 合作模式

### 2.1 合作模式分类

| 模式 | 深度 | 资源投入 | 适用场景 |
|------|------|----------|----------|
| **课程引进** | 浅 | 低 | 试点合作、资源有限 |
| **联合运营** | 中 | 中 | 正式合作、资源互补 |
| **深度共建** | 深 | 高 | 战略合作、长期发展 |

### 2.2 课程引进模式

**特点**: 学校引进 OWL 课程，由学校教师或 OWL 导师执教

| 维度 | 说明 |
|------|------|
| **合作内容** | 课程包、培训、教学支持 |
| **学校投入** | 场地、教师时间 |
| **OWL 投入** | 课程、培训 |
| **适用情况** | 试点阶段、预算有限 |

### 2.3 联合运营模式

**特点**: OWL 与学校共同运营创客空间

| 维度 | 说明 |
|------|------|
| **合作内容** | 空间建设、课程运营、师资培训 |
| **学校投入** | 场地、部分设备、教师 |
| **OWL 投入** | 设计方案、课程、培训、运营支持 |
| **运营模式** | 学校日常使用 + OWL 技术支持 |
| **适用情况** | 正式合作、中等投入 |

### 2.4 深度共建模式

**特点**: OWL 深度参与空间建设和日常运营

| 维度 | 说明 |
|------|------|
| **合作内容** | 全面规划、建设、运营 |
| **学校投入** | 场地、资金、政策支持 |
| **OWL 投入** | 规划设计、建设监理、运营团队、课程体系 |
| **运营模式** | 联合团队日常运营 |
| **适用情况** | 战略合作、高投入高产出 |

## 3. 合作筹备流程

### 3.1 完整流程

```
1. 意向接触 (1-2周)
   └── 了解需求、介绍方案
          ↓
2. 需求调研 (2-4周)
   └── 深入了解学校情况
          ↓
3. 方案设计 (2-4周)
   └── 定制合作方案
          ↓
4. 方案论证 (2-4周)
   └── 学校内部决策
          ↓
5. 协议签署 (1-2周)
   └── 正式合作协议
          ↓
6. 筹备建设 (4-12周)
   └── 空间建设、设备采购、人员培训
          ↓
7. 试运营 (4-8周)
   └── 试运行、调整优化
          ↓
8. 正式运营
   └── 常态化运营
```

### 3.2 需求调研要点

| 调研维度 | 具体内容 |
|----------|----------|
| **学校概况** | 规模、特色、发展规划 |
| **场地条件** | 可用空间、基础设施 |
| **预算情况** | 建设预算、运营预算 |
| **师资情况** | 可参与教师、培训需求 |
| **学生情况** | 年级分布、已有基础 |
| **时间安排** | 可用时间段、学期规划 |
| **期望成果** | 学校的核心期望 |

## 4. 空间建设

### 4.1 空间规划原则

| 原则 | 说明 |
|------|------|
| **安全第一** | 符合学校安全要求 |
| **因地制宜** | 充分利用现有条件 |
| **分步实施** | 可根据预算分期建设 |
| **灵活可变** | 空间功能可调整 |

### 4.2 空间配置建议

| 级别 | 面积 | 适合规模 | 核心功能区 |
|------|------|----------|------------|
| **基础版** | 50-80m² | 一个班级 | 教学区、制作区、展示区 |
| **标准版** | 100-150m² | 两个班级 | +工具区、材料区 |
| **进阶版** | 200m²+ | 多班/社团 | +专业设备区、独立项目区 |

### 4.3 设备配置建议

| 类别 | 基础版 | 标准版 | 进阶版 |
|------|--------|--------|--------|
| **电子** | Arduino套件、面包板 | +传感器套件 | +示波器、烙铁台 |
| **编程** | 电脑、Scratch | +Python环境 | +机器人套件 |
| **数字制造** | - | 3D打印机 | +激光切割机 |
| **手工** | 基础工具 | 手工工具套件 | +木工设备 |

## 6. 师资建设

### 6.1 教师角色

| 角色 | 职责 | 人员 |
|------|------|------|
| **空间负责人** | 整体协调、日常管理 | 学校教师 |
| **主讲教师** | 课程教学 | 学校教师/OWL导师 |
| **助教** | 教学辅助 | 学校教师/志愿者 |
| **技术支持** | 设备维护、技术问题 | OWL支持 |

### 6.2 教师培训计划

| 阶段 | 内容 | 时长 |
|------|------|------|
| **入门培训** | 理念、基础技能 | 2天 |
| **课程培训** | 具体课程执教 | 2-3天 |
| **进阶培训** | 深度技能、课程开发 | 按需 |
| **持续教研** | 定期教研活动 | 每月 |

### 6.3 协作教学模式

```
初期：OWL 导师主讲 + 学校教师助教
     (1-2个月)
       ↓
过渡期：双师协作教学
       (2-3个月)
       ↓
成熟期：学校教师主讲 + OWL 支持
       (持续)
```

## 7. 合作协议要点

### 7.1 权责划分

| 事项 | 学校责任 | OWL 责任 |
|------|----------|----------|
| **场地** | 提供、维护 | 规划建议 |
| **设备** | 采购、保管 | 选型、培训 |
| **师资** | 安排教师 | 培训、支持 |
| **课程** | 教学安排 | 提供、更新 |
| **安全** | 日常管理 | 标准培训 |
| **品牌** | 规范使用 | 授权、监督 |

### 7.2 知识产权约定

| 内容 | 归属 |
|------|------|
| OWL 现有课程 | OWL 所有，学校使用权 |
| 联合开发课程 | 共同所有 |
| 学生作品 | 学生/学校所有 |
| 教师教案 | 按贡献约定 |

### 7.3 费用模式

| 模式 | 适用情况 | 说明 |
|------|----------|------|
| **一次性授权** | 课程引进 | 课程包费用 |
| **年度服务费** | 联合运营 | 培训+支持+更新 |
| **共建分成** | 深度共建 | 按收入或成本分担 |
| **公益合作** | 特定项目 | 无费用或象征性费用 |

## 8. 常见挑战与应对

### 8.1 常见挑战

| 挑战 | 表现 | 应对策略 |
|------|------|----------|
| **师资不足** | 教师能力或时间不够 | 协作教学、外聘支持 |
| **课时有限** | 挤不出时间 | 融入现有课程、课后服务 |
| **预算受限** | 资金不足 | 分期建设、低成本方案 |
| **安全顾虑** | 学校担心安全 | 完善安全体系、保险覆盖 |
| **评价困难** | 难以量化成果 | 多元评价、过程记录 |
| **持续性差** | 合作虎头蛇尾 | 明确机制、定期复盘 |

### 8.2 成功要素

| 要素 | 说明 |
|------|------|
| **校长支持** | 获得学校领导层的认可和支持 |
| **教师热情** | 有热心参与的核心教师 |
| **资源保障** | 有必要的场地、设备、预算 |
| **机制明确** | 权责清晰、流程顺畅 |
| **持续投入** | 双方持续投入而非一次性 |

## 9. 集团校合作

### 9.1 集团校合作模式

```
集团总部
├── 统一规划、资源协调
├── 标准制定、质量监控
└── 师资培训、成果展示
         ↓
    ┌────┼────┐
    ↓    ↓    ↓
  校区A  校区B  校区C
  └── 各校区根据条件实施
```

### 9.2 集团校合作优势

| 优势 | 说明 |
|------|------|
| **规模效应** | 统一采购降低成本 |
| **资源共享** | 课程、师资、设备共享 |
| **相互借鉴** | 校区间学习交流 |
| **品牌效应** | 集团层面影响力 |
