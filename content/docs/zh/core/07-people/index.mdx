## Core 论断

M07 的全部人才论可压缩为五条第一性论断。任何招聘、培训、徽章设计或评估改造若违背其中之一，必须显式给出理由。

1. **领航员的核心能力是「自己继续学」。** 教学技能是工艺，求知姿态是骨架；骨架塌了工艺也无处安放。
2. **同侪 + 大导师 + 社区专家三层并存。** 单层依赖任一群体都会脆弱——同侪给共鸣、大导师给方向、社区给真实。
3. **目的决定干预方式。** 技能、文化、自治、留任各自需要不同的支持机制；混在一起喂鸡汤只会浪费每一类。
4. **学习者发起式优于指派。** 让学习者自选 mentor 比组织指派显著更有效；OWL 应建立 YIM 通道。
5. **培训是基础设施，不是事件。** 一次性研习营不能替代持续的反思赋权与社会-情感脚手架。

## 3E 映射

| 维度 | 本模块如何体现 |
|------|---------------|
| **Enlighten** 启发 | 「我也不知道，我们一起查」的求知姿态示范，让学习者看到学习的真实过程 |
| **Empower** 赋能 | 能力路径 + 培训体系 + 徽章 + 自评工具（如 MentorQ）让任何节点都能培养出合格导师 |
| **Engage** 联接 | 大导师网络 + 社区专家 + 远程导师让一个节点的好奇心能找到全球范围的回应者 |

## 关键证据

近两年的 mentoring 元分析、scoping review 与跨学科系统综述把 M07 的论断从口号抬升到证据：

**Mentor / coach / induction 概念分明，不可混用。** Ackermann 的 scoping review 系统区分了四类目的——技能掌握、文化适应、专业自治、留任——并指出当前文献的核心问题正是术语混淆<Cite id="M07-2025-TATE-Ackermann" />。OWL 导师培训应明确目的，按目的设计差异化路径，而非用一套"导师培训"覆盖所有需求。

**学习者发起式 mentoring 显著优于指派。** van Dam 等的三层级元分析（83 项研究 409 个效应量）显示：网络激活的整体效应不显著（d=0.11），但条件性显著——青年发起式 mentoring（YIM）d=0.46，学习者自选 mentor d=0.52，"指派型"几乎没有效应<Cite id="M07-2024-IJSRM-Network" />。OWL 的大导师对接应优先采用 YIM 通道。

**数字化导师在低资源场景尤其有效。** Iqbal 等基于 120 名教师的 PLS-SEM + 定性混合方法显示：mentoring 通过教学重构、反思赋权、社会-情感脚手架三条路径产生效应；数字/混合 mentoring 模型在低资源环境下能持续产生效果<Cite id="M07-2025-SciRep-DigitalMentor" />。OWL 的远程导师网络可直接对接此模型。

**Hub-and-spoke 是网络化导师的可复用模型。** Mangione 等识别出 10 个使能因素（共同目标、对照实践、资源访问、专业能力增强等），并提供了 MentorQ 自评工具以系统评估节点导师实践<Cite id="M07-2025-JeLKS-MentorQ" />。OWL 旗舰-区域-社区三级网络可直接复用 hub-and-spoke 模型与 MentorQ 自评。

**经典理论继续支撑能力模型。** Knowles 的成人学习理论<Cite id="M07-001" />、Schön 的反思性实践者<Cite id="M07-002" />、Lave & Wenger 的实践共同体<Cite id="M07-004" />、Shulman 的 PCK<Cite id="M07-005" />共同构成 OWL 能力路径的理论根基。
