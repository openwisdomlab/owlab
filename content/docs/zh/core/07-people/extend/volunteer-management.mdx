# 志愿者管理手册

## 概述

志愿者是创客空间宝贵的人力资源。他们不仅提供服务支持，更带来多元视角、专业技能和真实世界的连接。良好的志愿者管理能够创造志愿者、学习者和组织的三方共赢。

---

## 1. 志愿者价值

### 1.1 志愿者带来的价值

| 价值类型 | 具体表现 | 对组织的意义 |
|----------|----------|--------------|
| **人力支持** | 活动协助、技术支持 | 扩大服务能力 |
| **专业技能** | 各领域专业知识 | 丰富课程内容 |
| **社会连接** | 连接真实世界 | 增强项目真实性 |
| **多元视角** | 不同背景和经验 | 促进创新思维 |
| **社区纽带** | 连接家庭和社区 | 增强社区认同 |

### 1.2 志愿者获得的价值

| 价值类型 | 具体表现 |
|----------|----------|
| **意义感** | 贡献社会、帮助他人 |
| **技能发展** | 学习新技能、实践经验 |
| **社交网络** | 认识志同道合的人 |
| **职业探索** | 了解教育领域 |
| **证明材料** | 服务证明、推荐信 |

---

## 2. 志愿者类型

### 2.1 按来源分类

| 类型 | 来源 | 特点 | 适合任务 |
|------|------|------|----------|
| **大学生志愿者** | 高校合作 | 专业技能、时间灵活 | 活动执行、技术支持 |
| **家长志愿者** | 会员家庭 | 责任心强、关注安全 | 活动协助、安全监督 |
| **企业志愿者** | 企业CSR项目 | 专业背景、团队参与 | 专题分享、职业指导 |
| **退休志愿者** | 社区招募 | 经验丰富、时间充足 | 日常支持、传统工艺 |
| **校友志愿者** | 往期学员 | 熟悉环境、有归属感 | 学员辅导、活动协助 |

### 2.2 按参与深度分类

| 类型 | 参与频率 | 承诺程度 | 任务复杂度 |
|------|----------|----------|------------|
| **临时志愿者** | 单次活动 | 低 | 简单执行 |
| **定期志愿者** | 每周/每月固定 | 中 | 持续任务 |
| **核心志愿者** | 长期深度参与 | 高 | 复杂/负责任务 |

---

## 3. 招募

### 3.1 招募渠道

| 渠道 | 适合类型 | 优势 | 操作要点 |
|------|----------|------|----------|
| **高校合作** | 大学生 | 量大、专业对口 | 对接团委/社团 |
| **家长群** | 家长 | 信任度高 | 定期发布需求 |
| **社交媒体** | 多种 | 传播广 | 清晰说明 |
| **企业合作** | 企业员工 | 有组织 | 对接CSR部门 |
| **志愿者平台** | 多种 | 专业平台 | 发布详细信息 |
| **老带新** | 核心志愿者推荐 | 质量高 | 激励推荐 |

### 3.2 招募流程

```
1. 需求分析 → 2. 发布信息 → 3. 收集申请 → 4. 筛选面谈
       ↓             ↓             ↓             ↓
   岗位需求       渠道发布       表单收集        电话/面谈

5. 背景调查 → 6. 录用通知 → 7. 入职培训 → 8. 正式服务
       ↓             ↓             ↓             ↓
   涉未成年人必须    发录用函        完成培训       开始服务
```

### 3.3 招募信息模板

```markdown
## 🌟 创客空间志愿者招募

### 我们是谁
OWL创客实验室，致力于让每个人都能通过动手创造来学习和成长。

### 我们需要
- **活动助理志愿者**: 协助课程活动的组织和执行
- **技术志愿者**: 协助设备维护和技术支持
- **分享导师**: 分享专业领域知识（每月1-2次）

### 时间要求
- 每周X小时 / 每月X次活动
- 时间灵活可协商

### 你将获得
- 官方志愿服务证明
- 免费参加培训课程
- 结识志同道合的伙伴
- 优秀志愿者表彰

### 申请方式
填写申请表: [链接]
联系人: XXX (电话/微信)
```

### 3.4 志愿者申请表

| 信息类别 | 收集内容 |
|----------|----------|
| **基本信息** | 姓名、年龄、联系方式、职业/学校 |
| **技能特长** | 专业背景、技术技能、语言能力 |
| **时间意向** | 可服务时间、期望频率 |
| **动机** | 参与原因、期望获得 |
| **经历** | 志愿服务经历、相关经验 |
| **紧急联系人** | 姓名、关系、电话 |

## 4. 培训

### 4.1 培训层级

| 培训类型 | 对象 | 时长 | 内容 |
|----------|------|------|------|
| **入职培训** | 所有志愿者 | 2-4小时 | 组织介绍、安全、基础流程 |
| **岗位培训** | 按岗位 | 1-4小时 | 岗位职责、操作培训 |
| **进阶培训** | 核心志愿者 | 按需 | 深入技能、领导力 |

### 4.2 志愿者入职培训大纲

| 模块 | 内容 | 时长 |
|------|------|------|
| **组织介绍** | 使命愿景、价值观、工作内容 | 30min |
| **志愿者角色** | 职责、权益、期望 | 20min |
| **安全培训** | 安全规程、应急处理 | 45min |
| **服务流程** | 签到、活动流程、沟通渠道 | 30min |
| **实地熟悉** | 空间介绍、设备了解 | 30min |
| **Q&A** | 答疑交流 | 15min |

### 4.3 安全培训要点

| 主题 | 内容 |
|------|------|
| **基础安全** | 通用安全规则、紧急逃生 |
| **设备安全** | 观察但不操作、发现问题报告 |
| **儿童安全** | 保护政策、行为规范 |
| **应急处理** | 受伤处理、紧急联系 |

## 5. 日常管理

### 5.1 排班管理

| 环节 | 工具/方法 | 要点 |
|------|----------|------|
| **需求发布** | 在线排班表 | 提前1-2周发布 |
| **自主报名** | 共享表格/小程序 | 方便志愿者选择 |
| **确认提醒** | 自动/手动提醒 | 活动前1天提醒 |
| **签到记录** | 扫码签到 | 记录服务时长 |

### 5.2 沟通机制

| 渠道 | 用途 | 频率 |
|------|------|------|
| **志愿者群** | 日常沟通、活动通知 | 持续 |
| **个人沟通** | 反馈、关怀、问题处理 | 按需 |
| **月度会议** | 总结、交流、培训 | 每月 |
| **满意度调查** | 了解体验和建议 | 每季 |

### 5.3 服务记录

| 记录内容 | 用途 |
|----------|------|
| **服务时长** | 时数证明、奖励依据 |
| **服务岗位** | 能力匹配、发展规划 |
| **表现评价** | 反馈改进、晋升依据 |
| **培训记录** | 能力发展追踪 |

## 6. 激励机制

### 6.1 激励类型

| 类型 | 形式 | 适用 |
|------|------|------|
| **认可激励** | 感谢、表扬、展示 | 所有志愿者 |
| **发展激励** | 培训、技能提升 | 有发展意愿者 |
| **物质激励** | 小礼品、福利 | 适度使用 |
| **荣誉激励** | 证书、徽章、表彰 | 优秀志愿者 |
| **特权激励** | 优先参与、特殊体验 | 核心志愿者 |

### 6.2 服务时数证明

```markdown
## 志愿服务证明

兹证明 [姓名] 于 [时间段] 在 OWL创客实验室
参与志愿服务，累计服务时长 [XX] 小时。

主要服务内容:
- [服务内容1]
- [服务内容2]

服务表现评价: [优秀/良好/合格]

特此证明。

OWL创客实验室
[日期]
[签章]
```

### 6.3 徽章体系

| 徽章 | 获取条件 | 权益 |
|------|----------|------|
| **新星徽章** | 完成首次服务 | - |
| **十小时徽章** | 累计10小时 | - |
| **五十小时徽章** | 累计50小时 | 免费培训1次 |
| **百小时徽章** | 累计100小时 | 年度表彰 |
| **年度志愿者** | 年度评选 | 特别表彰+礼品 |
| **志愿导师** | 核心志愿者升级 | 导师待遇 |

### 6.4 年度表彰

| 奖项 | 标准 | 奖励 |
|------|------|------|
| **最佳贡献奖** | 服务时长+质量 | 证书+礼品 |
| **最具专业奖** | 专业贡献突出 | 证书+礼品 |
| **最佳新人奖** | 新志愿者表现出色 | 证书+礼品 |
| **团队协作奖** | 团队协作贡献 | 团队奖励 |

## 7. 志愿者保护

### 7.1 权益保障

| 权益 | 保障措施 |
|------|----------|
| **知情权** | 明确告知任务、风险、权益 |
| **安全保障** | 提供必要培训和保护 |
| **保险覆盖** | 服务期间意外保险 |
| **尊重隐私** | 保护个人信息 |
| **反馈渠道** | 提供反馈和申诉途径 |

### 7.2 风险管理

| 风险 | 预防措施 |
|------|----------|
| **人身安全** | 安全培训、保险、紧急预案 |
| **能力匹配** | 根据能力分配任务 |
| **过度服务** | 监控时长、鼓励休息 |
| **冲突处理** | 建立冲突解决机制 |

### 7.3 涉未成年人服务规范

| 规范 | 要求 |
|------|------|
| **背景调查** | 涉未成年人岗位必须通过背景调查 |
| **两人规则** | 避免单独与未成年人相处 |
| **行为边界** | 保持适当身体距离 |
| **报告义务** | 发现问题立即报告 |

## 8. 志愿者发展

### 8.1 发展路径

```
临时志愿者 → 定期志愿者 → 核心志愿者 → 志愿导师/团队负责
     ↓            ↓             ↓              ↓
  单次参与      固定服务        深度参与         领导角色
```

### 8.2 核心志愿者计划

| 项目 | 内容 |
|------|------|
| **选拔** | 定期服务3个月+、表现优秀 |
| **培养** | 额外培训、赋予更多责任 |
| **权益** | 培训优惠、优先参与、荣誉 |
| **责任** | 带领新志愿者、承担重要任务 |

### 8.3 志愿者转导师通道

| 条件 | 流程 |
|------|------|
| 核心志愿者满1年 | 申请评估 |
| 通过能力评估 | 导师培训 |
| 完成培训 | E0认证 |
| 实习期通过 | E1认证 |

## 9. 常见问题

### 9.1 问题与对策

| 问题 | 原因 | 对策 |
|------|------|------|
| 招募困难 | 渠道不足、信息不清 | 拓展渠道、优化信息 |
| 流失率高 | 缺乏归属感、激励不足 | 加强关怀、完善激励 |
| 服务质量不稳 | 培训不足、缺乏反馈 | 加强培训、建立反馈 |
| 时间难协调 | 沟通不畅 | 灵活排班、提前规划 |
