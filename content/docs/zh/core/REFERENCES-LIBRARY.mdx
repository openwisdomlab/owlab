## 引用格式规范

### APA 7 基本格式

**期刊论文**:
```
Author, A. A., & Author, B. B. (Year). Title of article. Title of Periodical, volume(issue), page–page. https://doi.org/xxxxx
```

**书籍**:
```
Author, A. A. (Year). Title of work: Capital letter also for subtitle. Publisher.
```

**书籍章节**:
```
Author, A. A. (Year). Title of chapter. In E. E. Editor (Ed.), Title of book (pp. xx–xx). Publisher.
```

**网页/在线报告**:
```
Organization Name. (Year, Month Day). Title of page. Site Name. URL
```

## 文献管理说明

### 添加新文献流程

1. 确认文献符合质量要求
2. 选择正确的主题分类
3. 按 APA7 格式录入
4. 标注与模块的关联
5. 提交审核

### 文献标签体系

| 标签 | 含义 |
|------|------|
| `#core` | 核心文献 |
| `#extended` | 延伸文献 |
| `#M01` - `#M10` | 关联模块 |
| `#theory` | 理论性 |
| `#practice` | 实践性 |
| `#policy` | 政策性 |
| `#standard` | 标准规范 |

### 文献更新机制

- **季度更新**: 添加新发表的重要文献
- **年度审核**: 审核文献时效性
- **按需更新**: 模块更新时同步更新文献
