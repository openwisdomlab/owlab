# 未成年人保护政策

## 概述

保护未成年人是OWL网络的首要责任和底线原则。本政策基于《联合国儿童权利公约》(UNCRC)及中国《未成年人保护法》，建立全面的制度性保护屏障，确保每一个参与OWL活动的孩子都能在安全、受尊重的环境中探索和成长。

**政策适用范围**:
- 所有OWL网络节点
- 所有与未成年人接触的人员（包括员工、志愿者、外聘讲师、家长志愿者）
- 所有涉及未成年人的活动（线下、线上）

---

## 与OWL框架的关系

| OWL原则 | 在未成年人保护中的体现 |
|---------|----------------------|
| **开放探索** | 在安全边界内鼓励自主探索，安全是开放的前提 |
| **容错成长** | 保护孩子免受侵害，让试错只发生在学习领域 |
| **同伴互助** | 建立健康的同伴关系模式，预防欺凌 |
| **社会连接** | 确保所有社会连接对孩子无害 |
| **意义驱动** | 帮助孩子在安全环境中寻找意义 |
| **自我效能** | 赋予孩子说"不"的权利和能力 |

## 1. 核心保护原则

### 1.1 四项基本原则

| 原则 | 内涵 | 实施措施 | 检查频率 |
|------|------|----------|----------|
| **儿童最大利益** | 所有决策以儿童利益为首要考量 | 决策前进行儿童影响评估 | 每项决策 |
| **家长知情同意** | 重要事项需获得家长书面同意 | 活动授权书、定期沟通机制 | 每次活动 |
| **最小化接触** | 避免成人与儿童不必要的单独接触 | 双人原则、开放环境设计 | 持续 |
| **零容忍政策** | 对任何侵害行为绝不姑息 | 立即报告、严肃处理、全网通报 | 持续 |

| 人员类别 | 调查内容 | 调查时机 | 有效期 |
|----------|----------|----------|--------|
| **全职员工** | 无犯罪记录证明、从业禁止查询、前雇主核实 | 入职前 | 每年更新 |
| **兼职人员** | 无犯罪记录证明、从业禁止查询 | 上岗前 | 每年更新 |
| **志愿者** | 背景问卷、推荐人核实、面谈评估 | 上岗前 | 每年更新 |
| **外聘讲师** | 机构资质核实、个人背景声明 | 每次活动前 | 单次 |
| **家长志愿者** | 背景声明、活动培训 | 参与活动前 | 单次 |

### 2.3 不予通过的情形

以下情形将**直接取消资格**，不予复议：

| 类别 | 具体情形 |
|------|----------|
| **刑事记录** | 性侵害类犯罪、暴力犯罪、虐待犯罪 |
| **行政处罚** | 涉及未成年人的行政处罚记录 |
| **从业禁止** | 被教育、医疗等行业禁止从业 |
| **诚信问题** | 申请材料存在虚假陈述 |
| **其他风险** | 推荐人核实发现重大风险信号 |

## 3. 双人原则与场景化保护

### 3.1 双人原则 (The Two-Adult Rule)

**核心规则**: 任何成人不应与单名儿童在视线受阻的封闭空间独处。

| 场景 | 要求 | 例外处理 |
|------|------|----------|
| **一对一辅导** | 必须在开放空间或有透明窗的房间 | 紧急情况除外 |
| **谈话/咨询** | 如需关门，必须有第三方在场或门有透明窗 | 无例外 |
| **接送** | 禁止单独接送非亲属儿童 | 紧急情况需提前获得家长授权 |
| **线上互动** | 禁止私信沟通，必须在群组或有记录的渠道 | 无例外 |
| **外出活动** | 成人与儿童比例不低于1:8 | 特殊活动需更高比例 |

### 3.2 身体接触边界

```
身体接触规范:
┌─────────────────────────────────────────────────────────┐
│  允许的接触                                              │
│  ├── 在公开场合的轻拍肩膀表示鼓励                        │
│  ├── 必要时纠正操作姿势（需事先告知）                    │
│  ├── 紧急情况下的保护性接触                              │
│  └── 儿童主动发起的合适身体接触（如拥抱告别）            │
├─────────────────────────────────────────────────────────┤
│  禁止的接触                                              │
│  ├── 任何私密部位的接触                                  │
│  ├── 让儿童感到不适的身体接触                            │
│  ├── 在私密空间的身体接触                                │
│  └── 超出活动必要范围的接触                              │
└─────────────────────────────────────────────────────────┘
```

### 3.3 特殊场景应对

| 场景 | 风险点 | 保护措施 |
|------|--------|----------|
| **过夜活动** | 住宿安排 | 同性别成人分区监护，禁止成人进入儿童房间 |
| **更衣/洗浴** | 隐私暴露 | 设置独立空间，成人不得进入 |
| **外出活动** | 脱离监护 | 严格点名制度，紧急联络机制 |
| **线上课程** | 屏幕共享隐私 | 培训儿童隐私保护，家长陪同 |
| **情绪疏导** | 独处风险 | 在开放空间进行，必要时请专业人员 |

- [ ] 已获得家长书面授权

- [ ] 移除可定位信息（校服徽章、家庭地址、学校名称）

- [ ] 图片无不当姿态或表情
- [ ] 符合发布平台相关规定

## 5. 强制报告机制

### 5.1 报告义务

**法律依据**: 《中华人民共和国未成年人保护法》第十一条规定，国家机关、居民委员会、村民委员会、密切接触未成年人的单位及其工作人员，在工作中发现未成年人身心健康受到侵害、疑似受到侵害或者面临其他危险情形的，应当立即向公安、民政、教育等有关部门报告。

**OWL要求**: 所有工作人员和志愿者发现或怀疑儿童遭受侵害，**必须在24小时内报告**。

### 5.2 需要报告的情形

| 类别 | 具体表现 |
|------|----------|
| **身体虐待** | 不明原因的伤痕、淤青、烫伤；对身体接触表现恐惧 |
| **情感虐待** | 自卑、退缩、过度焦虑；对特定成人表现恐惧 |
| **性侵害** | 年龄不相符的性知识或行为；对身体部位异常关注 |
| **忽视** | 卫生状况极差、营养不良；长期无人监护 |
| **欺凌** | 经常被孤立、财物被损坏；情绪异常波动 |

### 5.3 报告流程

```
强制报告流程:
┌────────────────────────────────────────────────────────────┐
│ 第1步：发现/怀疑                                            │
│ • 观察到伤痕或异常行为                                      │
│ • 接到儿童倾诉                                              │
│ • 收到第三方信息                                            │
└───────────────────────────┬────────────────────────────────┘
                            ▼
┌────────────────────────────────────────────────────────────┐
│ 第2步：记录                                                 │
│ • 时间、地点、当事人                                        │
│ • 客观描述所见所闻（不加主观推断）                          │
│ • 保留证据（照片、聊天记录等）                              │
└───────────────────────────┬────────────────────────────────┘
                            ▼
┌────────────────────────────────────────────────────────────┐
│ 第3步：上报                                                 │
│ • 立即向保护专员或最高负责人报告                            │
│ • 使用专用渠道（电话/邮箱/表单）                            │
│ • 24小时内完成                                              │
└───────────────────────────┬────────────────────────────────┘
                            ▼
┌────────────────────────────────────────────────────────────┐
│ 第4步：介入处理                                             │
│ • 负责人评估情况                                            │
│ • 必要时联系家长/学校                                       │
│ • 严重情况报警或联系民政部门                                │
```

### 5.4 举报人保护

| 保护措施 | 说明 |
|----------|------|
| **身份保密** | 举报人信息仅限必要人员知晓 |
| **反报复保护** | 严禁对举报人进行任何形式的报复 |
| **法律保护** | 协助举报人获得必要的法律保护 |
| **心理支持** | 为举报人提供心理咨询服务 |

## 7. 违规处理

### 7.1 违规分级

| 级别 | 情形 | 处理措施 |
|------|------|----------|
| **一级（轻微）** | 无意违反程序规定 | 提醒、补训 |
| **二级（中度）** | 多次违反程序或存在疏忽 | 警告、停职培训 |
| **三级（严重）** | 故意违反政策或造成风险 | 终止关系、全网通报 |
| **四级（触法）** | 涉嫌违法犯罪 | 报警、配合调查、永久禁入 |

### 7.2 处理流程

1. **接到举报**: 24小时内启动调查
2. **临时措施**: 必要时暂停当事人工作
3. **调查取证**: 由保护专员或第三方进行
4. **处理决定**: 根据调查结果做出处理
5. **申诉机制**: 当事人可在5个工作日内申诉
6. **记录存档**: 所有记录保存至少5年

## 常见问题

<details>
<summary>如果只有我一个成人和一个孩子在场怎么办？</summary>

应尽量避免这种情况。如果无法避免：
1. 保持在开放、可见的区域
2. 通知其他成人或家长
3. 尽快增加第二个成人
4. 保持门窗开放
</details>

<details>
<summary>孩子向我倾诉被侵害的事情，我该怎么做？</summary>

1. **保持冷静**: 不要表现出震惊或过度反应
2. **认真倾听**: 不要打断，不要质疑
3. **表示相信**: "谢谢你告诉我，这不是你的错"
4. **不要承诺保密**: "我需要告诉能帮助你的人"
5. **如实记录**: 使用孩子的原话
6. **立即报告**: 向保护专员报告
</details>

<details>
<summary>如何处理儿童之间的欺凌行为？</summary>

1. 立即制止欺凌行为
2. 分开当事人，分别了解情况
3. 安抚受害者，确保安全
4. 记录事件经过
5. 通知双方家长
6. 根据严重程度决定后续措施
</details>

<details>
<summary>家长要求查看活动照片，但照片中有其他孩子怎么办？</summary>

1. 可以提供只包含其孩子的照片
2. 如需提供包含其他孩子的照片，需获得其他家长同意
3. 可以对其他孩子的面部进行模糊处理
4. 解释隐私保护政策，争取家长理解
</details>
