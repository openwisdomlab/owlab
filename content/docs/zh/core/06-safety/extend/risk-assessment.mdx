# 风险评估矩阵

## 概述

风险评估是安全管理的核心工具。本文档建立了一套适用于 K-12 创新空间的风险评估方法论，帮助系统化地识别、评估和控制各类安全风险。

**核心理念**: 预防优于应对，系统化评估是有效预防的基础。

---

## 1. 风险评估框架

### 1.1 风险评估流程

```
风险识别 → 风险分析 → 风险评价 → 风险处置 → 监控与评审
    ↑                                               │
    └───────────── 持续改进 ←──────────────────────┘
```

### 1.2 评估周期

| 评估类型 | 频率 | 触发条件 | 负责人 |
|----------|------|----------|--------|
| **例行评估** | 季度 | 时间驱动 | 安全员 |
| **新设备评估** | 引入时 | 新设备到货 | 导师 + 安全员 |
| **活动评估** | 活动前 | 特殊活动计划 | 活动负责人 |
| **事故后评估** | 事故后 | 发生安全事件 | 安全员 + 负责人 |
| **年度全面评估** | 年度 | 年度安全审计 | 总部 + 节点 |

## 2. 风险识别

### 2.1 风险源分类

| 类别 | 风险源 | 典型风险 |
|------|--------|----------|
| **物理风险** | 设备、工具、材料 | 机械伤害、烫伤、切割 |
| **化学风险** | 清洗剂、焊接材料、实验试剂 | 化学灼伤、中毒、过敏 |
| **电气风险** | 电源、线路、电子设备 | 触电、短路、火灾 |
| **生物风险** | 微生物培养、食品材料 | 感染、过敏 |
| **人因风险** | 疲劳、误操作、能力不足 | 各类事故 |
| **环境风险** | 通风不良、照明不足、噪音 | 健康损害 |
| **组织风险** | 管理缺失、培训不足、规程不清 | 系统性风险 |

### 2.2 风险识别方法

| 方法 | 描述 | 适用场景 |
|------|------|----------|
| **检查表法** | 使用预设清单逐项检查 | 例行评估 |
| **现场观察** | 直接观察操作过程 | 活动期间 |
| **访谈法** | 与使用者、导师交流 | 深入了解 |
| **事故分析** | 分析历史事故/未遂事件 | 事后改进 |
| **头脑风暴** | 团队讨论识别潜在风险 | 新项目评估 |
| **流程分析** | 分析操作流程各环节 | 复杂操作 |

### 2.3 创新空间典型风险清单

#### 设备操作风险

| 设备类别 | 主要风险 | 危害后果 |
|----------|----------|----------|
| **3D打印机** | 高温喷头、机械夹伤、材料烟雾 | 烫伤、挤压伤、呼吸刺激 |
| **激光切割机** | 激光灼伤、火灾、有毒烟雾 | 眼部伤害、皮肤灼伤、中毒 |
| **电烙铁** | 高温烫伤、焊锡烟雾 | 皮肤烫伤、呼吸刺激 |
| **电动工具** | 切割、卷入、噪音、振动 | 外伤、听力损伤 |
| **CNC设备** | 高速切削、卷入、碎片飞溅 | 严重外伤 |
| **电子电路** | 触电、短路火花 | 电击、烧伤 |

#### 活动场景风险

| 活动类型 | 主要风险 | 危害后果 |
|----------|----------|----------|
| **常规课程** | 设备误操作、疲劳 | 轻微伤害 |
| **工作坊** | 人员密集、时间压力 | 操作失误 |

## 3. 风险分析与评价

### 3.1 风险评估矩阵

#### 可能性等级

| 等级 | 描述 | 发生频率参考 | 分值 |
|------|------|--------------|------|
| **1-极低** | 几乎不可能发生 | &lt;1次/10年 | 1 |
| **2-低** | 不太可能发生 | 1次/5年 | 2 |
| **3-中** | 可能发生 | 1次/年 | 3 |
| **4-高** | 很可能发生 | 1次/季度 | 4 |
| **5-极高** | 几乎必然发生 | 1次/月或更频繁 | 5 |

#### 严重性等级

| 等级 | 描述 | 后果参考 | 分值 |
|------|------|----------|------|
| **1-轻微** | 无伤害或极轻微 | 不适但无需处理 | 1 |
| **2-一般** | 轻微伤害 | 需简单急救，无需就医 | 2 |
| **3-中等** | 中度伤害 | 需就医但可完全恢复 | 3 |
| **4-严重** | 重大伤害 | 需住院或可能留有后遗症 | 4 |
| **5-灾难** | 危及生命或永久残疾 | 死亡或严重残疾 | 5 |

#### 风险矩阵

|  | 轻微(1) | 一般(2) | 中等(3) | 严重(4) | 灾难(5) |
|--|---------|---------|---------|---------|---------|
| **极高(5)** | 5(中) | 10(高) | 15(高) | 20(极高) | 25(极高) |
| **高(4)** | 4(低) | 8(中) | 12(高) | 16(高) | 20(极高) |
| **中(3)** | 3(低) | 6(中) | 9(中) | 12(高) | 15(高) |
| **低(2)** | 2(极低) | 4(低) | 6(中) | 8(中) | 10(高) |
| **极低(1)** | 1(极低) | 2(极低) | 3(低) | 4(低) | 5(中) |

**风险值 = 可能性 x 严重性**

### 3.2 风险等级定义

| 风险等级 | 风险值 | 颜色标识 | 处置要求 | 审批层级 |
|----------|--------|----------|----------|----------|
| **极低** | 1-2 | 绿色 | 可接受，日常监控 | 无需审批 |
| **低** | 3-5 | 蓝色 | 可接受，加强关注 | 导师确认 |
| **中** | 6-10 | 黄色 | 需控制措施 | 安全员审批 |
| **高** | 11-16 | 橙色 | 必须降低风险 | 负责人审批 |
| **极高** | 17-25 | 红色 | 不可接受，立即处理 | 总部审批 |

## 4. 风险控制措施

### 4.1 控制措施层级

按优先级从高到低排序:

```
1. 消除 Elimination
   ↓ 彻底去除危险源
2. 替代 Substitution
   ↓ 用低风险替代高风险
3. 工程控制 Engineering Controls
   ↓ 物理隔离、安全装置
4. 管理控制 Administrative Controls
   ↓ 规程、培训、标识
5. 个人防护 PPE
   最后一道防线
```

### 4.2 各层级控制措施示例

| 层级 | 示例措施 | 适用场景 |
|------|----------|----------|
| **消除** | 移除高风险化学品、取消危险操作 | 可替代的危险源 |
| **替代** | 用低温焊接替代高温、用安全剪刀替代美工刀 | 有低风险替代方案 |
| **工程控制** | 安装防护罩、联锁装置、通风系统、紧急停止按钮 | 设备固有风险 |
| **管理控制** | 操作规程、培训认证、工作许可、标识警示 | 人为因素风险 |
| **个人防护** | 护目镜、手套、口罩、护耳器 | 残余风险 |

### 4.3 典型设备风险控制

#### 激光切割机

| 风险 | 控制层级 | 控制措施 |
|------|----------|----------|
| 激光灼伤 | 工程 | 封闭式舱门、联锁装置 |
| 激光灼伤 | PPE | 激光防护眼镜 |
| 火灾 | 工程 | 烟雾检测、自动灭火 |
| 火灾 | 管理 | 禁止可燃材料清单、值守要求 |
| 有毒烟雾 | 工程 | 强制排风系统 |
| 有毒烟雾 | 管理 | 材料审批制度 |

#### 电烙铁

| 风险 | 控制层级 | 控制措施 |
|------|----------|----------|
| 烫伤 | 替代 | 低温焊接方案(适用时) |
| 烫伤 | 工程 | 烙铁架、自动休眠 |
| 烫伤 | 管理 | 培训认证、操作规程 |
| 焊锡烟雾 | 工程 | 排烟设备、通风 |
| 焊锡烟雾 | PPE | 防护口罩(高强度作业) |

## 7. 风险监控与改进

### 7.1 风险监控指标

| 指标 | 定义 | 目标 | 监控频率 |
|------|------|------|----------|
| 事故率 | 事故次数/活动人次 | &lt;0.1% | 月 |
| 未遂事件率 | 未遂事件/活动人次 | &lt;0.5% | 月 |
| 高风险暴露时间 | S3/S4设备使用时长 | 有监控记录 | 周 |
| 培训覆盖率 | 持证人员/应培训人员 | 100% | 季 |
| 整改完成率 | 已整改/待整改项 | >90% | 月 |

### 7.2 事故/未遂事件报告

**必须报告的事件**:
- 任何人员伤害(包括轻微)
- 设备损坏
- 火灾/烟雾触发
- 化学品泄漏
- "差一点"发生的危险情况

**报告流程**:
```
发现/发生 → 立即处置 → 24小时内报告 → 调查分析 → 整改措施 → 经验分享
```

### 7.3 风险评审机制

| 评审类型 | 频率 | 内容 | 输出 |
|----------|------|------|------|
| 月度回顾 | 月 | 事故统计、整改跟踪 | 月报 |
| 季度评估 | 季 | 风险清单更新、控制措施有效性 | 评估报告 |
| 年度审计 | 年 | 全面风险评估、体系有效性 | 审计报告 + 改进计划 |

### 7.4 持续改进

| 改进来源 | 改进机制 |
|----------|----------|
| 事故教训 | 根因分析 → 系统改进 |
| 未遂事件 | 早期预警 → 预防措施 |
| 检查发现 | 隐患整改 → 制度完善 |
| 行业实践 | 对标学习 → 引入创新 |
| 法规更新 | 合规审查 → 制度更新 |
| 用户反馈 | 收集分析 → 优化改进 |

| **柔性电子材料** | 化学挥发 | 成分分析 | 专用区域 |
| **生物材料** | 感染、过敏 | 无菌要求 | 生物安全柜 |
| **纳米材料** | 吸入风险 | 颗粒尺寸 | N95+、HEPA |
| **复合树脂** | 刺激、致敏 | 固化条件 | UV 防护、通风 |
| **可食用材料** | 过敏、交叉污染 | 成分标签 | 专用设备、清洁 |

### 9.5 远程/混合活动风险

| 风险类型 | 描述 | 控制措施 |
|----------|------|----------|
| **居家设备使用** | 无专业监护 | 仅限低风险设备、家长同意 |
| **网络安全** | 数据泄露、不当访问 | 安全平台、身份验证 |
| **屏幕时间** | 视觉疲劳、姿势问题 | 时间限制、休息提醒 |
| **心理隔离** | 缺乏社交互动 | 混合模式、小组活动 |
| **设备邮寄** | 运输损坏、不当使用 | 包装规范、使用指南 |

### 9.6 气候与环境风险

| 风险 | 影响 | 评估指标 |
|------|------|----------|
| **极端高温** | 设备过热、人员中暑 | 室内温度超35°C预警 |
| **极端低温** | 材料脆化、取暖设备风险 | 室内温度低于10°C预警 |
| **空气污染** | 呼吸系统影响 | AQI超150减少户外 |
| **电力不稳** | 设备损坏、数据丢失 | UPS 配置、自动保存 |
| **自然灾害** | 人员安全、设备损失 | 应急预案、疏散演练 |
