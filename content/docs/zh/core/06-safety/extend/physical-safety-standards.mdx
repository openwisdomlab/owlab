# 物理安全管理标准

## 概述

物理安全是创客空间运营的基石。创客活动涉及各类工具、设备和材料，潜在风险不容忽视。本标准建立系统化的物理安全管理框架，从消防设施、设备分级到个人防护装备，全方位保障每位参与者的人身安全。

**适用范围**:
- 所有OWL网络节点的物理空间
- 所有涉及设备操作的活动
- 所有参与者（员工、志愿者、学员、访客）

---

## 与OWL框架的关系

| OWL原则 | 在物理安全中的体现 |
|---------|------------------|
| **开放探索** | 安全是开放探索的前提，明确边界才能自由探索 |
| **容错成长** | 通过安全措施将"试错"的代价控制在可接受范围 |
| **同伴互助** | 安全监督需要同伴配合，培养相互关照的习惯 |
| **社会连接** | 安全标准与社会通用标准接轨 |
| **意义驱动** | 理解安全规则背后的意义而非盲目遵守 |
| **自我效能** | 掌握安全技能是自我效能的重要组成 |

## 1. 消防安全

### 1.1 消防设施配置标准

| 项目 | 标准要求 | 数量/覆盖 | 检查频率 | 责任人 |
|------|----------|----------|----------|--------|
| **灭火器** | ABC干粉，有效期内 | 每50m²一具 | 月检 | 安全员 |
| **烟感报警器** | 联网报警 | 全覆盖 | 月检 | 安全员 |
| **疏散通道** | 宽度≥1.0m | 至少两条 | 日检 | 值班人员 |
| **应急照明** | 断电后≥30分钟 | 全覆盖 | 月检 | 安全员 |
| **疏散指示牌** | 发光型或电致发光 | 通道转角处 | 月检 | 安全员 |
| **消防栓/水带** | 按规范配置 | 建筑要求 | 月检 | 物业/安全员 |

```

│  P - Pull (拔)                                          │
│      拔出安全销                                          │
├─────────────────────────────────────────────────────────┤
│  A - Aim (瞄)                                           │

└─────────────────────────────────────────────────────────┘

注意事项:
• 站在上风向或侧风向
• 与火源保持2-3米距离
• 如火势无法控制，立即撤离
```

### 1.3 疏散通道管理

| 要求 | 具体标准 | 违规处理 |
|------|----------|----------|
| **畅通** | 严禁堆放任何物品 | 立即清理，记录警告 |
| **标识** | 地面或墙面清晰标识 | 24小时内补充 |
| **宽度** | 主通道≥1.2m，次通道≥1.0m | 立即整改 |
| **门向** | 疏散门向外开启 | 立即整改 |
| **照明** | 正常+应急双重照明 | 24小时内修复 |

### 1.4 消防演练要求

| 演练类型 | 频率 | 参与人员 | 记录要求 |
|----------|------|----------|----------|
| **疏散演练** | 每季度 | 全员 | 时间、人数、问题 |
| **灭火演练** | 每半年 | 核心人员 | 操作考核 |
| **综合演练** | 每年 | 全员+消防 | 完整报告 |

## 2. 设备安全分级

### 2.1 四级分级体系

```
设备安全分级金字塔:
                    ┌───┐
                    │ D │ ← 专业级（仅限主管/专家）
                    │   │
                  ┌─┴───┴─┐
                  │   C   │ ← 高风险（资质+专人监管）
                  │       │
                ┌─┴───────┴─┐
                │     B     │ ← 中风险（培训+考核）
                │           │
              ┌─┴───────────┴─┐
              │       A       │ ← 低风险（自主使用）
              └───────────────┘
```

### 2.2 各级别详细说明

| 等级 | 风险特征 | 准入要求 | 监管方式 | 典型设备 |
|------|----------|----------|----------|----------|
| **A级** | 低风险，误操作不易造成伤害 | 阅读须知，签署声明 | 自主使用 | 3D打印机、电脑、低温热胶枪、手工工具 |
| **B级** | 中风险，可能造成轻微伤害 | 完成培训+考核通过 | 定期巡查 | 电烙铁、手持电钻、裁纸刀、小型切割机 |
| **C级** | 高风险，可能造成严重伤害 | 专项资质+使用登记 | 专人监管 | 激光切割机、台钻、带锯、雕刻机 |
| **D级** | 专业级，需要专业培训 | 仅限授权专家操作 | 全程陪同 | CNC铣床、车床、化学腐蚀设备 |

```
┌────────────────────────────────────────────────────────────┐

│ 2. 参加专项培训                                             │
│ 3. 通过安全评估                                             │
```

### 2.4 年龄与设备准入

| 年龄段 | A级 | B级 | C级 | D级 |
|--------|-----|-----|-----|-----|
| **8岁以下** | 监护下使用 | 禁止 | 禁止 | 禁止 |
| **8-12岁** | 可用 | 监护下使用 | 禁止 | 禁止 |
| **12-16岁** | 可用 | 培训后可用 | 监护下可用 | 禁止 |
| **16-18岁** | 可用 | 可用 | 培训后可用 | 监护下可用 |
| **18岁以上** | 可用 | 可用 | 可用 | 培训后可用 |

| 颜色 | 等级 | 标识内容 |
|------|------|----------|

| 🟠 橙色 | C级 | +监管要求、预约方式 |

### 3.3 安全禁忌 (Safety DON'Ts)

| 禁忌 | 原因 | 风险级别 |
|------|------|----------|
| **旋转设备禁戴手套** | 手套可能被卷入造成严重伤害 | 🔴 极高 |
| **长发必须盘起** | 防止被旋转部件卷入 | 🔴 极高 |
| **禁止穿拖鞋/凉鞋** | 防止落物砸伤或碎屑刺伤 | 🟠 高 |
| **禁止佩戴悬挂饰品** | 防止卷入旋转部件 | 🟠 高 |
| **禁止宽松衣物** | 防止被机械卷入 | 🟠 高 |
| **禁止在设备旁饮食** | 防止污染和分心 | 🟡 中 |

## 6. 事故应急响应

### 6.1 事故分级

| 级别 | 定义 | 典型情况 | 响应措施 |
|------|------|----------|----------|
| **I级（轻微）** | 无需医疗或仅需简单处理 | 轻微划伤、小面积烫伤 | 现场处理，记录 |
| **II级（一般）** | 需要医疗处理但无需住院 | 需缝合的伤口、二度烫伤 | 就医，报告上级 |
| **III级（较大）** | 需要住院治疗 | 骨折、严重烫伤 | 120急救，全面报告 |
| **IV级（严重）** | 危及生命或造成永久伤残 | - | 120急救，上报主管部门 |

```

                            ▼

│ 第3步：急救处理                                             │
```

## 7. 特殊环境安全

| 项目 | 要求 | 检查方式 |

| **喷漆区** | 专用喷漆房/排风 | VOC浓度 |

## 常见问题

<details>
<summary>如何判断设备应该归入哪个安全等级？</summary>

考虑以下因素：
1. **误操作后果**: 可能造成的最严重伤害
2. **操作复杂度**: 需要的技能和知识水平
3. **防护要求**: 需要的PPE类型和数量
4. **同类设备参照**: 参考其他空间的分级

建议保守评估，如有疑问归入更高等级。
</details>

<details>
<summary>PPE损坏或丢失怎么办？</summary>

1. 立即报告安全员
2. 在补充到位前暂停相关设备使用
3. 建立PPE消耗记录，定期补充
4. 对于频繁损坏/丢失的情况进行原因分析
</details>

<details>
<summary>学员拒绝佩戴PPE怎么处理？</summary>

1. 解释PPE的重要性和保护作用
2. 如仍拒绝，禁止使用相关设备
3. 记录情况，通知家长（如为未成年人）
4. 屡次违规可暂停使用资格
</details>

<details>
<summary>发生轻微伤害需要报告吗？</summary>

是的，所有伤害事故都需要记录。原因：
1. 为安全改进提供数据
2. 识别潜在的系统性问题
3. 法律合规要求
4. 作为保险理赔依据
</details>
