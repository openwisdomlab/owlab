# 工具借用管理

## 概述

允许会员借用工具可以延伸创客活动到家庭和其他场所，但需要良好的管理制度确保工具的可用性和安全性。本文档提供系统化的工具借用管理方案，帮助创客空间建立规范、高效、可持续的工具外借服务。

---

## 1. 借用制度设计

### 1.1 借用资格体系

| 资格等级 | 入门条件 | 可借范围 | 信用额度 |
|---------|---------|---------|---------|
| 见习会员 | 注册满1个月 | A类工具 | 200元以内 |
| 正式会员 | 注册满3个月+基础培训 | A、B类工具 | 500元以内 |
| 认证会员 | 技能认证+良好记录 | A、B、C类工具 | 1000元以内 |
| 核心会员 | 认证会员+贡献积累 | A、B、C类工具 | 2000元以内 |

**资格审核要点**

| 审核项 | 要求 | 验证方式 |
|-------|------|---------|
| 身份验证 | 真实有效身份 | 身份证件核验 |
| 技能认证 | 对应工具使用培训 | 培训记录查询 |
| 信用记录 | 无逾期/损坏未处理 | 借用历史查询 |
| 押金/担保 | 高价值工具额外要求 | 押金缴纳/担保人 |

### 1.2 可借用物品分类

| 类别 | 定义 | 典型工具 | 借用期限 | 管理强度 |
|------|------|---------|---------|---------|
| **A类** | 基础手工具，低价值 | 螺丝刀套装、钳子、锤子、卷尺 | 7-14天 | 登记即借 |
| **B类** | 电动/专业工具，中等价值 | 电钻、电动螺丝刀、万用表、热熔胶枪 | 3-7天 | 培训+登记 |
| **C类** | 贵重/专业设备，高价值 | 示波器、3D打印笔、专业焊接设备 | 1-3天 | 申请审批 |
| **D类** | 固定设备/危险品 | 台锯、激光切割机、化学品 | 仅限现场 | 禁止外借 |

**分类判定依据**

```
工具分类决策树:

                 工具分类判定
                      │
         ┌────────────┼────────────┐
         ↓            ↓            ↓
      安全风险    使用门槛     价值评估
         │            │            │
    ┌────┴────┐  ┌────┴────┐  ┌────┴────┐
    ↓         ↓  ↓         ↓  ↓         ↓
   高风险   低风险 需培训  无门槛  高价值  低价值
    ↓         │    │         │    │        │
  D类       └────┬────┘ └────┬────┘
  (禁借)         │            │
            培训+高价值    无门槛+低价值
                 ↓            ↓
               C/B类         A类
```

### 1.3 借用期限管理

| 工具类别 | 标准借期 | 可续借次数 | 单次续借时长 | 最长借用期 |
|---------|---------|-----------|------------|-----------|
| A类 | 14天 | 2次 | 7天 | 28天 |
| B类 | 7天 | 1次 | 3天 | 10天 |
| C类 | 3天 | 1次 | 2天 | 5天 |

**续借规则**

1. 需在到期前申请续借
2. 无其他人预约排队时方可续借
3. 续借期间仍需遵守使用规范
4. 超过最长借用期需归还后重新借用

## 3. 逾期管理

### 3.1 逾期预防机制

| 时间节点 | 操作 | 方式 |
|---------|------|------|
| 到期前3天 | 首次提醒 | 短信/微信推送 |
| 到期前1天 | 再次提醒 | 短信/微信/电话 |
| 到期当天 | 最后提醒 | 电话+短信 |

### 3.2 逾期处理梯度

| 逾期阶段 | 逾期时长 | 处理措施 | 后果 |
|---------|---------|---------|------|
| 宽限期 | 1-3天 | 每日提醒，限制继续借用 | 记录警告 |
| 轻度逾期 | 4-7天 | 暂停借用权30天 | 影响信用 |
| 中度逾期 | 8-14天 | 暂停借用权90天，正式催还 | 降低会员等级 |
| 严重逾期 | 15-30天 | 暂停会员资格，押金抵扣 | 限制入会 |
| 恶意逾期 | 30天+ | 按遗失处理，追究责任 | 法律手段 |

### 3.3 逾期催还话术

**首次提醒（到期前3天）**

```

```

## 4. 损坏与赔偿

### 4.1 损坏类型判定

| 损坏类型 | 定义 | 示例 | 处理原则 |
|---------|------|------|---------|
| 正常损耗 | 正常使用导致的磨损 | 钻头变钝、电池续航下降 | 不需赔偿 |
| 轻微损坏 | 不影响主要功能的小问题 | 外壳划痕、配件丢失 | 承担维修/配件费用 |
| 一般损坏 | 影响使用但可维修 | 功能故障、部件损坏 | 承担维修费用 |
| 严重损坏 | 无法维修或维修不经济 | 核心部件损坏 | 赔偿重置成本 |
| 遗失 | 工具无法归还 | 丢失、被盗 | 赔偿重置成本 |

### 4.2 赔偿金额计算

```
赔偿金额计算规则:

1. 正常损耗：0元

2. 轻微损坏：实际维修/配件费用

3. 一般损坏：维修费用
   - 若维修费用 > 50%重置成本，按严重损坏处理

4. 严重损坏/遗失：
   - 购买时间 < 1年：100%重置成本
   - 购买时间 1-2年：80%重置成本
   - 购买时间 2-3年：60%重置成本
   - 购买时间 > 3年：40%重置成本

5. 最低赔偿：不低于工具评估残值
```

### 4.3 损坏处理流程

| 步骤 | 操作 | 负责人 |
|------|------|-------|
| 1 | 归还时发现损坏 | 管理员 |
| 2 | 记录损坏情况（文字+照片） | 管理员 |
| 3 | 与借用人确认损坏事实 | 管理员+借用人 |
| 4 | 评估损坏类型和程度 | 管理员/技术人员 |
| 5 | 确定处理方式（维修/赔偿） | 管理员 |
| 6 | 计算赔偿金额（如需） | 管理员 |
| 7 | 与借用人沟通确认 | 管理员 |
| 8 | 执行赔偿/维修 | 借用人/管理员 |
| 9 | 结案记录 | 管理员 |

### 4.4 争议处理机制

**争议常见情形**

| 情形 | 处理原则 |
|------|---------|
| 损坏责任认定不清 | 借出时双方确认的状态记录为准 |
| 对损坏程度有异议 | 第三方技术评估 |
| 对赔偿金额有异议 | 提供市场比价参考 |
| 认为是正常损耗 | 参照同类工具正常使用状态 |

**申诉流程**

1. 借用人书面提出申诉（含理由和证据）
2. 管理员复核相关记录
3. 必要时请第三方评估
4. 空间负责人做出最终决定
5. 如仍有异议，可提交理事会裁决

## 5. 押金管理

### 5.1 押金制度设计

| 工具类别 | 押金标准 | 收取方式 | 退还时间 |
|---------|---------|---------|---------|
| A类 | 免押金 | - | - |
| B类 | 工具价值30% | 现金/转账/线上支付 | 归还确认后当日 |
| C类 | 工具价值50% | 现金/转账/线上支付 | 归还确认后当日 |

**押金替代方案**

| 方案 | 适用情形 | 操作方式 |
|------|---------|---------|
| 信用免押 | 核心会员、贡献者 | 累计借用记录良好 |
| 担保免押 | 机构会员、团体借用 | 机构担保函 |
| 积分抵押 | 积分丰富的会员 | 冻结相应积分 |

### 5.2 押金管理规范

**押金收取**
- 出具押金收据，注明工具、金额、借用人
- 押金专户管理，与运营资金分离
- 电子支付留存交易记录

**押金退还**
- 工具完好归还后及时退还
- 有损坏需扣除时，明细告知
- 退还方式与收取方式一致

**押金抵扣**
- 逾期费用抵扣（如有）
- 损坏赔偿抵扣
- 抵扣后余额退还
- 不足部分补缴

## 7. 特殊情况处理

### 7.1 紧急借用

| 情形 | 处理方式 |
|------|---------|
| 项目紧急需要 | 简化审批，事后补手续 |
| 非开放时间 | 授权人远程审批+临时取用 |
| 关键活动期间 | 预留专用工具不外借 |

### 7.2 长期借用

| 情形 | 适用对象 | 管理方式 |
|------|---------|---------|
| 项目周期长 | 进行中的长期项目 | 签订长期借用协议 |

| 远程会员 | 距离较远的会员 | 增加借用期限 |

- 明确借用期限和阶段性归还检查
- 定期状态报告要求
- 中途收回条款
- 保险和责任条款

### 7.3 团体借用

| 项目 | 要求 |

| 使用范围 | 明确活动/项目范围 |

## 8. 日常运营管理

### 8.1 值班管理员职责

| 职责 | 具体工作 |
|------|---------|
| 借还办理 | 办理借出和归还手续 |
| 工具检查 | 借出前和归还后检查状态 |
| 咨询解答 | 回答借用规则和工具使用问题 |
| 记录更新 | 及时更新借用记录 |
| 问题上报 | 发现异常情况及时上报 |
| 库存管理 | 定期盘点，补充耗材 |

### 8.2 定期检查制度

| 检查类型 | 频率 | 内容 | 负责人 |
|---------|------|------|-------|
| 日常检查 | 每日 | 当日借还记录核对 | 值班管理员 |
| 周度检查 | 每周 | 逾期跟进、热门工具状态 | 管理员 |
| 月度盘点 | 每月 | 全部工具清点、状态检查 | 管理员 |
| 季度维护 | 每季 | 工具保养维护、报废评估 | 技术人员 |

### 8.3 数据分析应用

**关键指标监控**

| 指标 | 计算方式 | 用途 |

| 逾期率 | 逾期次数/总借用次数 | 评估管理效果 |
| 损坏率 | 损坏次数/总借用次数 | 评估使用规范 |
| 热门工具 | 按借用次数排序 | 指导采购补充 |

## 常见问题

<details>
<summary>如何平衡外借和现场使用需求？</summary>

**供给侧管理**
1. 热门工具配置多套
2. 设置"仅现场"备用工具
3. 外借数量上限控制

**需求侧管理**
1. 预约制管理热门工具
2. 活动期间暂停外借
3. 优先保障现场活动需求

**动态调节**
1. 监控借用数据
2. 根据使用规律调整
3. 特殊时期临时调整政策
</details>

<details>
<summary>借用人声称归还但系统未记录怎么办？</summary>

**核实流程**
1. 查询值班记录和监控
2. 联系当时值班管理员
3. 检查工具实际位置

**预防措施**
1. 归还必须当面确认
2. 出具归还凭证
3. 重要工具双人确认

**处理原则**
1. 有证据支持：更正记录
2. 无法确认：双方协商
3. 系统记录为准（明确告知借用人）
</details>

<details>
<summary>如何处理借用人拒绝赔偿？</summary>

**沟通协商**
1. 再次说明借用协议条款
2. 提供损坏证据
3. 解释赔偿计算依据
4. 听取对方诉求

**协商方案**
1. 允许分期赔偿
2. 以服务时长折抵
3. 以同等工具替代

**最终手段**
1. 暂停会员资格
2. 押金抵扣（如有）
3. 必要时法律途径
</details>

<details>
<summary>如何激励会员按时归还？</summary>

**正向激励**
1. 良好借用记录积累信用
2. 信用高可享免押金
3. 信用高可借更多/更久
4. 年度"信用之星"表彰

**规则透明**
1. 明确逾期后果
2. 多渠道提醒
3. 给予合理宽限期

**便利归还**
1. 多时段开放
2. 代还机制（如有）
3. 快速归还通道
</details>
