# 思维的延伸 · L02

**Extended Mind: Conditions and Mechanisms of Cognitive Integration**

> 一句话立场：心智不是颅骨内的事件，而是脑-身-工具-社会构成的耦合系统。L02 关注「认知边界为何在外」，M05 关注「工具如何成为脚手架」。

---

## Core 论断

L02 的核心可压缩为四条理论命题。它们是 M05 工具实践层的认知基础。

1. **认知系统的边界由功能耦合定义。** Clark 与 Chalmers 的奇偶原则（Parity Principle）把"在世界中可靠承担认知功能的资源"算作认知系统的一部分。
2. **延伸通过身体、工具、符号、社会四条路径发生。** 具身认知、神经耦合、外部表征、分布式认知——四条路径都需要被显式设计。
3. **GenAI 重新定义了延伸的形态。** LLM 不仅延伸记忆与计算，还放大了语义可能性，引入"生成性不确定性"。
4. **延伸认知卫生（cognitive hygiene）是新教育核心。** 在工具值得多少信任、何时该卸载、何时该自洽，都是必须从小训练的元能力。

## 关键证据

近一年的 GenAI-延伸心智文献把 L02 从哲学命题升级为现实方法论：

**人类一直是混合思考系统。** Clark 在 *Nature Communications* 把延伸心智论扩展到 GenAI 时代：人不是"颅骨中的脑"，而是脑-身-工具-社会编织而成的混合思考系统；GenAI 不是异常入侵，而是延续多年来的工具延伸传统<Cite id="L02-2025-NatureComm-Clark" />。Clark 同时警告：必须从早期培养"延伸认知卫生"——在 GenAI 时代尤其重要。

**LLM 是延伸心智的具体技术实例。** Smart 在 *Synthese* 用 RAG 技术构建"Digital Andy"——把 Clark 自己的著作变成可问答的外部记忆——证明 LLM 的认知能力部分依赖"在线计算生态"而非纯内部参数<Cite id="L02-2025-Synthese-DigitalAndy" />。这给"工具是认知一部分"提供了可工程化的现实案例。

**「放大心智」是延伸心智的下一步。** Zenodo 上的 *From extended to amplified* 提出 LLM 不仅延伸还放大认知：从 Parity Principle 推广到 Amplification Principle；Neo-Otto 与 Generative Inga 思想实验展示了"导航生成可能性空间"的新认知模式<Cite id="L02-2026-Zenodo-AmplifiedMind" />。

**经典 1998 文本仍是基石。** Clark 与 Chalmers 的奇偶原则、Otto 与 Inga 的笔记本思想实验、对语言作为"外部认知设备"的论述<Cite id="L02-1998-AnalysisClark" />，是所有后续讨论的共同起点。

**M05 工具实践层证据可直接复用。** 关于 LLM 脚手架<Cite id="M05-2026-CompEduAI-LLMScaffolding" />、认知卸载<Cite id="M05-2025-PMC-CogOffload" />、批判性思维退化<Cite id="M05-2025-Microsoft-CritThink" />等实证研究为 L02 的"延伸认知卫生"主张提供具体证据。
