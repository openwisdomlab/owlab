# 可持续技术

**Sustainable Technology: Regenerative Design, Circular Economy & Green Maker Practices**

---

## 核心问题

> 技术的环境代价常被忽视——从稀土开采到电子垃圾。如何在技术教育中培养可持续意识？创客运动能否成为循环经济的实践场？

```
技术的生命周期影响

原料开采           制造           使用           废弃
────────────────────────────────────────────────────────
  矿山开采       工厂排放      能源消耗      电子垃圾
  生态破坏       水污染        计划淘汰      有毒物质
  冲突矿物       碳排放        过度包装      资源浪费

    ▼              ▼              ▼              ▼

【隐藏成本】   【隐藏成本】   【可见使用】   【隐藏成本】

设计挑战: 让全生命周期影响可见，并在设计中最小化
```

## 一、循环经济与设计

### 1.1 从线性到循环

```
线性经济 vs 循环经济

线性经济 (Take-Make-Dispose):
提取 ──► 制造 ──► 使用 ──► 废弃
 │                           │
 └───────── 资源流失 ─────────┘

循环经济 (循环利用):

         ┌──────────────────────┐
         │                      │
         ▼                      │
     ┌────────┐             ┌───┴───┐
     │  设计  │             │ 再制造 │
     └───┬────┘             └───┬───┘
         │                      │
         ▼                      ▲
     ┌────────┐             ┌───┴───┐
     │  生产  │             │  回收  │
     └───┬────┘             └───┬───┘
         │                      │
         ▼                      ▲
     ┌────────┐             ┌───┴───┐
     │  分销  │ ──────────► │  收集  │
     └───┬────┘             └───────┘
         │
         ▼
     ┌────────┐
     │  使用  │ ◄── 维修、共享、延长使用
     └────────┘
```

### 1.2 循环设计原则

**设计策略**：

| 策略 | 描述 | 例子 |
|-----|------|------|
| 为耐久设计 | 延长产品寿命 | 模块化设计、可升级 |
| 为维修设计 | 易于修理 | 可拆卸、标准件 |
| 为再制造设计 | 部件可重用 | 逆向物流、翻新 |
| 为回收设计 | 材料可分离 | 单一材料、清晰标识 |
| 为分解设计 | 生物降解 | 天然材料、无毒 |

### 1.3 Right to Repair 运动

**核心主张**：
- 用户有权修理自己购买的产品
- 制造商应提供维修手册和零件
- 反对"计划淘汰"策略

**教育意义**：
- 修理是学习的机会
- 理解"技术是人造物"
- 培养创造者而非消费者心态
- 资源节约意识

**OWL 实践**：

```
维修工作坊设计

类型 1: 维修咖啡馆 (Repair Café)
├── 社区成员带来坏物品
├── 志愿者协助维修
├── 学习发生在修理过程中
└── 社区建设 + 技能传递

类型 2: 拆解工作坊
├── 有计划地拆解旧设备
├── 学习内部结构
├── 材料分类与再利用
└── 揭开"黑盒"

类型 3: 升级改造 (Upcycling)
├── 旧物件改造为新用途
├── 创意与实用结合
├── 审美与功能重新定义
└── 减少浪费 + 创造价值
```

#### 菌丝体材料 (Mycelium)

**什么是菌丝体？**
真菌的根状网络结构，可以在有机废料上生长。

**应用**：
- 包装材料（替代聚苯乙烯）
- 建筑隔热材料
- 皮革替代品
- 家具材料

**DIY 菌丝体项目**：

```
菌丝体材料制作流程

Day 1-3: 准备
├── 基质准备（锯末、稻壳等）
├── 灭菌处理
└── 接种菌种

Day 3-10: 生长
├── 控制温湿度
├── 菌丝体扩展
└── 填充模具

Day 10-14: 成型
├── 停止生长（干燥/加热）
├── 脱模
└── 后处理
```

### 3.2 能量采集 (Energy Harvesting)

**从环境中获取能量的技术**：

| 能量源 | 技术 | 功率级别 | 应用 |
|-------|------|---------|------|
| 光 | 太阳能电池 | mW-W | 传感器、小设备 |
| 振动 | 压电材料 | μW-mW | 可穿戴、传感器 |
| 温差 | 热电效应 | μW-mW | 工业余热、体温 |
| RF | 射频能量采集 | μW | WiFi 供电传感器 |
| 生物 | 微生物燃料电池 | μW | 环境监测 |

**DIY 项目**：
- 水果电池（柠檬、土豆）
- 压电地砖（走路发电）
- Peltier 发电机（温差发电）
- 太阳能充电站

### 3.3 低功耗计算

**设计原则**：
- 选择低功耗微控制器（ESP32, nRF52）
- 睡眠模式优化
- 边缘计算减少通信
- 事件驱动而非轮询

**TinyML 教育**：
- 在微控制器上运行机器学习
- 低功耗 AI 应用
- Edge Impulse 等平台

全球电子垃圾流向

  消费                        ↓

| 升级 | 模块化升级、旧件再利用 |

### 4.3 冲突矿物与供应链伦理

**什么是冲突矿物？**
来自冲突地区、资助武装冲突的矿物（钴、钽、锡、钨、金）。

**存在于**：
- 智能手机
- 笔记本电脑
- 游戏机
- 几乎所有电子设备

**教育机会**：
- 供应链透明度研究
- 替代材料探索
- 消费伦理讨论
- 系统思维培养

```cpp
/*
 * OWL 智能能源监测站
 * 功能：三相电流监测、电压监测、功率计算、数据上传
 */

#include <WiFi.h>
#include <HTTPClient.h>
#include <Wire.h>
#include <Adafruit_SSD1306.h>
#include "EmonLib.h"  // 开源能源监测库

// WiFi 配置
const char* ssid = "OWL_LAB";
const char* password = "your_password";

// 数据上报端点
const char* serverUrl = "http://your-server.com/api/energy";

// 显示屏配置
#define SCREEN_WIDTH 128
#define SCREEN_HEIGHT 64
Adafruit_SSD1306 display(SCREEN_WIDTH, SCREEN_HEIGHT, &Wire, -1);

// 能源监测实例
EnergyMonitor emon1, emon2, emon3;

// 引脚定义
#define CT1_PIN 34  // 电流互感器1 (ADC1_CH6)
#define CT2_PIN 35  // 电流互感器2 (ADC1_CH7)
#define CT3_PIN 32  // 电流互感器3 (ADC1_CH4)
#define VOLTAGE_PIN 33  // 电压传感器 (ADC1_CH5)

// 校准值 (需根据实际调整)
const float CT_CALIBRATION = 111.1;  // 100A/50mA 互感器
const float VOLTAGE_CALIBRATION = 234.26;

// 数据结构
struct EnergyData {
  float voltage;
  float current1, current2, current3;
  float power1, power2, power3;
  float totalPower;
  float energyKWh;
  unsigned long timestamp;
};

EnergyData data;
float accumulatedEnergy = 0;
unsigned long lastMeasurement = 0;
unsigned long lastUpload = 0;

void setup() {
  Serial.begin(115200);

  // 初始化显示屏
  if (!display.begin(SSD1306_SWITCHCAPVCC, 0x3C)) {
    Serial.println(F("SSD1306 初始化失败"));
  }
  display.clearDisplay();
  display.setTextSize(1);
  display.setTextColor(SSD1306_WHITE);
  display.setCursor(0, 0);
  display.println("OWL Energy Monitor");
  display.println("Initializing...");
  display.display();

  // 初始化能源监测
  emon1.current(CT1_PIN, CT_CALIBRATION);
  emon2.current(CT2_PIN, CT_CALIBRATION);
  emon3.current(CT3_PIN, CT_CALIBRATION);

  // 连接 WiFi
  WiFi.begin(ssid, password);
  int attempts = 0;
  while (WiFi.status() != WL_CONNECTED && attempts < 20) {
    delay(500);
    Serial.print(".");
    attempts++;
  }

  if (WiFi.status() == WL_CONNECTED) {
    Serial.println("\nWiFi 已连接");
    Serial.println(WiFi.localIP());
  }

  lastMeasurement = millis();
}

void loop() {
  // 每秒测量一次
  if (millis() - lastMeasurement >= 1000) {
    measureEnergy();
    updateDisplay();
    lastMeasurement = millis();
  }

  // 每分钟上报一次数据
  if (millis() - lastUpload >= 60000) {
    uploadData();
    lastUpload = millis();
  }
}

void measureEnergy() {
  // 读取电压 (简化版，实际需要更复杂的采样)
  int voltageRaw = analogRead(VOLTAGE_PIN);
  data.voltage = map(voltageRaw, 0, 4095, 0, 250);

  // 读取三相电流
  data.current1 = emon1.calcIrms(1480);
  data.current2 = emon2.calcIrms(1480);
  data.current3 = emon3.calcIrms(1480);

  // 计算功率 (假设功率因数为0.95)
  const float PF = 0.95;
  data.power1 = data.voltage * data.current1 * PF;
  data.power2 = data.voltage * data.current2 * PF;
  data.power3 = data.voltage * data.current3 * PF;
  data.totalPower = data.power1 + data.power2 + data.power3;

  // 累计电量 (kWh)
  accumulatedEnergy += data.totalPower / 3600000.0;  // W -> kWh
  data.energyKWh = accumulatedEnergy;
  data.timestamp = millis();

  // 输出到串口
  Serial.printf("V: %.1fV | I: %.2f/%.2f/%.2f A | P: %.1fW | E: %.3f kWh\n",
    data.voltage, data.current1, data.current2, data.current3,
    data.totalPower, data.energyKWh);
}

void updateDisplay() {
  display.clearDisplay();
  display.setCursor(0, 0);

  display.println("=== OWL Energy ===");
  display.printf("Voltage: %.1f V\n", data.voltage);
  display.printf("Current: %.2f A\n", data.current1 + data.current2 + data.current3);
  display.printf("Power: %.0f W\n", data.totalPower);
  display.printf("Energy: %.2f kWh\n", data.energyKWh);

  // 绘制功率条形图
  int barWidth = map(data.totalPower, 0, 3000, 0, 128);
  display.fillRect(0, 56, barWidth, 8, SSD1306_WHITE);

  display.display();
}

void uploadData() {
  if (WiFi.status() != WL_CONNECTED) return;

  HTTPClient http;
  http.begin(serverUrl);
  http.addHeader("Content-Type", "application/json");

  // 构建 JSON 数据
  String jsonData = "{";
  jsonData += "\"lab_id\":\"owl-lab-001\",";
  jsonData += "\"voltage\":" + String(data.voltage) + ",";
  jsonData += "\"current_total\":" + String(data.current1 + data.current2 + data.current3) + ",";
  jsonData += "\"power\":" + String(data.totalPower) + ",";
  jsonData += "\"energy_kwh\":" + String(data.energyKWh);
  jsonData += "}";

  int httpCode = http.POST(jsonData);
  if (httpCode > 0) {
    Serial.printf("数据上报成功: %d\n", httpCode);
  }
  http.end();
}
```

**完整 Arduino 代码**：

```cpp
/*
 * OWL 智能水培控制器
 * 功能：pH/TDS/温度监测、自动营养液调节、水循环控制
 */

#include <Wire.h>
#include <LiquidCrystal_I2C.h>
#include <OneWire.h>
#include <DallasTemperature.h>

// 引脚定义
#define TDS_PIN A0
#define PH_PIN A1
#define TEMP_PIN 2
#define PUMP_RELAY 3      // 循环水泵
#define NUTRIENT_A_RELAY 4  // 营养液A
#define NUTRIENT_B_RELAY 5  // 营养液B
#define PH_DOWN_RELAY 6   // pH调节液

// 目标参数 (根据作物调整)
const float TARGET_TDS = 800;    // ppm (叶菜类)
const float TDS_TOLERANCE = 100;
const float TARGET_PH = 6.0;     // 理想pH
const float PH_TOLERANCE = 0.3;
const float TARGET_TEMP = 22.0;  // 理想温度

// LCD
LiquidCrystal_I2C lcd(0x27, 16, 2);

// 温度传感器
OneWire oneWire(TEMP_PIN);
DallasTemperature tempSensor(&oneWire);

// 状态变量
float currentTDS = 0;
float currentPH = 0;
float currentTemp = 0;
unsigned long lastCheck = 0;
unsigned long pumpStartTime = 0;
bool pumpRunning = false;

// pH 校准值 (需要实际校准)
const float PH_OFFSET = 0.0;
const float PH_SLOPE = 3.5;

// TDS 校准值
const float TDS_FACTOR = 0.5;

void setup() {
  Serial.begin(9600);

  // 初始化 LCD
  lcd.init();
  lcd.backlight();
  lcd.setCursor(0, 0);
  lcd.print("OWL Hydroponics");
  lcd.setCursor(0, 1);
  lcd.print("Initializing...");

  // 初始化温度传感器
  tempSensor.begin();

  // 初始化继电器 (低电平触发)
  pinMode(PUMP_RELAY, OUTPUT);
  pinMode(NUTRIENT_A_RELAY, OUTPUT);
  pinMode(NUTRIENT_B_RELAY, OUTPUT);
  pinMode(PH_DOWN_RELAY, OUTPUT);

  // 关闭所有继电器
  digitalWrite(PUMP_RELAY, HIGH);
  digitalWrite(NUTRIENT_A_RELAY, HIGH);
  digitalWrite(NUTRIENT_B_RELAY, HIGH);
  digitalWrite(PH_DOWN_RELAY, HIGH);

  delay(2000);
}

void loop() {
  // 每10秒检测一次
  if (millis() - lastCheck >= 10000) {
    readSensors();
    displayData();
    adjustNutrients();
    lastCheck = millis();
  }

  // 水泵循环控制 (每15分钟运行5分钟)
  managePumpCycle();
}

void readSensors() {
  // 读取温度
  tempSensor.requestTemperatures();
  currentTemp = tempSensor.getTempCByIndex(0);

  // 读取 TDS
  int tdsRaw = analogRead(TDS_PIN);
  float tdsVoltage = tdsRaw * 5.0 / 1024.0;
  // 温度补偿
  float compensationCoefficient = 1.0 + 0.02 * (currentTemp - 25.0);
  float compensatedVoltage = tdsVoltage / compensationCoefficient;
  currentTDS = (133.42 * pow(compensatedVoltage, 3) -
                255.86 * pow(compensatedVoltage, 2) +
                857.39 * compensatedVoltage) * TDS_FACTOR;

  // 读取 pH
  int phRaw = analogRead(PH_PIN);
  float phVoltage = phRaw * 5.0 / 1024.0;
  currentPH = PH_SLOPE * phVoltage + PH_OFFSET;

  // 串口输出
  Serial.print("Temp: "); Serial.print(currentTemp);
  Serial.print("°C | TDS: "); Serial.print(currentTDS);
  Serial.print(" ppm | pH: "); Serial.println(currentPH);
}

void displayData() {
  lcd.clear();

  // 第一行：温度和TDS
  lcd.setCursor(0, 0);
  lcd.print("T:");
  lcd.print(currentTemp, 1);
  lcd.print("C ");
  lcd.print("TDS:");
  lcd.print((int)currentTDS);

  // 第二行：pH和状态
  lcd.setCursor(0, 1);
  lcd.print("pH:");
  lcd.print(currentPH, 1);
  lcd.print(" ");

  // 状态指示
  if (abs(currentTDS - TARGET_TDS) <= TDS_TOLERANCE &&
      abs(currentPH - TARGET_PH) <= PH_TOLERANCE) {
    lcd.print("OK");
  } else {
    lcd.print("ADJ");
  }
}

void adjustNutrients() {
  // TDS 过低，添加营养液
  if (currentTDS < TARGET_TDS - TDS_TOLERANCE) {
    Serial.println("TDS 过低，添加营养液...");

    // 交替添加 A/B 液
    digitalWrite(NUTRIENT_A_RELAY, LOW);
    delay(2000);  // 注入2秒
    digitalWrite(NUTRIENT_A_RELAY, HIGH);
    delay(500);

    digitalWrite(NUTRIENT_B_RELAY, LOW);
    delay(2000);
    digitalWrite(NUTRIENT_B_RELAY, HIGH);

    Serial.println("营养液添加完成");
  }

  // pH 过高，添加pH降低液
  if (currentPH > TARGET_PH + PH_TOLERANCE) {
    Serial.println("pH 过高，调节中...");

    digitalWrite(PH_DOWN_RELAY, LOW);
    delay(1000);  // 注入1秒
    digitalWrite(PH_DOWN_RELAY, HIGH);

    Serial.println("pH调节完成");
  }

  // pH 过低警告 (需要手动添加pH升高液)
  if (currentPH < TARGET_PH - PH_TOLERANCE) {
    Serial.println("警告：pH 过低，请手动检查！");
    lcd.setCursor(10, 1);
    lcd.print("pH低!");
  }
}

void managePumpCycle() {
  unsigned long currentMinute = (millis() / 60000) % 20;

  if (currentMinute < 5 && !pumpRunning) {
    // 开启水泵
    digitalWrite(PUMP_RELAY, LOW);
    pumpRunning = true;
    Serial.println("水泵开启");
  } else if (currentMinute >= 5 && pumpRunning) {
    // 关闭水泵
    digitalWrite(PUMP_RELAY, HIGH);
    pumpRunning = false;
    Serial.println("水泵关闭");
  }
}
```

    def add_entry(self, category: str, activity: str,
                  quantity: float, unit: str, notes: str = "") -> CarbonEntry:
        """添加碳排放记录"""
        factor_key = activity.lower().replace(" ", "_")
        factor = EMISSION_FACTORS.get(factor_key, 0)

        if factor == 0:
            print(f"警告: 未找到 '{activity}' 的排放因子，使用 0")

        co2e = quantity * factor

        entry = CarbonEntry(
            date=datetime.now().strftime("%Y-%m-%d"),
            category=category,
            activity=activity,
            quantity=quantity,
            unit=unit,
            emission_factor=factor,
            co2e=co2e,
            notes=notes
        )
        self.entries.append(entry)
        return entry

    def get_summary(self, days: int = 30) -> Dict:
        """获取指定天数内的摘要"""
        cutoff = datetime.now() - timedelta(days=days)
        recent = [e for e in self.entries
                  if datetime.strptime(e.date, "%Y-%m-%d") >= cutoff]

        by_category = {}
        for entry in recent:
            cat = entry.category
            if cat not in by_category:
                by_category[cat] = 0
            by_category[cat] += entry.co2e

        total = sum(by_category.values())

        return {
            "period_days": days,
            "total_co2e_kg": round(total, 2),
            "by_category": {k: round(v, 2) for k, v in by_category.items()},
            "daily_average": round(total / days, 2) if days > 0 else 0,
            "equivalent_trees": round(total / 21, 1),  # 一棵树年吸收约21kg CO2
        }

    def generate_report(self, output_path: str = "carbon_report.png"):
        """生成可视化报告"""
        summary = self.get_summary(30)

        fig, axes = plt.subplots(2, 2, figsize=(12, 10))
        fig.suptitle(f'{self.lab_name} 碳足迹报告 (过去30天)',
                     fontsize=14, fontweight='bold')

        # 1. 分类饼图
        ax1 = axes[0, 0]
        categories = list(summary['by_category'].keys())
        values = list(summary['by_category'].values())
        colors = ['#FF6B6B', '#4ECDC4', '#45B7D1', '#96CEB4', '#FFEAA7']
        if values:
            ax1.pie(values, labels=categories, autopct='%1.1f%%',
                    colors=colors[:len(categories)])
        ax1.set_title('排放来源分布')

        # 2. 与目标对比
        ax2 = axes[0, 1]
        target_cats = list(self.monthly_targets.keys())
        target_vals = list(self.monthly_targets.values())
        actual_vals = [summary['by_category'].get(c, 0) for c in target_cats]

        x = range(len(target_cats))
        width = 0.35
        ax2.bar([i - width/2 for i in x], target_vals, width,
                label='目标', color='#95E1D3')
        ax2.bar([i + width/2 for i in x], actual_vals, width,
                label='实际', color='#F38181')
        ax2.set_ylabel('kg CO2e')
        ax2.set_xticks(x)
        ax2.set_xticklabels(target_cats, rotation=45)
        ax2.legend()
        ax2.set_title('目标 vs 实际')

        # 3. 每日趋势
        ax3 = axes[1, 0]
        daily_data = {}
        for entry in self.entries:
            if entry.date not in daily_data:
                daily_data[entry.date] = 0
            daily_data[entry.date] += entry.co2e

        if daily_data:
            dates = sorted(daily_data.keys())[-14:]  # 最近14天
            daily_values = [daily_data.get(d, 0) for d in dates]
            ax3.plot(range(len(dates)), daily_values,
                     marker='o', color='#6C5CE7')
            ax3.fill_between(range(len(dates)), daily_values,
                            alpha=0.3, color='#6C5CE7')
            ax3.set_xticks(range(0, len(dates), 2))
            ax3.set_xticklabels([dates[i][-5:] for i in range(0, len(dates), 2)],
                               rotation=45)
        ax3.set_ylabel('kg CO2e')
        ax3.set_title('每日排放趋势')

        # 4. 统计摘要
        ax4 = axes[1, 1]
        ax4.axis('off')
        summary_text = f"""
        📊 碳足迹摘要
        ─────────────────
        总排放: {summary['total_co2e_kg']} kg CO2e
        日均排放: {summary['daily_average']} kg CO2e

        🌳 相当于需要 {summary['equivalent_trees']} 棵树
           吸收一年才能抵消

        💡 减排建议:
        • 优先使用公共交通
        • 选择再生材料进行制作
        • 优化设备能效
        • 减少食物浪费
        """
        ax4.text(0.1, 0.5, summary_text, fontsize=11,
                verticalalignment='center', fontfamily='monospace',
                bbox=dict(boxstyle='round', facecolor='#DFE6E9', alpha=0.8))

        plt.tight_layout()
        plt.savefig(output_path, dpi=150, bbox_inches='tight')
        plt.close()

        print(f"报告已保存至: {output_path}")
        return output_path

    def save_data(self, filepath: str = "carbon_data.json"):
        """保存数据到文件"""
        data = {
            "lab_name": self.lab_name,
            "entries": [vars(e) for e in self.entries],
            "targets": self.monthly_targets
        }
        with open(filepath, 'w', encoding='utf-8') as f:
            json.dump(data, f, ensure_ascii=False, indent=2)

    def load_data(self, filepath: str = "carbon_data.json"):
        """从文件加载数据"""
        with open(filepath, 'r', encoding='utf-8') as f:
            data = json.load(f)
        self.lab_name = data.get("lab_name", self.lab_name)
        self.monthly_targets = data.get("targets", self.monthly_targets)
        self.entries = [CarbonEntry(**e) for e in data.get("entries", [])]
