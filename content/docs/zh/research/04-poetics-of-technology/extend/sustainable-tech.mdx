# 可持续技术

**Sustainable Technology: Regenerative Design, Circular Economy & Green Maker Practices**

---

## 核心问题

> 技术的环境代价常被忽视——从稀土开采到电子垃圾。如何在技术教育中培养可持续意识？创客运动能否成为循环经济的实践场？

```
技术的生命周期影响

原料开采           制造           使用           废弃
────────────────────────────────────────────────────────
  矿山开采       工厂排放      能源消耗      电子垃圾
  生态破坏       水污染        计划淘汰      有毒物质
  冲突矿物       碳排放        过度包装      资源浪费

    ▼              ▼              ▼              ▼

【隐藏成本】   【隐藏成本】   【可见使用】   【隐藏成本】

设计挑战: 让全生命周期影响可见，并在设计中最小化
```

## 一、循环经济与设计

### 1.1 从线性到循环

```
线性经济 vs 循环经济

线性经济 (Take-Make-Dispose):
提取 ──► 制造 ──► 使用 ──► 废弃
 │                           │
 └───────── 资源流失 ─────────┘

循环经济 (循环利用):

         ┌──────────────────────┐
         │                      │
         ▼                      │
     ┌────────┐             ┌───┴───┐
     │  设计  │             │ 再制造 │
     └───┬────┘             └───┬───┘
         │                      │
         ▼                      ▲
     ┌────────┐             ┌───┴───┐
     │  生产  │             │  回收  │
     └───┬────┘             └───┬───┘
         │                      │
         ▼                      ▲
     ┌────────┐             ┌───┴───┐
     │  分销  │ ──────────► │  收集  │
     └───┬────┘             └───────┘
         │
         ▼
     ┌────────┐
     │  使用  │ ◄── 维修、共享、延长使用
     └────────┘
```

### 1.2 循环设计原则

**设计策略**：

| 策略 | 描述 | 例子 |
|-----|------|------|
| 为耐久设计 | 延长产品寿命 | 模块化设计、可升级 |
| 为维修设计 | 易于修理 | 可拆卸、标准件 |
| 为再制造设计 | 部件可重用 | 逆向物流、翻新 |
| 为回收设计 | 材料可分离 | 单一材料、清晰标识 |
| 为分解设计 | 生物降解 | 天然材料、无毒 |

### 1.3 Right to Repair 运动

**核心主张**：
- 用户有权修理自己购买的产品
- 制造商应提供维修手册和零件
- 反对"计划淘汰"策略

**教育意义**：
- 修理是学习的机会
- 理解"技术是人造物"
- 培养创造者而非消费者心态
- 资源节约意识

**OWL 实践**：

```
维修工作坊设计

类型 1: 维修咖啡馆 (Repair Café)
├── 社区成员带来坏物品
├── 志愿者协助维修
├── 学习发生在修理过程中
└── 社区建设 + 技能传递

类型 2: 拆解工作坊
├── 有计划地拆解旧设备
├── 学习内部结构
├── 材料分类与再利用
└── 揭开"黑盒"

类型 3: 升级改造 (Upcycling)
├── 旧物件改造为新用途
├── 创意与实用结合
├── 审美与功能重新定义
└── 减少浪费 + 创造价值
```

#### 菌丝体材料 (Mycelium)

**什么是菌丝体？**
真菌的根状网络结构，可以在有机废料上生长。

**应用**：
- 包装材料（替代聚苯乙烯）
- 建筑隔热材料
- 皮革替代品
- 家具材料

**DIY 菌丝体项目**：

```
菌丝体材料制作流程

Day 1-3: 准备
├── 基质准备（锯末、稻壳等）
├── 灭菌处理
└── 接种菌种

Day 3-10: 生长
├── 控制温湿度
├── 菌丝体扩展
└── 填充模具

Day 10-14: 成型
├── 停止生长（干燥/加热）
├── 脱模
└── 后处理
```

```
自然启发解决方案

──────────────────────────────────────────────────────────

建筑降温                白蚁丘通风              被动式建筑通风

### 3.2 能量采集 (Energy Harvesting)

**从环境中获取能量的技术**：

| 能量源 | 技术 | 功率级别 | 应用 |
|-------|------|---------|------|
| 光 | 太阳能电池 | mW-W | 传感器、小设备 |
| 振动 | 压电材料 | μW-mW | 可穿戴、传感器 |
| 温差 | 热电效应 | μW-mW | 工业余热、体温 |
| RF | 射频能量采集 | μW | WiFi 供电传感器 |
| 生物 | 微生物燃料电池 | μW | 环境监测 |

**DIY 项目**：
- 水果电池（柠檬、土豆）
- 压电地砖（走路发电）
- Peltier 发电机（温差发电）
- 太阳能充电站

### 3.3 低功耗计算

**设计原则**：
- 选择低功耗微控制器（ESP32, nRF52）
- 睡眠模式优化
- 边缘计算减少通信
- 事件驱动而非轮询

**TinyML 教育**：
- 在微控制器上运行机器学习
- 低功耗 AI 应用
- Edge Impulse 等平台

```
全球电子垃圾流向

发达国家                   发展中国家
──────────────────────────────────────────────────
  消费                        ↓
    ↓                     非正规回收
  废弃                        ↓

问题:
- 全球每年 5000+ 万吨电子垃圾
- 仅 20% 被正规回收
- 珍贵材料流失
- 有毒物质泄漏

### 4.2 负责任的电子实践

**创客空间的材料责任**：

| 阶段 | 负责任实践 |

| 升级 | 模块化升级、旧件再利用 |

### 4.3 冲突矿物与供应链伦理

**什么是冲突矿物？**
来自冲突地区、资助武装冲突的矿物（钴、钽、锡、钨、金）。

**存在于**：
- 智能手机
- 笔记本电脑
- 游戏机
- 几乎所有电子设备

**教育机会**：
- 供应链透明度研究
- 替代材料探索
- 消费伦理讨论
- 系统思维培养
