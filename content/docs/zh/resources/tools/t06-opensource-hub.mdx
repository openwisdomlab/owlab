# T06 开源资源库

**Open Source Hub** | ✅ 已上线

---

## 定位

解决"不知道有什么好资源"的发现难题。汇集社区精选的开源硬件、软件、课程资源，让每个创客都能找到趁手的工具。

<Callout type="info">
**核心价值**：将 [M05 工具与资产](/docs/core/05-tools) 的开源生态和 [M04 课程与项目](/docs/core/04-programs) 的学习资源系统化整理，降低探索门槛。
</Callout>

---

## 核心功能

| 功能 | 描述 |
|------|------|
| **分类导航** | 硬件、软件、课程、社区四大类 |
| **难度分级** | 入门、进阶、专业三级标注 |
| **场景推荐** | 根据项目需求推荐资源组合 |
| **社区评价** | 真实用户的使用评价和技巧 |
| **更新追踪** | 追踪开源项目的最新动态 |

- 你的使用体验和评价
