# T03 游戏工坊

**Game Workshop** | 📅 计划中

---

## 定位

解决"学习无趣"的动力难题。用游戏化思维重新设计学习体验，让每个学习任务都变成一场有趣的冒险。

<Callout type="info">
**核心价值**：将 [M04 航线](/docs/core/04-programs) 的学习设计原则和 [M09 星轨](/docs/core/09-assessment) 的成长可视化方法游戏化，激发内在学习动力。
</Callout>

---

## 核心功能

| 功能 | 描述 |
|------|------|
| **学习路径地图** | 可视化的学习进度和分支选择 |
| **任务设计器** | 将学习目标转化为游戏任务 |
| **徽章系统** | 自动生成成就徽章和奖励 |
| **排行榜** | 健康竞争的激励机制 |
| **彩蛋设计** | 隐藏任务和惊喜发现 |

---

## 游戏化元素

基于游戏化学习研究：

| 元素 | 作用 | 设计原则 |
|------|------|---------|
| **积分** | 即时反馈 | 每个小行动都有回报 |
| **徽章** | 成就认可 | 可视化里程碑 |
| **排行榜** | 社交激励 | 健康竞争、避免焦虑 |
| **故事线** | 情境沉浸 | 赋予任务意义 |
| **随机奖励** | 惊喜感 | 保持探索热情 |

---

## 使用流程

```text
选择学习主题 → 设计任务链 → 配置奖励体系 → 发布给学习者 → 追踪进度
```

### 步骤详解

1. **选择主题**：确定要游戏化的学习内容
2. **拆解任务**：将学习目标分解为任务节点
3. **设计路径**：创建主线和支线任务地图
4. **配置奖励**：设置积分、徽章、解锁内容
5. **测试优化**：试玩并调整难度曲线
