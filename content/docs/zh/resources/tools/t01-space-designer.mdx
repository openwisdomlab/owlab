# T01 AI 空间设计器

**Space Designer** | ✅ 已上线

---

## 定位

解决"不懂设计"的空间规划难题。让没有建筑设计背景的创客空间运营者也能快速设计出专业的空间布局。

<Callout type="info">
**核心价值**：将 [M03 空间与环境](/docs/core/03-space) 模块的设计原则算法化，自动生成符合神经建筑学原则的布局方案。
</Callout>

---

## 核心功能

| 功能 | 描述 |
|------|------|
| **智能布局** | 基于功能区配比原则自动生成 |
| **3D 可视化** | 实时渲染 3D 效果图 |
| **Allen 曲线分析** | 优化协作距离布局 |
| **合规检查** | 自动检查消防通道宽度 |
| **导出分享** | 导出 PDF 平面图、3D 模型文件 |

---

## 循证参数

基于 2024-2025 前沿研究：

| 空间变量 | 最优参数 | 证据来源 |
|---------|---------|---------|
| 天花板高度 | 4-5m（神经生理峰值5m） | Kim & Kim 2025 |
| 光线色温 | 4500-5500K | 沈阳建筑大学 2025 |
| 协作距离 | 小于或等于8m（Allen曲线） | Tom Allen |
| 生物亲和性 | 曲线设计优于直角 | Li & Yin 2025 |

---

## 使用流程

```text
输入空间参数 → AI 生成布局方案 → 人工调整 → 合规检查 → 导出分享
```

### 步骤详解

1. **输入参数**：空间面积、形状、门窗位置
2. **选择定位**：教育型 / 社区型 / 专业型
3. **AI 生成**：自动生成 2-3 个方案
4. **人工调整**：拖拽调整功能区位置
5. **导出使用**：PDF 平面图 + 3D 模型
