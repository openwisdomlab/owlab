# 课程设计伴侣

**Curriculum Copilot** | 📅 规划中

---

## 定位

解决"备课难"的内容生产难题。帮助导师快速将一个模糊的想法转化为完整的 PBL 项目方案。

<Callout type="info">
**核心价值**：将 [M04 远航线](/docs/core/04-programs) 的 PBL/CBL 方法论算法化，让创意种子快速生长为完整课程。
</Callout>

---

## 核心功能

| 功能 | 描述 |
|------|------|
| **结构化创意** | 教师提供模糊想法，AI 补全为完整 PBL 流程 |
| **分龄适配** | 根据年龄段自动调整难度和活动设计 |
| **量规生成** | 根据课程目标自动生成评价维度表 |
| **材料清单** | 自动生成项目所需的材料和工具清单 |

---

## 课程框架

基于 [M04 远航线](/docs/core/04-programs) 的英雄之旅架构：

| 阶段 | 内容 | AI 辅助 |
|------|------|--------|
| **召唤** | 激发好奇，提出问题 | 生成驱动问题 |
| **启程** | 探索知识，获取技能 | 推荐学习资源 |
| **考验** | 动手实践，解决挑战 | 设计活动流程 |
| **回归** | 分享成果，反思成长 | 生成评价量规 |

---

## 使用流程

```text
输入创意种子 → 选择年龄段和时长 → AI 生成课程框架 → 人工调整 → 导出教案
```

### 步骤详解

1. **输入创意**：一句话描述项目想法
2. **设置参数**：年龄段、课时数、场地条件
3. **AI 生成**：完整课程框架 + 活动设计
4. **人工优化**：调整内容和节奏
5. **导出使用**：教案 + 材料清单 + 评价量规
