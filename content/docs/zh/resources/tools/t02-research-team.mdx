# T02 虚拟课题组

**Research Team** | 📅 计划中

---

## 定位

解决"找不到伙伴"的协作难题。让不同学校、不同城市的学生也能组建虚拟课题组，共同探究真实的科研问题。

<Callout type="info">
**核心价值**：将 [M02 新航道](/docs/core/02-governance) 的跨节点协作模式和 [M04 探索线](/docs/core/04-programs) 的研究性学习方法数字化，打破地域限制。
</Callout>

---

## 核心功能

| 功能 | 描述 |
|------|------|
| **智能组队** | 基于兴趣和能力的跨校匹配 |
| **角色分配** | 自动建议课题组角色分工 |
| **任务追踪** | 可视化的任务进度管理 |
| **协作空间** | 在线文档、视频会议、资源共享 |
| **导师连接** | 匹配领域专家作为远程导师 |

---

## 课题组角色

基于 [M07 领航员](/docs/core/07-people) 的能力模型：

| 角色 | 职责 | 能力要求 |
|------|------|---------|
| **项目负责人** | 统筹协调、进度管理 | 领导力、沟通能力 |
| **研究员** | 文献调研、数据分析 | 信息素养、批判思维 |
| **工程师** | 原型制作、技术实现 | 动手能力、技术素养 |
| **设计师** | 界面设计、可视化呈现 | 审美能力、创意思维 |
| **记录员** | 文档整理、成果归档 | 写作能力、细心 |

---

## 使用流程

```text
发布课题 → 智能匹配 → 组建团队 → 分配任务 → 协作研究 → 成果展示
```

### 步骤详解

1. **发布课题**：描述研究问题和目标
2. **智能匹配**：AI 推荐合适的跨校伙伴
3. **组建团队**：确认成员、分配角色
4. **任务分解**：制定研究计划和里程碑
5. **协作研究**：在线协同完成研究任务
6. **成果展示**：生成研究报告和展示材料
