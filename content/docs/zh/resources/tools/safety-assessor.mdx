# 安全评估向导

**Safety Audit Wizard** | 📅 规划中

---

## 定位

解决"不知道哪里有隐患"的安全排查难题。通过 AI 图像识别辅助安全检查。

<Callout type="info">
**核心价值**：将 [M06 安全舱](/docs/core/06-safety) 的三层安全防御体系算法化，降低安全检查门槛。
</Callout>

---

## 核心功能

| 功能 | 描述 |
|------|------|
| **图像识别** | 上传现场照片，AI 识别潜在安全隐患 |
| **检查清单** | 生成结构化的安全检查清单 |
| **整改建议** | 针对发现的问题给出整改建议 |
| **报告生成** | 生成可存档的安全评估报告 |

---

## 检查维度

基于 [M06 安全舱](/docs/core/06-safety) 的三层防御体系：

| 层级 | 检查重点 | AI 辅助 |
|------|---------|--------|
| **物理安全** | 通道、消防、电气 | 图像识别隐患 |
| **设备安全** | 防护、标识、维护 | 检查清单生成 |
| **人员安全** | 培训、操作、应急 | 培训记录核查 |

---

## 使用流程

```text
上传现场照片 → AI 分析识别 → 查看隐患列表 → 获取整改建议 → 导出报告
```

### 步骤详解

1. **上传照片**：多角度拍摄空间照片
2. **AI 分析**：识别潜在安全隐患
3. **人工确认**：核实 AI 识别结果
4. **获取建议**：针对性整改方案
5. **导出报告**：可存档的评估报告
