# 智能规划向导

**Smart Planner** | 🚧 Beta 测试中

---

## 定位

解决"不知道买什么"的采购难题。帮助运营者快速生成符合预算的设备清单和成本预测。

<Callout type="info">
**核心价值**：将 [M05 工具与资产](/docs/core/05-tools) 的分级配置原则和 [M08 运营手册](/docs/core/08-operations) 的成本结构算法化，实现一键生成采购方案。
</Callout>

---

## 核心功能

| 功能 | 描述 |
|------|------|
| **预算匹配** | 输入预算自动生成最匹配的设备清单 |
| **一键生成** | 输入"50万预算，小学"自动配置 |
| **TCO 预测** | 预测未来 3 年的维护耗材成本 |
| **BOM 导出** | 生成可直接用于采购的物料清单 |

---

## 分级配置

基于 [M05 工具与资产](/docs/core/05-tools) 的 Lite/Pro/Ultra 体系：

| 级别 | 适用场景 | 预算范围 | 典型配置 |
|------|---------|---------|---------|
| **Lite** | 社区站点、起步阶段 | 5-15万 | 基础电子+3D打印笔+Chromebook |
| **Pro** | 标准学校节点 | 15-50万 | Arduino+3D打印机+激光切割 |
| **Ultra** | 旗舰实验室 | 50万+ | 工业级设备+CNC+专业仪器 |

---

## 使用流程

```text
输入预算和定位 → 选择配置模板 → 自定义调整 → 生成BOM和概算
```

### 步骤详解

1. **输入基础信息**：预算、面积、目标人群
2. **选择配置级别**：Lite / Pro / Ultra
3. **自定义调整**：增减特定设备
4. **生成方案**：BOM 表 + 3年 TCO 预测
5. **导出使用**：Excel 采购清单
