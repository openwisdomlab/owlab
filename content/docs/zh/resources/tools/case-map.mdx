# 全球案例地图

**Global Case Map** | 📅 规划中

---

## 定位

解决"想看看别人怎么做"的学习需求。可视化浏览全球创客空间案例。

<Callout type="info">
**核心价值**：将 [M02 新航道](/docs/core/02-governance) 的全球网络可视化，促进经验分享和社区连接。
</Callout>

---

## 核心功能

| 功能 | 描述 |
|------|------|
| **地图浏览** | 在地图上浏览全球案例分布 |
| **多维筛选** | 按地区、类型、规模筛选 |
| **案例详情** | 查看详细的案例介绍和图片 |
| **联系方式** | 获取案例负责人的联系方式 |

---

## 案例分类

| 类型 | 描述 | 典型案例 |
|------|------|---------|
| **学校型** | 嵌入学校的创客空间 | 中小学 STEM 实验室 |
| **社区型** | 开放的公共创客空间 | 图书馆、社区中心 |
| **专业型** | 面向专业领域的空间 | 生物实验室、电子工坊 |
| **企业型** | 企业内部创新空间 | 创新实验室、研发中心 |

---

## 使用流程

```text
打开地图 → 筛选条件 → 选择感兴趣的案例 → 查看详情 → 联系交流
```

### 步骤详解

1. **打开地图**：查看全球案例分布
2. **设置筛选**：地区、类型、规模、预算
3. **浏览案例**：点击标记查看简介
4. **深入了解**：查看详细介绍和图片
5. **建立联系**：获取联系方式，请求访问
