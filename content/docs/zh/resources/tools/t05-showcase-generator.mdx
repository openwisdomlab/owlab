# T05 作品展示生成器

**Showcase Generator** | 📅 计划中

---

## 定位

解决"成果难以呈现"的展示难题。让每个学习者的作品都能以专业的方式展示出来，获得应有的认可。

<Callout type="info">
**核心价值**：将 [M09 成长轨](/docs/core/09-assessment) 的成果可视化和 [M04 探索线](/docs/core/04-programs) 的项目输出自动化，让每个创作都能闪闪发光。
</Callout>

---

## 核心功能

| 功能 | 描述 |
|------|------|
| **海报生成** | AI 生成专业项目海报 |
| **摘要撰写** | 自动生成研究摘要和简介 |
| **PPT 模板** | 一键生成展示演示文稿 |
| **视频脚本** | 生成项目介绍视频脚本 |
| **社交分享** | 优化各平台分享格式 |

---

## 输出格式

支持多种展示场景：

| 场景 | 输出格式 | 用途 |
|------|---------|------|
| **学校展览** | A1海报、展板 | 科技节、开放日 |
| **线上分享** | 社交媒体图片 | 朋友圈、公众号 |
| **学术交流** | PDF摘要、PPT | 研究报告、答辩 |
| **竞赛申报** | 标准申报书 | 各类科创竞赛 |
| **档案归档** | 标准化文档 | 成长档案袋 |

---

## 使用流程

```text
上传项目资料 → 选择展示场景 → AI 生成初稿 → 人工调整 → 导出分享
```

### 步骤详解

1. **上传资料**：项目描述、图片、视频、数据
2. **选择场景**：海报、PPT、摘要等
3. **AI 生成**：自动生成专业排版
4. **人工调整**：修改文案、调整布局
5. **导出分享**：多格式导出、一键分享
