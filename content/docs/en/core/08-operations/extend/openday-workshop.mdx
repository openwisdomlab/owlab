# Open Days and Workshop Operations

> Events are the heartbeat of a maker community. They bring people through the door, create shared experiences, and build the relationships that sustain the space long-term.

---

## Core Value

### Events as Community Lifeline

Regular, well-designed events serve multiple functions:

| Function | Description |
|----------|-------------|
| **Recruitment** | Attract new members and raise awareness |
| **Engagement** | Deepen existing members' connection |
| **Learning** | Structured skill development opportunities |
| **Showcase** | Celebrate and display community achievements |
| **Culture** | Reinforce community values and norms |

### The Significance of Programmatic Events

Events create rhythm and ritual in community life. Consistent scheduling (e.g., "First Saturday open day") builds habits and expectations that sustain engagement over time.

## Open Day Operations

### Design Principles

#### Goal Alignment

| Goal | Target Audience | Success Metric |
|------|----------------|---------------|
| **Awareness** | General public | Attendance, media coverage |
| **Recruitment** | Potential members | Sign-up rate |
| **Engagement** | Current members | Participation depth |
| **Partnerships** | Organizations, sponsors | Meetings, agreements |

#### Core Experience Design

Every visitor should:
1. **See** something inspiring
2. **Touch** something (hands-on interaction)
3. **Make** something to take home
4. **Connect** with someone in the community

### Open Day Types

| Type | Frequency | Audience | Scale |
|------|-----------|----------|-------|
| **Regular open day** | Monthly | General public | 30-100 visitors |
| **Themed open day** | Quarterly | Targeted groups | 50-200 visitors |
| **Grand opening** | Annual | Everyone | 200+ visitors |
| **Partner event** | As scheduled | Specific partners | Varies |

### Regular Open Day Process

#### Pre-Event (1 Week Before)

| Task | Responsible | Deadline |
|------|------------|---------|
| Promotional materials posted | Communications | T-7 days |
| Activity stations planned | Program team | T-5 days |
| Materials and supplies prepared | Operations | T-3 days |
| Volunteer briefing | Event lead | T-1 day |
| Space cleaned and arranged | All | T-1 day |

#### Event Day

| Time | Activity | Notes |
|------|----------|-------|
| 9:00-10:00 | Setup and final preparation | All hands |
| 10:00-10:15 | Welcome and safety briefing | At entrance |
| 10:15-12:00 | Open exploration + guided activities | Multiple stations |
| 12:00-12:30 | Show-and-tell (member projects) | Central area |
| 12:30-13:00 | Feedback collection and cleanup | All hands |

#### Post-Event (Within 1 Week)

| Task | Responsible | Deadline |
|------|------------|---------|
| Photo/video processing | Communications | T+2 days |
| Follow-up with new contacts | Community | T+3 days |
| Feedback analysis | Program team | T+5 days |
| Event report | Event lead | T+7 days |

### Open Day Experience Project Design

#### Design Principle: Cyclical, Not Linear

Activities should be joinable at any time:
- No required starting point
- Multiple skill levels accommodated simultaneously
- 15-30 minute completion windows
- "Finished" is defined by the participant

#### Recommended Experience Projects

| Project | Duration | Age Group | Materials |
|---------|----------|-----------|-----------|
| LED greeting card | 15 min | All ages | Paper, LED, conductive tape, battery |
| 3D pen creation | 20 min | 7+ | 3D pen, filament |
| Vinyl sticker design | 15 min | 7+ | Vinyl cutter, transfer tape |
| micro:bit interactive toy | 30 min | 10+ | micro:bit, accessories |
| Laser-engraved keychain | 20 min | All (supervised) | Pre-cut wooden pieces |

## Workshop Operations

### Workshop Type Matrix

| Type | Duration | Depth | Audience | Frequency |
|------|----------|-------|----------|-----------|
| **Intro workshop** | 1-2 hours | Basic | Beginners | Weekly |
| **Skill workshop** | 2-4 hours | Intermediate | Members | Bi-weekly |
| **Project workshop** | 4-8 hours | Advanced | Experienced | Monthly |
| **Intensive** | 1-3 days | Deep | Dedicated | Quarterly |

### Workshop Design Framework

#### ADDIE Model Application

```text
Analysis → Design → Development → Implementation → Evaluation
    │          │          │              │               │
  Needs    Learning   Materials     Delivery        Feedback
  assess.  objectives  prep         + facilitation   + improve
```

#### Learning Objective Design

Use the format: "By the end of this workshop, participants will be able to..."

| Level | Verb | Example |
|-------|------|---------|
| **Remember** | List, identify, name | List 3 types of 3D printing materials |
| **Understand** | Explain, describe | Explain how FDM printing works |
| **Apply** | Use, demonstrate, create | Create a simple 3D model in Tinkercad |
| **Analyze** | Compare, troubleshoot | Troubleshoot a failed print |
| **Evaluate** | Judge, recommend | Evaluate which material suits a project |
| **Create** | Design, build, invent | Design an original functional object |

### Workshop Execution Process

#### Preparation Phase (T-2 Weeks)

| Task | Details |
|------|---------|
| Content finalization | Learning objectives, outline, materials list |
| Instructor preparation | Rehearsal, backup plans |
| Participant registration | Sign-up, prerequisites communication |
| Materials procurement | All materials, spares, backup equipment |
| Space arrangement | Station setup, equipment testing |

#### Day-of Time Allocation

| Segment | Duration | Content |
|---------|----------|---------|
| **Welcome** | 10% | Introduction, safety, expectations |
| **Demonstration** | 20% | Instructor shows technique/concept |
| **Hands-on** | 50% | Participants practice and create |
| **Sharing** | 10% | Show work, discuss, get feedback |
| **Reflection** | 10% | What did you learn? What's next? |

#### Instructor-to-Assistant Ratio

| Workshop Size | Instructor | Assistants | Ratio |
|--------------|-----------|-----------|-------|
| 1-8 | 1 | 0-1 | 1:8 |
| 9-16 | 1 | 1-2 | 1:8 |
| 17-24 | 1 | 2-3 | 1:8 |

### Workshop Quality Control

#### Teaching Quality Standards

| Criterion | Indicator |
|-----------|----------|
| **Preparation** | Materials ready, equipment tested, backup plans |
| **Delivery** | Clear instructions, good pacing, engaging |
| **Differentiation** | Supports different skill levels |
| **Safety** | Safety briefing, proper PPE, supervised equipment use |
| **Outcomes** | Participants achieve learning objectives |

#### Feedback Collection Template

```text
Workshop Feedback Form:
1. Overall satisfaction (1-5 stars)
2. Was the difficulty level right for you? (Too easy / Just right / Too hard)
3. What was your favorite part?
4. What would you improve?
5. Would you attend another workshop? (Yes / No / Maybe)
6. Would you recommend to a friend? (1-10)
7. Any other comments?
```

## Special Event Design

### OWL Night (Maker Evening)

- **Format**: Evening event (18:00-21:00)
- **Activities**: Open making, music, food, themed challenges
- **Atmosphere**: Relaxed, social, creative
- **Frequency**: Monthly

### Science Mystery Night

- **Format**: Research-themed escape room / detective game
- **Duration**: 2-3 hours
- **Age**: 10+
- **Goal**: Make science exciting and accessible

### Maker Marathon (Hackathon)

- **Format**: 24-48 hour intensive making event
- **Teams**: 3-5 people
- **Theme**: Announced at start
- **Output**: Working prototype + presentation
- **Frequency**: Quarterly or semi-annually

### Mobile Fab Lab

- **Format**: Equipment-loaded vehicle visiting communities
- **Duration**: Half-day to full-day
- **Goal**: Bring making to underserved communities
- **Activities**: Pre-designed quick-make projects

## Event Operations Management

### Event Calendar Planning

#### Annual Rhythm

| Quarter | Focus |
|---------|-------|
| **Q1** | New year launch, planning, recruitment |
| **Q2** | Skill-building workshops, partnerships |
| **Q3** | Summer programs, community events |
| **Q4** | Showcases, celebrations, reflection |

#### Monthly Rhythm

| Week | Event Type |
|------|-----------|
| **Week 1** | Regular open day |
| **Week 2** | Skill workshop |
| **Week 3** | Community gathering / show-and-tell |
| **Week 4** | Special event or planning |

### Resource Management

#### Instructor Resource Pool

- Maintain a database of available instructors
- Track instructor specialties and availability
- Rotate instructors to prevent burnout
- Invest in instructor development

#### Materials Standardization

- Standard kits for recurring workshops
- Pre-measured, pre-cut material packages
- Inventory tracking and reorder points
- Sustainable/recycled materials when possible

### Risk Management

| Risk | Mitigation |
|------|-----------|
| **Low attendance** | Multi-channel promotion, RSVP tracking |
| **Equipment failure** | Backup equipment, alternative activities |
| **Safety incident** | First aid kit, trained responders, clear protocols |
| **Weather (outdoor)** | Indoor backup plan |
| **Over-enrollment** | Waitlist system, repeat popular events |

## Impact Assessment

### Event Effectiveness Metrics

| Metric | Measurement | Target |
|--------|-------------|--------|
| **Attendance** | Head count | Meet capacity target |
| **Satisfaction** | Post-event survey (1-5) | >= 4.0 average |
| **Conversion** | Visitors → members | >= 10% |
| **Retention** | Return visitors | >= 30% |
| **NPS** | Net Promoter Score | >= 40 |

### Event Retrospective Template

```text
Event Retrospective:
├── What went well?
├── What could be improved?
├── Unexpected challenges?
├── Participant feedback highlights?
├── Key metrics vs. targets?
├── Action items for next time?
└── Overall assessment (1-5)?
```

## 2024-2025 Event Operations Trends

### Hybrid Event Models

| Dimension | In-Person | Online | Hybrid Strategy |
|-----------|----------|--------|----------------|
| Workshops | Hands-on making | Design and theory | Theory online, making in-person |
| Showcases | Immersive experience | Broad reach | Physical exhibit + virtual gallery |
| Community | Deep connections | Convenience | Core events in-person, extras online |

### AI-Powered Event Operations

- AI-assisted event planning and scheduling
- Personalized event recommendations for members
- Automated follow-up communications
- Data-driven event optimization

### Immersive Event Experiences

- AR-enhanced scavenger hunts during open days
- VR showcase of member projects
- Interactive installations that respond to visitors

### Livestream and Asynchronous Content

- Livestream workshops for remote participants
- Record and share workshop content for on-demand viewing
- Social media content from events extends reach

---

## Common Misconceptions

1. **"One amazing event is enough"** — Consistency matters more than spectacle. Regular small events build community better than rare large ones.
2. **"Events run themselves"** — Every successful event requires careful planning, execution, and follow-up.
3. **"More attendees = better event"** — A small, deeply engaged group often creates more value than a large, passive crowd.
