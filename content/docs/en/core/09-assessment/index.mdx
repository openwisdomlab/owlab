## Building Standards

<Callout type="info">
**Core Question**: How can growth become visible? The answer is not "measure more," but "measure what truly matters." Traditional assessment is like taking a snapshot — capturing only a static moment. OWL assessment is like filming a documentary — recording the entire journey from curiosity to creation.

**Key Insight**: The most valuable assessment happens the moment learners realize "I've come so far." Our job is to make that moment happen often.
</Callout>

### Core Principles

| Principle | Meaning | Verification |
|-----------|---------|-------------|
| **Assessment as Learning** | Preparing for a defense yields more than testing — reflection, organization, and articulation are themselves deep learning | Learners can say "while preparing my presentation, I realized I understood more than I thought" |
| **Growth Portfolio** | Every learner has a traceable growth trajectory — not a score curve, but an evolution of works | Randomly selected portfolios clearly show change over 3 months |
| **Multiple Evidence** | Reject single scores — works, journals, recordings, peer feedback cross-verify | Any capability judgment supported by at least 2 different types of evidence |
| **Three-Layer Service** | Learning assessment serves individuals, operational assessment serves nodes, impact assessment serves society | All three stakeholder groups can find data answering their core questions |
| **Theory of Change** | A complete logical chain from inputs to impact — making social value provable and communicable | Can explain in 3 sentences "what we did -> what changed -> why it matters" |

| S9 | Theory of Change Established | ToC document completeness | Basic version | Complete version | Evidence-based version |
| S10 | Impact Stories | Annual in-depth case studies | >= 1 | >= 3 | >= 5 |

### Parameter Metrics

| Metric | Calculation | Healthy Range | Warning Threshold |
|--------|-----------|--------------|-------------------|
| **Evidence Validity Index** | (Multi-source verified capability judgments / Total judgments) x 100% | 70-90% | &lt;60% calibration of assessment standards needed |
| **Growth Visibility** | (Portfolios clearly showing progress / Total portfolios) x 100% | 60-80% | &lt;50% evidence collection needs improvement |

| Pattern | Applicable Context | Key Practice |

### Assessment Tools

**Growth Visibility Check** — Randomly select 3 member portfolios:
- [ ] Can see a time span of >= 3 months
- [ ] Can see >= 2 different types of evidence
- [ ] Can clearly point to "this is progress"
- [ ] The learner themselves agrees the portfolio reflects genuine growth

**Assessment as Learning Verification** — Observe one presentation defense:
- [ ] The learner made new discoveries or gained understanding during preparation
- [ ] Q&A triggered deep thinking rather than simple recall
- [ ] The learner begins sentences with "I discovered..." rather than "I remember..."

**Theory of Change Completeness Check**:
- [ ] Inputs (what resources we invest) clearly described
- [ ] Activities (what we do) logically connected to inputs
- [ ] Outputs (direct results) quantifiable and trackable
- [ ] Outcomes (capability changes) supported by evidence
- [ ] Impact (long-term changes) reasonably inferred

The essence of educational assessment is not ranking, but feedback. In innovative education, we face a timeless challenge: how do we measure "creativity" and "confidence" — qualities that seem immeasurable? This module constructs a "learning-operations-impact" three-layer assessment system, advocating a shift from "merely measuring what is easy to measure" (such as scores) to "measuring what truly matters."
