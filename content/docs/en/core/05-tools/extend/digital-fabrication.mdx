# Digital Fabrication Toolchain

## Overview

Digital fabrication bridges the gap between digital design and physical creation. This document maps the complete toolchain — from initial concept to finished object — providing guidance on software selection, workflow integration, and educational applications.

**Core Workflow**:
```text
Idea → Digital Design → Manufacturing Preparation → Fabrication → Post-Processing → Output
       (CAD/modeling)    (Slicing/CAM/toolpath)     (Machine)     (Finishing)
```

---

## Part One: Digital Fabrication Workflow Framework

### 1.1 Complete Workflow

```text
                    Design Phase
                        │
         ┌──────────────┼──────────────┐
         ↓              ↓              ↓
    3D Modeling     2D Design     Electronics
    (CAD/sculpt)   (Vector/raster)  (EDA/PCB)
         │              │              │
         ↓              ↓              ↓
                Preparation Phase
         ┌──────────────┼──────────────┐
         ↓              ↓              ↓
     Slicing        Laser prep      CAM/toolpath
    (3D print)     (cut/engrave)    (CNC milling)
         │              │              │
         ↓              ↓              ↓
               Fabrication Phase
         ┌──────────────┼──────────────┐
         ↓              ↓              ↓
    3D Printing    Laser Cutting    CNC Machining
         │              │              │
         └──────────────┴──────────────┘
                        ↓
               Post-Processing
           (Sanding, painting, assembly)
                        ↓
                   Final Output
```

### 1.2 Key File Formats

| Format | Type | Used For |
|--------|------|---------|
| **.STL** | 3D mesh | 3D printing, CNC reference |
| **.OBJ** | 3D mesh (with color) | 3D printing, rendering |
| **.STEP/.STP** | 3D parametric | Engineering, CNC |
| **.SVG** | 2D vector | Laser cutting, vinyl cutting |
| **.DXF** | 2D vector | Laser cutting, CNC |
| **.GCODE** | Machine code | 3D printer, CNC instructions |
| **.GERBER** | PCB layout | PCB manufacturing |

## Part Three: Manufacturing Preparation Software

### 3.1 3D Print Slicing Software

#### FDM Slicing Software

| Software | License | Difficulty | Features |
|----------|---------|-----------|----------|
| **PrusaSlicer** | Open source | Intermediate | Excellent defaults, multi-material |
| **OrcaSlicer** | Open source | Intermediate | Bambu Lab + others, AI features |
| **Cura** | Open source | Beginner-Intermediate | Wide printer support |
| **Bambu Studio** | Free | Beginner | Bambu Lab printers |

#### Slicing Parameter Learning Path

```text
Beginner:   Layer height, infill %, supports (yes/no)
Intermediate: Wall count, top/bottom layers, retraction, speed
Advanced:    Variable layer height, pressure advance, support placement
Expert:      Custom start/end GCODE, material profiles, multi-process
```

### 3.2 Laser Cutting Preparation

#### Software Tools

| Software | License | Best For |
|----------|---------|---------|
| **LightBurn** | Commercial ($) | Most laser cutters |
| **LaserGRBL** | Free | GRBL-based lasers |

#### Design Specifications

| Parameter | Cut | Engrave |

| **Material margin** | 2-3mm from edge | N/A |
| **Kerf allowance** | 0.1-0.3mm (material-dependent) | N/A |

## Part Four: Materials and Process Reference

### 4.1 3D Printing Materials

#### FDM Materials

| Material | Properties | Best For | Safety |
|----------|-----------|---------|--------|
| **PLA** | Biodegradable, easy to print | General prototyping | Safe for all ages |
| **PETG** | Strong, water-resistant | Functional parts | Safe, no toxic fumes |
| **ABS** | Impact-resistant | Engineering parts | Requires ventilation |
| **TPU** | Flexible, durable | Wearables, grips | Safe |

### 4.2 Laser Cutting Materials

| Material | Cut | Engrave | Safety Notes |
|----------|-----|---------|-------------|
| **Plywood** | Yes | Yes | Use exhaust |
| **Acrylic** | Yes | Yes | Well-ventilated |
| **Cardboard** | Yes | Light | Fire risk |
| **Leather** | Yes | Yes | Fumes require ventilation |
| **PVC/Vinyl** | NO | NO | Releases toxic chlorine gas |

### 4.3 CNC Materials

| Material | Difficulty | Tools | Notes |
|----------|-----------|-------|-------|
| **MDF** | Easy | Standard bits | Dust collection required |
| **Plywood** | Easy-Medium | Standard bits | Watch for delamination |
| **Hardwood** | Medium | Sharp bits, slow feed | Beautiful results |
| **Aluminum** | Advanced | Specialized bits | Coolant recommended |

## Part Six: Workflow Integration

### 6.1 Typical Project Workflow Examples

#### Example 1: 3D Printed Phone Stand

```text
Tinkercad (design) → STL export → PrusaSlicer (slice) → 3D Printer → Sanding → Done
```

#### Example 2: Laser-Cut Organizer Box

```text
Inkscape (2D design) → SVG export → LightBurn (laser prep) → Laser Cutter → Assembly → Done
```

#### Example 3: Arduino IoT Device

```text
KiCad (PCB design) → Gerber export → PCB mill/order → Solder components → Program → Test → Done
```

### 6.2 Cross-Tool Collaboration Matrix

| From \ To | 3D Print | Laser Cut | CNC | Electronics |
|-----------|----------|-----------|-----|------------|
| **3D Print** | - | Jigs, fixtures | Molds | Enclosures |
| **Laser Cut** | Flat parts | - | Templates | Panel mounts |
| **CNC** | Molds | - | - | PCB milling |
| **Electronics** | Smart objects | Control panels | - | - |

## Common Misconceptions

1. **"Start with professional software"** — Complexity kills motivation. Start simple (Tinkercad) and progress naturally.
2. **"The machine does all the work"** — Digital fabrication requires significant design and preparation skills.
3. **"More tools = better learning"** — Master one workflow well before expanding.
